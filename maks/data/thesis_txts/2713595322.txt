Summary
In the year 2020 covid-19 breaks out in the world and send the whole world into
lockdown.
Never have there been developed a vaccine so fast and needed so fast
all over the world. It is expensive to develop medicine and due to that companies
there work in the biopharma industry try to be as cost-eÔ¨Äicient as possible. This
project will attempt to simulate and optimize the upstream process for monoclonal
antibodies (mAB). The simulation is done in MatLab using ode45. For optimization
will Casadi be used as a plugin there can be used to help solve non-linear problems
(NLP) and optimal control problems (OCP). the optimization algorithms there will
be implemented in casadi are direct single shooting, direct multiple shooting and
direct collocation. We will see that for the simple model will there be possible to
replicate results but where when we try for a more advanced model it starts to be
more complicated. For optimization will we see that there are some problems with
single shooting and we need to use an option for the NLP solver so it only does
an approximation but this problem does not seem to be an issue for direct multiple
shooting and direct collocation. With the optimization does it look like we end up
producing up to 200 g over two weeks‚Äô time just with simple bang-bang control.
