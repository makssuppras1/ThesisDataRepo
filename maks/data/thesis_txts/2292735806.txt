Abstract
On-line monitoring and control of key variables in industrial production pro-
cesses are of high priority with respect to quality assurance and cost-reduction.
In this thesis mathematical methods for monitoring, control and predictive
maintenance are derived. This is done for continuous processes and extended to
batch processes.
A state space model for continuous industrial processes is developed. This model
is used as a framework for the mathematical methods.
On-line monitoring
through multivariate statistical process control is developed with control charts
as the main statistical tool. The Hotelling T2 control chart is implemented as
a monitoring tool for the key variables. Regulation of the process is derived
though model predictive control. Matlab simulations show that a process regu-
lator does not interfere with the process monitoring.
This thesis is part of a project about quality control of injection molding pro-
cesses.
The processes are multiphase, and batch disturbances variate from
batch-to-batch.
These are some of the challenges in monitoring and control
of batch processes. The state space model is extended to batch processes and
includes batch-to-batch disturbances. A new method for batch-to-batch distur-
bance ﬁltering is developed. The Kalman ﬁlter is applied to the state space
model for the batch-to-batch evolution. The ﬁltered states are used to calculate
the ﬁltered disturbances. The controller predictions are improved by includ-
ing the ﬁltered disturbance. This way better performance of the controller is
achieved. Future research should focus on further development and testing of
the new ﬁltering method.

Chapter 1
Introduction
On-line monitoring and control of key variables in industrial production pro-
cesses are of high priority with respect to quality assurance and cost-reduction.
Process monitoring can determine whether the key variables are at their optimal
values or if the process is out of control. An out of control process will result in
decreasing quality of the products and production safety issues. On-line moni-
toring enables faster detection of faults compared to oﬀ-line monitoring. Faster
fault detection time will decrease the number of defective products leading to
reduced production cost and improved production safety. Control of the key
variables ensures operation at optimal values and corrections can be made if the
process is turning out of control.
1.1
On-Line Monitoring
On-line monitoring is monitoring of key variables while the process is running.
It enables faster detection of process faults, but requires fast computation of the
monitoring algorithm. This is because the monitoring time scale is in seconds for
many real-time industrial processes. Most industrial processes are multivariate
and the key variables of interest are usually correlated. Monitoring of correlated
variables are done through multivariate Statistical Process Control(SPC). Qual-
ity control charts are the main statistical tool. The three most known control
charts are: 1) The Hotelling T2 control chart, 2) The multivariate Cumulative
Sum(CUSUM), and 3) The multivariate Exponential Weighted Moving Aver-
age(EWMA). Hotelling T2 uses a summary statistic to produce a single test
==PAGE:8==
2

value each time an observation is acquired. The summary statistic takes the
variable correlation into account. The test value is compared to a threshold in
the control chart to determine if the process is in control. Hotelling T2 is the
most commonly used control chart in the industry, but is insensitive to small
shifts in the key variables. In those cases the multivariate CUSUM by Crosier
(1988) and Pignatiello and Runger (1990) or the multivariate EWNA by Lowry
et al. (1992) can be used. Reducing the variable space dimensions can be done
through Principal Component Analysis(PCA) or Partial Least Square(PLS).
Bersimis et al. (2007) show how PCA can be used to make a process model
from historical data of the process variables. Quality variables are much harder
than process variables to measure in real time. In case quality variables are
available, a process model can be made by applying PLS to both the historical
process variables and product quality variables. Monitoring of new observations
is done by checking how the observation ﬁts the PCA or PLS model.
PCA models do not ensure independent observations as discussed in a paper
by Mercer et al. (2002). The threshold in the control chart is based on the
assumption that the observations are independently and normal distributed.
Monitoring of serial correlated observations will result in a higher number of
false alarms, longer fault-detection time and missed out-of-control signals. Mer-
cer et al. (2002) model historical data with a Canonical Variate Analysis(CVA)
state space model instead of the PCA model. A valid threshold is ensured by T2
monitoring of the uncorrelated residuals from the CVA model mismatch. After-
wards PCA can be applied to the residuals to reduce the dimensions. The CVA
state space model is also used by Negiz and Cinar (1997). Here T2 monitoring is
applied on the state variables. This is not recommended by Mercer et al. (2002)
since the state variables are shown to contain serial correlation. Residual-based
monitoring is the best way to ensure a valid threshold.
1.2
Model Predictive Control
Industrial process control can be implemented through Model Predictive Con-
trol(MPC). MPC is process control based on a dynamic model for the given pro-
cess. The model is state space based and is derived from either ﬁrst-principles,
empirical models or from data-ﬁtting.
MPC ties in well with residual-based
monitoring because control and monitoring can be based on the same state
space model for the process. Time is saved in the process modelling phase since
only a single model is needed. Figure 1.1 illustrates the input and output in
MPC. In MPC the input and output of the past is known. These variables are
used within the process model to predict the future output for a given time
horizon. MPC is an optimization process where the diﬀerence between the pre-
dicted output and the reference output is minimized.
The prediction of the

output is controlled by optimal input sequence for the future time steps. The
optimal input for the next time step is implemented and a new output becomes
available. At each time step the minimization process is repeated to ﬁnd the
optimal input sequence.
The minimization problem is constrained by the prediction models. Input and
output constraints can be added. The derived state space models are of high
importance in the prediction of the future output values. Mismatch between
the model and the actual process decreases the MPC controller performance.
A survey on industrial MPC has been made by Qin and Bagwell (2003). They
highlight some of the challenges in implementing MPC in industrial processes.
Especially how to chose the key variables and model the process are the hard
part. There is no given solution since the various industries have diﬀerent re-
quirements and set-ups for their production. In addition, most industries have
to tackle the problems related to process disturbance. The disturbances can
be included in the state space models to obtain oﬀ-set free control. This can
account for model mismatch and unmodelled disturbances. Combining distur-
bance modelling with MPC has been done by Pannocchia and Rawlings (2003).
They show how oﬀ-set free control can increase the performance of the process
control for certain cases. They further more describe how the number of inte-
grated disturbances should be equal to the number of measurements to ensure
oﬀ-set free control.

1.3
Monitoring, Control and Predictive
Maintenance
There are two main goals for this thesis. The ﬁrst goal is to present mathe-
matical methods for on-line monitoring, control and predictive maintenance for
continuous industrial processes. These methods will be derived by applying SPC
and MPC to a state space framework. Throughout the thesis the required steps
for applying monitoring, control and predictive maintenance will be summarized
in small lists. This is done to obtain an overview of the implementation process.
Matlab algorithms of key aspects in the mathematical methods will be shown.
This thesis is part of a project about quality control of injection molding batch
processes. Thus, the second goal is to extend control of continuous processes to
apply for industrial batch processes. To reach this goal the knowledge gained
from implementing control for continuous processes will be combined with a
state space model for batch processes. A new method for batch-to-batch dis-
turbance ﬁltering will be derived.
Chapter 2-7 contains the implementation of the mathematical methods on con-
tinuous processes. In Chapter 2 the state space representation is developed. The
mathematical methods will be based on a stochastic state space model with cor-
related process and measurement noise. This is a more general model than the
model by Mercer et al. (2002). In Chapter 3 the Kalman ﬁlter is applied to
the state space representation. The ﬁlter estimates the states of the process
and predicts the next observations.
Independent residual-based observations
are produced this way. The control chart threshold is thereby ensured to be
valid in the monitoring of the process. The independent observations are used
in Chapter 4 to monitor the process. Monitoring of the observations with the
Hotelling T2 control chart is derived. In Chapter 5 process control is derived.
The optimization problem for input regulation of the process is solved. This
is done for both the unconstrained and the constrained case. In Chapter 6 a
compact state space model for evolution of a closed-loop system is developed.
In Chapter 7 predictive maintenance of the system is developed. Simulation of
a four tank system will be used to show the application of the mathematical
methods for monitoring, control and predictive maintenance. In Chapter 8 in-
jection molding is reviewed. This is done to give a better understanding of the
challenges in modelling and control of an industrial batch process. In Chapter 9
control of batch processes is derived. The control methods from Chapter 5 are
extended in Chapter 9 to account for batch-to-batch dynamics. Batch-to-batch
dynamics are modelled in state space to develop a new method for disturbance
ﬁltering.

2.2
Independent Observations
Measured observations should be monitored to check if the industrial process
is in control. The measured observations yk in Equation 2.1b depend on the
previous observations yk−1.
The measurements are serial correlated as they
depend on each other. The threshold in the control charts is invalid for serial
correlated observations as discussed by Mercer et al. (2002) in Section 1.2. In
Chapter 3 the Kalman ﬁlter developed by Kalman (1960) is applied on the
stochastic linear time invariant state space representation. This is done to ensure
valid thresholds in the control charts. The Kalman ﬁlter estimates the state of
the process based on the input uk and measured output yk. The state estimate is
used to compute the innovation ek. The innovation is the diﬀerence between the
estimated output and the measured output. These innovations are independent
and normal distributed, and thereby satisfy the assumptions for SPC charts.
An innovation is the same as a residual. The monitoring of observations in this
thesis is residual-based.
A more statistical approach of ensuring independent observations includes the
following.

(ARMAX) is ﬁtted to a time series of the observations yk. The ARMAX model

Multivariate Statistical
Process Control
In Chapter 3 independent observations were obtained by applying the Kalman
ﬁlter to the state space representation. In Chapter 4 monitoring of industrial
processes are derived through Multivariate Statistical Process Control(MSPC).
MSPC can determine if an observation here referred to as a sample has the same
mean vector as the sample population. The process is out of control if changes
occur in the mean vector. MSPC is performed for two reasons: 1) Many variables
can be monitored with MSPC at the same time, and 2) It enables monitoring
of correlated variables. Variables of interest are usually correlated and this has
to be taken into account in the process monitoring. The two reasons suggest
the use of a summary statistic, that is a function of all variables and implicitly
considers the correlation among them. The statistical test value is calculated
based on all the p variables. Only a single control chart needs to be monitored
to check for changes in the mean vector. A control chart consists of the plotted
test values and the threshold. Figure 4.1 shows a sketch of a control chart. A
test value is computed each time a new sample becomes available. The test
value is plotted and compared to the threshold to monitor if the process is in
control.

4.4
Hotelling T2 and Principal Component
Analysis
An industrial multivariate process contains often more than ten variables. A
high number of variables increases the computation time of the statistical test.
A way to reduce the dimension space of the variables is to apply Principal
Component Analysis(PCA) described byMason and Young (2002) and Kourti
and MacGregor (1995). Inversion problems of the covariance matrix is another
reason for performing PCA. High correlation between the variables makes the
matrix singular and inversion becomes a problem. PCA is a orthogonal transfor-
mation of the observations into linear uncorrelated principal components. The
ﬁrst component accounts for the largest variance in the reduced system and
the second for the second largest variance and so on until all the variance are
accounted for. The number of principal components are equal to the number
of variables. Reduction of the dimensions can be done by excluding the com-
ponents that only account for a small amount of the variance. The retained
number of principal components are less than the number of variables. Fig-
ure 4.7 shows an example of PCA in two dimensions. Most of the variance is
accounted for along the line. The two dimensions can be reduced to one di-
mension by projecting the points onto the line. This simple example illustrate
how PCA can reduce the number of dimensions. The observations should be
standardized before PCA is applied to avoid scale dependency of the variables.
The innovations ek are standardized into Yk by using the estimated mean and
covariance from Equation 4.3 and 4.4. The statistical test in Equation 4.6 is

Chapter 7
Predictive Maintenance
In some of the industrial production processes the production machines are
maintained regularly.
The maintenance of the machines ensure production
safety by preventing machine breakdown.
The maintenance is also done to
maintain the quality level of the products. Production downtime is very costly
so the timespan between maintenances should be maximized without decreas-
ing production safety and product quality. The timespan between maintenances
are often ﬁxed based on ﬁrst-hand knowledge of the production machines. The
timespan is found by subtracting a time margin from the known time limit of
the machine. This is to avoid safety and quality issues. A ﬁxed timespan is not
the best choice if the wear on the machines changes with time. In such cases
predictive maintenance is a better choice. Predictive maintenance is the ability
to decide when to do maintenance based on process variable predictions. In
Chapter 7 predictive maintenance is derived based on Kalman predictions.
7.1
Kalman Predictions
The static Kalman ﬁlter derived in chapter 3.2 is also capable of predicting more
than one step ahead. It can be used to predict n steps ahead and should be
chosen correspondingly to the discretized time step to give the wanted time span
for the prediction. The state predictions are used to make measurement output

8.1
Introduction to Injection Molding
Zhou (2013a) deﬁnes injection molding as: "A repetitive process in which melted
(plasticized) polymer is injected(forced) into a mold cavity or cavities, packed
under pressure, and cooled until it has solidiﬁed enough"Zhou (2013a, pp. 3).
Injection molding is a batch process because it is a repetitive process and the
same machine can produce diﬀerent products. Diﬀerent products are made by
equipping diﬀerent molds. A batch of products is produced in each repetitive
cycle also called a shot. An injection molding batch process is multiphase. This
should be kept in mind when the process is modelled, monitored and controlled.
The batch phases are: 1) Melting of thermoplastic, 2) Injection of plastic into
the mold cavity, 3) Packing and cooling of plastic and 4) Ejection of the cooled
plastic parts.
The injection molding process is complicated because many variables are in-
cluded. The variables are divided into four groups: 1) Material properties, 2)
Machine variables, 3) Process variables and 4) Part quality. Kamal et al. (2009)
and Chen and Turng (2005) list variables from the four groups and show how
some of them are linked. Table 8.1 presents the four groups of variables. The

8.2
Batch Monitoring
Many of the processes in the chemical and manufacturing industry have inherit
batch structure. The same product is made by a series of machines, and diﬀer-
ent batches of the product are made. Principal component analysis was review
in Section 1.1 as a possible way of process modelling. The historical data from
batch processes has three dimensions, but PCA only works in two dimensions.
Multi-way PCA(MPCA) has been developed by Wold et al. (1987) to enable
dimension reduction of three dimensional data-arrays. Monitoring of batch pro-
cesses using MPCA are presented by Nomikos and MacGregor (1994). Many
batch processes are inherently multistage or multiphase as shown in Section 8.1.
An overview of diﬀerent techniques, like multiblock and phase-separation, for
modelling of such processes is given in a survey by Yao and Gao (2009). They
conclude that multistage or multiphase features should be taken into considera-
tion when modelling using MPCA. This will give a better understanding of the
batch process and ensure product quality.
A framework for state-space monitoring of batch processes has been developed
by Lee and Dorsey (2004). They show that state-space monitoring enhances
the ability to: 1) Detect small shifts in the mean vector and 2) Detect changes
in the correlation structure between batches. The state space framework does
not exclude the use of MPCA as a modelling tool. A MPCA model must be
transformed into a state space model to be used in the framework.
8.3

the part quality. To successfully reach the goal it has to be remembered that the
control process has three levels. An example can be seen in Table 8.1 where the

machine and process variables have to be implemented ﬁrst before the variables
can be linked to implement part quality control. State of the art injection mold-
ing control is described in a review by Chen and Turng (2005). They conclude
that there is already precise built-in control system for the machine variables.
Process disturbances can however introduce faults that the built-in control can-
not detect and control.

on-line monitoring of part quality will greatly increase the performance of on-

On-line product quality monitoring is done in a paper by Gordon et al. (2014).

equations. They use data from the nozzle pressure sensor. The simulated vari-
ables are then used in the control of the injection molding process.
MPC is widely used in injection molding for batch control of single process
variables. Control of the melt temperature can be done through diﬀerent ap-
proaches. Huang et al. (1999) have developed adaptive generalized predictive
control of the melt temperature. The adaptive controller incorporates parameter
learning based on changes in material properties. The parameters for the gener-
alized predictive control are updated as the process varies. Another approach is
made by Gerber et al. (2006). They couple computational ﬂuid dynamics with
MPC. The governing equations are used to produce temperature data from sen-
sor inputs. The inputs and temperatures are ﬁtted to an ARX model. Diﬀerent
process conditions gives diﬀerent models. The MPC algorithm can then call
and use corresponding models for the given conditions. Huang et al. (1999)
and Gerber et al. (2006) show two diﬀerent approaches to batch control. One
is to use learning control where the controller updates itself. The other is to
use multiple models for the diﬀerent scenarios. The ﬁrst approach is preferred
because it is the most general. The second approach requires a big database of
the diﬀerent model parameters.
8.4
Batch-to-Batch Dynamics and Control
Huang et al. (1999) used adaptive MPC to adjust for material composition vari-
ations. Material variation is categorized as a batch-to-batch dynamic. Batch-to-
batch dynamics can be refereed to as batch-to-batch disturbances. Temperature
sensitive material characteristics, diﬀerent material properties, hydraulic system
sensitivity to oil temperature and material variation are all categorized as distur-
bances. Process control becomes more diﬃcult if non-controllable disturbances
are present. The controller must be extended to account for disturbance changes
to ensure the performance level. Batches have a repetitive nature and it can
be exploited with learning control to adapt to dynamic disturbances. Diﬀer-
ent injection molding controllers have been investigated by Furong Gao and Yi
==PAGE:60==
54
Injection Molding Processes
Yang in Kamal et al. (2009, Chapter 12) and Zhou (2013b, Chapter 12-15).
They show how to use adaptive control and MPC for batch processes. More
importantly it is shown how to apply Iterative Learning Control(ILC) to injec-
tion molding processes. ILC takes advantages of the repetitive nature in batch
processes. It uses information from the last time step to improve the current
time step.
There is both direct and indirect ILC. In direct ILC the control
law is updated based on the previous time step. In indirect ILC the controller
parameters are updated. ILC is compared to repetitive-control and run-to-run
control in a survey by Gao et al. (2009a). ILC is found to be the optimal learn-
ing control for batch processes. ILC is extended to two dimension space in a
paper by Gao et al. (2008). The injection molding batch process is optimized
both in time within a batch and from batch-to-batch. Batch process control in
both time and batch direction has also been developed by Bonne et al. (2014),
Xiong et al. (2005) and Lee et al. (1999). They also combine ILC with MPC
like Gao et al. (2008), but state space models are used as basis instead. The
state space models are transformations of the input/output model for the batch
process. They use the tracking error as the states in all the state space models.
That way the batch-to-batch disturbance is eliminated.
8.5
Potential Application
The potential applications of applying the mathematical methods to injection
molding processes are reviewed. The ﬁrst application is monitoring of variables.
It can simplify the job of ensuring that the process is in control. It can hopefully
lead to faster detection time of faults. This is a cost-reduction for the company
since fewer products have to be scrapped. The second application is variable
control. It should ensure less faults by stabilizing the process. It could also give
faster start up time for a new production of batches. The control system could
simplify the process of creating a process window and thus reduce the number of
shots in the DOE. This will also be an automation of the process, which reduces
the requirement of a specialist. Less specialist means reduces educational and
payment cost. It will furthermore reduce the vulnerability of the production as
the company do not have to rely on a few specialists. The last application is
the predictive maintenance of the machine. The injection molding machine is
maintained when it has ran for a ﬁxed time to avoid wax accumulation in the
mold. The downtime is costly because the machine cannot produce during the
maintenance. The process window can also change during downtime as men-
tioned earlier. Prediction of the variables based on the mathematical methods
can be used to decide when to maintain the machine. The predicted time before
maintenance is required is hopefully longer than the ﬁxed time that is currently
used. This will reduce the number of maintenances.

A shrinking horizon state space model of industrial batch processes has been
derived. The parameters where made time dependent because a batch can be
divide into several phases. The dynamic Kalman ﬁlter was extended to account
for a known disturbance and time dependent parameters.
An example of a
batch process were made by linearizing the start up of the four tank system.
Input regulation was derived for a batch process with a known disturbance.
Designing the parameters for the regulation problem was shown in Matlab. The

Step 6: Filtering and regulation of batch processes

4. Set up the initial values for the process

(a) Within batch regulation using ﬁltered disturbance

Chapter 10
Conclusion
Mathematical methods for on-line monitoring, control and predictive mainte-
nance for continuous industrial processes have been developed. All the methods
were based on a state space model for industrial processes. The Kalman ﬁlter
was applied to the state space model to compute independent normal distributed
residuals. Process monitoring was implemented through Multivariate SPC with
control charts as the statistical tool. The Hotelling T2 control chart was applied
as a summary statistic to simplify monitoring of all the variables in the process.
The independent residuals ensured valid thresholds of the control charts. Con-
trol of the industrial process was implemented through MPC. Input regulation
was achieved by solving the minimization problem of the future residuals. The
minimization problem with the state space model as equality constraint was
solved both with and without input and output constraints. The ﬁrst entry of
the solution was the optimal input for the next time step. Output predictions
from the Kalman ﬁlter was used to determine the timespan before next machine
maintenance. It is possible to reduce the number of maintenances with predic-
tive maintenance and thus valuable production time can be gained.
Lists of the diﬀerent stages in each of the mathematical methods were shown
to gain an overview of the implementation process. Simulations in Matlab of a
four tank system showed the applications of the mathematical methods. The
simulations showed that faults due to disturbance changes in the system could
be detected. Input regulation of the system did not make it more diﬃcult to
detect a fault on the control chart.
Regulation of continuous processes have been extended to industrial batch pro-
cesses. The batch dynamics was modelled with a shrinking horizon state space
model. The parameters were made time depended to account for multiphase
==PAGE:79==
73
batch structure. A batch-to-batch disturbance model was combined with the
shrinking horizon state space model for a complete batch-to-batch model. A
new method for disturbance ﬁltering was derived purely based on MPC instead
of a combination of MPC and ILC. In this method the Kalman ﬁlter was ap-
plied to ﬁlter the states of the batch-to-batch model. The ﬁltered disturbance
was calculated from the ﬁltered states. The controller predictions could then be
improved by including the ﬁltered disturbance. This way better performance of
the controller can be achieved.
Future research should focus on further development and testing of the new ﬁl-
tering method. This include testing on a simulated batch system and comparing
of the performance with the current ILC controllers. Potentially this can lead
to a more precise and reliable way of monitoring and controlling of industrial
batch processes.
