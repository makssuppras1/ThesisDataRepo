Summary
The recent advances in deep learning have achieved promising results on many
biomedical image segmentation benchmarks, addressing the problem of learning
from a massive amount of data. However, the diﬃculty in acquiring suﬃcient
training data in medical image analysis domain becomes an obstacle for training
a deep neural network. Therefore, to produce a scalable solution with data-
eﬃciently learning, we present an active learning framework for recognizing
objects in images using multitask deep neural network.
To formulate the active learning framework, we ﬁrst present a two-tasks deep
neural network for object recognition in medical images using deep residual
network (ResNet) and deep contour aware network (DCAN). Due to the fact
that the acquisition of diﬀerent types of labels (bounding box label or full mask)
will incur various annotation cost, a three-tasks deep neural network, which can
perform object recognition, contour detection, and bounding box prediction is
proposed for giving active learner the possibility to obtain diﬀerent types of
label. Then to enable the evaluation of the informativeness for the images in
terms of object recognition task, the model uncertainty, which can be extracted
by dropout training the proposed deep neural network, is incorporated in the
proposed acquisition functions. The active learning framework can query the
most useful images for training the deep neural network, therefore the model
can achieve similar performance with less amount of data.
The 2015 MICCAI Gland Challenge dataset is utilized to demonstrate the per-
formance of the proposed framework. Our results show the signiﬁcance of the
deep residual network (ResNet) in medical image analysis domain because it can
ease the training and improve the generalization. Apart from that, the design
of the three-tasks deep neural network can further facilitate the performance of
the model in comparison to the single-task deep neural network. Then with the
proposed active learning framework base on the deep neural network, we achieve
the best performance using 55 out of 85 training images for Gland Challenge
dataset.

Chapter 1
Introduction
In histopathological image analysis, the morphology of histological structures,
such as glands and nuclei, has been routinely adopted by pathologists to assess
the malignancy degree of adenocarcinoma.
Accurate detection and segmen-
tation of these objects of interest from histology images is an essential pre-
requisite to obtain reliable morphological statistics for quantitative diagnosis
[CQYH16]. Recent advances in deep learning have achieved promising results
on many biomedical images segmentation benchmarks such as Hep-2 cell image
classiﬁcation [GWZZ15], nuclei segmentation [SZC+15] [VvDP16], and gland
cell segmentation [CQYH16] [XLL+16] [BKH16].
However, it is known that deep neural networks need a large amount of labeled
data to be able to learn their large number of parameters [GIG17]. Besides it
is almost always better to have more data considering the accuracy of CNN is
often not saturated with increasing data size [SS17]. Manually label medical
images can incur a great deal of annotation cost and eﬀort due to the fact that
only the trained biomedical experts can annotate medical images. Therefore
data-eﬃcient deep learning has been an area of interest in the medical image
analysis domain, which is asking how to learn in a complex domain for producing
the scalable solution without requiring deep learning model to be trained with
large quantities of data. Active learning is one of the standard paradigms to
address this question.
==PAGE:18==
2
Introduction
In active learning, the goal is to achieve high accuracy using as few labeled
data as possible [Set09]. The active learner seeks the most informative data
by evaluating the uncertain level for each data point which assumes the most
uncertain data point will be most critical to be labeled. The studies led by
[YZC+17], [GIG17] and [WZL+17] have illustrated that they obtained a similar
model performance with less amount of training data by utilizing deep neural
network based active learning. We thus design an active learning framework
base on the deep neural network to illustrate the capacity of learning a complex
model without requiring large quantities of data.
1.1
Related Work
We consider the related work in the following categories separately.
Brieﬂy,
there are two challenges regarding deep neural network based active learning.
First, the deep neural network needs to be fast and easy to train so that the
time between two annotation stages can be acceptable. It also needs to have a
good generalization performance when only a small amount of training data are
available so that the predictions are reasonable. Then from the active learning
point of view, the methods for representing the informativeness of the data point
is of major importance since the proper selection criteria can lead performance
gain with less queried data. Therefore the related work is discussed below.
Deep Neural Network
Deep neural networks are driving advances in image recognition related tasks
with a powerful feature representation capacity. [RFB15] won the 2015 ISBI cell
tracking challenge by a large margin compared to other work with a U-shape
neural network, which applied a long-skip connection between down-sampling
and up-sampling path for extracting multi-level context and precise spatial in-
formation. Instead of only optimizing the network by one metrics like [RFB15],
[AKEH17] proposed a two-tasks neural network for tracking cells by ﬁrst propos-
ing the bounding boxes of cells and then selecting cells. This work outperforms
other existing cell tracking methods by taking advantage of the shared feature
representation in multitask learning. However directly training these long-skip
connection networks could be diﬃcult especially when only a few training data
are available.
[CQYH16] proposed a two-tasks network which also utilized multi-level con-
texture extraction for objects recognition in gland cell images and achieved the
state-of-art performance on 2015 MICCAI gland cell challenge. To deal with the
vanishing gradient problem, auxiliary deep supervision, which has been veriﬁed

1.1 Related Work
3
by [LXG+14] [WLTL15] that it can eﬀectively ease the training for a really deep
network, is utilized. Based on [CQYH16], [XLW+16] added another bounding
box detection channel to assist the prediction of the object, they also accom-
plished promising results. Nevertheless, the bounding box detection network
that they used [XLW+16] was outperformed by [HGDG17] and [LDG+16] be-
cause of a more powerful feature representation and localization. In comparison
to the previous multitask learning networks, which assign an equal ratio to the
loss from each task, [BH17] utilized a weighted multi-loss objective function for
learning the weight of the loss in relation with diﬀerent tasks. They achieved
encouraging results which indicate that the performance of multitask learning
can be improved further by fully investigating and utilizing task relationship.
Since the depth of the network has already been veriﬁed as a crucial element
in terms of performance as shown in [IS15] and [SLJ+14], [HZRS15] designed
a deep residual network (ResNet) by short-skip connection for improving the
depth of network as well as making training easier. They have achieved the
state-of-art performance in all ﬁve main tracks of the ImageNet and COCO
2015 competitions. Based on [HZRS15], [HZRS16] has proposed a new residual
unit with the utilization of batch-normalization layer, which makes training
easier and improves generalization. More importantly, [DVC+16] extended the
fully connected network (FCN) by short-skip connection as ResNet [HZRS15]
and long-skip connection as U-Net [RFB15] on 2D EM segmentation challenge.
They proved that the addition of short-skip connection can help to update
the deeper convolutional layers, which cannot be updated in the normal CNN
because of gradient vanishing problem in a really deep FCN. They have achieved
the state-of-art performance on EM segmentation challenge without any post-
processing methods. This has pointed out the signiﬁcance of the deep residual
learning.
Active Learning Query Strategy
Active learning has been widely studied and most of the early work can be found
in the classical survey of Settles [Set09], which covers diﬀerent query scenarios
such as membership query synthesis, stream-based selective sampling, and pool-
based sampling. The present study only focus on pool-based sampling since a
large number of unlabeled images are available and can be used as unlabeled
pool set.
In pool-based sampling, each data point in the pool set needs to be given an un-
certainty score. There are diﬀerent methods for evaluating the informativeness
of each data point. [CM05] and [SC08] utilized the least conﬁdent acquisition
function for extracting uncertainty based on a linear-chain conditional random
ﬁeld (CRF) [LMP01] for information extraction task. However, the least con-
ﬁdent acquisition function neglects the remaining label distribution. [SDW01]
==PAGE:20==
4
Introduction
constructed the marginal sampling for considering the marginal diﬀerence be-
tween the most likely label and second most likely label. Nevertheless, the least
conﬁdent and marginal sampling will have the same eﬀect if the problem is a
two-class classiﬁcation since they both consider to choose the data point which
has class posterior close to 0.5. A more generalized uncertainty measure for
representing the informativeness of the data is obtained by the calculation of
entropy [Sha48].
The acquisition functions above only consider the relative uncertainty within
diﬀerent class labels without considering the model uncertainty, which can be
more generalized for describing the overall uncertainty for all the class labels.
Therefore, [YZC+17] utilized bootstrap for expressing the overall uncertainty
on gland cell image dataset. They have demonstrated that they achieved state-
of-art performance with 50% of training data. However, training the model with
large amount of parameters requires high regularization power when only few
training data are available. Dropout training deep neural networks has been ver-
iﬁed by [GG15] that it can both improve the regularization power and represent
the model uncertainty. Thus [GIG17] came up with a new and eﬀective Bayesian
active learning network by using dropout layers for high dimensional input data.
They achieved signiﬁcant improvements on MNIST data and ISIC2016 skin can-
cer diagnosis task. This is the method that is utilized in this work.
Except for the estimation of informativeness, lots of practical challenges also
need to be considered. In comparison to the selection of queries in serial, i.e.,
one at a time, [GIG17] illustrated that the selection of more than 5 queries per
time is much more computationally eﬃcient and performance eﬀective compared
to only 1 query per time. Considering labeling cost, [TWH07] proposed new
adaptive stopping criteria for information extraction task. The query process
is stopped if the disagreement between diﬀerent max-entropy classiﬁers is close
to 0. This work indicates that the additional annotations for the data points,
which can be classiﬁed very well by the model, will not deliver any apparent
further model performance improvement.
However, the acquisition of a fully labeled image is always expensive. Therefore,
[VG09] has proposed to predict the trade-oﬀbetween the eﬀort and information
gained as the net worth, which can be used to rank the unlabeled and partially
labeled images. By using this method, they have actively improved the model
performance with minimal manual intervention. Similar to [VG09], [KULF17]
has introduced an intelligent annotation dialogue for bounding box annotation
by box veriﬁcation and manual box drawing. The oracles only need to draw
a box manually if all the proposed boxes are veriﬁed that they do not include
objects. This work can largely speed up the annotation compared to manual
box drawing alone or box veriﬁcation alone.
Thus these two works indicate
that the utilization of partially labeled data and the prediction from the model

can further decrease the labeling cost and also improve the model performance,
which motivates multitask active learning.
In multitask active learning, the same data instance can be labeled in multiple
ways for diﬀerent tasks. Consequently, it attempts to assess the informative-
ness of a query with respect to all learners involved. [RTHR08] selected the
queries based on the simply summed uncertainty value for each task in a two
task NLP. However, this might over-estimate the informativeness of some data
points. [HY10] tested two diﬀerent query strategies: selﬁsh approach only de-
pends on the uncertainty score for one task, benevolent approach depends on
the uncertainty score for all the tasks. They demonstrated that the combination
between selﬁsh approach and benevolent approach had the best performance.
Nevertheless, it is pretty diﬃcult to estimate the timing when selﬁsh or benev-
olent strategy should be applied.
1.2
Contribution
Based on the discussion and review of the related work, a multitask deep neural
network is designed for recognizing objects in medical images. Then an active
learning framework based on this multitask deep neural network is built for
investigating the capacity of learning a complex model with less amount of
data. It is hypothesized that:
1. The replacement of the encoder network from DeepLab [CPK+16] to
ResNet [HZRS16] in deep contour aware network (DCAN) [CQYH16] can
achieve a better accuracy and higher training speed for object recognition
task in medical images. This is veriﬁed in section 3.4.
2. Multitask learning can boost the performance of its sub-task compared to
single-task learning. This is demonstrated in section 3.6.

2.1.3
Training
When employing the deep neural networks in medical image analysis task, one
of the main challenges is the insuﬃciency of the high-quality training data with
accurate annotations, due to the expensive cost and complicated acquisition
procedures. Compared to this situation in medical images, much more training
data can be available in the natural image analysis task. Moreover, the previous
studies [SCL+16], [KH16], and [CWF+13] have veriﬁed that transfer learning
can alleviate the problem of insuﬃcient training data. Thus the encoder layers
in our network are initialized with the pre-trained weight from natural image
analysis task.
As for training DeepLab-DCAN, the pre-trained DeepLab2 weight on PASCAL
VOC 2012 [EVGW+] dataset speciﬁcally for semantic image segmentation is
utilized. As for the ResNet-DCAN, the weight which is trained on ILSVRC-2012-
CLS image classiﬁcation dataset [RDS+15] is adopted3. All the layers in the
encoder part are initialized with the pre-trained weight. Then the convolutional
and deconvolutional layers in the decoder part are initialized by a truncated

The basic idea behind the feature pyramid network (FPN) [LDG+16] is that the
feature map with lower resolution has a stronger semantic feature representation
with lower accuracy on the location, while the higher resolution feature map has
low semantic feature representation but with high localization accuracy. Thus
if the feature maps with diﬀerent resolution can be combined, it can have more
powerful accuracy concerning both feature representation and localization.
The input of FPN consists of a series of feature maps {C2, C3, C4, C5} at several
scales. Each of them is sub-sampled by a factor of 2 and has a relative stride of
{4, 8, 16, 32} in comparison to the original input (C1 is not used because of the
memory limitation). Then a 1 × 1 convolutional layer without any non-linearity
is applied on each of these feature maps to make sure all of them have the same
output dimension d (d = 256). To achieve the goal of improving both feature
representation and localization power, each feature map is up-sampled by a
factor of 2 using bi-linear up-sampling and is merged with the corresponding
bottom-up feature map by element-wise addition as shown in Figure 2.6 (since
C5 does not have the corresponding bottom-up feature map, we simply attach a
1x1 convolutional layer to produce P5). This process is iterated until the ﬁnest
resolution map P2 is generated. Finally, a 3x3 convolution without any non-
linearity is applied on each merged map to generate the ﬁnal feature map for
reducing the aliasing eﬀect of up-sampling. These maps {P2, P3, P4, P5} are
utilized as the input for the Fast R-CNN detector. As for the region proposal
network (RPN), a max-pooling layer is appended on feature map P5 to produce
P6, which has a higher receptive ﬁeld size and higher stride (64) so that it can
make sure the scales of the bounding box are broad enough to cover the objects
with various size.

2.3
Extracting Uncertainty in Active Learning
The present study only considers pool-based active learning.
It means the
learner has access to a pool of unlabeled data from which to select points for
annotation. To select the most informative point, a score to represent the in-
formativeness needs to be assigned to every point in the pool set. A high score
means knowing the ground truth of this data point is critical for improving
model performance. On the contrary a low score means the trained model can

already reveal enough information for this point, which indicates no labeling
eﬀort needs to be made for this point. Therefore such utility function needs to
capture the conﬁdence level or in other words the uncertainty of the model in
making decisions for data points.
There are two kinds of uncertainty that can be modeled and measured:
1. The relative uncertainty between class labels. The predicted probability
from model output can be utilized to describe the conﬁdence level re-
garding if this speciﬁc data point belongs to class k1 compared to other
classes.
2. The overall uncertainty for all class labels. This uncertainty can be also
called as model uncertainty.
It measures how much can the model be
trusted in predicting a speciﬁc class k generally.
The predicted probability from model output can be used to measure the rela-
tive uncertainty between class labels. As for extracting overall uncertainty for
all class labels, dropout layer is applied. Dropout has been demonstrated by
[GG15] to be a useful tool as a Bayesian approximation for representing model
uncertainty.

2.4
Acquisition Functions in Active Learning
This section introduces diﬀerent acquisition functions for measuring the infor-
mativeness of the data points. This thesis deals with object recognition task
in images, which means it is a pixel-wise classiﬁcation problem. It indicates
that each pixel can obtain an uncertainty value, and the aggregation of the un-
certainty value determines the informativeness of an image. In this thesis, we
only consider representing the informativeness of an image by merely calculating
the sum or the mean of the uncertainty value regarding foreground-background
predictions over all the pixels in that image. However, since an image consists
of millions of pixels, the aggregation method that is utilized may cannot ade-
quately represent the informativeness of this image. The inﬂuence of aggregation
methods is discussed in section 4.4.2.

two-tasks network ResNet-DCAN. Two possible reasons could be: First, it is
diﬃcult to perform equally well on every task when all the task-speciﬁc layers are
trained from scratch in this complex network since the total loss is the one which
is back-propagated. In other words, for example, assume if only one task has
excellent performance, while the other two tasks have acceptable performance.
Then it indicates the total training loss may have already converged.
As a
consequence, it is diﬃcult to continue to optimize the model for ﬁnding a better
local minimum. Therefore, some sub-tasks cannot achieve better performance.
Another reason is that the ﬁrst two tasks foreground-background classiﬁcation
and contour detection are pixel-wise classiﬁcations. However, for the bounding
box detection network, the cls classiﬁer works as an image-level classiﬁcation,

seeing that each proposed box is classiﬁed to be 1 (include an object) or 0 (not
include an object). The reg layer learns the coeﬃcients for ﬁtting the proposed

characteristics of each task may also add diﬃculty for training ResNet-DCAN-BD
end-to-end.
As regards the inﬂuence of diﬀerent training scenarios on ResNet-DCAN-BD, it

task-speciﬁc layers already have a good performance in classifying objects, and
the bounding box detection network does not inﬂuence them. Thus even if the
tasks have diﬀerent characteristics, ﬁne-tuning allows the network to have a good

are ﬁne-tuned again based on ResNet-DCAN-BD (ﬁne-tune) model, but since the
purpose of this work is not for competition, we did not continue to ﬁne-tune the

Experimental Setup
The performance of the active learning framework using the deep neural network
is evaluated with 2015 MICCAI Gland Segmentation (GlaS) Challenge Con-
test dataset. The initial active learning framework use ResNet-DCAN (ResNet-
50) as the foundation, since it is fast and easy to train and also it has an
equivalent object recognition performance compared to the three-tasks network

ResNet-DCAN-BD.
To enable the extraction of model uncertainty, 16 dropout layers are added at
the last unit of each block in the encoder ResNet-50. In the training time, the
stochastic forward pass is only applied once in each iteration with a dropout rate
of 0.5. For estimating the uncertainty and evaluating the model performance,
25 MC dropout samples are generated per image. The uncertainty estimation
for the images in the unlabeled pool set U is explained in section 2.4.
As
for the evaluation of the model on test data, the averaged predicted probabil-
ity over 25 MC dropout samples is measured as the ﬁnal predicted probabil-
ity for both foreground-background classiﬁcation and contour detection. Then
the foreground-background and contour prediction are fused and post-processed
with the second method in section 2.1.4. F1 score and Dice index are utilized
to give the quantitative results for describing model performance.
We split the 85 training images to be 10 labeled training images in the labeled
set L which are utilized to train the initial model, 65 unlabeled images in the
unlabeled pool set U from which to select queries based on acquisition function
and 10 validation images. The initial training set L includes 5 benign images
and 5 malignant images to keep the uniform distribution between benign and
malignant cases. To prohibit the situation that a bad initial model will hurt
the performance of the acquisition functions, we use a multi-start strategy for
training the initial model. In other words, we select the best initial model in
repeated 5 experiments with the same training images based on the performance
on validation dataset. Then this initial model is regarded as the starting point
for all the acquisition functions and random selection as shown in the ﬁrst step
of algorithm 1.
In each acquisition step, 5 images are queried and removed from the unlabeled
pool set U based on the estimated uncertainty score. Then the selected images,
as well as their ground truth labels, are added into the labeled training set. After
that, a new model is obtained by re-training from scratch1 with the updated
training set. Training from scratch is for avoiding local optimal and confusing
model performance improvement with an improvement resulting from simply
using longer (cumulative) optimization time [GIG17]. The model is trained with
a max epoch 1300, mini-batch size 5, initial learning rate 0.001, L2-regularization
parameter λ 0.001, and deep auxiliary supervision decay every 300 epochs (w
is decay by 0.1 every 300 epochs until it is totally dropped after 900 epochs
as shown in equation 2.4). The exponentially decay steps for learning rate is
diﬀerent regarding a diﬀerent number of the training images.

a pixel a high uncertainty value if the model is uncertain and also ambiguous
about, as shown in equation 2.15a (since we have diﬀerent prediction results in
diﬀerent MC dropout samples). It indicates that those pixels, which are the
neighbors of the contours, satisfy the ﬁrst requirement since the model is un-

uncertainty value concerning averaged predicted probability. However, they fail
to satisfy the second requirements, since the model is likely to produce diﬀerent
predictions in each MC dropout samples, which means they will also have a
high averaged uncertainty. Therefore, the high uncertainty in terms of averaged
predicted probability and the high averaged uncertainty are canceled out by
equation 2.15a compared to Dropout VarRatio and Dropout Entropy.

model can produce a reasonable predicted probability, then the uncertainty es-

relation between predicted probability, the accuracy of the predictions as well
as the estimated uncertainty, the connection between the number of wrongly
and correctly classiﬁed pixels with respect to the estimated uncertainty is taken
into account as shown in Figure 4.9 and Figure 4.10. Only acquisition functions

each of them also has 90% of conﬁdence for belonging to class k, it means there
should be around 900 pixels which are correctly classiﬁed, and 100 pixels which
are wrongly classiﬁed. Therefore theoretically the ratio between the number of

Conclusions
5.1
Summary and Discussions
In this thesis, we present the framework of using active learning for image data
with deep neural network. We propose the use of active learning in a multitask
deep neural network for object recognition task in images. The experimental
results demonstrate that the proposed active learning framework allows the
model to have a similar performance with less amount of data, which means
the annotation eﬀort and cost can be highly decreased because of data-eﬃcient
learning.
In our work, we combine the deep residual network [HZRS15] with deep con-
tour aware network (DCAN) [CQYH16] for object recognition task. It is shown
that the utilization of deep residual network can ease the training, increase the
training speed as well as improve the generalization. Therefore the performance
of the model using deep residual network highly surpasses the state-of-art per-
formance [CQYH16].
It is known that the annotation for diﬀerent types of labels (bounding box label
or full mask) will arouse various labeling cost for oracles. In order to allow the
active learners to query diﬀerent types of label, we design a multitask deep neural
network for object recognition, contour detection and bounding box prediction.
==PAGE:92==
76
Conclusions
The experimental results demonstrate that the shared feature between each task
can boost its sub-task performance in comparison to the single task deep neural
network.
Most of active learning acquisition functions rely on model uncertainty, however
deep learning methods mostly focus on the relative uncertainty within diﬀerent
class labels.
To enable the estimation of model uncertainty, dropout layers
[GG15] are added in the deep neural network. Then this model uncertainty is
incorporated in each acquisition function for measuring the informativeness of
the images. The experimental results demonstrate that acquisition functions
can signiﬁcantly outperform random selection after enough acquisition steps.
Further more, our work has shown that with the proposed the active learning
framework, we can query the most informative images with which the deep
neural network can be trained to achieve a similar performance in comparison
to the model which is trained with full amount of training data.
This work further discusses the relation between the predicted probability, es-
timated uncertainty and accuracy of the prediction. The acquisition functions
tend to outperform random selection when the predicted probability are highly
correlated to the accuracy of the prediction. Besides, we also consider the in-
ﬂuence of data eﬀectiveness on the performance of acquisition functions.
It
indicates if the images in the pool set can be highly diﬀerentiated by the es-
timated uncertainty, then the most uncertain images selected by acquisition
functions can rarely have the chance to be also selected by random selection.
As a consequence, acquisition functions tend to outperform random selection.
5.2
Further Work
This thesis work can open up many opportunities for the use of active learning
in deep neural network. In our work, we only consider to use dropout layer for
the extraction of model uncertainty, other approaches i.e., bootstrap can also
be utilized to express model uncertainty.
Besides, object recognition in images is a pixel-wise classiﬁcation task, which
indicates that the uncertainty value for all the pixels in an image need to be
aggregated to represent the overall uncertain level of this image. In our work,
we have only considered to sum up the uncertainty value over all the pixels.
However, it is observed from the experimental results that actually only a part
of pixels in each image are most uncertain. By this way, we mean that an image
can be divided into three part based on the uncertainty value: most uncertain
region, accepted uncertain region and most certain region. Therefore if only
==PAGE:93==
5.2 Further Work
77
the most uncertain region in an image is selected for the annotation, then the
labeling cost can be highly reduced. This maybe achieved by using the bounding
box prediction, or maybe divide the image into diﬀerent patches and only select
the most uncertain patch or create a new aggregation method to strengthen the
most uncertain area.
Then the investigation of the relation between predicted probability, estimated
uncertainty and the accuracy of the prediction indicates that there could be
an "open window" for the active learning. It means active learning will have a
similar performance in comparison to random selection inside of this "open win-
dow". However, active learning can signiﬁcantly outperform random selection
after. Therefore, it would be interesting to identify when active learning should
be used instead of random selection.

Kruskal-Wallis test
The Kruskal-Wallis test [Con99] is utilized for testing if the population from
diﬀerent groups are identity. The assumptions for Kruskal-Wallis test include:
1. All samples are random samples from their respective populations
2. In addition to the independence within each sample, there is mutual in-
dependence among the various samples.
3. The measurement scale is at least ordinal
4. Either the k population distribution functions are identical, or else some
of the populations tend to yield larger values than other populations do.
The hypothesis for Kruskal-Wallis test is:
H0: All of the k population distribution functions are identical;
H1: At least one of the populations tends to yield larger observations than, at
least one of the other populations.
