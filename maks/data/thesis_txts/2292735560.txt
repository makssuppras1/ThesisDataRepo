Abstract
This thesis describes the implementation and use of a method to update a
mathematical model for a vibrating system based on experimental data. The
method is initially tested on simulated data where di↵erent sti↵ness and mass
is applied to test the robustness of the method. Thereby the method is applied
to a real system supplying experimental data. The overall objective is to obtain
a method which can update a damped rotating system, and locate the position
of deviating physical parameters in the analytical model. Additionally using the
method as an identiﬁcation method for unknown characteristics of rotor dynamic
systems.
1
Introduction
Model updating can be described as an adjustment of an analytical model based upon
experimentally measured data. The main goal of model updating is to represent the
dynamic behavior of a system more accurately.
In the ﬁeld of dynamic structures
several updating techniques has been proposed, but as described by Imregun[3], two
approaches are most widely used. The Inverse eigensensitivity method(IESM) and the
Response function method(RFM). The main di↵erence between these two methods is
the basis of updating. Whereas the IESM considers the eigenvalues and eigenvectors
obtained from the modal analysis of measured response functions, the RFM considers
the frequency response function directly. In this thesis the RFM is used to update
an experimental model. In section(2) di↵erent updating techniques are described with
their advantages and disadvantages.
Updating a vibrating system can hence be done with a variety of methods with the
main goal of obtaining an accurate analytical model. When updating a rotor dynami-
cal model, another important beneﬁt of the updating process is being able to determine
physical parameters of rotor dynamical elements, such as bearings, discs and gears.
The chosen method must therefore be able to update nodal parameters in the mathe-
matical model in order to be useful for a rotor dynamic system. This is only possible
for methods that have the ability of locating the error in the analytical model based
upon the experimental data hence this is also a criteria to choose the best updating
method.
In this report the RFM method is implemented and tested on a simple vibrating system
and also a new application for the method is used to determine physical parameters
mounted on the system. Thereafter gyroscopic e↵ects are added to the method and
the measurement data from a rotor with ﬂuid ﬁlm bearings and ball bearings are used
to update the mathematical model.

2.1
Anti Resonant Frequencies [4]
The motive for this method is the resulting inaccuracy when using modal information
such as mode shapes and natural frequencies. Even though the natural frequencies can
be calculated accurately the mode shapes tends to be accurate within 10%. There-
fore this method is based on anti resonant frequencies which are more accurately than
mode shapes. To identify the anti resonant frequencies the method uses transmissibil-
ity functions, which are deﬁned as the ratio between measured outputs of two di↵erent
positions of the structure. Hence the method does not require measurement of excita-
tion forces which is an advantage in terms of model updating, since an issue in model
updating is to reduce the dependence upon measurable excitation forces. It is though
necessary to know the position of the excitation.
According to Meruane[4] recent studies have shown that transfer anti resonance makes
the method very unstable, thus the anti resonance from point FRFs must be used.

2.3
Inverse Eigensensitivity Method(IESM) [3]
The IESM is based on ﬁrst obtaining an eigen description of the structure using modal
analysis of measured response functions. Thereby a di↵erence vector is introduced as
the di↵erence between the analytical and experimental eigenvectors and eigenvalues.
The model is linearized and a set of p-values are introduced as shown in equation(2.12)
where the sensitivity matrix is dotted by the p-values in order to obtain the di↵erence
vector.
S · p
=
δ
(2.12)
The p-values are used as updating coeﬃcients and the analytical model can thus be
updated by multiplication of the p-values. The method is hence dependent of modal
analysis which requires the user to be able to measure mode shapes of the structure.
Since equation(2.12) is solved by the least squares method the solution is not unique,
since di↵erent initial conditions will produce, equally correct, but di↵erent solutions.
In the paper[3] the method was tested on a simple system consisting of a plate in a
free-free condition and proved successful to update the mathematical model.
Advantages
• Can cope with a noisy experimental data
• Good location of error
• Easy to introduce new parameters
• Can be applied to a system with a theoretical response deviating a lot from the
experimental one.

Disadvantages
• Modal analysis is required
• The solution is not unique
2.4
Response Function Method(RFM) [2, 3]
The RFM uses the response functions directly and a set of over-determined system
equations is obtained. The method introduces a set of p-values, as the IESM, to update
the analytical model and the iterations are done by solving the over-determined system
of equations shown in equation(2.13) where G (!) represents the sensitivity matrix
dotted by subsequently the FRFs of the analytical model and the experimental FRFs.
∆↵(!) denotes the di↵erence in FRFs between the analytical and experimental model,
hence the main goal of the method is to minimize this right-hand side of the equation.
G (!) · p
=
∆↵(!)
(2.13)
The p-values multiply on updating parameters chosen by the user such as elements
and nodes in the model, which allows for locating the errors in the analytical model.
Equation(2.13) is solved for the vector p, in order to obtain the p-values, by the least
squares method hence the solution for this method is not unique as is the case with
the IESM.
In the papers[2, 3] the method was tested on simple structures as plates and beams in
a free-free condition and proved successful.
Advantages
• Good location of error
• Easy to introduce new updating parameters
• No need to use modal analysis
• Good at ﬁne tuning the updating process e.g. after the IESM has updated the
system with large deviations
Disadvantages
• The solution is not unique
Based on the review of the di↵erent methods the appropriate method for this report
is the RFM since the method provides a presumable easy way to introduce updating
parameters in the form of p-values. It is not necessary to perform a modal analysis
but it should be kept in mind that the solution is not unique. As large deviations
between the theoretical and experimental response is not expected for the cases in this

thesis, it is also considered that the updating will be a tuning process for which the
RFM is a good approach. Imregun[3] proposes to use both the IESM and the RFM
if the updating process should be fully stable. The IESM can then roughly correct
the model and thereafter the RFM can approximate the model to ﬁt the experimental
data.
Since for this report the idea is to test and use an updating method for the purpose
of updating a rotor, which has a well-deﬁned FE-model, the RFM is considered to
be suﬃcient in order to update the model. Also this will provide a foundation for
future work in the ﬁeld of model updating where the IESM can be implemented as a
supplement to the RFM.

4.4
Updating a damped structure with noise
When experimental data are measured it is inevitable that noise in the data is present.
The noise is generated from both measuring equipment and from uncertainty when
performing the experiment.
To test the method against noisy data it is necessary
to generate fake noise in the simulated data. Equation(4.9) shows the principle of
generating noise. The factor ✏denotes the user-deﬁned maximum noise allowed in the
model in %.
Hx.noise (!)
=
Hx.initial (!) · (1 + r · ✏)
(4.9)
The simulated data are multiplied by a normally distributed random factor r and
thereafter multiplied by a user-deﬁned maximum noise. The factor r is preferred to
be in the range r = [−1; 1] and since it is known that at least 99.7% of the numbers
will be within µ + 3σ it is chosen to set σ = 1
3, and use the built-in Matlab-function:
randn.
Matching the FRFs
The method is tested with noisy data for di↵erent settings of ✏. The added nodal
sti↵ness and mass are in the same position as described earlier. Also the cross-section
properties of the simulated data are maintained. Due to the randomness of the factor
r the noisy data will not be the same by each run of equation(4.9). Therefore it is
chosen to do 5 runs for each setting of ✏. The resetting of ✏is continued to examine
di↵erent settings of ✏. The maximum level of noise is set to 20%, since it is considered
to be the maximum amount of noise which could be present in a real test provided the
test is done carefully.
The average number of iterations, before convergence, for each setting of ✏is plotted
in ﬁgure(4.9). On top of each column the standard deviation for the 5 runs per ✏
is displayed. The standard deviation shows that for some ✏-values there was a large
di↵erence between the number of iterations the method took to obtain convergence.
The lower standard deviation the better the column height represents the average
number of iterations for the current ✏-value. High values of σ occurs for high values
of ✏, which is also expected since equation(4.9) generates random values within an
increasing range for an increased value of ✏.

8
Conclusion & Future aspects
After reviewing state-of-the-art methods within the ﬁeld of model updating the re-
sponse function method was chosen as the updating method for this thesis.
The
method proved to work for simulated data and updated a mathematical model con-
taining mass, sti↵ness and damping. A new application to identify nodal parameters
were implemented and tested on the structure with simulated data which also gave
promising results. The method were thereafter used to update a structure using ex-
perimental data obtained by subsequently using an impact hammer and a shaker. It
was possible to update the model using either the hammer or the shaker data, if only
sti↵ness and mass were considered in the model.
Some speculations regarding the method:
• When structural damping was introduced the shaker data, did not update as
well as the impact hammer data.
This might have to do with complications
regarding the test setup for the shaker, since the coupling between the system
and the shaker might have introduced additional sti↵ness or damping to the
system. Even though additional sti↵ness and even damping might have been
added to the system the method should be able to update the model, hence this
is regarded as a weakness of the method.
• The convergence of the method has proven to be dependent of the discretization
and range of frequencies in which the response function is acquired from the
experimental data. This means that when applying the method di↵erent settings
for the !-range must be tested in order to obtain convergence.
• A reason that the method in some cases do not ﬁnd convergence might also
be caused by the fact that experimental data are subject to incompleteness of
the measured frequency response functions.
A review of papers[2, 3] within
this subject also suggest that the incompleteness of measured FRFs is a major
obstacle in order to make updating methods converge.
Nevertheless the method was used to update an additional sti↵ness added to a node
on the structure, which proved that the method could potentially be used as an iden-
tiﬁcation method for nodal parameters. It were though discovered that the equivalent
sti↵ness of the beam in position of an added sti↵ness might be of importance of the
updated result, since the method uses the least squares method to solve the objective
function, thus more, equally correct, solutions can be achieved.
After the method proved to work for a structure a new application for the method was
the implementation of gyroscopic e↵ect in order to update a mathematical model of
a rotor. As for the structure the new application was initially tested with simulated
data which gave good results.
The method tended to match the FRFs worse for
higher natural frequencies when gyroscopic e↵ects were added. The method’s ability
to update nodal parameters were once again tested and now both damping and sti↵ness
were updated, since this would mimic a journal bearing mounted in a node. Finally the
82
==PAGE:88==
method was used to update a test-rig with a rotor and journal bearings for di↵erent
rotational speeds.
Speculations regarding updating a rotor:
• The nodal updating proved to update the nodal parameters in the direction of
the excitation force better than in the perpendicular direction. This is assumed
to be caused by the use of a numerical method, but should be taken into account
when using the method to predict nodal parameters.
• It was found that the quality of the data might have had an inﬂuence of the meth-
ods ability to update, since for some rotational speeds the horizontal direction
did not update and for others the vertical did not update correctly.
• The approach to deal with complex p-values seemed to complicate the solution
process when using experimental data for a rotor.
The nodal parameters for a mounted tilting pad journal bearing were updated and
compared with another identiﬁcation method, which proved that the updated param-
eters were in good correspondence.
8.1
Future aspects
The main obstacles regarding the RFM are as follows:
• Incompleteness of experimentally measured frequency response functions
• Introducing damping with experimentally measured frequency response functions
• Using a suﬃcient !-range
To address the problem with incompleteness of the measured FRFs, other methods
to complete the FRFs should be investigated.
In this thesis two approaches were
tested namely interpolation of measured FRFs and substitution of analytically ob-
tained FRFs. It was proven that substitution of analytically obtained FRFs gave the
best results in the cases treated in this thesis, but other methods to complete the
measured FRFs, might prove more successful.
Another approach to deal with complex p-values should be investigated in order to
update a rotor dynamic system. In general the introduction of damping to the system
complicates the method and makes the solution process unstable.
In order to improve the experimentally obtained FRFs the method could be expanded
to update using two columns from [↵X (!)] at the same time. The method was initially
derived using the entire [↵X (!)], but as stated by Imregun[2] only one column would be
suﬃcient for the method to update a mathematical model. However it is believed that
including more than one column would provide more information about the physical
system, hence the method might ﬁnd convergence more easily.
83
==PAGE:89==
The cross-coupling e↵ect caused by updating a rotor, might also inﬂuence the results
of updating. Therefore it is suggested to investigate the e↵ects of obtaining the FRFs
in the direction of which they have the largest magnitude. By doing so the response
from the system would be more signiﬁcant, thus providing more information about the
dynamics of the system.
