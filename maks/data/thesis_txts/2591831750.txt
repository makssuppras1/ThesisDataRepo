Abstract
As the need for better mobile communication network rises both for 4G and 5G tech-
nology, planning is essential to ensure necessary coverage and capacity for the end user.
Current literature relates terrain and radio propagation by using categories of terrain
or probability function. For an easy prediction, models using available terrain data are
desired.
In this report, it is investigated how available terrain data (LIDAR) can be used to
determine Line Of Sight (LOS). LOS determination, based on a direct view and based
on Fresnel geometry, as this describes the radio propagation, are developed. A method
for mapping Fresnel geometry is shown and used to approximate LOS as a 2D plane.
Based on the two ways of determining LOS, path loss predictions are carried out and
compared to measured results from a drive test. The drive test was carried out at DTU
campus in Lyngby, Denmark and the predicted values are based on a mapping between
recorded coordinates and LIDAR data.
To reduce the complexity deep learning is used to predict and scale LOS prediction
for more efficient computations at similar accuracy level.
Deep learning models are
trained to predict LOS and path loss based on the available data and a super resolution
method is used to increase resolution of predicted values.
Based on the proposed models it is found that using terrain data to predict radio
conditions is useful in terms of planning. LOS is mapped in a useful manner and re-
lated to path loss prediction using terrain observations. Deep learning can assist in the
prediction and provided the right data model path loss based on LIDAR data.

CHAPTER 1
Planning and deploying
mobile communication
networks
The use of mobile networks, mobile phones and data transmissions over radio links have
increased drastically in recent years[10]. The usage of these technologies is available
due to access points, such as a WiFi hot spot, Evolved NodeB (eNB), etc.
As the
development of these technologies continues, new deployments of access points will follow.
These deployments are to be carried out in a smart and enlightened fashion, such that
resources are not wasted and/or users are unable to connect to the service.
As the data demands increase, not only will it be necessary to upgrade the existing
4G network, but also roll out the next generation network, namely the 5G New Radio
(NR) network. 5G is capable of slicing the use cases and more effectively use the fre-
quency bands, as well as utilizing now unused frequency space for data transmissions.
To avoid excessive use in terms of materials, power and frequency to provide coverage
and capacity for end users, one would have to obtain an idea of how signals from mobile
communication technologies propagates in a given environment. This also works the
other way around, where an operator of a mobile network invest resources in providing
end users better connectivity, but fails to do so in case of wrong modelling. In terms of
network planning, finding suitable locations for an eNB is essential, as each deployment
is costly to the operator and as a consequence of that also to the end user[41].
Modelling the propagation of radio waves is key in planning mobile networks, but is
also not straight forward. The signal is very dependent on the environment it travels in.
To accommodate the issue of mapping the propagation to a specific environment
this report investigates how to use Light Detection and Ranging (LIDAR) data and
LOS modelling to predict radio environments. Many conventional methods use proba-
bilistic models to model the environment rather than actual terrain data[31]. This is
a convenient approach, allowing the model to be of general use, rather than being of
limited use. Other approaches require access to detailed information about the terrain,
such as most ray-tracing methods use. While terrain specific, more general libraries than
the ray-tracing allows, can be used for a simpler approach.
Throughout this report path loss models are investigated and by using information
about LOS learned from LIDAR data, path loss predictions are made. LIDAR data for
all of Denmark exist and thus is LIDAR data a general approach, which can be used in

1.2
4G LTE
4G or LTE is a standard developed by Third Generation Partnership Project (3GPP)
describing a wireless transmission system[36]. 4G is deployed in large scale and while
the next generation of transmission technology is around the corner, it could takes years,
even, decades before 4G is obsolete. Though 4G is available in most of Denmark[42]
this is not the case everywhere in the world and thus investigating radio properties of
4G is important. Given that 4G is under further development, newer editions are also
deployed and current network updated and upgraded. A sketch of a 4G LTE network
is shown in Figure 1.1. The Figure showcases how User Equipment (UE) connect to
the internet through 4G. In an LTE network, the Mobility Management Entity (MME)
controls the eNB while theServing Gateway (sGW) and Packet Data Network Gateway
(PGW) provide the connection to the internet. While these are all interesting aspects
of an LTE network, this project is about the radio link between the eNB and the UE.
4G operates on frequencies from 400MHz to 5000MHz, though 800-2600MHz is the
most common in Denmark. Frequencies are important as the attenuation of a signal,
and thus the quality at the receiver, is heavily dependent on frequency. The frequencies
are sub divided in bands of up to 20MHz width. A single cell may use up to 20MHz

better signal from another eNB.
The measurements of interest in this project is the RSRP as the RSRP describes
the received power over the entire band since the reference measurements are present
with each resource block and RSRP is calculated over the entire bandwidth. Though
the RSRQ might describe the received quality better, predicting it requires information
about transmission powers of each nearby eNB as interference has to be taken into
account.

The modulation scheme choice depends on the
Channel Quality Indicator (CQI). CQI calculation is vendor specific as different hardware

1.3
5G NR
5G NR is the next generation of mobile networks[37]. The peak download rate can be
up to 20Gbit/s. It has some similarities to 4G, as it uses OFDM to divide subcarriers.
The subcarriers however can be divided into smaller intervals in the time domain. The
largest difference is the introduction of new frequencies. 5G is split in 3 main categories,
namely low-band, mid-band and high-band. Low-band operates on frequencies from
600-900MHz, mid-band at 2.5-4.2GHz and high-band at 24-47GHz. Some frequencies
in between are excluded as they are reserved for other purposes, e.g. radar, emergency
calls, licence free spectrum, etc. The low band frequencies are similar to the ones used in
4G and the resulting speed increase is not huge, but as the frequency is low, the coverage
is great. The mid-band ranges from 2.5GHz to 4.2GHz, but is more uncommon. This
frequency range gives download rates of up to 1Gbit/s. For the high-band or millimeter
Waves (mmW) frequencies the range decreases significantly but the download rates are
in some cases tripled compared to mid-band.
This means that 5G can use new frequencies to achieve higher speeds, thus providing
better capacity for the end user. It is important when deploying 5G to analyse how an
area is covered at these new frequencies as terrain or other factors block or alter the
signal in a way, such that the end user is not experiencing the promised capacity and
services.
Aside from the new frequencies does the 5G standard introduces some new features
and possibilities, such as Internet of Things (IoT), Machine to Machine Communication
(M2M), Ultra Reliable Low Latency Communication (URLLC) and Massive MIMO
(mMIMO). As some of these enable rather critical uses, such as autonomous cars[27],
accurately predicting the radio environment is important. Otherwise can critical infras-
tructure not be assumed reliable. Accurate channel modelling is thus of great importance,
otherwise can a provider not guarantee the promised URLLC network to actually be
ultra reliable or have low latency. If the signal fails to transmit it is neither low latency
or reliable as retransmissions would increase the latency and the failure in the first place

It is therefore very important to study channel modelling.

CHAPTER 2
LOS, path loss and Fresnel
geometry
Since predictions of new and existing mobile communication networks are important
to ensure the promised coverage and capacity, literature on this subject is investigated.
Path loss predictions are an important trade in the field of mobile communications. This
chapter investigates path loss models and LOS approximations and relates them to each
other.
2.1
Path loss
Accurate path loss predictions are a foundation for deployment of mobile networks. Path
loss predictions gives a realistic insight to determine if a radio link is feasible or under
which circumstances it can be.. The path loss can also be used to estimate the expected
amount of data a radio link can carry. This enables and ensures consistency in a mo-
bile radio network before deployment as well as handling changes in the environment.
Changes could be a new large building, that changes the radio environment, higher user
demand, power outages, etc. Based on changes one can then remodel the radio environ-
ment and investigate if the current deployment is sufficient or other measures needs to
be taken into account.
By using path loss models one is thus able to account for predicted events (e.g.
the construction of a new building) as well as unanticipated events (e.g. cable cuts)
without having to set up a test station (with the appropriate licence) and carry out large
experiments to best account for a change in the radio environment since deployment of
the network.
2.1.1
Path loss metric
A path loss model describes the expected attenuation of a waveform emitted from an
antenna, received by another antenna.
As commercial radio environments on Earth
rarely takes place in vacuum, the electromagnetic wave will interact with its surroundings
as it propagates. The level of interaction is dependent on many factors, such as the
frequency of the wave and the medium it travels through. Especially objects in the
direct path can be problematic, and so can scattering and interference. Unfortunately
==PAGE:24==
8
2 LOS, path loss and Fresnel geometry
it is more complex than that as the geometry of the emitting and receiving mediums
(antennas) play a large part. Antennas tend to emit signals in all directions. This can
cause scattering along the path of propagation and cause self interference, but also give
constructive interference if the reflected waves are in phase. When this (self) interference
results in attenuation it is referred to as multipath fading.
When the signal is attenuated due to large still objects, such as the earth, large
buildings or obstacles invariant of time, it is referred to as large scale fading, shadowing
or slow-fading. Attenuation from small time variant obstacles referred to as scattering,
small scale fading or fast fading. Other effects such as the Doppler effect can also occur
if either the transmitting or receiving end is moving. If transmitter and receiver are
approaching each other the signal will be shifted to a higher frequency and vice versa
when transmitter and receiver are distancing each other. Usually though, it will be
the receiver, e.g.
the mobile user, who is moving either away from or closer to the
transmitter. A brief description is given in appendix A.
Radio links can in general be described with a link budget calculation:
Prx = Ptx + Gtx + Grx −PL −loss
(2.1)
Equation 2.1 describes the received power Prx as a sum of the transmitted power Ptx and
antenna gains Gtx and Grx with the path loss PL and other losses subtracted. Other
losses can be modelled in the path loss term PL, such as thermal noise, but other sources
of loss, namely loss in cable are usually modelled separately.

this project is not to go through and evaluate all of them, but as they provide a funda-

and maintaining a mobile communications network. A way to predict the radio envi-

into account, but can showcase the received power at the end user and thus reveal if a

easy.

implementation is done in a worst-case scenario. Due to the complexity of the model,
however, ”every day” use is not recommended as many factors have to be estimated
numerically and local variations as weather and refractivity of the earth are included.
Simplifications exists[46] making it more accessible, but the model as a whole relies on
classifying terrain rather than applying the terrain data directly.
2.1.2.3
Other path loss models
Supplementary models are a category of models, that only add or modify existing models,
and as such is unable to be used as a stand alone implementation. While integrating
supplementary models for the purpose of more accurate models is a respectable goal, the
scope of this report is not integrating supplementary models in existing models. The
supplementary path loss model are thus only of academic interest in the view of this
project.
Stochastic models often add random variables for time and/or frequency variance
to simulate small scale fading. In many cases this include some Gaussian distributed
attribute with a mean of zero. As this project is mostly focused on large scale fading
properties, the stochastic models are not further elaborated on here.
Ray tracing models[47] provide good estimates, but relies on high resolution data of
terrain, obstacles and material to accurately predict the radio environment. Libraries
for ray tracing models are not widely available and can be hard to assemble as most
buildings materials has to be known/estimated. Ray tracing is thus not investigated
further as the process of assembling the necessary data is likely to be of more trouble
than the obtained results are worth.
Active measurement models assume that predictions using prior information is insuf-
ficient and uses measurements and interpolations techniques to predict the radio envi-
ronment. For optimal use, and easy/cheap deployment, a goal of this project is to use
terrain data to predict the radio environment, thus omitting doing expensive measure-
ments. In a scenario of deploying a new base station, several location may be available

2.1.3
Modeling RSRP from path loss
As path loss is not directly measured in the LTE standard one has to do a work around
using RSRP to be able to compare results of model work with measurements of deployed
networks. If one had the proper licenses and equipment it would be possible to measure
the path loss based on the known transmitted signal and the received signal. With only
the received signal available, however, fair estimations can be made.
RSRP is a signal in the LTE standard and is used as a reference power over the entire
band. In each frame a reference power signal is transmitted and the resulting power (in
dBm) can then be used to evaluate signal quality and give important information about
signal strength and which modulation to use. In the case of predicting a RSRP value
a fair estimation can be done by applying Equation 2.1 as it describes the received
power. To do a fair estimation of RSRP all one need, besides the estimated path loss,
is transmitter power and the antenna gains. According to [16] a transmitter gain of
17dBi and receiver gain of 0dBi is the most common. The transmission power ranges
from 41dBm to 49dBm based on scenario and bandwidth. Using these values in the link
budget from Equation 2.1 yields the estimated RSRP.
As only 4 out of 7 by 12 symbols in a LTE resource block carry the reference signal
used to asses RSRP the cell load affect the measured RSRP as the RSRP is measured
over the entire bandwidth. In [34] a difference in the order of up to 15dBm is reported
between a loaded and empty frame depending on interference.

2.2
LOS
Two large standardization institutes, namely 3GPP and The Institute of Electrical and
Electronics Engineers (IEEE), both use LOS and NLOS channel models to predict path
loss[45]. LOS is a widely used term in the literature. The definition is, however, loosely
defined. For the most part LOS is when there is an unobstructed view in a straight line
between the transmitter and the receiver. This definition is easy to comprehend as one
can verify whether or not LOS is present by observing with the naked eye. However
some literature uses another definition for LOS. As the transmitted signal is not visible
light, but radio waves of much lower frequencies another factor comes into play, namely
the Fresnel zone. In these scenarios an obstructed LOS can still be classified as LOS if
only part of the Fresnel zone is occupied.

2.2.1.1
Diffraction and knife edge diffraction
As waves propagate through space they are diffracted by the obstacles they hit. The
diffraction will lead to a power loss as not all secondary waves of the diffraction are
diffracted around the obstacle. The received power is therefore a vector sum of the
energy from all Fresnel zones not obstructed. As a rule of thumb, as long as the first
Fresnel zone is unobstructed, the diffraction loss is rather minimal. For microwave links
it is practice to allow up 55-60% of the first Fresnel zone to be obstructed.

Summary
Previous studies have investigated the connection between terrain and path loss for
mobile communications.
Most of these, however, tend to use meta data about the
terrain or subdividing them into terrain type. Those applying terrain data directly rely
on complex terrain data. A simpler approach could be to use LIDAR data as terrain
data. This also requires a way to model LOS, which can be done in a simple way or
from a Fresnel geometry perspective. The Fresnel geometry is proved to have valuable
information about the radio propagation as it can describe the increase in path due to
obstacles and diffraction. Combining the properties of the easy accessible LIDAR data
and the Fresnel geometry to predict LOS and based on that predict path loss is thus
expected to provide good results.

The problem at hand
The literature presented show that using Fresnel geometry and terrain data provides
useful knowledge about the radio environment for the terrain. Using a straight line to
determine LOS is interesting to investigate further and compare to a soft state LOS,
which takes the Fresnel geometry into account. The goal for this project is thus
Based on LIDAR data to predict a LOS measure, which can be utilized to model the
radio environments for 4G/5G mobile communication systems.
LIDAR is the terrain data chosen due to the high availability and resolution. The mobile
communications technologies 4G and 5G are chosen as they make up the current and
future technology for most data transfers. Connecting the properties of the terrain with
the performance of a mobile network thus yields great use in a real world application
scenario. Otherwise could modelling at best or worst case scenarios lead to heavy over
dimensioning of the network capabilities in some areas and under dimensioning in others.
Both 4G and 5G reuses frequencies and allow for micro and pico-cells to cover ”holes”
in the network and thus is identifying these location important, if those holes are to be
covered. As described can this modelling turn out to be rather complex and the power
of deep learning is investigated, as it is expected to assist or even improve the results.

transmitter location has a lot to say for how a LOS map looks. A solution is to place
the transmitter at the center of the image for all training images as this gives the same
starting point for finding LOS without giving the placement of the transmitter as an
input. This has drawbacks as the terrain might have a ”valley” at that location. If that
is the case then the transmitter is highly unlikely to be placed there in the first place and
the LOS prediction is likely to be all NLOS. Another option is to place the transmitter
in the corner, but as the CNN works with filters of sizes [1×1, 5×5] (typically), the
transmitter might by so far away from the locations that correct LOS prediction is
unlikely.

Super resolution is in short when the amount of pixels in an image is increased. The
ideal super resolution algorithm is able to fill in the blank spots. What to fill in, in the
blank spots is of course not straight forward, but previous work has proved deep learning
with CNN can yield impressive results[23]. By extracting key features, the algorithms
are able to fill in the blanks and produce quite astonishing results.

The depth of the input image is typically 3 (RGB) or 1 (gray scale). The feature map is
a representation of the images given as inputs, but the feature map can have a smaller or
larger dimension compared to the input picture. The feature map represent the features
of interest in found in the images. As not all pixel are of interest mapping to a feature
map smaller that the input images often makes the most sense. The feature map is a
product of a chosen filter (or kernel). The filter extract a single value from and area at
same size as it self. Typically a filter will be of size 2 by 2, 3 by 3, 4 by 4 or 5 by 5. As
the features extracted from the filter are still of significant size they are usually further
reduced using a pooling layer. The pooling layer pools each feature from the filter size
to the pooling size, e.g. a 3 by 3 filter would have a 3 by 3 feature and by using a pooling
layer with a 2 by 2 filter, the feature is now reduced to 2 by 2. usually the pooling is a
Max Pooling, taking the largest value as it is considered the most important. However,
depending on the application, other pooling methods may be of use.
After pooling and reducing the feature map, a mapping between the features and
the desired output is needed for the algorithm to be of use as a whole. This is usually
done with some scaling function or layer mapping from the feature map to an image of
the desired size.
To control all the filters and the mapping, an optimizer is needed. The optimizer
modifies the network to increase performance, e.i. lowering the error. The Adam opti-
mizer[24] is used in this project. It chosen because it was designed for deep learning, is
easy to implement and performs well. Other optimizers could also have been considered,
but evaluating which is the best optimizer is somewhat out of scope for this project. The
Adam uses an adaptive learning rate for each parameter. The learning rate describes
the rate each gradient changes. Having a large learning rate makes the search for a min-
imum of the error function quick. A small learning rate will most likely find a smaller
minimum at the cost of more computing time and the risk of finding a local minimum
rather than the global minimum. The adaptive learning rate combines these properties
by finding the (presumed) global minimum and then tuning it afterwards.

4.2.1
GPS basics
GPS is a service operated by United States Space Force. It is a system of satellites,
very commonly used in navigation systems such as navigation for cars and in many
mobile phones. It is one of several types of Global Navigation Satellite System (GNSS).
Other examples include the Russian GLObal NAvigation Satellite System (GLONASS),
European Galileo or the Chinese BeiDou Navigation Satellite System (BDS). All systems
work by having a constellation of satellites in orbit and receivers on Earth capable of
calculating their position from the broadcast signals of the satellites.
The satellites utilizes accurate atom-clocks, synchronized with atom-clocks on the
surface of the earth. To determine a position 4 dimensions is needed, namely a position
in 3D space and a timestamp, e.i. the 4th dimension. Thus a receiver must receive
transmissions from at least 4 satellites.
With signals from more than 4 satellites a
fitting method is used as there are 4 unknown but more than 4 inputs. In the broadcast
signal of the GPS satellites are signals of their trajectory and by calculating backwards
from the timestamp received, a position is known[14].

here illustrated how a reflection is used as the received signal, meaning, that the signal

5.6
Deep learning
Though one can obtain the results presented above quite fast, a more accurate yet faster
approach was desired. Thus it seemed appropriate to implement a machine learning
algorithm.
In the field of machine learning an extremely large amount of methods
are available.
While all has pros and cons, the choice fell on a neural network.
A
neural network can tackle complex problems fast and efficient. One of the drawbacks is
the hidden layers. These layers account for larger parts of the results, but from their
structure an understanding of how the problem is solved is hard to obtain. Thus such
a network would rather provide accurate result than a deeper understanding of how to
solve the problem in an efficient way. Another reason for choosing neural network is
their ability to handle images through convolutions, which again can be optimized on a
Graphics Processing Unit (GPU) for efficiency. The advantages of this method outweigh
the drawbacks.
5.6.1
From LIDAR to LOS
Though efficient, deep learning networks, have limits as to how large pictures they are
able to handle. This limits the use, but different proof of concepts were tried out. Since
the drawback of the implementation in Matlab was run time at full scale, training the
network to predict where LOS would be, seemed appropriate. The deep learning method
would have to predict a binary matrix of same dimension as the input picture. This could
in theory be any size, but training on too many parameters at once, would most likely
not lead to good results and thus a smaller cutout was chosen. These cutouts were 64 by
64 points. Since this was in an early stage a proof of concept was more important than
comparable results. Especially given the computation time, where a laptop computer
fast and efficient could train the model sufficiently. The cutouts was found by splitting
the 5000 by 5000 points full scale image. Due to the high resolution each picture would
vary some, but not dramatically change between locations. For simplicity it was also
chosen to place the transmitter at the center of the picture and at a height of the average
height of the picture plus 30m. The center coordinate was the bottom left of the four
center coordinates as both side length of each picture was an even number. A training
set was thus calculated with the cutout and the resulting LOS matrix available. In total
11,000 picture were made to ensure a sufficiently large training and test mass.
The implementation of the deep learning model was done in PyTorch[32]. With 64 by
64 cutouts as training data a network was trained in a supervised manner. The network
used the Adam optimizer [24]. The other hyper parameters of this model can be found
in Table 5.1.
The results yielded from this method was of little use. Most of the time the prediction
would just predict all locations to be LOS or all to be NLOS. As the MSE did not drop
below ≈0.08, seen in Figure 5.20, there was no gain compared to downscaling the LIDAR
map and evaluating those (Figure 5.18). A similar approach was tried on a soft state
LOS map in the same way to see if a softer slope and more possible outcomes, e.i. [0:1]

does as such separate itself from the function it is trying to predict. The deep learning
model showed that height profile analysis can predict RSRP to some extend, but more
work would be needed to get conclusive results on this. One would probably benefit by
adding the information about frequency as well as training on far more data.
5.6.4
Deep learning summary
Using deep learning as a tool is not straight forward in all cases. Predicting LOS directly
based on LIDAR proved difficult. As the algorithms working on the input relies on filters
of a small size, compared to the size of the input, determining LOS was difficult as LOS
can depend on obstacles far away. The same principle came into play when trying to
use super resolution to scale LOS maps rather than calculating them from LIDAR data.
Here the filters once again struggled to map the features extracted to LOS. Especially
the edges proved hard, but more clever use of filters and padding is expected to solve
this issue. Using an extracted height profile to determine LOS proved viable, though the
total amount of steps required to calculate a full path loss map was not reduced. Trying
to map the RSRP directly based on the obtained data from the drive test turned to an
over fitting as a rather small amount of data was available. Common for all the deep
learning done here, is that one of the most important factors, namely the frequency, are
not given as an input and thus none of the models are expected to generalize for different
frequencies, but only on the frequency they were trained. For future appliance this could
by solved by giving the frequency as in input. Especially for the RSRP prediction from
the drive test is this of particular use as the data then could be separated.

5.7
5G in relation to the obtained results
As earlier described, the most relevant part of 5G, for this project, is the new frequencies.
The work and results described earlier in this chapter all describe frequencies for 4G LTE,
though most of the frequencies are also used in 5G. The results obtained could in theory
be applied to mmW as the same physical laws apply.
For the LOS prediction, the method would be the same for mmW frequencies. The
direct LOS assessment is, of course, frequency free, but as 5G allows higher frequency, the
resulting Fresnel zones, become smaller. This means that for high enough frequencies
using direct LOS is very close to investigating the Fresnel geometry. The soft state
approach thus become obsolete for high enough frequencies, though further investigation
would have to be carried out to confirm this.

5.8
Summary
With the proposed model it was shown that mapping LOS from LIDAR data was possible.
Using Fresnel geometry a soft state LOS mapping was also carried out. From the path
loss investigation and comparison with the drive test results a connection between using
Fresnel geometry and path loss is found. This connection is hard to to quantify from
the data and models at hand, but is still useful as a guide in a deployment scenario.
It was also shown, that scaling can prove useful result as one can scale back given the
proper algorithms. Using deep learning powered super resolution scaling back from a
scaled-down picture was more effective than using a standard library function. However
was a deep learning algorithm struggling to predict LOS directly from a LIDAR map,
but when provided same information as the function it tried to replicate it performed
very well. As the deep learning algorithms were tested without frequency considerations
it is uncertain how they would respond to having that as an input, but it can be assumed
to be doable as their potential in general is huge.
Using deep learning, LIDAR and Fresnel geometry can help predict the radio quality
of an area, but more work would have to be carried out to ensure accurate predictions.

CHAPTER 7
Conclusion
In deployment of future mobile communications networks, dimensioning coverage and
capacity requires insight in terrain and radio propagation. In this report this issue is
investigated and it is shown that LIDAR data can be used as terrain data. LIDAR
data has the advantage of being easily accessible compared to similar data sets of equal
resolution. Based on LIDAR data it was shown that LOS could be approximated and
incorporated in a model predicting RSRP. LOS based on a straight line behaved differ-
ently than soft state LOS which takes the terrain into account using Fresnel geometry.
The soft state LOS was thus dependent on obstacles not in the direct LOS, emulating
actual radio propagation as diffraction changes the propagation path.
Compared to measured RSRP values at the same locations, it was evident that the
models developed were able to follow the measure RSRP in trend, but lacked accuracy
and stochastic terms to completely simulate the measured RSRP values. However, the
proposed models can replicate some characteristics of RSRP and is thus applicable in a
deployment scenario.
As the models can be evaluated for frequencies from 0.5GHz-100GHz the results are
applicable for both 4G and 5G as both operate in the frequency space. It would be
beneficial to investigate other path loss models as well to be more conclusive, since only
one was implemented.
By lowering resolutions of the input, much faster computation times were achieved
at the cost of accuracy, as the resolution dropped. This can, however, be omitted by
the proposed super resolution method using deep learning. The super resolution model
can scale a LOS map from which the path loss and thus RSRP is predicted. Using the
LIDAR data directly to map LOS with a deep learning method proved difficult, due to
the nature of the CNN used. If provided the same data as the analytic model, e.i. the
height profile of the terrain, a deep learning model was able to predict LOS. As the deep
learning models were only trained for one frequency, they need modifications to include
the frequency or a model for each frequency is needed. From this approach RSRP was
shown to be predicted directly from the height profile, but with the small volume of
data, conclusive results were not achieved though the method seemed promising.
The work carried out shows it is beneficial to use LIDAR data to predict LOS assisted
by deep learning for a better understanding of terrain effect on a mobile communication
network as 4G or 5G. Though not conclusive, the resulting modelling tells about the
expected signal strength at the receiver and thus planning deployment of mobile networks
can be done in an enlightened manner. At the end user promised services can thus be
enabled.
Whether it is delivering band width, low latency, enabling IoT or critical
infrastructure, the modelling ensures that deployed mobile communication networks are
