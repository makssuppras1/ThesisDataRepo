ABSTRACT
This report contains a detailed analysis of diﬀerent clustering techniques that have been
applied on equivalents representing a part of a large power system. The given power system
where the dynamic analysis are carried out contains the detailed model of the German
transmission system whereas, the external system representing the neighbouring countries
are replaced by a set of equivalents called dynamic power injectors.
One of the main aim of this thesis is to reduce the number of injectors surrounding the study
system by applying diﬀerent clustering techiques. Therefore algorithms were developed
in order to identify and cluster the data based on three diﬀerent N-1 contingencies. In
each of the contingencies a large generator within the study system is tripped and the
power ﬂow along the lines connecting the external system to the study system is noted.
Clustering was carried out either by using k means or hierarchical techniques.
Diﬀerent methods for determining the optimal number of clusters as well as the quality of
clusters were discussed.
It was concluded that for the given methods the k means algorithm produces a better set
of clusters in comparison to hierarchical clustering. The drawback of the used methods
were that it tends to produce losses in the system which slightly aﬀects the response of
the clustered system from the full system. Due to these drawbacks, the third method of
clustering which involves grouping the DPIs based on the country of origin. Despite this
method not using either k means or hierarchical clustering, it produces the best set of
results that matches the response of the unreduced system.

INTRODUCTION
In this chapter a brief introduction to the background and purpose for performing this
master thesis is described.
1.1
Background
An increase in globalisation has lead to varoius countries becoming dependent on one
another for diﬀerent commodities. In the case of the electricity market due to the ever
expanding nature of the power system supported by this force of globalzation, it has become
a norm nowadays to see large shares of power being traded between two or more countries.
This can be seen within Europe itself where diﬀerent countries within the European Union
transfer power between them so as to maintain a balanced system.
With the increasing size and complexity of the power system, ensuring the security and
stability of such a system becomes a signiﬁcant task for the transmission system operators
(TSO) of the countries involved. Since the power system consist of various components
such as generators, transformers, transmission lines among others, a fault/failure in any of
the components may lead to potential blackout. An example can be seen from the blackout
that occured in September 2003 where the western part of Denmark and south of Sweden
were aﬀected due to malfunction in a substation equipment. Due to this 4,700 MW and
1,850 MW of loads were disconnected in Sweden and Denmark respectively [1].
To avoid a potentail blackout, the TSO of each country carry out certain stability studies
to evaluate the impact of diﬀerent fault contingencies on its’ power system network. In
order to carry out these dynamic studies, detailed model of the generator need to be
known. Depending on the regulations within a country, the generator owner has obligations
to provide the data for the modelling. The output of such studies also depends on the
characteristics of the generators from a neighbouring country. But carrying out studies on
such a large system is impractical and time consuming.
In order to overcome this, a model reduction approach was performed in [2] where, the
power system of the neighbouring countries were represented by a set of equivalents called
dynamic power injectors. These dynamic power models emulate their inertial and primary
frequency response of the external system. Depending on the size of the country, the
number of injectors used for performing these studies would vary.
On reducing the number of injectors by grouping them together based on certain criteria,
the computational time needed for performing these studies could reduce considerably.
Along with the reduction in the number of equivalents, it would also help in optimal tuning
of the parameters.

1.2
Thesis Objective
The main objective of this thesis is to apply and compare diﬀerent clustering techinques
which can be used to group the dynamic power injectors. Despite representing this large
system using injectors, tuning them individually is a time consuming process. Hence by
reducing the number of such injectors ,it would be easier to tune them in order to perfectly
replicate the external system.
The system on which the dynamic studies are performed, henceforth refered to as the
study system consists of the detailed model of the generators as well as the transmission
grid. For this thesis the study system was taken to be Germany and the countries that
surround Gernmany represent the external system. Most of the code for segregating the
DPI data was written in Matlab. The output from Matlab was then used to run the
power ﬂow using Newton Raphson in the software called Artere. The data obtained from
the load ﬂow were then used in order to run the dynamic simulations on another software
called Ramses.
Dynamic studies were performed taking three contingencies into consideration where a
generator within the study system is tripped. The clustered data obtained by using these
contingencies as the basis is then tested for other faults in the study system to check
whether the clustered system provides the same inertial and frequency response as that of
the external system.
Some of the task involved in this thesis are:
• Obtaining a tuned model from the untuned reduced model that has been provided
• Development of algorithms for clustering the dynamic power equivalents
• Evaluation of the diﬀerent clustering approaches by comparing the dynamic system
response with that of the reduced and the unreduced model
• Investigate an approach for optimal tuning of the model parameters of the dynamic
equivalent

Learning Objectives

1. Apply a simple curve ﬁtting approach to determine parameters of a power system
equivalent
2. Discuss the impact of the power system equivalents on the frequency response
3. Describe diﬀerent clustering techniques
4. Explain the approach used for clustering the power system equivalents and list its
limitations
5. Evaluate the quality of simulation of the clustered power system model

2.1
Power system used
The power system model used for performing dynamic analysis was developed by the
European Network of Transmission System Operators for Electricity (ENTSO-E). The full
(unreduced) model developed in this case represents the large power system of Continental
Europe seen in ﬁgure 2.1. This unreduced model can be divided into three diﬀerent regions
namely the study system, the buﬀer zone or boundry system and the external system. As
mentioned earlier, the study system where analysis are carried in this project is Germany
which comprises of approximately 6500 buses, 800 generators and 9000 branches. The
voltage at the buses in this study system vary from 1 kV to 400 kV which is the standard
throughout Europe.
From ﬁgure 2.2 it is seen that the study system is enclosed by a buﬀer zone. The buﬀer
zone is created so as to ensure that during the reduction process, the components which
lie outside the study system retain their impact on the voltages within the study system[3].
The buﬀer zone is connected to the study system with the help of S-B tie lines whereas
the transfer of power between the buﬀer zone and the external system takes place through
the B-E tie lines.
Since it is impractical and time consuming to carry out dynamic studies on the entire
system, in [4] a model reduction approach is presented where the B-E tie lines are replaced
by a set of equivalents. The DPIs provide the same amount of active and reactive power
that would have been provided by the B-E tie lines.

2.2
DPI modelling/Equivalent representation
A Dynamic Power Injector is an equivalent which is used to emulate the inertial and
frequency response of the external system when the study system is subjected to
disturbances. The dynamic power injector injects or extracts certain amount of active
power ﬂow from the study system but the supply of reactive power remains constant. The
block diagram of the DPI can be seen in the ﬁgure below :
Figure 2.4. DPI block diagram[3]
The input for such a conﬁguration is the parameter ω which represents the weighted average
rotor speed of all the machines present in the reduced system. Due to the possibility of
inter-area mode of oscillations occuring in the reduced system, a low pass ﬁlter is used at
the input to cancel out these eﬀects. The constant ωref is equal to the COI speed in the
initial system state and in most cases equal to 1.0 pu[3].

3.1
k means Clustering
The k-means clustering also called as Lloyd’s algorithm is an unsupervised learning
algorithm that is used to obtain a speciﬁed number of clusters ’k’ from a given set of
observations. These observations can either be single dimensional or multi-dimensional
with each row representing an object to be clustered an each column representing an
attribute of the object. Clustering is usually performed around a set of data points called
as the centroids.
The the main goal of k means is to assign each object to a centroid such that the distance
between them is minimal. A well known criteria for evaluating the clustered stucture
formed is the sum of squared errors where the distance between each object to its’ centroid
is calculated by the squared Euclidean distance formula:
d(x, c) = (x −c)(x −c)′
where x is the location of the data point and c is the location of the centroid.
In Matlab this clustering algorithmn is performed by using the function kmeans. By
default, this function uses the k means ++ algorithm to ascertain the initial set of values
for the centroid.
For a given data set X the k means ++ algorithm operates as follows [6]:
1. An observation is selected arbitarily to be the ﬁrst centroid.
2. The remainder of the centroids are selected randomly from the data set with
probability :

P
xϵX
D(x)2
where the numerator denotes the distance between each point to a given centroid.
3. Step 3 is repeated until ’k’ number of centroids have been selected
After the initial set of centroids have been determined the k means algorithm proceeds as
follows:

1. The distance between the objects and each centroid is computed using the distance
measure mentioned above
2. Each point is assigned to a speciﬁc cluster based on this distance
3. A new centroid location is determined by taking the average of the objects belonging
to a particular cluster
4. Steps 1 through 3 are repeated until each object is retained in the same cluster or
maximum number of iterations have been reached
Therefore the output of k means clustering is that we obtain ’k’ number of clusters
containing data that are close to one another. But since a number of clusters can vary, an
optimal number of clusters need to be determined. The two methods which can be used
for determining the optimal number are :
• Elbow method
• Average Silhouette method

Hierarchical clustering
Hierarchical clustering can be of two types namely:
• Agglomerative hierarchical clustering
• Divisive hierarchical clustering
The diﬀerence between the two is that the agglomerative hierarchical clustering technique
follows a bottom-up structure where initally each object represents an individual cluster.
These objects are merged in a binary manner with one another until a structure is formed
where all of them ﬁt under a single cluster. On the other hand the divisive method separates
the already clustered data sets into individual clusters. In this thesis the agglomerative
hierarchical clustering was used.

individual data points. The objects can either be clustered by specifying the number of
centroids required or by cutting the dendrogram at a desired similarity level.

CONCLUSION
5.1
Results
The objective of this thesis was to test diﬀerent clustering techniques which can be applied
to equivalents representing a part of the power system. In order to ﬁnd which would be
the most suitable, clustering was carried out on a reduced dynamic study model where
Germany was taken as the study system.
The kmeans and hierarchical techniques were applied on the data obtained from two
methods : 1.) change in power along the B-E tie lines and 2.) overall change in power
at the DPI buses. Another method that was used, is to cluster the DPIs based on the
country of origin. The performance of these three methods have been investigated for the
N-1 contingency scenario where only a generator within the study system is tripped . By
making this comparison it helps to determine which would be the most suitable method
for clustering the data when other study models are taken into consideration.
For the ﬁrst method, what can be deducted from the results is that applying either kmeans
or hierarchical clustering on individual contingencies provides a better set of clusters in
comparison to when they are applied on the data for multiple contingencies. This can be
seen from the the fact that the diﬀerence in power ﬂow along the S-B lines for both the
scenarios applied is smaller. But the drawback of kmeans is that, the formation of quality
clusters is always dependent on the intial set of centroids determined by the kmeans ++
algorithm. On the other hand the clusters formed by hierarchical clustering is always
constant.
In the DPI based clustering kmeans was applied to a data set consisting of 369 data points
for single contingency scenario. Hence it was observed that the number of unclustered data
is fewer is comparison to the previous method. But it has to be noted that the number of
clusters does not determine the quality of clustering as the diﬀerence in steady state power
along the S-B tie lines is higher when compared to the single contingency scenario of the
ﬁrst method.
On analyzing the country based clustering it was found that it gives the least amount of
power diﬀerence along the S-B tie lines. A drawback of using this type of clustering is that
during the transient period, the power production of the generators in the study system
do not reﬂect the production in the unreduced system.
Thus an overall conclusion that can be made is the country based clustering provides the
set of clusters which represents the external system most accurately. The reason why
kmeans or hierarchical clustering does not provide a good representation of the external
system in this case is because the external system is taken to be made out of individual
elements. Therefore clustering provides a mixture of diﬀerent power systems which leads
44
==PAGE:53==
5. Conclusion
to increased losses. It was also observed that there is not signiﬁcant reduction in the
simulation time as the total number of buses are decreased by a small margin.
5.2
Future work
1. Optimal tuning of DPI parameters :
In this thesis the DPI paramters were tuned manually in order to ﬁt the COI response
of the unreduced system. Therefore diﬀerent approaches can be investigated where
the response of the clustered system ﬁts perfectly with that of the unreduced system.
2. Clustering of B-E tie lines based on country : Since it was observed that
clustering tie lines taking the full system into account produces losses. Clustering
can be performed seperately for the tie lines belonging to diﬀerent countries. This
would ensure that the losses in the system are reduced as in the case of country based
clustering.
3. Performing clustering based on diﬀerent distance measure :
Since k means and hierarchical clustering were performed based on a default distance
measures in Matlab. Diﬀerent distance measures can be used to check if better
quality of clusters can be obtained.
4. Identifying good set of cetroids for kmeans :
Since the ﬁrst set of centroids for kmeans are determined on the basis of probability
by using the kmeans ++ algorithm. investigations can be carried out in order to ﬁnd
the global minimum for the objective function rather the current local minimum.
