Abstract
Three dimensional phononic crystals with low frequency phononic band gaps
can be used to isolate test objects from vibrations within certain frequency
ranges. This can be of interest in e.g. metrology labs that wants to ﬁlter
away noise caused by low-frequency vibrations.
The crystals analyzed in
this thesis are based on strut-based lattices modelled by frame structures
as additive manufacturing now oﬀer opportunities to manufacture complex
strut-based lattices.
The phononic crystals will be designed by applying
topology optimization on frame structures to maximize phononic band gaps.
The aim of this thesis is to develop and implement a topology optimization
code to maximize phononic band gaps using frame structures.
The topology optimization algorithm described in this thesis is based on a
ﬁnite element program using beam elements. The underlying theory of the
ﬁnite element analysis is presented including beam elements and transforma-
tion of beam element of arbitrary direction. To model phononic band gaps
the Bloch-wave representation is implemented by applying periodic Floquet-
Bloch boundary conditions to the design problem. The result is a diﬀeren-
tiable eigenvalue problem that has to be solved. This requires the calculation
of the sensitivity for an eigenvalue. Moreover, multiple eigenvalues must be
dealt with as these often arise in symmetric structures as treated in this
thesis.
The topology optimization algorithm is implemented in MATLAB 2017a.
The optimization formulation is formulated using the double bound formu-
lation to ensure smooth, continuous and diﬀerentiable functions that can
be solved by the gradient based optimization algorithm Method of Moving
Asymptotes. An active set strategy is implemented to reduce the number of
constraints and for problems with no initial band gap the zipper-method is
used to ease the optimization.
It is demonstrated how the topology optimization program can be used to
design phononic crystals using simple cubic frame structures. Systematic
studies are conducted to test the possibilities and limitations of the program.
Phononic crystals with low-frequency phononic band gaps below 1304 Hz,
which is 20% lower compared to similar results reported in the literature.

Resumé
Tredimensionelle phoniniske krystaller med lavfrekvente phononiske båndgap
kan bruges til at isolere arbejdsemner fra vibrationer i bestemte frekvensom-
råder. Det kan udnyttes i f.eks. målelaboratorier, hvor man ønsker at ﬁltrere
støj væk fra sine resultater, som opstår på grund af vibrationer af udstyret.
De phononiske krystaller, som bliver analyseret i dette speciale, er baseret
på rammestrukturer, fordi industrielle 3D-fremstillingsmetoder nu tilbyder at
kunne fremstille komplekse rammekonstruktioner. De phononiske krystaller
bliver designet ved at anvende topologioptimering af rammestrukturer til at
maksimere phononiske båndgap. Formålet med dette speciale er at udvikle
og implementere en kode, som netop kan maksimere phononiske båndgap
ved at optimere på rammekonstruktioner.
Topologioptimeringsalgoritmen, som er beskrevet i dette speciale, er baseret
på et ﬁnite element program, som bruger bjælke elementer. Den nødvendige
ﬁnite element teori, som skal bruges til at forstå specialet er præsenteret,
herunder bjælkeelementer og transformationer af bjælkeelementer med arbi-
trær orientering. For at modellere phononiske båndgap er Blochs bølgefunk-
tion implementeret ved at påføre periodiske Floquet-Bloch randbetingelser til
problemet. Dette resulterer i et diﬀerentiabelt egenværdiproblem, som skal
løses. Dette kræver, at man udregner sensitiviteten for en egenværdi. Yder-
mere, skal problemet som opstår ved sensitiviteter af multiple egenværdier
løses, da de tit opstår som følge af symmetriske strukturer, som strukturerne
analyseret i dette speciale.
Topologioptimeringsalgoritmen er implementeret i MATLAB 2017a. Formu-
lering af optimeringsproblemet er baseret på to-grænse metoden for at opnå
glatte, kontinuere og diﬀerentiable funktioner, som kan løses med den gra-
dientbaserede optimeringsalgoritme Method of Moving Asymptotes. En ak-
tiv mængde strategi er formuleret for at reducere antallet af begrænsninger,
som optimeringsalgoritmen modtager og for optimeringer, som starter uden
et båndgap er lynlås-metoden implementeret for at lette optimeringen.
Det er bevist, hvordan topologioptimeringsprogrammet kan blive brugt til
at designe phononiske krystaller i simple kubiske strukturer. Systematiske
studier er udført for at teste programmets muligheder og begrænsninger.
Phononiske krystaller med lavfrekvente båndgap er præsenteret med båndgap
startende ved 1304 Hz, hvilket er 20% bedre end lignende resultater hidtil
præsenteret i litteraturen.

Chapter 1
Introduction
The work conducted in this thesis is motivated by research made by scientists
in a metrology lab at the University of Nottingham. They are searching for a
structure that can isolate a work piece from low frequency vibrations, when
it is tested [Syam et al., 2017a]. One way to achieve this is by tailoring the
geometry of a lattice structure, so the dynamic properties are changed to
hinder vibrations in certain frequency ranges.
As research in additive manufacturing (AM) has increased over the last
decades, the possibilities for producing lattice cells have increased. This al-
lows for more complex structures to be manufactured, which might not have
been possible previously. Therefore, there is an interest in exploring, whether
strut-based lattice structures are suitable for band gap maximization. The
idea of this thesis is to use topology optimization to design lattice struc-
tures with a geometry that tailor the mechanical properties of the structure
as desired. The objective is to investigate whether it is possible to achieve
such a design using frame structures. If so, the structure could be used as
a passive vibration ﬁlter in e.g. metrology labs. No one have to the authors
knowledge yet used the same techniques in attempt to achieve this goal. In
the following sections selected literature on the subjects covered in the thesis
will be presented.
1.1
Lattice structures and frame designs
The structures analyzed in this thesis will be strut-based lattice structures.
In Figure 1.1 an example of a strut-based lattice structure is shown. Small
lattice structures for structural purposes have been under heavy research for
the last three decades. The research has mostly been related to metal foams
in search of lightweight materials, but also for band gap purposes. Band gaps
or phononic band gaps, are frequency ranges, where acoustic or elastic waves
cannot propagate through the given structure. The theory of band gaps will
be described in detail later. The concept of Bravais lattices was introduced in
the ﬁrst edition of [Kittel, 2005] published in 1953. This is the same year as
Leon Brillouin published [Brillouin, 1953], where he introduced the concept

1.2
Photonic and phononic band gaps
A way to achieve the desired geometry of the lattice is to design it, so it
has a band gap. The research on photonic and phononic band gaps started
already before the beginning of the 20th century. The basics of electromag-
netic theory of light was documented already in [Rayleigh, 1881] 1881 by

Lord Rayleigh. In [Bragg and Bragg, 1913] the reﬂection of X-rays was de-
scribed, creating the foundation for bragg-scattering, which is the one of the
most common reasons for the existence of band gaps. The early research
has mainly been concerning Photonic band gaps, which is band gaps related
to electromagnetic waves or light waves.
One of the pioneering publica-
tions was [Yablonovitch, 1987] suggesting the ﬁrst photonic crystals. Based
on theory described in [Joannopoulos et al., 2008], ﬁrst published in 1995,
and in [Yablonovitch, 2001] beam mirrors, wave guides, optical ﬁbers, and
several others industrial applications have been developed. The research in
phononic band gaps, which is band gaps related to acoustic waves or elas-
tic waves, has not been as extensive as in photonic band gaps. Although,
it has increased during the past few decades. Already in 1992 [Sigalas and
Economou, 1992] modelled phononic band diagrams and showed the exis-
tence of band gaps for elastic and acoustic waves.
Later on, [Kushwaha,
1996] investigated phononic band structures extensively, showing band gaps
for a lot of diﬀerent composite combinations. [Vasseur et al., 1998] experi-
mentally veriﬁed the existence of phononic band gaps in 2D for composite
materials and [Ping et al., 2003] experimentally veriﬁed the existence of 3D
band gaps, also in composite materials. [Martinsson and Movchan, 2002]
showed that phononic band gaps existed for several diﬀerent unit cells based
on lattice structures. [Phani et al., 2006] made a deeper study of phononic
band gaps in periodic 2D lattice structures, showing relations between the
diﬀerent modes and the emerging band gaps.
[Warmuth and Körner, 2015] modelled phononic band gaps in 2D quadratic
and 3D cubic strut-based structures. They showed how hindering of speciﬁc
eigenmodes could form band gaps in simple cubic lattices. This was done by
bending the struts in shapes resembling speciﬁc mode shapes. These results
also led to the belief among the author of this thesis that shape and topology
optimization can yield signiﬁcant results regarding band gap optimization.
The results form [Warmuth and Körner, 2015] were experimentally veriﬁed
in [Warmuth et al., 2017].
Within the ﬁeld of metrology, vibration suppression is also of key interest.
In [Syam et al., 2017a] they proposed diﬀerent strut-based lattice designs
for vibration isolation. In [Syam et al., 2017b] they showed how the natural
frequencies depends on cell size, number of tessellations, and volume fraction,
concluding that a larger unit cell or reduced volume fraction reduces the
natural frequency, due to the decrease in stiﬀness. In [Maskery et al., 2019]
triply periodic minimal surface (TPMS) lattices were studied for the purpose
of producing phononic band gaps. Here a parameter study was also made
in order to tailor the properties correctly to tune the band gaps as desired.

1.3
Optimization in general
In this thesis topology optimization will be used to design the lattice struc-
ture. The optimization methods used in this thesis have been used for various
design applications since the 1980’s. Topology and shape optimization have
been used to design a variety of products within many diﬀerent ﬁelds of
physics through time. In mechanics they have mostly been applied to struc-
tural optimization, where most often the stiﬀness to weight ratio has been of
concern. The topology optimization method was ﬁrst reported by [Bendsøe
and Kikuchi, 1988]. Since then, these methods have been used extensively
in many optimization problems. In 1995 [Bendsøe, 1995] published a book
describing the diﬀerent methods behind structural topology and material
optimization. This book was later revised in [Bendsøe and Sigmund, 2004].
Although, this only documents topology optimization.
The topic of shape optimization is also well documented. However, the term
has been attributed diﬀerent meaning in the literature. In this thesis, shape
optimization is understood as optimization, where the designs variables are
the coordinates of the nodes, as described in [Christensen and Klarbring,
2009].
The methods can be used in various applications. This spans from meta-
materials with negative poisons ratio [Larsen et al., 1997] and highly negative
thermal expansion coeﬃcient [Sigmund and Torquato, 1997] to the Architec-
tural designs of Mutsuro Sasaki. As computational power has increased over
the years the problems that can be solved have also grown larger. The largest
topology optimization yet known was performed by [Aage et al., 2017]. In
this publication a whole aircraft wing was optimized without having to split
it in sections revealing many intriguing design features. Topology and shape
optimization of lattice structures are also well known and broadly used for
stiﬀness optimization. Especially, in the ﬁeld of construction engineering op-
timization of truss and frame structures have been investigated. The theory

1.4
Optimization of phononic band gaps
Previously, it was stated that no one has yet conducted analysis similar
to the one shown in this thesis. However, optimization of band gaps has
been reported many times. In 2003 [Sigmund and Jensen, 2003] conducted
a topology optimization of phononic band gaps for a 2D phononic structure
with two material phases. They showed a procedure for such an optimization
algorithm revealing that topology optimization could be used as an eﬀective
tool to increase the size of band gaps. The problem of the sensitivity analysis
of multiple eigenvalues was also addressed in this article, which is a common
issue when doing optimization of problems involving eigenvalues. [Seyranian
et al., 1994] proposed a solution to this problem, where a new eigenvalue
problem is formulated to separate the sensitivities of the multiple eigenvalues.
[Olhoﬀet al., 2012] used topology optimization to maximize band gaps for
a cantilever beam studying various beam designs and the resulting eﬀects
on the dispersion curves. In [Pedersen, 2006] several band gap optimizations
are presented both utilizing topology optimization and shape optimization
of plates. [Vondrejc et al., 2016] performed a shape optimization of phononic
band gap structures using the homogenization approach. However, the study
was only performed 2D with a triangular mesh. The same team of researchers
that produced the results used for veriﬁcation have made both topology and
shape optimization for phononic band gaps shown in [Wormser et al., 2017].
The optimizations are made for lattice structures similar to the ones pre-
sented in this thesis. However, the optimization is only performed in 2D.
The results are then extended to 3D, but a full 3D optimization is not con-
ducted. A 3D optimization topology optimization for phononic band gaps
is conducted in [Lu et al., 2017]. Although, the optimizations performed in
this paper is for a 2-phase composite material using a solid mesh and not
on frame structures using beam elements as in this thesis. To the authors
knowledge, no one has yet tried to use topology and shape optimization on
frame structures using beam elements to maximize phononic band gaps in
3D.

Thesis outline
The following thesis studies band gap maximization of frame structures using

Chapter 2
Theory of band gap modelling in
a FEM perspective
The theory of band gap modelling in a ﬁnite element perspective include sev-
eral topics. It includes the general ﬁnite element theory, including the topics
such as dynamics and Floquet-Bloch theory. The geometry also plays an im-
portant role. The geometries studied and optimized in this thesis are based
on Bravais Lattices, which is symmetric lattice structures, where the most
common are the cubic structures well known from atomic crystal theory. Fur-
thermore, there is the physical theory of the band gaps itself, which is well
known from the ﬁeld of photonics. All these topics will be further introduced
in the following chapter.

Finite element formulation and transfor-

2.1.1
Beam element
The element used for the optimizations presented in this thesis is a 3D beam
element. This element is chosen, because the strut-based lattices, which this
thesis concerns, can be well represented by a 3D beam element. For the
optimizations a circular cross section is used. Although, for the veriﬁcation
in Chapter A a square cross section of the elements are used. The issues
involved, when dealing with the diﬀerent cross sections will be discussed later
in this chapter. The 3D beam element has six degrees of freedom (DOF) in
each node, three translational and three rotational, respectively. The beam
element is deﬁned in a local coordinate system with its primary axis on the
x-axis, as shown in Figure 2.1 along with the 12 DOF.
There exist several types of beam elements reﬂecting the diﬀerent beam the-
ories.
The two most common are the Bernoulli-Euler beam element and
Timoschenko beam element. The elements reﬂect the two theories they were
named after, which means that the Timoschenko elements consider shear

Brilluoin zone
Brillouin zones are symmetric zones with special properties. In [Kittel, 2005]
the ﬁrst Brillouin zone is described as follows:
The ﬁrst Brillouin zone is the smallest volume entirely en-
closed by planes that are the perpendicular bisectors of the recip-
rocal lattice vectors drawn from the origin.
The volume enclosed is also called Irreducible Brillouin Zone and is deﬁned
in the reciprocal space. This is a Wigner-Seitz cell of the reciprocal lattice,
which means it is the volume enclosed by the lines of the perpendicular bi-
sector. Hence, the ﬁrst Brillouin zone is the smallest volume, where no other
lines are crossed, which can be observed as the red area in the middle in
Figure 2.5 for a simple square(SS) lattice. The SS lattice is the 2D anal-
ogous to the SC in 3D, which is used due to the simplicity of the ﬁgures.
The ﬁrst Brillouin zone is very important in modelling band gaps, because
the symmetric properties can be used to reduce the problem, which saves a
lot of computational time. For undamped problems the band gap problem
can be even further reduced by only analyzing the boundaries of the ﬁrst
Brillouin zone. Thus, by only analyzing the boundaries of the ﬁrst Brillouin
zone all eigenmodes are exited and a full band diagram becomes visible. Al-
though, this has not to the authors knowledge been scientiﬁcally proven, it
is considered good practice, when dealing with band gaps modelling.

2.2.3
Band gaps
The foundation for understanding Floquet-Bloch wave theory in beams have
now been established through the theory of Floquet-Bloch boundary condi-
tions, Bravais lattices and Brillouin Zones. This leads to the introduction
of the theory for the band gaps itself. Band gaps have many applications
as discussed previously. Especially, within the ﬁeld of photonics they are
widely used for wave guides, light mirrors etc. Phononic band gaps are band
gaps in elastic or acoustic waves. These can serve the same applications for
elastic and acoustic waves as photonic band gaps can for light waves and
electromagnetic waves. In this thesis the purpose is to design a unit cell that
can serve as a vibration isolator by designing it, so it has a band gap.
Band gaps arise as energy gaps in the electron band in periodic structures.
They are found by solving the eigenvalue problem stated in Eq. (2.1.9) for
diﬀerent wave vectors k. Typically, the wave vectors are deﬁned as points

3.1.5
Mapping
Previously, it was stated that only points located on the border of the ﬁrst
Brillouin zone are included in k and hence only these points are analyzed.
Although, to be able to only analyze the borders of the ﬁrst Brillouin zone,
symmetry of the structure has to be maintained for every iteration in the
optimization loop.
This means that the optimization can only alter the
structure symmetrically. If the symmetry of the ﬁrst Brillouin zone is not
maintained, then the assumption of symmetry that the theory of Bravais
lattices is based on, no longer holds. This means that instead of only looking
at the ﬁrst Brillouin zone, the whole structure would have to be analyzed,
which is very expensive computational wise. To ensure that symmetry of the
structure is preserved only the nodes and elements in the ﬁrst Brillouin zone
are used as design variables. These nodes and elements corresponding to
1
32
of the 3D unit cell are then mapped to the rest of the structure as observed in
Figure 3.3. The red structure that can be observed is the ﬁrst Brillouin zone.
The large structure in blue is the full structure after the mapping process
with the ﬁrst Brillouin zone marked in red. The mapping is done before the
FEA, so the FEA is calculated for the whole structure, but the MMA only
receives and returns the design variables related to the ﬁrst Brillouin zone.
This of course impose some restrictions regarding the design of the unit cell,
as it must be symmetric.
The bookkeeping of this mapping process it not trivial. First of all, it is
important in the individual steps of the mapping that the nodes and elements
placed on the respective center line of rotation or mirroring, are not copied.
This would create double nodes and elements, which would cause fatal code
errors to occur. Furthermore, one has to keep track of the nodes and elements
in order to sum the sensitivities in the right manner.
Hence, it must be
controlled, which node/element in the ﬁrst Brillouin zone all nodes/elements
originate from in the full design. This information is obtained once and stored
in a matrix, which is then used for summation. The summation is forward
for the design variables, meaning that the mapped design variable will have
the same value as the original. For the sensitivities backward summation
is used, meaning that the sensitivities for the mapped design variables are
summed, so the MMA only receives the original amount of sensitivities equal
to the number of original design variables.

3.2
Topology optimization
The optimization method used in this thesis is called topology optimization.
As the name imply the method optimizes the topology of the structure.
Typically, design variables in solid mechanics topology optimization problems
are often a relative density for continuum mechanics or the cross-sectional
area. Such approaches are commonly called material distribution methods
[Bendsøe and Sigmund, 2004]. For the topology optimizations conducted in
this thesis the design variables are the radius of the beam elements, hence the
cross-sectional area. The minimization problem for the topology optimization
is stated as in Eq. (3.1.23). For the topology optimization problem N is equal
to the number of elements. For the topology optimization conducted in this
thesis only circular beam elements are considered.
The equation for the radius of the cross section as a function of the design
variable xi is:

The formulation for an optimization using a standard topology optimiza-

to a design problem with the objective of maximizing phononic band gaps.
The objective function used an alternative formulation called the bound-
formulation to smoothen the optimization problem, as the original formu-
lation were not smooth and hence not solvable by the gradient based opti-
mization algorithm used called MMA. The optimization problem furthermore
utilized an active set strategy to reduce the number of constraints that the

4.1.3
Band sweep study
A second study test the robustness and performance of the program by mak-
ing a band sweep. The frequency of the band gap is desired as low as pos-
sible, as the objective is to create low frequency band gaps. Furthermore,
the higher the modes chosen the more frequencies must be calculated. A
rule of thumb states that twice the amount of needed frequencies should be
included, which cost computationally. Thus, the lower the modes of the band
gap the better. Considering these two reasons leads to testing the programs
ability to produce band gaps at lower modes.
A parameter sweep of optimizations starting from the third band and then
increasing the lower band is performed to test if the program is able to open a
band gap. The reason that the parameter sweep is started at the third band
and not the ﬁrst band, is that a band gap cannot occur below the acous-
tic cut-oﬀ. The acoustic cut-oﬀis the maximum frequency of the highest
acoustic mode. A band gap cannot occur below any acoustic mode, because
acoustic modes have a zero frequency at the Brillouin zone center at the high
symmetry point Γ. The unit cells that have been tested in this thesis all
have three acoustic modes, thus band three is chosen as the lowest staring
band. Furthermore, many of the low natural frequencies are multiple of three
due to the three directions of geometry symmetry. Thus, the translational
and rotational modes will often be multiple in pairs of three for the unit
cells analysed in this thesis. Therefore, the band chosen for the test it not

4.4

In Section 2.1.1, it was mentioned that diﬀerent types of beam elements ex-

Euler beam elements. These elements are made with an assumption of long
and slender beams meaning the slenderness ratio given in Eq. 2.1.1 is above
ten. However, as the hierarchy of the basis mesh is increased this assumption
is more and more violated. The elements have been deliberately allowed to
violate this assumption, to provide more freedom to the optimization algo-
rithm to make it easier to produce band gaps. This is not a problem for the
optimization, however the accuracy of the results can be compromised. To
test the size of the error, the ﬁnal result of the optimization in Section 4.2.3
are reproduced using Timoschenko beam elements. The third hierarchy is
chosen as this is the analysis that violates the assumption of slender beams
the most and hence is the worst case. The analysis is conducted using the
same parameters as the optimization shown in Section 4.2.3.
The band diagram calculated using Timoschenko beam elements is shown in
Figure 4.15. The band gaps calculated using Timoschenko beam elements
have a relative size of 0.873 between the ninth and the tenth band and a
relative size of 0.250 between the sixth and the seventh band. For the large
band gap, this is a deviation of -9.6% compared to the results obtained using
Bernoulli-Euler beam elements shown in Figure 4.11a. For the smaller band
gap, the deviation is 1.2% compared to the original. The center frequency
for the larger band gap using Timoschenko beam elements corresponds to
7030Hz, which is a deviation of -11.3% compared to the band gap calculated
using Bernoulli-Euler beam elements. The lower band gap has a center fre-
quency, which corresponds to 3431Hz, which is a deviation of -2.3% compared
to the band gap presented in Figure 4.11a.
It can be concluded that using Timoschenko beam elements yield a signiﬁ-
cant change in the magnitude of the results. However, the band structure in
Figure 4.15 looks similar to the band diagram obtained from the optimiza-
tion in Figure 4.11a. Thus, it is concluded that the Bernoulli-Euler beam
elements works for the design purposes and proof of concept of the program
implemented in this report. Although, to obtain more accurate results Tim-
oschenko elements should be used.

Chapter 5
Conclusion and outlook
This thesis has investigated the use of topology optimization of frame struc-
tures for the purpose of designing phononic crystals with low-frequency phononic
band gaps. Systematic studies have been conducted to show the possibilities
and limitations of the implemented program.
5.1
Thesis summary
The theory needed to model 3D phononic band gaps in a ﬁnite element per-
spective was presented. This includes beam elements to model frame struc-
tures. Moreover, it was described how the Bloch wave representation was
applied through Floquet-Bloch boundary conditions. The periodic bound-
ary conditions led to the concept of phononic crystals simulated by frame
structures. The concept of band diagrams was introduced with the more
important topic of phononic band gaps.
It was presented how a topology optimization framework was applied to a de-
sign problem maximizing phononic band gaps in frame structures. An active
set strategy was implemented to reduce the number of constraints. Moreover,
for problems without a band gap and the initial design the zipper-method
was introduced to ease the optimization. The optimization formulation was
based on the bound-formulation to make the functions smooth, continuous
and diﬀerentiable, which is essential in gradient based optimization. To solve
the optimization problem the gradient based non-linear optimization algo-
rithm MMA was used.
The ﬁrst optimizations were performed on a SC scaﬀold structure. The op-
timization maximized an existing band gap resulting in a band gap with a
relative size of 0.860, which was a tenfold increase. Furthermore, a consis-
tency study and a band sweep study were made to test the code’s limita-
tions. It was shown that the simple mesh did not oﬀer enough freedom to
the optimization algorithm, which led to that none the optimizations proved
successful in regard to producing a band gap.
To increase the ability for the optimization algorithm a hierarchical tetra-
hedral meshing tool was implemented to create more advanced basis mesh.
67
==PAGE:81==
CHAPTER 5. CONCLUSION AND OUTLOOK
For the second and third hierarchy the optimizations proved successful and
produced band gaps with a relative size of 0.078 and 0.966, respectively.
Several studies based on the hierarchical meshes were conducted to test the
performance of the code. A mesh reﬁnement study was performed on the
ﬁrst hierarchy mesh showing that increasing the number of elements in the
optimization could lead to producing band gaps that were not found using
the same, but coarser basis mesh. The eﬀect of using Timoschenko beam
elements instead of Bernoulli-Euler beam elements was tested, which showed
a signiﬁcant deviation in terms of the magnitude of the relative band gap
size. The conclusion was that to obtain more accurate results Timoschenko
beam elements should be used for future optimizations. A band sweep study
for the third hierarchy mesh showed that it was easier for the optimization
program to separate two bands, where the modes were not multiple. Finally,
a study was made to test if it would make a diﬀerence if the elements with
minimal radius were removed from the mesh and the optimization continued.
It was shown that removing the minimum radius elements increased the per-
formance of the optimization. The optimization program was able to increase
the band gap size with 31.5%, when the elements were removed and the op-
timization continued. This led to the conclusion that removing the elements
had a signiﬁcant inﬂuence. However, for some optimizations the structural
integrity was lost, in which case the unit cell designs are impractical for real
applications.
5.2
Outlook
During the work period ideas for further investigation have come to mind.
The following list is some of the topics that were not included in this thesis
due to time limitations.
• Implementation of shape optimization.
The implementation of this
optimization algorithm was originally a part of the scope for this thesis.
However, it was not possible to implement the algorithm within the
work period. It is believed that shape optimization will provide good
results too, as this optimization will have great power to inﬂuence the
modes.
• Timoschenko beam elements to be included in optimization. It was
shown that the results calculated using Bernoulli-Euler beam elements
have a signiﬁcant deviation from results calculated using Timoschenko

beam elements. Thus, to get more accurate results Timochenko beam
elements should be included.
• Further studies of the consequence of reﬁning the basis mesh. This
study was only conducted for the ﬁrst hierarchy as time only allowed
for this analysis to be performed. However, it would be interesting to
see if the same improvement could be found for reﬁning the basis mesh
of the higher hierarchies.
• To get a better understanding of the dynamics and how the modes limit
the optimization algorithm it is recommended to further analyze the

5.3
Conclusion
The main objective of the thesis was to investigate if frame structures are
suitable for designing structures with low-frequency phononic band gaps.
The goals were reached and it was demonstrated that by using a topology
optimization framework, designs can be created with low-frequency phononic
band gaps starting as low as 1304 Hz. Furthermore, it was found that it was
easier for the optimization program to separate non-multiple modes and that
the optimization often utilizes the concept of concentrating the mass to use
it as a resonator to produce band gaps. However, more studies are needed for
full understanding of the mechanisms in designing low-frequency phononic
band gaps in frame structures.

Appendix B
Shape optimization
B.1
Shape optimization formulation
The second optimization method is called shape optimization. The overall
framework for shape optimization is the same as for topology optimization.
The diﬀerence is the choice of design variables. In topology optimization,
the design variables are chosen as a geometry or material parameter. This
makes no changes to ﬁnite element mesh. However, for shape optimization
the design variables are chosen as the coordinates of the nodes in the mesh.
This means, that while shape optimization cannot change the geometrical
parameters of the cross section or the material, it can alter the mesh, to
search for the most optimal structure for the design problem.
The opti-
mization formulation for the shape optimization problem looks similar to the
optimization formulation for topology optimization. The formulation can be
observed in Eq. (3.1.23). However, the number of design variables N is no
longer equal to the number of elements as for topology optimization. Now
the design variables are equal to the number of coordinates, which is three
coordinates for each node using a Cartesian coordinate system. The design
variables, which for clarity is now called αij instead of xi, are still subject to
the box constraint between 0 and 1.
