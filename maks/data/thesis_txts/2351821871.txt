Summary (English)
In this thesis I will empirically examine how robust recurrent neural networks are
against randomized added weight perturbations. More speciﬁc, I will conduct
experiments on a variant known as the long short-term memory (LSTM) block
to examine the eﬀect of applying layer exclusive weight perturbations and as a
whole. I contribute to the existing understanding of recurrent neural networks
with a method for estimating the expected change in loss as a function of the
increased imposed noise.
I conclude that it is possible to estimate the growth in error to a surprising
large magnitude of imposed noise and that there is a layer speciﬁc diﬀerence in
sensitivity to the added noise.

Chapter 1
Introduction
A recurrent neural network (RNN) is a special type of artiﬁcial neural network
(ANN) (or just neural network) that in contrast to the latter allows cyclic con-
nections to exist. This means that information can persist in the network and
form more complex mappings between the input and output signals. As such,
RNNs have successfully been used in modeling a variety of machine learning
problems such as natural language processing and video frame labeling.
Unfortunately, RNNs are known to fail learning long term dependencies when
the gap between the input signal and previous processed signals becomes too
large. This issue is known as the vanishing gradient problem which motivates
the much improved long short-term memory (LSTM) blocks. As originally in-
troduced by [HS97] the LSTM block addresses the vanishing gradient issue by
introducing gates and an internal cell state. As such, the time related input
signal is stored as a complex inner representation in one ore more of the cell
states which can be used for later reference. Since the input signal has no clear
indication for when to reset the cell states, [GSC00] has later introduced the
forget gate that completes the modern structure of the LSTM block.
ANNs are currently subject to a lot of ongoing research to fully discover why
they have proven to be so successful in a variety of applications. In particular
research regarding stability and robustness of ANNs are related to the topic of
this thesis. Early studies by [SR87] have shown that ANNs by construction are
==PAGE:14==
2
Introduction
surprisingly robust against weight perturbation. Studies in ConvNets (CNNs)
have also shown robustness against weight perturbation [MAA+16] [CBD15]
which also has proven to be depth speciﬁc sensitive [AK16].
This thesis continues this topic with focus on recurrent neural networks and in
particular LSTM blocks. I will investigate the gate speciﬁc responses to weight
perturbations and try to predict the change in loss that is induced.
Here follows a brief overview of the content of this thesis. In Chapter 2 the
necessary background material for neural networks is covered with a particular
emphasis on LSTM networks as covered in Section 2.3.2. To complement the
theory a simple single cell LSTM network is described as an example in Section
2.3.3.
In Chapter 3 the methods used in the experiments are described which includes
the method for estimating the change in loss as well as an description of how
to train neural networks in general. In Chapter 5 the two major experiments
of this thesis are shown including model training, damage experiments and loss
estimations. As a leading experiment, the simple example from Section 2.3.3
is used to illustrate the eﬀect of applying damage to the network weights. In
Chapter 6 the results are elaborated on and in Chapter 7 a selected set of
conclusions are drawn.

2.1
Supervised learning
Let us consider the supervised learning approach. Let (x, t) be an input-target
pair where x is an element of the input space X and t an element in the target
space T. A set of input-target pairs used in training is referred to as training
samples in S and a disjoint set S′ is referred to as the testing samples (hold-
out set) and used to evaluate the obtained performance. Elements from both
S and S′ are assumed to be drawn independently from the same input-target
==PAGE:16==
4
Theory
distribution DX×T . In some cases the training set S is split into a training
set and a validation set. The purpose of the validation set is to monitor the
performance of the model during the training process and interrupt the learning
when an overﬁtting behavior is detected.
The goal in supervised learning is to minimize an error measure E that captures
the deviation between the target values and the output values produced from the
input values. A traditional supervised learning task is for example a regression
problem where both input and target values are real-value numbers and the goal
is to predict the following number in a sequence. For this the error measure
known as the sum-of-squares (or Euclidean distance) is used to measure the
distance (or error) between the predicted and target values.

2.1.2
Multiclass logistic regression
Continuing from the linear regression modeling to the multiclass logistic regres-
sion problem, I will now show how to formulate error functions for binary and
multiclass classiﬁcation problems.
For the binary classiﬁcation problem we start by formulating the target t as a
vector of two elements. Given an observation x that belongs to class C0 the
corresponding index (i.e index 0) of t is then equal to one and with index 1
equal to zero. Likewise for an observation belonging to class C1 index 1 of t is
equal to one with index 0 equal to zero.
We can interpret t as a vector of probabilities but is only allowed values at
the extremum ie. either 0 or 1. For K > 2 classes we can now extend this
formulation to a 1-of-K coding scheme where the k’th index has the value 1
and the rest 0. That is, all tk values are zero except tj that has the value 1 for
class Cj.
Now that we have a method for formulating the target vector of a binary and
a multiclass classiﬁcation problem I will now show how to use this for modeling
classiﬁcation problems in neural networks. Se eg. Section 2.2.1 for the deﬁnition
of a neural network.

2.2
Neural Networks
Artiﬁcial neural networks (or simply neural networks) was originally developed
with the intention to mimic the information processing capabilities of biologi-
cal systems (McCulloch and Pitts, 1943; Widrow and Hoﬀ, 1960; Rosenblatt,
1962; Rummelhart et al., 1986). However, we now know that these systems
have little in common with the real biological processes and the model has in-
stead claimed its popularity but its capability of solving practical applications
of pattern recognition.
The most successful model in context of pattern recognition is the multilayer
perceptron network which is a feed-forward neural network. It consists of layers
with ﬁxed sets of units with forward directed connections that combines the
==PAGE:20==
8
Theory
diﬀerent units from one layer to the other. Apart from the input units, the
units are commonly supplied with a non-linear activation function that deﬁnes
how excited each unit is given the input and passes this signal further through
the remaining network layers.
In the following subsections I will introduce the basic knowledge required for
modeling supervised classiﬁcation using the traditional multilayer perceptron
network. Although, the scope of this thesis covers solely the much more com-
plicated recurrent neural network architecture, the following section is intended
as a universal tool for understanding the recurrent deviation.

squares error measure we will continue this case with the formulation of error
backpropagation and how to utilize it.

procedure of minimizing an error function by adjusting the parameters sequen-
tially. This yields two distinguishable stages that involves calculating derivatives

2.3
Recurrent Neural Networks
A recurrent neural network (RNN) deviates from the traditional architecture of
neural networks with inner loops that allow information to persist. This means
that the network has an inner interpretation of the input it has previously seen
which it uses for future outputs. Likewise, as from a human perspective we
don not have to begin every task from scratch and thus only has to recall the
required memory to complete a certain task.
An application for this architecture is for instance the task of classifying what
kind of event is happening in a movie at every frame. For traditional Convo-
lutional Neural Networks (CNNs) it is unclear of how to persist information of
events between each classiﬁcation to inform later ones. RNNs addresses this
issue by remembering the reasoning about previous events and thus capable of
solving this examples.
In Figure 2.2 the basic structure of a RNN is shown where the left most ﬁgure is
the traditional simpliﬁcation of the RNN and its equivalent unrolled illustration
is shown to the right. The input xt is presented to the network at time t which
yields an output ht known as its hidden states.
The natural architecture of the RNN is thus closely related to sequences of
input data and has successfully been applied in a variety of applications such
as speech recognition, natural language processing and image captioning. Men-
tionable is [GDG+15] that shows how RNNs can be used as an image generation
architecture on the MNIST and CIFAR datasets which great results.
Unfortunately, RNNs struggle in practice with learning long-term dependencies
which has been addressed in depth by [Hoc91] and [BSF94]. If the gap between
the current and previous related states becomes to large, the network will fail
to retrieve the correct output.
If we consider a natural language processing
==PAGE:26==
14
Theory
problem, where the gap is small, the ﬁnal word in the sentence "the clouds
are in the sky" is easily retrieved. This is because the context is clear from
the previous few words and the word "sky" is the desired output. If the gap
becomes too large and thus requires more context, it would not be possible to
predict the ﬁnal word in: "I speak ﬂuent Danish".
This is also known as the vanishing gradient problem for neural networks. The
problem occurs when an activation function that squashes its input signal into
a very non-linear output space (such as tangent hyperbolic) is applied. For very
positive or negative input signals the tangent hyperbolic activation function
yields either a value close to −1 or 1 which results in a gradient close to zero.
This becomes much worse when the depth of the network increases, mapping
the nonlinear signals into an even smaller range in the following layer and so
on. This renders both the traditional neural network and the RNN vulnerable
for the vanishing gradients but renders RNNs much more aﬀected due to the
commonly much greater depth.
2.3.1
Back propagation through time
In Section 2.2.2 I described how to formulate the derivatives of the parameters
for a two layer perceptron network using the chain rule.
It turns out that
calculating the derivatives of a recurrent neural network is achieved by repeating
this procedure from the top most layer (regarding time) to the ﬁrst layer. Thus,
back propagation of the error through time can be considered a repeating process
of the chain rule as already described.
2.3.2
Long Short-Term Memory Networks
A long short-term memory (LSTM) network is a special type of recurrent neural
network that is capable of learning long-term dependencies which was originally
introduced by [HS97] and later reﬁned by many other. The LSTM avoids the
long-term dependency problem by nature with the introduction of a gate con-
trolled cell memory architecture. The eﬀectiveness of this architecture is re-
ﬂected by the vast amount of successfully implemented applications with a few
mentionable papers such as [DAHG+15], [SSB14], [GMH13] and [VXD+14].
A modern choice of LSTM architecture is a construction of four gates and an
internal cell state referred to as the networks internal memory. The four gates
exists within the block and are referred to as the input, output, forget and
candidate gate. The cell state is determined by a combination of the signals

vanishing gradient problem as mentioned earlier. By the introduction of the
forget gate [GSC00] the LSTM block became able to eﬀectively learn when to
forget its internal states and thus handle long term dependencies.

3.1.2
Regularization
The number of hidden units H ∈N of a neural network is known as a free
variable that can be tuned in order to achieve the best predictive performance.
==PAGE:39==
3.1 Network training
27
Since the number of weights and biases used is controlled by H we need dif-
ferent techniques to ﬁnd an optimal balance between the under-ﬁtting and the
over-ﬁtting behavior. Finding such a balance ensures that we achieve the best
generalization performance on unseen data.
For regression problems with neural networks, a general approach is to use a
large number of hidden units and then control the complexity of the network
by adding a weight decay term to the error function. A cost function controlled
with a weight decay can be formulated as,
˜E(w) = E(w) + λ
2 w⊤w
(3.10)
with w as the set of weights and λ as the weight decay control parameter.
As such, the eﬀective model complexity is determined by the regularization
parameter λ and found using eg. cross-validation.
Likewise, Dropout is commonly used as a method for regularizing the network
parameters [SHK+14]. During training the parameters are reset with a prob-
ability deﬁned by the dropout ratio and normalized to compensate for the re-
duced set of eﬀective parameters. In sentiment analysis and natural language
processing applications build on recurrent neural networks, it has been shown
that applying a dropout across the embedding and input weight layers improves
learning and makes the network generalize better [Gal15].
In Section 5.2.1 and Section 5.3.1 I examine the relationship between a variety
of hidden units H and dropout ratios used for training.
3.1.2.1
Early stopping
Early stopping is perhaps the simplest method used to improve generalization
of the model. The idea is to split the data set into two parts known as the
training and validation sets and use the validation set for model evaluation
exclusively. Thus, only the training set is used for training the model and the
cost function is evaluated at regular intervals during training on the validation
set. During training the validation and training set errors typically decreases
with the training samples until a certain point where the validation error begins
to rise. This behavior is known as overﬁtting on the training data and should
be avoided as much as possible. By monitoring the validation error we interrupt
the learning when a rise is detected.
One drawback of early stopping is the sacriﬁced training samples used for val-
idation which can result in a reduced performance if the training set is small.
==PAGE:40==
28
Methods
However, for the experiments in this thesis both datasets are reasonably large
with an almost equally distribution of samples belonging to each class. In both
experiments I apply a 10% validation split during training chosen at random
locations along the samples of the original training set. The validations error
is evaluated after each epoch of learning and the training is interrupted if the
error has not improved after 3 epochs.
3.1.3
Weight initialization
Weight initializations in neural networks plays an important role on the con-
vergence of its parameters. Choosing a weight initialization range too small,
compared to the signal, makes the learning slow as a consequence of the small
gradients. On the other hand, choosing an initialization range too large forces
the signal from the activation functions to be step-like, meaning e.g. an out-
put of plus and minus one for the hyperbolic tangent function, yielding in a
slow learning. Common choices for weight initialization are a normal distribu-
tion with mean µ = 0 and standard deviation σ = 1 or a uniform distribution
between two close boundaries.
As for the training schemes used in this thesis the recommended weight ini-
tialization of a LSTM is determined by a uniform distribution as formulated by
[GB10]. The cell states are initialized using a orthogonal initialization method as
described by [SMG13] which is the recommended method. The fully-connected
layer is initialized using the same uniform distribution as for the LSTM param-
eters.
Furthermore, it is recommended to force an initial forget gate bias of 1 during
training of LSTM networks to provoke the learning of long term dependencies
[Zar15]. By setting the initial forget gate bias to 1, the network will force the
activation signal from the forget gate high and thus provoke the cell states to
keep its values (see e.g.
Section 2.3.2 for more detail).
When the learning
progresses the bias is likely to decrease and enable more complex learning to
occur.

3.3
Deep learning frameworks
It is essential to use a well supported deep learning framework for training
and experimenting with diﬀerent network architectures. Documentation and
included examples are important to get started developing networks within such
frameworks.
Only a couple of open-sourced and well supported frameworks
currently exists that keeps getting updated and maintained. The most popular
frameworks are BVLC Caﬀe, Tensorﬂow and Theano.
==PAGE:44==
32
Methods
Since I have encountered all the mentioned frameworks during this thesis I will
in the following elaborate on some of the advantages and disadvantages between
these.
BVLC Caﬀe [JSD+14] was originally developed by Jia, Yangqing during his
PhD at UC Berkeley.
The framework is known for its computer vision and
image retrieval performance and its capability to utilize CUDA (and cuDNN)
for optimized learning time. It is written in C for low level performance and
accessed through a Python module known as PyCaﬀe. A key property of Caﬀe
is the included low level implementation for CUDA accelerated computation,
designed mainly for building ConvNets.
In fact, many state-of-art performance results has been accomplished using
Caﬀe. Mentionable is the ﬁndings of [SZ14] with state of the art performance
on the ImageNet ILSVRC-2014 submission achieving top-5 validation set error
rate of 7.1% and a test set error rate 7.0%1.
One advantage of Caﬀe is the inclusion of diﬀerent image retrieval and clas-
siﬁcation examples with access to the blobs containing layer activations and
parameter derivatives during training. Unfortunately, Caﬀe has a complicated
installation process to gain fully advantage of the external libraries such as
CUDA and cuDNN. Only recently has the support for recurrent layers been
added (to the GitHub repository) but with no further documentation of how to
utilize these layers.
The Tensorﬂow framework, which was developed by Google’s Brain Team and
made public available November 2015, is by far the best documented framework
available. Tensorﬂow, like Caﬀe, can utilize many diﬀerent external libraries
(like CUDA, cuDNN and cuBLAS) but is much easier to install on diﬀerent
conﬁgurations. Additional frameworks build on top of Tensorﬂow, like Keras,
allows a much easier high level programming language to further accelerate the
development of neural networks. Unfortunately, accessing the internal parame-
ters and derivatives during training is not as easy as with Caﬀe.
For the results in this thesis, I ended up implementing all experiments in both
Caﬀe and Tensorﬂow (through Keras) in order to answer implementation related
details which were crucial for the later reported ﬁndings. I originally started
deloping in Caﬀe but faced many diﬃculties relating recurrent neural networks
and was forced to move to Tensorﬂow due to the lag of documentation. Due to
the inaccessibility of the derivatives through Keras I ended up using Keras for
training only and Caﬀe for the damage experiments and loss function estima-
tions.

same bipolar separation and thus reduces the rating scale to be either positive

review that is considered to be neutral or slightly positive/negative ensuring a
clear separation of the two classes.
A typical positive review looks like the following sample: "The art direction,
pacing and cinematic style of this ﬁlm seem to come from another time, more
distant than the eighties and, in some ways, even than the second world war.

sympathy of the character racked and ruined by the cruelties of that order, of
necessity trapped in the enclosed world of the theatre;"
Negative example: "Really, we’re talking about mysticism, possession, and a
killer on the loose here - not a bad recipe for trash cinema. Unfortunately, there’s
nothing here to make it even "good" trash; when joined to the aforementioned,
the bad acting and not-so-special eﬀects are just that - bad acting and not-so-
special eﬀects."

of word indexes (integers) for easier processing. To extract the essence of the

ﬁfth most commonly used word across all reviews (training and test set). For

5.1
The parenthesis example
The parenthesis example is constructed with the intention to visualize the eﬀect
of applying random weight perturbations on the diﬀerent weight layers. The
fact that the network structure is comparable to the following cases further
motivates this small experiment.
Using the found network parameters and the two sampled sentences, the internal
cell state is easily visualized at every time step. This is shown in Figure 5.1 with
a dotted line representing th discriminant between the two classes. The cell state
is initialized to zero.
The discriminant is obtained by the fact that only a single hidden state is
produced and passed through the fully-connected layer with two output units
(with no hidden layers). Calculating the discriminant is a matter of solving an
equation with a single unknown variable.
As shown in Figure 5.1, the cell state is positively aﬀected when the ’)’ char-
acter is presented as the input forcing the cell state over time to become more
positive. Cell activation values much greater than the discriminant (0.155) ren-
ders the posterior probability of class C0 much higher than class C1 and vice
versa for values lower than the discriminant.

5.2
Sequence classiﬁcation on MNIST
Sequence classiﬁcation on the MNIST dataset is achieved by the unconventional
slicing of the images as described in Section 4.1. The reasoning of such is to
provoke the use of long term dependencies in a time related perspective and thus
suitable for a LSTM network. One can think of this method as a line scanning
process, starting from the top of the image and progresses downwards.
5.2.1

5.3
Polarity sentiment analysis on LMRD
The above shown results on the MNIST dataset clearly indicated a robustness in
the RNN-LSTM structure against random weight perturbations. This motivates
the repeating inspection of robustness against weight perturbations in a much
more complex case.
Performing document level sentiment analysis remains a signiﬁcant challenge
due to the complex relationship between sentences. Many existing studies in-
terpret the content by the semantic or syntactic relationships of the document
to extract descriptive features.
Traditional machine learning approaches ex-
tract dense feature descriptors to each document and utilizes Support Vector
Machines (SVMs) to perform the classiﬁcation [PLV02]. Unfortunately, such
methods fails to capture semantic information between sentences [TQL15] and
thus motivates the studies of sentiment analysis using recurrent neural networks.

Chapter 6
Discussion
Here follows the discussion related to the obtained results in the previous chapter
and a project-wide perspective regarding the applied methods.
In terms of robustness against random weight perturbation, I expect, following
the results from [SR87], that the response of the added noise is small for relative
small noise signals and increases shortly after with a parabolic or liner trend.
A range of which diﬀerent damage experiments yields the same loss is thus
considered as an indication of robustness against noise.
6.1
Sequence classiﬁcation on MNIST
The handwritten digits of the MNIST (or NIST) dataset were most likely not
developed with the intention to be used as a time sequence of image slices. The
majority of reported performance measures on the MNIST dataset is also based
on more straightforward machine learning approaches such as SVM, MLPs or
ConvNets. Nevertheless, the MNIST dataset has proven to perform better than
most of the mentioned methods, on the unconventional RNN-LSTM structure
while maintaining eﬀectiveness by keeping the model parameters low compared
to eg. ConvNet based solutions.

Network training and parameter tuning
By the importance of obtaining a fully-trained network for the damage and loss

a saddle point was detected at around 90 hidden units and a dropout ratio of

Despite the large eﬀort of determining the optimal set of parameters by the
polynomial ﬁt as shown in Figure 5.2 I decided to use these results as an indicator
for potential parameter optimality. This was chosen due to the fact that only 63
network estimations was used to ﬁt the parameters of the plane which renders
it prune to a potential overﬁtting behavior. I reckon that a more correct ﬁt is
obtained by increasing the set of network estimations in the selected intervals
with the additional use of cross-validation for each estimate.
This yields a
highly time consuming process which is equally ineﬃcient. As such, I decided
to use 80 hidden units with no dropout for the later experiments hoping this
conﬁgurations was close to a true optimal conﬁguration.
Training multiple networks using the chosen set of parameters yielded in the
averaged learning curve as shown in Figure 5.3. Monitoring the validation loss
yielded no overﬁtting behavior but began settling after 16 epochs of learning.
As such, the network with the best generalization performance was achieved
and used for the following damage and loss estimation experiments.

6.3
Project-wide considerations
A couple of important project related considerations follows here.
The aim of this study was to uncover a potential resilience to random weight
perturbations in recurrent neural network with a particular focus on the LSTM
deviation. As such two main experiments have been conducted using the MNIST
and LMRD datasets by empirically measuring the response in loss to the added
noise. Measured responses that were stationary for small quantitative of layer
relative added noise was considered resilient while layer responses that were
unaﬀected was considered non-essential.
Overall, the LSTM architectures did show a surprisingly robustness against
weight perturbation which i reckon to be a direct result of the feed-forward
styled construction. As described in Section 2.3 a recurrent neural network can
be compared to a very deep feed-forward neural network.
Regarding the method for estimating loss as described in Section 3.2 I recog-
nize three potential sources of error (see also [AK16]).
First, I replaced the
expected change in loss function given by the random weight perturbation with
the expected value over the ensamble of random weight perturbations. This is
justiﬁed by the fact that the loss is a sum over a very large number of training
samples such that an self-averaging occur. Thus, I render the importance of
this simpliﬁcation limited and made the important simpliﬁcation of the error
estimate using only the diagonal elementals of the Hessian.
The second source of error is the estimation of loss is related to the second order
expansion of the error function. However, I found evidence in Figure 5.7 and
Figure 5.15 that the error follows a quadratic trend, rendering this source of
limited importance.
The third source of error is the use of the Gauss-Newton approximation method.
A main justiﬁcation of the use of this is the assumption of a well-trained network
has been achieved. This is also the reasoning of the large part of the experi-
mentation is focused on training the model parameters. If the neural network
has not reached this state, the ﬁrst order derivatives of the error function are
not neglectable and our assumption is violated.
Finally, we are left with one question to discuss. What is it that renders recur-
rent neural networks this resilient against random weight perturbations? I am
convinced that this is partly due to the natural similarity to the traditional feed-
forward construction of MLP networks which are known to be robust against
small weight perturbations. In general, neural networks are known to be quite
==PAGE:72==

eﬀective to construct a distributed representation of its input signals. I am con-
vinced to have found evidence in the conducted damage experiments that parts
of the network are more reﬁned in terms of interpreting the input in a problem
solving perspective. Adding noise to those layers will cause a larger impact on
the error function than compared to the rest of the network parameters. Un-
aﬀected network parameters will still contain important information and thus
maintain a predictive capability.

Chapter 7
Conclusion
In this thesis I have empirically studied the eﬀect of applying random weight
perturbations to the diﬀerent parts of a recurrent neural network with the in-
tention to reveal robustness against such attacks. In addition to this I have
covered the inner functionalities of the long short-term memory block of which I
have used to obtain the diﬀerent networks used in the experiments. Along with
the diﬀerent damage experiments I have described a method for predicting the
expected change in error value as the function of the added noise.
I found evidence of a layer speciﬁc diﬀerence in response to the added weight
perturbations and an indication of a surprisingly large robustness against weight
perturbation. The proposed method for predicting the expected change in error
function yielded very adequate results for even large magnitudes of noise.
