Summary
Modelling grouped data can be undertaken in more than one way. In some cases
a fixed effect model with parameter estimate for each group and in other instances
random effects, that allow for parameters characterizing the variation between groups.
In this project,the focus has been on mixing the two methods in Generalized Linear
Mixed Effects Model and Hierarchical Generalized Linear Models respectively. These
models were tested on two different data sets against each other and the influence
of both fixed and random effects were discussed.
Likelihood theory was used to
determine the performance of the models.

CHAPTER 1
Introduction
Having data put into groups is a common occurrence, whether it is patients in a
medical trial being put into groups based on sex or treatment, animals being grouped
which litter they belong to etc. When modelling grouped data, there is more than
one way to go about it.
Some times a regular fixed effect model with parameter
estimates for each group will do and in other instances random effects, that allow for
parameters characterizing the variation between groups will prove more useful.
Within this study there will be two different data sets, containing grouped data,
that will be examined separately. The first data set is from 1951 and concerns the
number of defective lids based on each batch in production. Here it is of interest
to see if the methods we will apply can help describe the variation of defective lids
between each batch. With regards to the second data set it consists of the official
numbers, of new Covid-19 cases and deaths of people who had Covid-19 at the time
of death. The data is provided by the European Centre for Disease Prevention and
Control’s official website. For this data it is of interest to model the variation of
deaths from country to country.
For both sets of data the methods applied will be Generalized Linear Mixed Effects
Models and Hierarchical Generalized Linear Models, as they are able to handle both
the fixed and random effects, should the need arise.
To carry out computations and to further the statistical analysis the statistical soft-
ware R has been utilized.

Theory
Generalized linear models (GLM) have an established place within statistics. They
extent the capabilities of statistical models to include response variables that are not
quantitative, continuous or normally distributed. However, in an effort to expand
the use of GLMs, they have extensions, more specifically extensions that address the
challenges which arise with grouped data.

Such a table illustrates a regular example of classification. Typically speaking a pa-
rameterization of grouped data consists of a parameter that contains a level for each
unique group in the data, in other words, defined as a regular fixed effect. It does,
however, occur that it is more suitable to consider the within-group variations of the
data and not the specific level relative to the entire population. In order to account
for the within-group variation, the fixed effect for each level will be exchanged for
what is called a random effect. A model containing only random effects is called a
random effects model and a model incorporating both fixed and random effects are
referred to as mixed effect models.
Generally speaking mixed effect models allows the user to describe the dependence
between observations within and between groups, by assuming at least one latent

are random, thus random effects [MT10, p. 158].

Above a Gaussian Mixed model is formulated. The key take away here, is that mixed
effect models are constructed in what is called a hierarchical structure, as equation
2.1 & 2.2 shows. This entails that the model have a first and second stage. The first
stage (equation 2.1) is the conditional distribution of Y given the outcome of the
random effects, u.The second stage (equation 2.2) is the assumed distribution of the
random effects.
Equation 2.1 & 2.2 clearly states that U and u are normally distributed, thus they
are what is called a hierarchical normal distribution model.
Because of this they
are considered a special case of the more general class of models, the General Linear
Mixed Effects models (LMM). Which are defined as follows in equation 2.3 :
Y = Xβ + ZU + ϵ

spectively.The error term ϵ ∼N(0, Σ) and random effects U ∼N(0, Ψ) assumes
normality.
With respect to mixed models LMMs are not in a completely different position as
General Linear Models are with respect to Linear Models. They constitute the base-
line models and they assume normal distributions, and because of this extensions that
allow for the response variable to assume other distributions are needed. General Lin-
ear Models have the extensions of Generalized Linear Models that allow for a broader
variety of distributions. For LMMs there is more than one extension that allows for
the user to assume other distributions. One extension is Generalized Linear Mixed
Effect Models that allows for non-normal response variable distributions, but assumes
normality for the randoms effects. An other option is Hierarchical Generalized Linear
Mixed models, that further allows for both response variable and random effects to
assume other distributions. Both such options will be examined.

2.3
Hierarchical Generalized Linear Models
Hierarchical Generalized Linear Models (HGLMs) are one further extension of the
Generalized Linear Models (GLMs). Where Generalized Linear Mixed Effects mod-
els (GLMMs) allows for the linear predictor of a GLM to incorporate random effects
with normal distributions, in addition to the fixed effects. While this is useful for
specifying correlations among the random effects, they ability to use other distribu-
tions for random effects enables the models to be useful in a greater variety of cases
[LNP18, p. 173] . In 1996 Lee and Nelder extended GLMMs into HGLMs, in which
the distribution of random effects are extended to an arbitrary distribution, more
precisely the conjugate distribution of the response y [LN96]. Thus HGLMs serve as

Data
The data examined in this thesis is comprised of two different data sets. Both data
sets contain data where it is of interest to estimate the within group variance of a
variable and as such lend themselfs well to mixed effect models.

Defective Lids Data
The first data set is a distribution of defective lids in sampled lots of 770 lids each, and
a total of 229 lots (Source, J.H. Ford (1951)). The process of sampling the lids from

of defective lids per lot y, the number of observations per lot (n = 770), and the
number of each lot - no data cleansing will be needed. All three variables are discrete
variables. Below, in Figure 3.1 the distribution of defective lids is shown.

CHAPTER 5
Discussion
The comparison between Generalized Linear Mixed Models and Hierarchical Gener-
alized Linear Models proved in the case of the Lids data to come out in favor of
the Generalized Linear Mixed Model. Considering that the Hierarchical Generalized
Linear Model should be more flexible when modelling the random effects in, this was
not expected. For the comparison of the methods to be considered successful, on
the Covid-19 data, it would be necessary to be able to correctly apply the conjugate
distributions for multiple random effects. This will need to be looked into. Regarding
the performance of the Generalized Linear Mixed Model it was seen that the early
ramp up of deaths was predicted too slow, and that the model generally speaking
might need more than time and number of deaths as variables to distinguish between
countries better.
With respect to the lids data set, any further improvements in predictions would
likely have to build upon the existing Hierarchical Binomial-Beta distribution model,
following the Polya Distribution. Even though the model feel off near the tail end
with too low expectations for samples with a high number of defective lids, it was still
the runaway best performer. However, as there is little other information from the
study, considering it was done in 1951, the Polya distribution would either need to be
extended in some shape or one would have to formulate another random effects model.
With respect to the Covid-19 data there are many ways to expand the study. One
way of expanding the study would be to incorporate variables that would describe
the strategy of each country. It would add a depth and nuance that could take into
account whether or not a country was fast to enforce a lock down, or how compre-
hensive the lock down was. This could possible help catch the slow ramp up problem,
as described in the Results section. Furthermore, as the model proved insufficient in
detecting the spikes in deaths on given days it might be desirable to construct a model
with a time interval at one week in place of one day, possibly increasing accuracy.
Lastly it could be of interest to model the assumed mortality of Covid-19.
A bi-
nomial model that considers both the number of cases of Covid-19 and the number
of deaths of people with Covid-19 at the time of death. It would ultimately serve as
a different measurement, as it would not measure the amounts of deaths, but more
the ratio of survivors/mortality.

CHAPTER 6
Conclusion
This study set out to employ techniques that would allow for grouped data to be in-
vestigated using both fixed effects and random effects. This was done on two different
data sets.
With respect to the data set concerning defective lids the Generalized Linear Mixed
Effects model performed the best of the two methods concluding that the individual
batch of lids did have an effect on the number of defective lids in said batch. However,
both methods proved to be inferior to the previously developed Polya distribution.
For the data concerning Covid-19 cases and deaths, it was effectively only possible
to construct the same model twice. For the Generalized Linear Mixed Effects model
everything went fine and the model was successfully computed, finding the individual
country did have an effect on the number of deaths with being Covid-19 positive.
As for the Hierarchical Generalized Linear Model the software needed to conduct the
computing of the Gamma distributed random effects, proved incapable of doing so
for more than one random effect. Thus it was effectively a Generalized Linear Mixed
Effects model, as the random effects were assumed normally distributed in the Hier-
archical Generalized Linear Model. The investigation in the Results section proved
that they modelled identically.
