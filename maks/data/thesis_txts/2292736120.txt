Abstract 
 
The estimation of wave spectrum is critical for safe voyage and safe cargo 
transportation. 
In this dissertation, the method of estimating a wave train using the Kalman filter is 
analyzed. The estimation is done using simulated data. This is because by simulating 
data the form of the wave elevation is known as it is chosen by the user. In that way it 
is easy to validate if the wave elevation is reconstructed correctly. The examined vessel 
is a US Navy destroyer where only the heave motion is analyzed. The estimation is 
conducted under the assumption of zero forward speed. 
The implementation is done using the software Matlab R2014b which is kindly 
provided by the department of Mechanical Engineering at DTU. Three cases are 
analyzed using the simulated data. The first case is done using one frequency where the 
wave elevation is reconstructed perfectly. The second case is done by using two 
frequencies to realize how to scale up the problem. In this case, the reconstruction of 
the wave elevation is also done successfully. The last case is done by using five 
frequencies, which is a step to make the code more general and in the end, work for an 
arbitrarily number of frequencies. This would be required in case of real ocean wave 
spectra were to be estimated. At the last case, the reconstruction of the wave elevation 
is done successfully as well. In all three cases, the accuracy of the estimation procedure 
reaches 99%.

1 Introduction 
 
1.1 Motivation 
 
Sea state estimation is important for marine operations. In particular, applications that 
involve decision support systems where, sea state estimates (wave height, wave 
elevation, etc.) can be used to prevent: seasickness of the passengers, damage of the 
cargo or even cargo loss due to large accelerations, local structural damage due to 
slamming, green water on deck, propeller emergence, loss of stability due to cargo 
shifting etc.(Nielsen 2005). Another application benefiting from the knowledge of sea 
states includes dynamic positioning, where the sea state estimates can be used for feed-
forward control. In that way, both station-keeping behavior and fuel consumption are 
improved (Tannuri et al. 2003). Therefore, the two main reasons sea states should be 
estimated are for safety and operational reasons. 
 
 
1.2 Background 
 
In order to understand the method of estimating a wave train using Kalman filtering, it 
must first be clarified where this theory is based on as well as explain the reason why a 
new method is used. This procedure is actually one of the methods that are used in order 
to estimate a sea state using measured ship responses. It is based on the concept of wave 
buoy analogy. In this method, the ship itself is considered a wave buoy, having motion 
sensors onboard and measuring the wave spectrum. Of course, the wave estimations 
carried out by ship responses are not as accurate as the measurements from a real wave 
buoy. This is mainly because of two reasons; firstly, because the motions monitored by 
ships are filtering due to the high inertia of ships hull, compared to the relatively small 
size of a wave buoy. The second reason is that ships are moving with a forward speed 
when the estimations are carried out. Therefore, the results from ship responses and 
wave buoys cannot be the same; however, this analogy is a good approximation 
(Nielsen 2007). 
The mathematical models that have been used the most for the estimation procedure 
based on the wave buoy analogy are the Bayesian Modelling and the Parametric 
Modelling. The Parametric Modelling is based on a summation of a number of 
parameterized wave spectra. On the other hand, the Bayesian Modelling estimates the 
directional wave spectrum in a number of discretized points of the wave field, which is 
divided into a set of frequencies and a set of directions (Nielsen 2006). 
 
Both of these methods are based on the estimation of wave spectral densities by 
minimizing an error measure. Either these spectral densities can be estimated directly 
from the given data or they can be calculated using the estimated wave spectrum. 
However, a bottleneck for any of the methods is that minimizing the error requires

iterative procedures. Moreover, this estimation procedure requires also a reasonable 
period of time in the order 10-15 minutes (backwards) to produce stable results of the

ceases. However, the most noticeable difference compared to the other methods is that 
using the Kalman filter, there is no need to perform any cross-spectral calculation since 
the wave amplitudes can be estimated directly (Pascoal et al 2009). 
 
 
1.3 Literature Survey 
 
In order for the decision support systems to work as effective as possible (at least from 
operational point of view), the wave spectrum at the location of the moving vessel needs 
to be estimated accurately and updated continuously. Fortunately, nowadays there are 
some methods to estimate the wave spectrum. One of these methods is by using moored 
wave rider buoys. This method is quite reliable and accurate however, it gives an 
estimation at a fixed position in the ocean. (Nielsen 2010). Other methods such as 
satellite measurements and wave radars are also accurate and they give an estimation 
in a quite wide area. However, these methods require complex computational hardware 
as well as high installation cost (Tannuri et al 2003). One of these methods is by using 
Acoustic Doppler Current Profilers (ADCPs). The ADCPs are not only able to estimate 
the wave spectrum but also do a prediction regarding the direction of the current. This 
method is reliable and it gives an estimation in a wide area however, this has also a high 
installation cost (Pandian et al 2010). A commercial product which uses the information 
of the radars and conducts a wave estimation analysis is the Wave Monitoring System 
(WaMoS II). This system has been developed by the German GKSS Research Center. 
This hardware consists of a conventional marine radar, an A/D converter, to convert 
the analog radar video signal into digital and a computer for storage and processing of 
the data (Borge et al 2000). Another product which is able to do a wave estimation is 
the Wavex. This system is developed by the company MIROS and it is able to predict a 
3-dimensional wave spectrum. Wavex uses an X-band radar for the estimation 
procedure (http://www.miros.no). Another method for sea state estimation is by using 
motion sensors. This method only requires some sensors onboard the ship, which 
monitor the ship responses.

One method for estimating the ocean wave directional spectrum using ship responses 
has already been developed and presented in the paper by Pascoal and Soares (2009) 
Kalman filtering of vessel motions for ocean wave directional spectrum estimation. 
This method uses as input data, the measurements from motion sensors (see Figure 1) 
which can easily be installed at any kind of vessel since the cost of these sensors is quite 
low. Therefore, this method is ideal for providing initial estimates to offline estimation 
procedures and to give spectral updates under quickly changing weather conditions. 
The Kalman filter is used to filter the noise and provide the updated estimates. Other 
advantages of this method, comparing to the previously mentioned, are that, it is easy 
to formulate (as it only requires the implementation of the Kalman filter into a 
computer) and also gives accurate estimations in a wide area (in contrast to wave rider 
buoys).  
 
 
1.4 Problem statement and scope 
 
The goal of this thesis is to use the method developed in the article by Pascoal and 
Soares (2009) for  the estimation of a regular and unidirectional wave train. This method 
is based on the Kalman filter. By assuming a wave amplitude, a frequency and a phase 
angle, the wave elevation is calculated as well as the two components x1 and x2 which 
are part of the wave elevation. Then, using given RAO’s, the responses of the ship are 
simulated. Finally, these responses are entered into the Kalman filter and by calculating 
the new components (x1, x2) the wave elevation is reconstructed.

2.2 Regular Waves 
 
Since the estimation problem is restricted only in regular waves, this section should be 
mentioned. It introduces the theory of regular waves and it is based on the theory 
presented by Nielsen (2010).  
 
2.2.1 General 
 
In general, regular waves never occur into real oceans and seas. However, they form 
the basis to study the irregular waves. Besides, irregular waves based on the assumption 
that they are a superposition of regular waves with different frequencies, amplitudes, 
directions and phases. Regular waves can be produced artificially in towing tanks in 
order to carry out hydrodynamic tests like resistance measurement and seakeeping. 
Therefore, the importance of this kind of waves is noticeable. 
 
 
Figure 3. Regular Waves (Nielsen 2010) 
From Figure 3, it can be seen that a regular wave has the following parameters: 
ζ is the surface elevation (it is not shown in the figure). 
ζ0  is the wave amplitude. The wave amplitude is the vertical distance between a crest 
or a trough and the water surface. Later, the wave amplitude will be referred also with 
the letter A. 
H is the wave height which is twice the wave amplitude. 
λ is the wave length. The wave length is the horizontal distance between one crest or 
trough and the next one. 
c is the velocity of an individual crest in the x direction. It is also called celerity.

3 The Kalman Filter 
 
The Kalman filter is a computer algorithm that uses a series of measurements observed 
over time, containing white noise, and produces estimates of unknown variables that 
tend to be more precise than those based on a single measurement alone. In general, 
the Kalman filter is a set of mathematical equations that implement a predictor-
corrector type estimator that is optimal in the sense that it minimizes the estimated 
error covariance (Welch, Bishop 2001). 
 
3.1 History 
 
The Kalman filter is named after Rudolph E. Kalman, who in 1960 published his 
famous paper describing a recursive solution to the discrete-data linear filtering 
problem (Kalman 1960). 
Rudolph E. Kalman was born in 1930 in Budapest, Hungary and in 1943 he immigrated 
to the USA. He is an electrical engineer widely known as the creator of the modern 
control theory and the system theory. His research reshaped the field of control 
engineering and placed the groundwork for future research and innovation (Galeazzi 
2014). Kalman has won several awards for his commitment to science the most 
prestigious of which are the Charles Stark Draper Prize (2008) from the American 
National Academy of Engineering “For the development and dissemination of the 
optimal digital technique (known as the Kalman Filter) that is pervasively used to 
control a vast array of consumer, health, commercial and defense products” and the 
American National Medal of Science from the US President Barack Obama: “The 
impact of Kalman filtering on all areas of applied mathematics, engineering, and 
sciences has been tremendous” (Galeazzi 2014). 
 
Kalman filtering has a lot of different applications, it is used extensively in the area of 
autonomous or assisted navigation. The most important and well known application is 
when NASA used the Kalman filter to solve the problems related to determining 
satellite orbits. As NASA made the Kalman filter directly part of the US Space 
programme, the organization didn’t hesitate to use it in big missions like Ranger, 
Mariner and of course Apollo and actually very successfully. Consequently, the Kalman 
filter knowledge passed into the commercial airlines and aircraft industry mainly for 
navigational use (AFOSR 2011) 
The Kalman filter came into use after the famous Wiener filter. The Wiener filter was 
quite remarkable, it was a filter weighting function in the continuous case, or a set of 
weight factors in the corresponding discrete problem, that means that the past values of 
the input should be weighted in order to determine the present value of the output and 
that’s the optimal estimate. However, despite the extensive use of the Wiener filter and 
its useful applications, the filter was not able to solve a more complicated time-variable, 
multiple-input/output problem. Therefore, since 1960 the Kalman filter is used for more

complicated problems such as vector modeling of the random processes or recursive 
processing of the noisy measurements (Brown, Hwang 1996). 
Therefore, some advantages and applications of the Kalman filter are the following: 
 Removes “noise” from series of data 
 Easy to formulate and implement given a basic understanding. 
 Optimally estimate the state of a system over time 
 Target tracking by radar 
 Global positioning systems

3.3 Noise – Stochastic processes 
 
This section is based on Chapter 2.1 from the book of Brown and Hwang (1996), 
Introduction to Random Signals and Applied Kalman Filtering. 
The first thing to be distinguished – clarified in this section is the difference between 
the deterministic signals (or noise) and the random ones. 
Deterministic is the signal which is 100 percent predictable in the time span which 
interest the researcher. One example could be: 
 
𝑥(𝑡) = 10 sin 2𝜋𝑡 
(16) 
 
In the above example a sinusoidal wave is shown. As it is obvious, all the parameters 
are known of course in a given time span. 
On the other hand, a random signal (or a stochastic process) has at least one element 
of chance associated with it. Therefore, it is absolutely not predictable (at least from a 
deterministic point of view). One example, which is also the case in this project is the 
wave elevation of a regular wave:

(17) 
 
Here, the parameters of wave amplitude and frequency are defined by the user.

5 Conclusion 
 
The goal of this thesis is to find a method to estimate regular wave trains using Kalman 
filtering. This is because wave trains are the basis for a sea state estimation. Decision 
support systems use sea states in order to provide data about the route and then the 
captain can change the course of the ship for the sake of safety. 
After the presentation of the theory of regular waves, the outcome is that the wave 
elevation must be estimated. In order to do so, the responses of a ship need to be 
simulated. For that reason, the RAO’s from a destroyer vessel are used. Then, the 
Kalman filter applies. The Kalman filter is an optimization algorithm which has the 
ability of filter the noise from a signal, and provide an updated “version” of it by using 
new estimates. In the end, the wave elevation is reconstructed. 
The estimation of the new wave elevation is done at zero forward speed. Three cases 
are analyzed. One initial case with only one frequency which is done in order to 
understand how the Kalman filter works and to check actually if it works correctly. The 
wave elevation is reconstructed totally. At the second case, two frequencies are used in 
order to see how it is possible to scale up the problem. After understanding this point, 
the wave elevation estimation takes place. The results are again acceptable as the wave 
elevation is totally reconstructed. At the last case, five frequencies are added in the 
algorithm. If this case works, then the Kalman filter works for a random number of 
frequencies. At this case, the wave elevation is also reconstructed completely.  
One step to enrich the present work, would be to continue the analysis like in Pascoal 
and Soares (2009) and include more responses such as roll, pitch and yaw 
simultaneously. Then, continuing, like in the above paper, and estimate a 
multidirectional wave spectrum. 
 
5.1 Further work 
 
Further work would be to add some forward speed i.e. 20 knots. In the end, it would be 
really interesting to see the same work but instead of using synthesized data, using real 
data.
