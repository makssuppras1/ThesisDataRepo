Abstract
In the п¬Ғeld of turbo machinery it is desired to construct a system where oil consumption and
spill in the product is limited. In some industries it is now a requirement that there is no oil
leak. The search for a system where no oil is used leads to a bearing system using only air as
for instance gas bearings or foil bearings. The types of bearings use the pressurized air to
generate a lifting force needed to stabilize the rotor system. Another bearing type not using
any oil is a magnetic bearing. The magnetic bearing system needs an active control system to
ensure that the shaft is stabilized during operation.
In this project a mathematical model able to describe the dynamics linked to a п¬Ӯexible rotor
levitated in a pair of magnetic bearings accurately is wished to be established. This will make
it possible to implement an LQG-control design. The mathematical elements used to describe
the global mathematical model are established and validated. Then the LQG-control design is
used to calculate and implement a model-based controller which is then evaluated
performance wise. Lastly the response from a run-up of the shaft is wished to be simulated to
deп¬Ғne the expected operational range for the system. Therefore an estimation of the
unbalance force acting on the rotor has been carried out using experimental data.
Through the report it is concluded that the elements contributing to the mathematical model
are in good agreement with the experimental results. The elements are adding up to an
established mathematical model able to describe the system dynamics well. An examination
of the eigenvalues of the designed LQG-controller shows that the system is stabilized in at
least the operational range from 0 [Hz] to 1000 [Hz] that is if the system dynamics remain
linear. The conclusion from the estimation of the unbalance force is that it is not accurate to
place the unbalance purely in the rotor laminates considering the present rotational speed
only. However the study shows that the unbalance force is possible to estimate using the
displacement signals from an experiment.

Motivation
In collaboration with GL-Turbo a п¬Ӯexible rotor system supported by Active Magnetic
BearingвҖҷs (AMBвҖҷs) was built during the master studies of the author. Before the start of the

Challenges regarding the control design
The challenges speciп¬Ғcally concerning the design of the model-based controller is a limited
amount of resources at the FPGA target of the Compact-RIO from National Instruments.
A high order model-based controller is therefore tricky to implement on the system. The high
order state space model controller is needed to describe the bending modes from the shaft and
states in the electrical part of the system. It was therefore realized early in the process that a
controller also able to react to the foundation dynamics cannot be implemented. However a
model containing the dynamics of the foundation is used in simulations such that the
dynamics are accounted for when the controller performance is evaluated.
Another challenge regarding the system occurs due to the design. The nodes for the mode
shapes of the shaft are poorly placed which will be shown when the modeshapes of the shaft
are examined. In the design the nodes are located close to the center of the magnetic bearing.
The damping added from the control design is dependent on the movement of the shaft which
is lowest at the nodes. Therefore the mathematical model needs to be accurate in order to be
able to design an LQG-controller able to stabilize the bending modes.
In addition the placement of the eddy current sensors measuring the position of the shaft has
caused some diп¬ғculties while commissioning the test rig. The placement of the sensors results
in a non-collocated measurement of the shaft in the bearing [1, p. 323]. This means that the
shaft is moving in the opposite direction in the bearing than what is measured from the
sensors. This occurs when a node is present between a sensor and the magnetic bearing.

manufacturing processes. To emphasize the parameters to be optimized and the reasoning a
description of these is made.
вҖў Ki, Ks - While evaluating the manufacturing of the stator geometry for the radial
magnetic bearings it was found that the grinding process caused them to be oval. This
means that at some legs the gap is smaller than expected while at other legs the gap is
bigger than expected.
Also the grinding of the rotor laminates proved that some laminates deformed
plastically. This means it is highly possible that the laminates have deformed elastically
during the grinding. This will cause a smaller gap than expected since the removed
material from the laminates is lesser than expected.

12.a
Sensitivity
As the sensitivity to a system is evaluated it is assumed that process variation, model
discrepancies or other disturbances to the system can be lumped in a point of the model and
from that point fed into the system. In this system the process variance of the changing
rotational speed will need to be taken into account as well as the noise and unbalance force.
Therefore the sensitivity of the control design is evaluated two places in the system namely
for the current in the magnetic bearings and the input to the position controller. The change
in rotational speed is thought of as an uncertain parameter, which can take a value from 0
[Hz] to 700 [Hz] which is the desired range of operation. The sensitivity from the input signal
of the current to the AMBвҖҷs shows the same response as for the unbalance force. The
sensitivity function states the ampliп¬Ғcation of a given disturbance from the system itself.
Diп¬Җerent classiп¬Ғcations of the sensitivity of the system are described through standards. A
class A system will not amplify the disturbance more than a factor of 3 or 9.5 [dB].
In п¬Ғgure 35 a comparison between the sensitivity for the LQG-control design and PID-control
design is made. In the п¬Ғgures a number of occurrences of the system is shown all having a
diп¬Җerent value of ПүOp. It is noticed that the singular values of both of the system violates the
class A. However as the rotational speed of the shaft increases the sensitivity peak at 20 [Hz]
for the LQG-control design is minimized eventually getting below the classiп¬Ғcation A limit. It
is noticed that the PID-control design will not be able to comply with the classiп¬Ғcation. Also
comparing the two control designs it is evident that the amplitude for the п¬Ғrst bending mode
of the system is much smaller for the LQG-design than for the PID-controller. The diп¬Җerence
is a symptom of the PID-control design neglecting the dynamical contribution from the п¬Ғrst
bending mode of the shaft.
In п¬Ғgure 36 the sensitivity for the input to the position and current set points for the
LGQ-controller is visualized. The п¬Ғgure describes the ampliп¬Ғcation of the noise of the sensors
through the system dynamics. These sensitivities are not as important as the sensitivity from
a disturbance to the system when a classiп¬Ғcation is made. However it reп¬Ӯects that the noise
of the displacement sensors is ampliп¬Ғed by a factor of approximately 6 at 20 [Hz] while the
noise from the current sensors are not ampliп¬Ғed more than by a factor of approximately 2.

Discussion
The scope of the project was to get the shaft to rotate stable in the operating range of 0 [Hz]
to 700 [Hz]. This goal however was not achieved п¬Ғrstly because of a limit of the power supply
to the ampliп¬Ғers. The ampliп¬Ғers however need to be redesigned for being capable of handling
a suп¬ғciently larger power supply. In the following a discussion on the important elements of
the report is deduced and a suggestion to further work is given.
14.a
Finite element model of the shaft
From the п¬Ғnite element model of the shaft it is noticed that a discrepancy between the
placement of the nodes predicted and in reality. Also the eigenfrequency of the п¬Ғrst bending
mode seemed a bit oп¬Җwhich seems to be corrected as the shaft is levitated with the PID-gains
found in table 2. However if the gains of the control system are wished to be magniп¬Ғcently
changed the п¬Ғnite element model should maybe be revised. The model of the shaft consists of
approximately 70 elements which is a high number for such a short shaft. This has been the
reason for using Timoshenko beam elements in the п¬Ғrst place. The Timoshenko matrices
from [6] which from an evaluation of the gyroscopic eп¬Җects in appendix A п¬Ғgure 44 yields a
wrong result. As the matrices from section 4.a.1 are evaluated it is also shown in appendix B
that the results do not agree with the ones found in the article. This is also the case for the
stiп¬Җness matrix. An example evaluating the suggested matrices to be used for the
Timoshenko element is found in appendix B п¬Ғgure 46. The results are compared to a
Bernouilli beam element. It is concluded from the test that the matrices found in the article
match the Berouilli element. The discrepancy for the gyroscopic eп¬Җect is due to a too high
number of elements which results in the elements becoming too small. The reason for the
disbelief in the article is based on lots of typos meaning that what was stated could be wrong.
The alternative gyroscopic matrix is used to generate the results in п¬Ғgure 9 where the tuning
factor of 1.7 is added. This is npt able to be explained reasonably other ways than the
elements have been chosen too short.
14.b
Finite element model of the foundation
The transfer function from corner 1 to 2 in п¬Ғgure 14 shows that the damping of the
foundation might have been somewhat wrongly estimated. Here the error of using п¬Ғve points
instead of three around the resonances is not present. The discrepancy is based on the fact
that the most damped eigenfrequencies of the measured were left out of the estimation. The
proportional damping based on a linear contribution from the mass and stiп¬Җness is however
not able to describe a highly damped frequency just next to a little damped eigenfrequency if
there are more than two frequencies in the estimated set.

Normally when one knows that there is noise in the system a band of the standard deviation
to the expected noise is drawn in a simulation in the time domain. This band is used to

Unbalance estimation
The expected amount of residual unbalance after the balancing at GEA-niro seems from
experiments to be higher in reality. The reason for this could be because the shaft is not
rotating around the same axis as the one the shaft was balanced around or that the unbalance
is located elsewhere than in the rotor laminates. To examine this an input matrix for every
translational degree of freedom along the shaft needs to be build and the maximal singular
value at every frequency should be evaluated in the same way as in п¬Ғgure 38.
14.e
Suggestions to further work
First of all a reduction of the electrical subsystem would be very beneп¬Ғcial to look at.
Currently the subsystem holds 32 states alone which is a pair of 16 of the same states. This
seem ridiculous when the rotor system is reduced from 592 states to 16.
It would also be beneп¬Ғcial to look for other methods on how to more accurately estimate and
implement the damping to the foundation. If a wider frequency range than 0 [H] to 500 [Hz]
regarding the foundation model is wished to be established these should be compared with
the ANSYS model and the reduced foundation model.
Also a better estimate for the residual unbalance of the shaft will need to be implemented and
tested to correctly estimate the voltage needed to drive the current in the coils. Essentially
designing the voltage supply to be able to fulп¬Ғll all of the operational area.
As it is stated during the observer design a thoroughly analysis of the noise has been avoided
in this report. Therefore if one would like to design a more correct observer for the system
this task would need to be carried out.
15
Conclusion
From the project it can be concluded that a mathematical model able to simulate the system
dynamics well has been established. This has been successful through a validation of the
elements needed to build the mathematical model.
An optimal control design in form of an LQG-controller has proved to be able to stabilize the
system successfully in the operational range from 0 [Hz] to 400 [Hz]. Using linear theory the
controller is able to operate between 0 [Hz] and 1000 [Hz].
It has been shown how the experimentally obtained displacement signals from the system can
be used to estimate the residual unbalance of the system. The estimation of the residual
unbalance has also shown that the assumptions on the location of the residual unbalance is of
importance.
