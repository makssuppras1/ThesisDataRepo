Abstract  
This project revolves around the use numerical tools, to determine steels plastic material parameter from tensile tests. 
These parameters are determined by an inverse analysis, where a guess for the material parameters are used to perform 
the numerical analysis. The results from this analysis are compared to the experimental results and based on the 
comparison the material parameters are updated. During this project a program is created which performs the inverse 
analysis, automatically. Besides performing the inverse analysis automatically, the following parameters are 
determined: 
 
Young‟s modulus 
 
Proportionality limit 
 
Yields stress 
 
Ultimate stress 
To validate the performance of the program, a total of 40 tensile tests were conducted. It was relevant to have a variety 
of experiment to verify that the program was fully generalized. It was therefore chosen to use two different steel types 
S235 and S355. Besides the common Dogbone specimen, the use of a simplified specimen type was investigated; this 
specimen was chosen to be a Disc with a hole. 20 of each type were tested, consisting of an equal number of each steel 
type. The geometric stability for the Disc with a hole specimen was investigated with regards to; the hole size and 
eccentricity. It was observed that the eccentricity of the hole did not influence the results of the analysis, but there were 
observed a difference between the two different hole sizes. Further investigations are required to determine the origin of 
this difference.  
The 20 Dogbone experiments were used to validate the precision of the inverse analysis. The experiments were 
analyzed both analytical and numerical. The results showed a reasonable agreement between the results from the two 
analysis methods. Likewise were the results from the Disc with a hole specimens compared to the Dogbone specimens 
and showed a reasonable agreement. 
The method used for the parameter update is based on the method for the manual inverse analysis. Each specimen was 
analyzed with regards to both engineering and “true” stress-strain. Through processing of the experimental data various 
problems, related to the chosen method, were observed. These problems have to be addressed for further use of the 
program.   
 
The necking phase is difficult to replicate with only a few point. The strain point did not sufficiently fit the 
data to fit the necking phase for some experiments. This was a complication for the engineering analyses. 
 
For cases, where the majority of the curve fit the experimental data and a small part is far from the 
experimental data, here is the improvement per iteration below the tolerance and the update is stopped. This, 
even though a part of the curve, does not fit the experimental data. 
 
The analysis has a long calculation time. The accuracy has to be reduced, in order to have a reasonable 
calculation time. 
Three alternative methods were presented: known shape, shape combination and a stepwise method. These methods 
could improve the results from the inverse analysis, and avoid the above mentioned problem. Implementation and 
testing of these methods are required to be able to assess their full potential. 
The program created to perform the automatic inverse analysis is intended for further development and use. For this 
reason the user-friendliness was in focus during the creation of the program. The report contains descriptions for the 
majority of the scripts and the theory, which the scripts are based on. Other measures has been taken to improve the 
user-friendliness, e.g. added a user manual. Furthermore, the major part of the program is created in Matlab, which 
makes it easy to implement new functions or alter old ones.

1 INTRODUCTION 
 
This report revolves around the processes associated with 
tensile testing in regards to determine the plastic material 
properties. Two types of stress-strain relations are considered 
in regards to the plastic properties: the engineering that is 
based on the original geometry and the Cauchy or “true” 
stress/strain, which considers the geometric changes. The 
current methods for obtaining these material properties from 
experimental data are either by an analytical analysis or by a 
manual inverse analysis.  
The analytical equations for the “true” stress and strain are 
approximations to the actual “true” stress and strain. On   
Figure 1-1  the precision of the approximation are illustrated, 
the analytically determined stress/strain is processed in 
ABAQUS, and the resulting force/displacement curve is 
compared to the experimental data. The comparison shows a 
clear deviation and for this reason is the inverse analysis 
preferred, when determining the plastic material properties. 
The inverse analysis uses a numerical tool to determine the force-displacement curve from a given set of plastic 
parameters. The force-displacement curve is then compared with the experimental data and the plastic parameters are 
updated based on the fit between these. This process continues until the output from the numerical tool match the 
experimental data. In this report two following subjects are investigated. 
 
 
The possibility of automatizing the inverse analysis is investigated in this document. 
 
 
The preferred test specimens are either flat or round Dogbone specimens. The possibility of yielding similar 
results with a simpler test specimen is investigated in this document.   
The current method for performing an inverse analysis is through manual updating of the material parameters for the 
plastic behavior. This method imply a substantial level of idle time awaiting completion of analysis, before subsequent 
estimate and evaluation of the result can be conducted and the required decision for material parameter update and rerun 
of analysis can be taken. An automatized process would release resources to work on other parts of the project while 
awaiting conclusion of the analyses. Further benefits of an automated progress would be that various alternative 
methods could be applied; leading to an improvement of the results and the program could in addition work throughout 
the night.  
Currently the preferred specimen type for tension tests is the Dogbone specimen. This specimen type is selected since 
the stress is evenly distributed through the cross section at the center of the specimen, which makes it possible to 
analyze analytically. The downside of using this specimen type is that the carving can be time consuming. For this 
reason is a simpler specimen type considered. When performing an inverse analysis with a numerical tool, it is 
unnecessary to know an analytic solution, and it is therefore possible to analyze simpler specimen types with any given 
stress distribution. The requirements for the simplified specimen type are, that the carving of these specimens can be 
performed using basic machinery and that the accuracy is on level with the Dogbone specimens. The exactness is based 
on the similarities of the determined stress-strain curves, and the material is homogeneous and identical, hence the 
acquired stress-strain curves will be identical. The chosen specimen type is a simple Disc with a drilled hole. The results 
from this specimen type will be compared to the common Dogbone specimens, which are analyzed both analytical and 
numerical. The outlining of the two specimen types is shown on Figure 1-2. Furthermore the stability of this simplified

2.1 STATISTICAL ACCURACY 
The statistical analysis hypothesis testing is performed in order to determine the necessary number of experiment 
needed to get accurate results. The analysis consists of four steps.  
Step 1: Formulate the statement to be judged.  
A correct statement, which is significant for the experiment elements, is formulated. This statement is known as the null 
hypothesis 
 
Step 2: Find the experiment statistic.  
The standard deviation related to the parameter formulated in the null hypothesis is a prerequisite for analyzing the null 
hypothesis. This has to be known and obtained by previous experiments. 
 
Step 3: Develop the operating rule.  
In this step, different choices have to be made. 
The first decision is on a level of significance. This level can take values between 0 and 1 - often one of the following 
three values are chosen 0.1, 0.05 and 0.01. The level of significance is a parameter, that determine how far from the null 
hypothesis statement (from step 1) the result can be, to be deemed correct. The level of significance is thereby a 
parameter, which determines the risk of making a type 1 error. A type 1 error is when a true null hypothesis is rejected.  
Then an operation rule has to be selected. This rule determines how the acceptance area around the null hypothesis is 
distributed. Many different operation rules can be chosen, but the three common rules are:  
Rule A: the acceptance area is symmetric around the value in the null hypothesis.  
Rule B: the acceptance areas center is moved to a higher or lower value than the null hypothesis statement. This rule 
can be used if a result, which is lower than the value of the value stated in the null hypothesis, is more critical than if it 
was larger.  
The last rule considered is Rule C. In this rule is the value stated in the null hypothesis considered a limit to the 
acceptance area and the area is distributed on one side of the value. This is an extreme version of Rule B.  
 
Step 4: Apply the operation rule with the experiment statistics.  
In this final step, the operation rule from step 3 is applied to the experiment statistics from step 2. With the known 
standard deviation, the acceptance area can be determined and thereby the accuracy of the experiment, using a given 
number of specimens.

2.3.4 ELEMENT CHOOSING 
When creating an ABAQUS model, it is important to choose elements which fits the given model type. 3D stress 
elements are used for a 3D static test model. These elements have to be specified concerning first- or second-order and 
full- or reduced-integration.  
First- or second-order elements 
The choice of element-order is based on calculation time and accuracy for the given model. The calculation time is 
based on the number of nodes used for the model. All element shapes are available in both first and second order. An 
illustration of the elements and their node positions are shown on Figure 2-6. 
 
FIGURE 2-6 – FIRST AND SECOND ORDER ELEMENTS 
The first-order quadratic elements, C3D8, used eight nodes located in each corner. With these elements, is the strain  
modeled as constant through the elements. Due to the constant strain, an increased number of elements can be required 
for accurate results.  
 
The second-order quadratic elements, C3D20, require 20 nodes per element - an additional node between each node. 
These elements can model a linear change in strain, which is useful with bending-dominated problems. The chosen 
elements should be the elements, which require the least amount of nodes to fulfill the accuracy requirements.     
Full- or reduced-integration elements 
Like with the order of the elements is, the choice of whether to use full- or reduced-integration,  based on which method 
yield accurate results with the shortest calculation time. The reduce-integration elements, C3D20R, use eight integration 
points, where the fully integrated elements use 27 points. Besides the difference in calculation and accuracy, each type

has their advantages and disadvantages. The reduced-integration elements can suffer from “Hourglassing” and the fully

Hourglassing 
Hourglassing is an issue, which can occur when using first-order, reduced-integration elements. These elements only 
have one integration point, which allows the elements to distort in an hourglass like shape, as shown on Figure 2-7, 
which leads to zero strain calculations at the integration points. In ABAQUS this issue is reduces by an hourglass 
control function, this function does not remove the issue, only reduce the risk of occurrence. A reasonably fine mesh 
should be used to reduce the risk further. This issue occurs in models subjected to bending, and if this issue occurs in 
the model, then counteract would either be increase of the mesh refinement or by switching to the fully integrated 
elements. 
 
FIGURE 2-7 – EXAMPLE OF HOURGLASSING 
Shear and volumetric locking 
The fully integrated elements can suffer from a locking behavior. The first-order fully integrated elements risk shear 
locking when subjected to bending. This is caused by the numerical formulation, which can introduce non-existing 
shear strains. Elements subjected to shear locking will have an increased stiffness in bending and therefore faulty 
results. 
The occurrence of volumetric locking is based on the material properties. If the material is incompressible, fake 
pressure stresses develop at the integrations points, which will cause the element to behave too stiff. This is common for 
elastic-plastic materials, when second-order fully integrated elements are used. This can often be avoided by increasing 
the mesh refinement at the regions where large plastic strains occur.

considered converged and thereby stable, when the result no longer changes when the mesh is increased.

Discontinuities 
The discontinuities in a FEM analysis are the difference in the requested parameter, at a single node from the 
surrounding elements. Small discontinuities are equivalent to similar results at each element and are thereby a good 
indicator for the precision of the model results. As a rule of thumb, the discontinuities should not exceed 5%.

correct.

the cross section are constant. This position is decided to be a distance equal to the width away from the load position.

the force is applied.

3 EXPERIMENTS 
3.1 OVERVIEW 
Two different specimen types have been chosen for the experiments, the specimens are illustrated on Figure 3-1. The 
conventional Dogbone specimen, this type is commonly preferred due to evenly distributed stress across the cross 
section, which makes it possible to analyze analytical. The downside of the specimen type is that the shape is complex 
and time consuming to cut out and in addition require special machinery for preparing the required quantity of 
specimens within available timeframe. The second specimen type is a simple Disc with a drilled hole. This type has an 
uneven stress distribution at the yielding position, thus require numerical tools to analyze. However, the production of 
the specimen is a lot simpler than the Dogbone specimen, thus faster to produce. 
 
 
FIGURE 3-1 – ILLUSTRATION OF THE TWO SPECIMEN TYPES 
 
The argument for selecting these two specimen types are that the Dogbone specimen can be analyzed both numerical 
and analytical and the precision of the numerical analysis is thereby validated, while the Disc with a hole specimen is 
chosen due to its simplicity and outcome can, with the results from the Dogbone specimens, be validated by 
comparison.  
It is chosen to use two different steel classes for both specimen types, s235 and s355, to ensure that the numerical 
analysis are not case specific and that the steel class is irrelevant for the analysis precision. It is determined in section 
3.2, that 10 of each specimen type are required for statistical accuracy. With four different specimens, this results in 40 
specimens in total. 
In order to secure the stability of the analysis, regarding the Disc with a hole specimen, it is chosen to use two different 
hole diameters. The chosen diameters are 8mm and 12mm, which is picked according to the investigations, performed 
in section 3.3.2. Furthermore the stability of the analysis with regards to the eccentricity of the hole is investigate by 
introducing a noticeable eccentricity in 1 of each hole diameter for each steel class. 
The majority of the experiments are performed using the general method, which is fast and has an easy setup, while it 
lack local behavior results in the plastic phase. A few initial tests using ARAMIS which provide very detailed results, 
but the setup is more complicated and time consuming. Initial intention to perform the experiments using the two 
different methods, was adjusted in preliminary phase and the ARAMIS method abandoned due to broken test machines 
and time-constraints. Descriptions of the two methods can be found in section 3.4, 0 and 3.6, and ARAMIS descriptions 
are moved to Appendix A – ARAMIS description as all performed experiments included in this report use the general 
method.  
It is chosen to use two different displacement rates during the experiments. A low displacement rate during the elastic 
phase, this both protects the extensometer and yields more detailed results in this phase. For the plastic phase is a high 
displacement rate used. Large deformations with little variation occurs during this phase, thus detailed data sampling is

3.4.1 GENERAL METHOD 
The general experiment method tracks the displacements from the piston movement, and the applied force from a load 
cell. Furthermore, strain gauges and/or extensometers are used to track the local deformation. In order to track useful 
local deformations, it is necessary to have a clearly defined position to measure the deformation.   
Using the general method has advantages, but also disadvantages. The predominant advantages are the fast experiment 
procedure due to the simple setup, and precise results in the elastic phase. The disadvantages are that it only provides 
longitudinal deformations. The plastic phase is in addition not precise since the extensometer gives the average 
extension over a certain length and has to be un-mounted for large deformations.

3.6.2 GENERAL METHOD 
1. Setup the program to a slow and fast displacement rate; use the slow to avoid damaging the extensometer. 
2. Mount the correct clamps and insert the specimen 
3. Mount the extensometer on the specimen, the extensometer is mounted at a position, where the stress/force

4. Start the specified loading program 
5. Wait for yielding to start 
6. Remove the extensometer to avoid damage 
7. Change to the fast displacement rate 
8. Wait for failure 
9. Measure the cross sectional area of the failure point, connect the two parts of the specimen and measure the

4.1 PROGRAMS 
4.1.1 SPECIALIZATION 
When creating a script, it is important to use programs, which are designed to solve the specific problem. For this 
particular script there are multiple problems; FEM, programming and mathematics. The FEM part is handled by 
ABAQUS, the programming is mainly handled by Python and the mathematics is handled by Matlab. 
4.1.2 HOW THEY WORK TOGETHER  
Different programs, which run separately, are used and they need to be able to corporate. None of the variables and 
output data can be directly transferred to the other program. It is therefore necessary to integrate via text-files, and dump 
the output into a text document, which is then imported and interpreted by the other programs. Via this method, the 
input data from Matlab can be read by Python and used in the FEM analysis, while output data from Python 
subsequently can be read and analyzed in Matlab, which then delivers new input data.    
4.2 GENERALIZATION 
In order to automatize the process, everything in the ABAQUS model has to be generalized and specified based on 
input variables. The first step is to create a dummy model. This model contains all the elements of the final model, but 
everything is model specific. Then each part of the model script is reviewed and generalized. This is done by 
introducing variables like height, width and thickness, which is used to define the model. If e.g. a box is considered, 
then this box will have random numbers defining each corner in the dummy model and these numbers are then 
substituted by the variables. Furthermore the box can be loaded with a pressure on a surface. In the dummy model, this 
surface will be identified by a number, which is not constant and might change when the model changes. It is therefore 
necessary to assign a name to the surface, identified by a find command. This command finds the surface at specific 
coordinates defined by the introduced variables. This is done for each surface, edge and node used for the model.  
Subsequently should all of the parameters like Young‟s modulus, be substituted with a variable name. This makes it 
possible to alter if necessary. With all of this done, it is possible to alter the model as desired by changing the input 
variables, but it is only applicable for box models. In order to make other model type, the same method has to be 
applied to that dummy model.

5 POST-EXPERIMENT THEORY 
5.1 MATERIAL PARAMETERS 
5.1.1 YOUNG‟S MODULUS 
The Young‟s modulus defines the stress-strain slope for the elastic part. From the experiments can, the stress-strain 
relation be derived from the piston movement and the applied force, but the piston movement consists of two parts: 
deformation of the specimen and of the test machine. It is therefore necessary to measure the local deformations during 
the experiment to determine the Young‟s modulus. The experimental data should then be altered to resemble the 
determined Young‟s modulus and thereby removing the deformation of the test machine from the piston movement. 
The Young‟s modulus is thereby determined from the strain, measured at a position, where the relation between the 
force and stress is known. 
The experimental data is then calibrated to fit the Young‟s modulus. This is done in three steps: 
 
Find the difference between the displacement at the proportionality limit (see section 5.1.2) from ABAQUS 
simulation with the new E-modulus and the experiment data. 
 
Calibrate data below the proportionality limit with a linearly increasing difference from zero force to the 
proportionality limit. 
 
Calibrate data above the proportionality limit with the difference at the proportionality limit. 
5.1.2 PROPORTIONAL LIMIT 
According to (David Roylance u.d.), can the proportional limit is defined as the stress where slope on the 
force/displacement curve deviates from the linearity. In order to determine the proportional limit from the experimental 
data, is the position on the force/displacement curve, where the first significant deviation from the elastic path occurs, 
defined. The stress derived at this position defines the proportional limit.  
In order to automatically determine the proportional limit, a methodology has to be composed. It is chosen to adopt the 
principles of the inverse analysis. In order to make it work, then only a small part of the plastic phase is going to be 
included in the calculation of the RMSE. The position of the proportional limit is approximated by determine the 
position on the force/displacement plot, where the first significant change in inclination occurs. The smaller the part of 
the included plastic phase is, the more accurate the proportional limit can be determined, since the proportional limit 
position is approximated. The part above the proportional limit positions, which is included in the analysis, should be 
large enough to make sure in includes the plastic part.   
5.1.3 YIELD STRESS 
The yield stress is determined as the 0.2 % strain offset method. This method define the yield stress, as the stress, where 
a line from 0.2 % strain and an inclination equal to the Young‟s modulus, intersects with the stress-strain curve.  
5.1.4 ULTIMATE STRESS 
The ultimate stress is the highest stress that can be carried by the material. During which phase the ultimate stress is 
found depends on whether the engineering or “true” stress is considered, as described in section 5.2.1. When 
considering the engineering stress, the ultimate stress defines the initiation of the necking phase. When necking occurs, 
the cross section drastically decreases in a single area, which reduces the load bearing capacity. The engineering stress 
is based on the original area and thereby the load bearing capacity. The stress will therefore peak at the initiation of the 
necking phase. When the “true” stress is considered, then the ultimate stress occurs just prior to fracture. In reality does 
the hardening continue after necking and the only decrease in strength is caused by fracture. In both cases, the ultimate 
stress is determined as the largest derived stress.

In order to determine the material parameters, the trial and error approach is used. An estimated guess of material 
parameters, based on the experiment results, are used in the numerical analysis and the results are compared to the 
experiment results. Based on the error between the numerical analysis and the experimental results, the material 
parameters are updated and another analysis is performed. This process is repeated until a satisfying result is acquired. 
A satisfying result can be acquired in a low number of tries, when the analysis is based on the initial geometry, since 
there is a linear relation. When the change in geometry is taken into account, then the relation becomes nonlinear, and 
the number of tries, needed to acquire a satisfying result, increases. Due to the repetitive process and computational 
requirements of the numerical model, this is a resource and time-consuming process. 
Automatic process 
In order to make the process automatic, it is necessary to make a script, which can compare the numerical results with 
the experimental results. The parameter update is based on human assessments in the manual process and it is therefore 
necessary to formulate a methodology, which can be used to update the parameter.  
Compare results 
The result comparison is based on the Root mean squared error method (RMSE) which is described with equation (5-9). 
This is a common and proven method for model verifications. The method differentiates between small and large errors, 
giving a few large errors a greater impact than multiple small errors. The deviation between small and large errors 
secure that the best fit is the one following the overall behavior of the experimental results. This can pose an issue when 
comparing to experimental results, since the results can contain local spikes, which will be excluded in the numerical 
model, where these spikes can be dominating the RMSE result. In order to avoid this, certain repercussions can be 
taken, either by altering the experimental results and hereby remove the spike or to exclude large local error in the 
summation.

Basis of the parameter update 
The parameter update is based on the built-in MATLAB function “fminsearch” this function finds the minimum value 
of a given objective function. The advantage of using “fminsearch” is that it can minimize the objective function based 
on multiple input parameters. Creating an objective function, with stress-strain relations at given strain positions as the 
input parameters and the RMSE as output data, allows the “fminsearch” function to determine the optimal strain/strain 
relations and the given strain positions. In order to get good results, the chosen strain positions are crucial and should be 
chosen where the significant changes or peaks occur in the stress-strain relation. This method is time consuming since 
each function evaluation requires a full ABAQUS analysis. In order to reduce the calculation time, the number of 
iterations required, should be reduced. This is done by optimizing the initial estimated guess.

Optimization variables 
The speed and accuracy of the analysis is dependent on the chosen variables. Each variable increase the number of 
function evaluations per iterations and thereby has a large influence on the speed of the analysis. For this reason, is it 
necessary to decrease the number of variable while maintaining a suitable accuracy. The following method is used for 
this inverse analysis and other methods will be discussed in section 7.  
Point method: this method is based on the method for the manual inverse analysis, where various points are chosen on 
the stress-strain curve to describe the entire curve. When this method is used, then one variable are used for each point 
where the stress is varying at the chosen strain point. Using this method requires a reasonable amount of variables, but 
the points are locked at the strain points that are chosen based on a general stress-strain curve. This can have a 
significant influence on the accuracy, if the experimental data differs too much from the general stress-strain curve. The 
strain points can be released by including them as variables in order to overcome this issue. As a result, each point will 
contribute with two variables, which will double the number of variables and thereby greatly increase the calculation 
time. This solution has a good accuracy, but a long calculation time. In order to keep some flexibility in the strain point, 
without adding too many variables, the point can be controlled by a single variable. This method is a good combination 
of the two previous methods. The flip side of this method is that the strain points are locked together and cannot find the 
optimal position for the individual point. The combined method is chosen for the inverse analysis due to its reasonable 
amount of variables and flexibility in strain points, which increase the accuracy.

7 FUTURE WORK 
This section contains ideas for how to improve the automatic inverse analysis. The ideas are focused on analysis speed 
and accuracy and how to improve the user friendliness. None of the following ideas have been implemented or tested. 
They are however based on the observations made throughout the creation and testing of the program scripts. 
7.1 ANALYSIS SPEED AND ACCURACY 
With the current method of describing the stress-strain curve, the only way of increasing the accuracy is to increase the 
number of points or lowering the tolerance for the “fminsearch” function. Both of these methods would have a huge 
negative impact on the analysis speed. Furthermore were various issues with the current method discovered during 
processing of the results! Different methods of describing the stress-strain curve are for this reason considered and 
described in the following section.   
Known shape 
A known and well-described shape, which should fit the experimental results, are used for this method. This shape is 
then corrected to fit the given experiment with the use of three variables: strain direction multiplier, stress direction 
multiplier and a rotator. These variables should work as shown on Figure 7-1, where the red line indicates the known 
shape, the green is affected by the strain direction multiplier, the blue is affected by the stress direction multiplier and 
the cyan is affected by the rotation.  
 
 
 
FIGURE 7-1 – SHOWING THE EFFECT OF THE THREE MULTIPLERS IN THE KNOWN SHAPE METHOD 
This method can fit to any shape with the use of only three variables, which leads to a fast calculation time. The 
downside is that the base shape has to be well known and described.  
Shape combination 
This method uses various shapes to describe the stress-strain curve. If the stress-strain curve from the manual analysis 
of the s235 ”true” stress-strain are considered. The first part of the plastic phase can be considered to follow an arch, 
which can be controlled by one variable - the radius. The next part is considered linear and could be modeled according 
to an inclination. This is shown as a principle drawing on Figure 7-2, where the red lines indicates the shapes used to

Stepwise method 
This method is inspired by the Newton‟s method, described in section 2.3.3. The curve is divided into steps, and a line,

then the program continues to the next step using the inclination of the linear line on the stress-strain curve for the 
previous step, according to the initial estimated guess. The progress of this method is shown as a principle drawing on

guess=[c * x(2:end).^n];

9 CONCLUSION 
 
The possibility of automatizing the inverse analysis was investigated. By composing a method that allows Matlab and 
ABAQUS to collaborate, it was possible to update the material parameters, which is the core of the inverse analysis. 
The methodology used for the parameter update was based on the method used for the manual inverse analysis. This 
method was proven practicable and it was documented, that the program was able to generate the material parameters 
that could replicate the major part of the experimental data. However, the method has challenges and issues, which need 
to be addressed: 
 
The necking phase is difficult to replicate with only a few point. The strain point did not sufficiently fit the 
data to fit the necking phase for some experiments. This was a complication for the engineering analyses. 
 
For cases, where the majority of the curve fit the experimental data and a small part is far from the 
experimental data, here is the improvement per iteration below the tolerance and the update is stopped. This, 
even though a part of the curve, does not fit the experimental data. 
 
The analysis has a long calculation time. The accuracy has to be reduced, in order to have a reasonable 
calculation time. 
The method for material updating has to be optimized, to address these issues. Three alternative methods have been 
presented:  
- 
The known shape method, which uses three multiplier, x-axis, y-axis and rotation, to alter a known shape to fit 
the experimental data.  
- 
The shape method, which combines geometric shapes to define the shape of the stress/strain curve.  
- 
The stepwise method, which divides the force/displacement curve into steps and determine one step at the 
time. 
 Each method has the potential to solve previously stated issues. The stepwise method is considered the most promising, 
as it does not require any previous knowledge of the stress/strain curve. However, the calculation time for this method 
can be an influencing factor, as the number of variables is not reduced.  
 
The possibility of using a simplified specimen type was investigated. It was chosen to use a Disc with a hole as the 
simplified specimen. This type has an easy manufacturing process that did not require special machinery. In order to 
validate the use of the simplified specimen, the experiments were analyzed using the automatic inverse analysis. The 
results from the analysis were compared to both the analytic results and the results, yielded from the inverse analysis of 
the Dogbone specimen. This comparison showed coherence between the results and the use of the simplified specimen 
and is therefore valid. Furthermore was the stability of the geometry of the simplified specimen investigated, 
concerning eccentricity of the hole and radius of the hole. The resulting stress/strain curves from the eccentric and 
centric hole placement was compared. This comparison showed that the eccentricity of the hole did not influence the 
results. When the specimens with the two different hole radiuses where compared, was a difference in the results 
discovered. The difference is deemed to be cause by the model used for the ABAQUS analysis. The current model is 
based on and verified for the 4 mm radius hole. Further investigations are required to confirm the origin of the 
difference.  
During the creation of the program, which performs the automatic inverse analysis, the user friendliness was included as 
a success criterion. All of the user options are located in one script. Various verification figures are printed during the 
analyses and all of the result files from the given experiment are located in a separate folder. The combination of these 
measures and a user manual has resulted in a user friendly program. Furthermore is the processing performed in Matlab, 
which makes is easy for the user to implement additional features to the program.  
The overall conclusion is that it was possible to create a program which was able to automatically perform an inverse 
analysis, on any given specimen. With the easy accessibly codes the possibilities for optimization could lead to a 
program which very accurately determine the “true” material properties for elasto-plastic material

11 APPENDIX A – ARAMIS DESCRIPTION 
 
Experiment method 
The experiment method using ARAMIS is similar to the General method, but instead of using strain gauges and 
extensometers to track the local displacements, ARAMIS uses digital images to track the behavior of the specimen. In 
order to track the behavior, the specimens need to be painted in a distinct manner. This is done using white with 
unevenly distributed black dots. The progress of the experiment is then recorded with a digital camera, whereupon the 
images from the camera can be analyzed to gain a complete picture of the specimens‟ behavior. 
Using the ARAMIS method has huge advantages and disadvantages. The data received are relatively easy to extract. It 
is not necessary to know the specific position of the failure, as is required with strain gauges. It can capture non-uniform 
full-field deformations. If a large number of specimens are going to be tested, it is relatively fast, but if only a few 
specimens are tested, the setup and calibration is time consuming compared to other methods. Consistent lighting is 
necessary. 
 
Experiment setup 
The setup for this experiment consists of two parts. The first part is the machine that loads the specimens and is 
connected to a computer, which then controls the loading and logs, the applied load and the resulting displacement. The 
second part is the ARAMIS system, consisting of two cameras that points at the specimen as shown on Figure 11-1. 
These cameras are connected to the computer that controls the ARAMIS program. The dashed box around the specimen 
shows the area of which the ARAMIS program can measure displacements. 
 
 
FIGURE 11-1 – ARAMIS CAMERA SETUP FROM (GOM 2005) 
A facet size with a small overlap has to be chosen for the program setup. The facet is a section of the picture and each 
picture is divided into facets of a given size. The facets are set to overlap with the chosen number of pixels. This makes 
it possible for the program to connect the facets in the correct manner.

Experiment Procedure 
1. Setup the tripod with the two cameras mounted on it, follow the specifications of the precise arrangement of 
the cameras with respect to; distance to the specimen, distance between cameras, angle of cameras and 
pointing perpendicular to the specimen. 
2. Paint the specimens, first the specimens are spray painted white, then small black dots are sprayed onto the 
specimens, it is important to have an even ratio between black and white and to have no large areas of a 
specific color. 
3. Calibrate the ARAMIS, for this the calibration plates are used. These plates are placed at the position of the 
specimens and turned around in numerous positions. This is done so that the program is absolutely certain of 
the position and angle of each camera. 
4. Setup the program to a specific displacement rate. 
5. Place the specimen in the machine. 
6. Verify that ARAMIS has a clear view of the specimen 
7. Start measuring with ARAMIS. 
8. Start the loading procedure. 
9. Wait for failure to occur after which the measuring and loading is stopped. Post process the images in 
ARAMIS and extract the desired results

function  [uPL,FPL]=find_uPL(expcurve,Modelfile,expnumber) 
     
uinc=0.00001; % set strain increment 
% start at 0.02% strain to avoid unwanted breaks at low strain 
n=0.0002/uinc; 
FPL=expcurve((n)*uinc); % calculate force at initial strain increment 
ratio=expcurve(n*uinc)/(n*uinc); % calculate initial ratio 
% the while loop breaks when the next point deviates more than 5% from the 
% propotionality line and thereby indicates the limit 
while FPL>n*uinc*0.95*ratio 
    n=n+1; 
    ratio=sum(expcurve((1:n-1)*uinc))/sum((1:n-1)*uinc); 
    FPL=expcurve(n*uinc); 
end 
uPL=n*uinc;

13 APPENDIX C – RESULT FIGURES 
This appendix contains the stress-strain curves and best-fit figures from each experiment. The figures are represented 
both with and without geometric changes. The numbering of the specimens refers to the number on the specimen tables 
given is section 3.7. 
Best-fit figures 
For some of the best-fit figures is the original experimental curve represented rather than the calibrated curve. This is 
purely a graphical error. If the elastic phase does not coincide, then this is the reason. 
Stress-strain curves 
For some of the stress-strain curves was the fitmethod “smootheningspline” used. This was a leftover from earlier script 
testing and was changed when discovered. The issue is purely graphical, since all calculations are based on the points, 
thus it is chosen not to perform new analyses to correct the issue.   
The stress-strain curves are based on the stress-strain points. These points can exceed the maximum strain leading to 
situations, where these point does not influence the analysis. This is common for the engineering stress-strain curve, and 
the curves has, for this reason, to be reevaluated to correctly represent the stress-strain curve.

    Af=MP3(:,10)*1e-6; 
    MP=MP*1e-3; %convert to m from mm 
    % for expnumber=5 
    for expnumber=1:size(MP,1) 
        filename=[num2str(steeltype) '_' num2str(expnumber) '.csv']; 
        %     filename=['b1-h1-1fx1.csv']; 
        % read the files using the real_files script 
        [pistonmovement,Load,strain]=read_files(filename); 
        stop 
        for i=1:length(MN) 
            MN2{i}={num2str(MN{i})}; 
        end 
        % find and define the relevant specimen parameters 
        idx = find(strcmp([MN2{:}], 'Lc')); 
        Lc=MP(expnumber,idx); 
        idx = find(strcmp([MN2{:}], 'Wc')); 
        Wc=MP(expnumber,idx); 
        idx = find(strcmp([MN2{:}], 't')); 
        t=MP(expnumber,idx); 
        % define the original cross section 
        A0=t*Wc; 
        % calculate the engineering stresses and strains 
        sigma_eng=Load/A0; 
        eps_eng=pistonmovement/Lc; 
%         nu=0.5; 
        % calculate the ”true” stresses and strains 
        sigma_true=sigma_eng.*(1+eps_eng); 
        eps_true=log((Lc+pistonmovement)/Lc); 
         
        %% fracture 
        % determine the frature stresses and strain from the measures 
        % fracture area 
        sigma_frac=(max(Load)/Af(expnumber)); 
        eps_frac=log(A0/Af(expnumber)); 
        % find the position of necking 
        [~,index]=max(Load); 
        % substitude the post necking part with a linear line between 
        % necking and fracture 
        sigma_true=vertcat(sigma_true(1:index),sigma_frac); 
        eps_true=vertcat(eps_true(1:index),eps_frac);
