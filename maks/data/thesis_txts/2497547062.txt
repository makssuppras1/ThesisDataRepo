Summary (English)
The Danish strategy for reaching zero carbon emission is based on variable renew-
able energy. The variability of power production makes it benecial to introduce
energy exibility into systems consuming power. Allowing demand to follow the
supply instead of supply matching demand.
This thesis investigates the use of
predictive control to estimate the energy exibility possible using storage capacity
within sewer systems.
For this purpose is a model predictive control algorithm developed which considers
the cost of electricity as a second objective. Where the main objective is prevent-
ing waste water overow.
The model developed for the algorithm is a lumped
greybox model, describing the waste water ow within the sewer system using a
small number of dierential equations, where the description of precipitation in-
clude stochastic elements.
With the aim of using the model to investigate dierent sewer systems is a strategy
for adapting the model to historical data formulated. Such that an estimate of
obtainable energy exibility can be obtained by simulating the system with the
developed model predictive control algorithm

Introduction
The Danish strategy for reaching zero carbon emission include a lot of variable
renewable energy, such as wind turbines [3].
In 2019 47% of power came from
wind turbines [4]. The challenge of variable energy is the lack of control over when
electricity is produced. This makes it benecial to introduce energy exibility into
systems, using electricity when it is available. Allowing demand to follow the supply
instead of supply matching demand.
The focus in this thesis is energy exibility controlling waste water storage within
sewer systems.
The main purpose of this storage is to prevent overow due to
rain. Here emptying the storage as fast as possible ensures it is ready for the next
rainfall. However, when the weather forecast is clear there is no urgency to empty
the storage leaving room for considering exibility in energy consumption.
The aim is to create a framework that can be applied to any sewer system to es-
timate obtainable energy exibility. To accomplish is a model predictive control
algorithm that considers energy exibility developed, together with a lumped model
describing the waste water within sewer systems. This allow control considering
energy exibility to be simulated. To encapsulate a specic system are Greybox
modelling tools presented for adapting the model to the historical data using pre-
possessing and maximum likelihood estimation with the toolbox CTSM-R.
Greybox modelling is dependent on the data available, so the tools presented are

Sewer System Model
In the simplest form, a combined sewer system consists of sewer pipes transporting
waste water from collection to a waste water treatment plant by gravity.
This
system is often expanded with pumping stations, as there is a limit to how deep in
the ground it makes sense to place sewer pipes. These lift the waste water, so it
can continue to ow by gravity just under the surface.
Overow channels are included to improve on the system. These act as safety valves
when system capacity is reached, leading waste water out untreated in a controlled
manor to avoid damage by ooding.
To minimize the need for overow, storage can be added to the system.
This
creates a buer for the connected pumping station,such that overow isn't required
to prevent ooding before storage capacity is reached. The storage can take the
form of basins and storm drains. Storm drains are generally larger pipes placed
next to regular sewer pipes, with connection at the top. This allow waste water
to ow into the storm drains just before the sewer pipes reach maximum capacity,
both minimizing the risk of ooding and increasing storage.
The combination of pumpings station and storage is what allows control over the
system, the buer gives the option of allowing ow towards the treatment plant or
not, without causing overow, creating a point of control in the system. These are
the nodes where energy exibility can be a consideration by storing waste water.

Greybox modelling
To obtain an estimate of energy exibility for a specic sewer system. The models
needs to be adapted to the system. This is done by creating a model using the
sewer system structure, then adapting the model to historical data, by maximizing
the likelihood that the data could be a realization created using the model. Two
methods are used for the maximum likelihood estimation, the toolbox CTSM-R
and Least squares method.
To ensure the parts of the model describe the trends in the data they where designed
to describe, require minimizing the eect of the marginally stable volume balance
model and to isolate the trends in the data, such that the model can be estimated
in steps.
The prediction error is not expected to be symmetrical around zero,
as the model is a lumped model. So errors would accumulate in the marginally
stable volume balance model. This leads to a bias in the estimate for describing
the short term prediction.
However, the volume balance does not contain any
parameters, so the dierential equation can be used to estimate the inow as part
of the preprocessing of the data.

3.1

Model predictive control
Model predictive control is used to investigate the exibility of the sewer system.
This type of control work by turning the task of controlling the system into an
optimization problem. A system response model is used to describe the eect of
any control action, so controlling the system becomes the optimization problem of
selecting the control actions that t the response model best.
As change isn't instant, the best control action at the present is dependent on the
future. To account for this, the optimization problem is extended to determine the
best sequence of control actions for a prediction horizon, then use the rst of these
actions. 'Best' is quantied by a cost function that describe any control sequence
with a single number, such that poor control is penalized with a high number.
The minimum of the cost function therefore result in the optimal control sequence,
assuming the cost function represent the system perfectly.
A receding horizon is used, which have a constant prediction length, as the end
point of the control sequence change with the starting point. So when new data
becomes available, the optimization problem stays the same except for the state of
the system and disturbances. Allowing the control strategy to be described by the
algorithm:
1. Obtain new data.
2. Adapt model to new data.
==PAGE:40==
28
Model predictive control
3. Predict disturbance for horizon.
4. Solve optimization problem to obtain control sequence
5. Apply rst control action in sequence to system
6. Wait for new data.
The term disturbance is dened as any part of the system that are not aected by
control, i,e,. Runo from precipitation and sewage ow
Both step 2 and 3 of the algorithm revolves around making the model ready to
be used to solve the optimization problem. Adapting the states to new data en-
sures that the initial state used in the prediction is as close to the real system as
possible, while predicting the disturbances to the system beforehand simplies the
optimization as these predictions does not change with the control sequence. This
also means that any limitation put on the response model does not apply to the
description of the disturbances. As the model is tted with CTSM-R the ltering
and prediction supplied can be used to obtain both.
The challenge using this type of control is the optimization problem. Robust and
fast control is more important than the optimal control sequence is found, as per-
fect information is not available. Without robustness, the implementation might
work one moment but not the next, making the results unreliable [10]. To ensure
robustness, the control problem is constrain to convex programming as robust and
fast algorithms are readily available.

Convex Programming is optimization on a convex set, which are sets where any
point on a line between two arbitrary points in the set is also in the set. For a
linear system response model, the convex set requirement mean equality constraints
must be linear on the variables and inequality constraints must be concave on the
variables. Plus the cost function must be convex [7].
A convex function satises the triangle inequality,
f(αx1 + (1 −α)x2) ≤αf(x1) + (1 −α)f(x2),
∀α ∈[0, 1].
(4.6)
Where a function f is concave if -f is convex.
The system response model (4.5) does not have any equality constraints. However,
the eect constraints on the volume changes shape depending on the state of the
system. So they are not necessarily concave at all times. This is solved by softening
the these constraints in section 4.2.2.

Chapter 6
Discussion and Conclusion
The aim of the thesis was to develop a framework that can be applied to any sewer
system to estimate obtainable energy exibility. To this end has a exible model
been developed to describe sewer structure. This structure t any sewer system that
satises the assumption of each node having one connection downstream. However,
there might exists sewer systems where pumping stations, connected to same stor-
age capacity, have dierent routes downstream to the waste water treatment plant.
Cases such are not encompassed in the design of the model of sewer structure.
A strategy for using historical data to adapt the node model and connections was
formulated. Due to quality of data, was the t of the model of ow between nodes
evaluated. However, the model of the node structure was adapted to data and
the eect of the prediction error on control investigated. Here it was found that
using the prediction of inow lead to a negligible dierence in the simulation when
compared to control where the disturbance is assumed known. This assume the
t of Amager west node is representative and it does not include errors due to
predictions of the precipitation, the data are radar measurements, not prediction
of precipitation up to a day forward in time. While this source of error has not
been included, the simulations included a large inow not explain in the precipita-
tion data, which lad to a signicant prediction error, this did not lead signicant
dierence response in the control algorithm.
The control algorithm was shown to functions as designed for controlling a single
==PAGE:70==

node. Allowing simulation of the system, where energy exibility is a consideration
when the system constraints allows it. However, the design does not include any
consideration of waste water storage composition, only the volume of inow. So in
dry weather conditions the algorithm will stile store waste water when the cost is
high. This limits the use of the control algorithm to sewer systems where this is
not a problem or simulations using data with enough precipitation for it to not be a

Based on these considerations it can be concluded that there is room for improving

without the ability to simulate the system over an extended period of time, little
faith can be put into the estimated reduction in cost.
