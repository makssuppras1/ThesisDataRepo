Abstract
The concept of maximum sustainable yield is well known in the world of fisheries
science. It tells the biomass state where the surplus production of a species is maxi-
mized. This surplus can be harvested and the biomass state returns to the maximum
sustainable yield state and will produce the same surplus again, creating an optimal
harvest policy. This project aims to develop an optimal simultaneous harvesting pol-
icy while also considering stochastic fluctuations. First, an optimal harvest policy
was generated for a stochastic logistic growth model, also known as the single species
model. The optimal harvesting policy is generated by solving the Hamilton-Jacobi-
Bellman equation, which is derived from the single species model.
A discounting
factor was added to the single species model to see if there are any notable changes
in the policy. A stochastic Rosenzweig-MacArthur model with harvest was used as
the predator-prey system. The optimal harvesting policy generated for the system
will also be found by solving the Hamilton-Jacobi-Bellman equation. It was solved
with backstepping using a partial differential equation solver called FEniCs. Once
the method to generate an optimal harvesting policy for the stochastic Rosenzweig-
MacArthur system is derived, several test cases were conducted to see how the optimal
harvesting policy would work in different situations. The initial results show that it is
possible to generate a harvesting policy for the cases. The harvesting policy considers
the noise in the cases by not harvesting all the surplus that is produced, therefore
allowing some uncertainty. In most cases the prey survives and is harvested, while
in some cases the predator becomes extinct, which could be interpreted as it is not
always optimal to let the predator survive.

CHAPTER 1
Introduction
In a world where nearly every renewable resource is exploited, a sustainable harvest-
ing policy is vital for the resource to not become eradicated and consequently not
profitable for the harvester.
This is directly related to the maximum sustainable
yield.
In fisheries science the maximum sustainable yield (MSY) is a classic concept. The
goal of the MSY is to keep the stock at the biomass level where the surplus production
(the yield) of the stock is maximized in the long term. This surplus can then be
harvested continuously without affecting the future surplus production(ICES 2018a).
Schaefer (1957) showed that for a logistic growth equation the maximum sustainable
yield would be obtained if the biomass of the stock was equal to half of the carrying
capacity (the maximum sustainable biomass level). Harvesting the surplus produced
at the MSY state would keep the system constantly in the MSY state. This is a good
example of a sustainable harvesting policy. The MSY is however not automatically
the optimal harvest state for the system, that is the state that would generated
the highest profit. Schaefer (1957) shows this by introducing economic factors that
connect to the harvest, such as the profit from the catch or cost of the catch. He
then uses these economic factors to find the optimal harvesting effort, which will
maximize the profit for the system. The optimal harvesting effort is a constant effort
that removes a fraction of the biomass from the system and will eventually bring the
system to the state where the profit is maximized and will stabilise there.
Using an constant harvesting effort policy works well for a deterministic model,
the surplus is always known in each state.
Bousquet et al. (2008) states that in
the past, models that estimated the MSY for real life species, ignored environmental
uncertainties which in turn led to an overoptimistic harvest decision and caused stock
collapses. Like Ewald and Wang (2010) states, this is caused by the fact that these
species do not grow according to a deterministic model. Hoggarth et al. (2006) states
that a stochastic varying term should introduce uncertainty to the surplus model1.
This can be interpreted as introducing environmental noise to the model, transforming
it to a stochastic model. The surplus model becomes more complex and it is not
possible to derive the MSY using the same method that is used for the deterministic
model.
The MSY biomass state, as well as the optimal harvesting effort for the
stochastic model has been found in Bousquet et al. (2008). Previous studies have
focused on constant effort strategies, that is the constant effort that removes a fixed

biomass from the system. This is highly restrictive, one should be able to exploit
resource more if the abundance was relatively high and in the same sense it should
be exploited less if the abundance is relatively low.
Thygesen (2018) has developed an optimal harvesting strategy for the stochastic
logistic growth equation. The harvesting policy is dependent on the state of the sys-

less if the abundance is low. The policy is determined by using the Hamilton-Jacobi-

expected future profit of the system while it assumes that the control (harvest) is
optimal. Then the value function can be used to compute the optimal policy. The
Hamilton-Jacobi-Bellman is solved dynamically.
In reality almost all fisheries are predator-prey system. Knowing that, it is some-
what surprising that the single species situation is so extensively used and considered.
A good example of predator-prey model is the Rosenzweig-MacArthur model, a mod-
ified version of the well known Lotka-Volterra system (Thygesen 2019). A harvesting
policy for the stochastic system has not been explored in detail and for this reason
it is worth to examine further. The addition of stochastic environmental noise to
the Rosenzweig-MacArthur model introduces uncertainty to the model. Like in the
stochastic logistic growth model generating an optimal simultaneous harvesting pol-
icy for the stochastic Rosenzweig-MacArthur model should be possible to attain by
solving the Hamilton-Jacobi-Bellman equation for the system.
1.1
Project Overview
The main objective for this project is to develop a way to generate an optimal si-
multaneous harvest strategy for the stochastic predator-prey system, in this case,
the stochastic Rosenzweig-MacArthur system. The Hamilton-Jacobi-Bellman equa-
tion for the system is used to generate the optimal harvest strategies. The solution
from the Hamilton-Jacobi-Bellman equation is a value function which considers an
economic utility function for the stochastic system that the equation is derived from.
The value function can be used to compute the optimal policy for the system.
The project is divided into three main parts. The first part, in order to gain a
better understanding of the situation, a stochastic single stock model will be evaluated
and an optimal policy generated for the system. Additionally, a discounting term will
be introduced to the Hamilton-Jacobi-Bellman equation. The second part is where
the optimal harvest policy for the stochastic Rosenzweig MacArthur with harvest will
be generated. This requires developing a numerical method to solve the Hamilton-
Jacobi-Bellman. The third part is where the assessment of the policies are generated
with several case examples used to test the policies. This project aims to answer the
questions raised:
• How the Hamilton-Jacobi-Bellman equation, that is derived from a two dimen-
sional predator-prey system, is solved.

CHAPTER 2
Literature Review
As the objective of this project is to generate optimal harvest strategies, the literature
cited provides a valuable insight on what has been accomplished in the field of optimal
control for the single species case. The maximum sustainable yield (MSY) has been
subject to many studies, most of them relating to fisheries management. An article
written by Schaefer (1957) is one of the fundamental paper in the field of fisheries
management.
Schaefer demonstrated how an sustainable harvesting policy could
be generated in a deterministic model by harvesting the surplus produced at each
time step, developing what is called the Schaefer model.
He also shows that an
optimal harvest policy could be derived for the deterministic model that maximizes
the profit for the harvester while considering economic factors such as the profit per
unit and cost of each catch. These optimal harvesting policies do not necessarily
have to be at the state that produces the largest surplus (the maximum sustainable
yield). Plourde (1970) continues exploring the optimal harvesting policy further for
the deterministic model. There he describes that if production costs are introduced
to a logistic growth model with harvest, the optimal policy would want to increase
the steady state biomass and thus produce less surplus to harvest.
Bousquet et al. (2008) extends the Schaefer model to include an environmental
noise, thus developing a stochastic Schaefer model with a constant harvest.
The
authors derive stationary distribution for the stochastic system. They also develop
expressions to compute biological reference points for the stochastic system. Ewald
and Wang (2010) explore the sustainable yield a bit further. They develop a way
to compute the maximum expected sustainable yield for a stochastic logistic growth
equation, by maximizing the expectation of the equilibrium distribution for the sys-
tem. They also develop optimal harvest policies in cases where a risk aversion factor
is introduced and case where the risk is minimized given an minimum expected sus-
tainable yield.
Bordet and Rivest (2014) also study the MSY for stochastic surplus models, specif-
ically the Pella Tomilson model and the Fox model. They compute the biological ref-
erence points, namely the state where MSY is achieved, for the stochastic models and
thus generating an optimal harvesting strategy that avoids extinction. Their findings
show that the optimal harvesting strategy in the stochastic system harvests less then
the MSY in the deterministic models, meaning it is considering the noise level whilst
deriving the biological reference points. The articles that have discussed the stochas-
tic system have it in common that they are all considering a fixed constant effort

A logistic growth equation is widely used in cases where the growth of a single species
is described without any interaction of harvest or predation. By adding the harvest
term, one has to bear in mind that if the harvest is too large, it may result in
negative growth and therefore make the model unsustainable. This means that the
harvest has to be carefully chosen. A solution to a deterministic model is relatively
simple, a harvest which is equal or less than the surplus produced at each time step is
sustainable and can be easily found. The stochastic model, on the other hand, needs
to be examined further.

3.2.1
Extension: Discount Rate
Using discounting in economics is rather common. Discounting calculates the present
value of the potential future value. The present value of the harvest at time t is
worth more than at time t + 1, thus making the harvest more valuable at time t if
the discounting factor is positive. The HJB equation (eq. 3.12) is modified to include
this term. The equation should match the discounting term, thus forcing the optimal
policy profit to be equal to the rate of return if everything was harvested and the
only income would be interest (Marín-Solano and Navas 2009).

3.3.1
Backstepping
This is a control problem where t < T and the terminal conditions are V (x, T) = 0.
That is, the harvesting stops at time T. Then the system is solved by backstepping
the implicit Euler method until convergence is reached. Convergence is reached when
the value function that is generated in each time step does not change significantly
between steps while t →−∞.
When convergence is reached the solution is at its steady state and the numerical
solution of the optimal policy is therefore ready to be used in a closed loop system.

CHAPTER 4
Two Interacting
Species
The two interacting species in this case means the interaction of a prey and a predator,
the predator consumes the prey to grow. With the absence of a predator, the prey has
a logistic growth without a harvest term. Almost every species in the world has some
kind of predator so it would make sense to include a predator in the model. Better
yet include a predator and harvesting term for the predator as well as the prey. In
real life this happens often, for example in the fishing industry the Atlantic cod and
capelin can be found in the same region and the cod preys on the capelin (Hilmarsson
et al. 2015). To make the model even more realistic a stochastic noise term should
introduce uncertainty. The end goal is to develop an optimal simultaneous harvesting
policy for the two interacting species in a stochastic system.
4.1
Deriving the Model
To begin with, it is important to understand what is going to be modelled. The
model is supposed to describe two interacting species.
The prey grows according
to the logistic growth model with an intrinsic growth rate while the growth of the
predator depends on the biomass of the prey. These two growth differential equations
have to connect somehow. This is where the functional response is introduced. The
functional response is, in essence, the capture rate of the predator as a function of
the prey biomass, that is how much prey the predator can capture.
There are a
few types of functional responses. The functional response that will be used in the
predator-prey model will be a type II3 (Gotelli 2018).

Boundary Conditions
The HJB equation is defined on the positive quadrant. For the sake of computing
efficiency, the boundary is truncated.
By truncating the boundary two different
boundary conditions are introduced, a Dirichlet and a Neumann boundary bound-
ary conditions. A Dirichlet boundary, specifies a value at the boundary following a
known function, while a Neumann boundary conditions is when the derivative of the
unknown is given at the boundary (Peirce 2018). A Dirichlet boundary condition is
used where the line is purple in figure 4.1. Then at the boundary for the truncated
domain (dashed green line), a Neumann boundary condition is imposed like in the
single species case. The Dirichlet boundary is imposed on the purple line because at
the boundary where y = 0 it means that the predator is eradicated from the system.
The prey is still left in the system and can be harvested, meaning the model changes
to a single species model which is described in Chapter 3. The Dirichlet boundary
condition then computes the value function for the single species model at the bound-
ary which can be used to compute an optimal policy for the model. If the tables
are turned and the prey becomes eradicated (at x = 0) the predator will lose their
growth term and will eventually become extinct. This means the resource can still be
exploited until it becomes naturally extinct and at the boundary where the prey is
extinct the Dirichlet boundary conditions computes the value function for the system

Adjusting the Fixed Predator Mortality
The deterministic Rosenzweig-MacArthur converges either to an equilibrium point or
a limit cycle (May 1972). By constructing a bifurcation diagram where one parameter
is varying in the model, one should be able to tell where the system stops being in a
limit cycle and starts to converge to the equilibrium point. A bifurcation diagram for
the deterministic Rosenzweig-MacArthur model is computed and can be seen in figure
5.27. It uses the parameter values that are described in table 5.1 with the exception of
the mortality rate, µ, which is used as the varying variable in the bifurcation diagram.
The bifurcation diagram is in a chaotic state at the left boundary, which will evolve
into period halving bifurcations which then leads to order. Where the period halving
bifurcations happens the population dynamics will end in a limit cycle, when there
is order however the system converges to an equilibrium point. From the bifurcation
diagram it is evident that the model starts to converge to the equilibrium point when
the fixed mortality rate for the predator is around 0.3. The deterministic example
from section 5 will produce a limit cycle as the mortality rate is 0.15. This is is
confirmed in figure 5.1a.
Fixing the mortality rate to µ = 0.4 should eliminate
the limit cycle and converge to the equilibrium point. Adjusting the mortality rate
will generate a new equilibrium state for the system. The equation to compute the
equilibrium state has already been defined in equation 5.2c and by inserting the
parameters from table 5.1 and the new mortality rate, the equation will yield the
equilibrium state of:
{x = 0.6667, y = 0.3333}

5.5
Uncertain Stock Assessment
The policy should be tested for robustness. An example of a robustness test is to
examine how the policy behaves in the case of an uncertain stock assessment. Marine
institutes8 determine the stock size and give advice on how much should be harvested
each year. Even though the institute possess many sophisticated researching tools,
there can always be some uncertainty in the stock assessment. Now the base model
is an optimal harvesting policy, it gives an advice on how much to harvest given
the state of the biomass in the system, just like the marine institutes. The policy
generated for the predator and prey should be tested for robustness by observing how
it handles the uncertain stock assessment. This case uses the same value function as
defined in the base model case (section 5.1) and the same optimal harvest policy for
the prey and predator. For this uncertain assessment case it is assumed that the stock
of the prey is overestimated by 70% while the predators biomass is underestimated
by 20%. This can by implemented in the stochastic Rosenzweig-MacArthur model by
rewriting the harvesting syntax in order to test the robustness:
fx(1.7x, 0.8y)
(5.10a)
fy(1.7x, 0.8y)
(5.10b)
This makes sure that the harvesting policy gets the wrongfully estimated biomass
while the biomass state of the system is true.

is assuming uncertainty and does not harvest a drastically large biomass. As the
predator is underestimated, it is allowed to grow almost unhindered, but the policy
recognises that the biomass is growing and harvests it, but not enough to keep it
from growing further. This will naturally affect prey as the predator is consuming
the already over harvested prey which will end in an low abundance for the prey
which in turn affects the growth of the predator by causing a stock collapse. One
might even say it is a vicious cycle. The population dynamics cycle has more extreme

CHAPTER 6
Conclusion
The aim of this project was to develop a way to generate optimal simultaneous harvest
strategies for the stochastic predator-prey system. This would prove to be a technical
challenge and starting at that point would result in a fairly steep learning curve. To
get familiarised with the content in question the single species case was studied, and
an optimal harvest strategy was developed for the stochastic single species model.
Harvesting policies for a deterministic single species model are well known in the
literature and can be found in Schaefer (1957), for example. The article also discusses
harvesting policies that include economic factors, and it shows that it not always
optimal to harvest at the MSY state, sometimes it is more profitable to harvest with
a lower or higher state in the long run. Adding a noise to the single stock model turned
it into a stochastic differential equation. The Hamilton-Jacobi-Bellman equation is
a method that can be used to solve control problems given a utility function. The
Hamilton-Jacobi-Bellman generates a value function which depends on state and time.
This value function is then used to develop the optimal harvesting policy. This method
was used to solve the control problem, that is generating an optimal harvest strategy
depending on the state of the species, for the stochastic single species model.
In
an extension to the single species problem, a discounting factor was added to the
Hamilton-Jacobi-Bellman equation.
The Hamilton-Jacobi-Bellman equation for the single species was transformed into
an advection-diffusion form as the available software assumes an advection-diffusion
form. After the transformation it is solved by using the implicit Euler method as the
terminal condition are known, V (T) = 0. After the back stepping process was over,
and a steady state was reached, the optimal harvesting policy was generated. Then
the optimal harvesting policy was tested on a stochastic single species model and
compared to a constant effort policy. The results indicate the optimal policy works,
the species is not eradicated, and it continuously generates profit for the harvester.
The constant deterministic harvesting policy is, however, the harvester of sorrow in
the stochastic system, as it eradicates the species and therefore, does not generate
any profit. Without the noise, the constant effort policy would eventually converge
to the MSY state, but that is not the case in a stochastic world.
Furthermore,
the discounting extension was observed by generating five optimal harvesting policies
with different discount rates. This includes the harvesting policy without discounting.
The result from the policy shows that the value function, which is the expected future
profit of the system while assuming the control is optimal, decreases as the discount
==PAGE:72==
64
6 Conclusion
factor increases. This also means that the harvesting effort increases in the optimal
harvesting policy as the policy favours the short term profit more than the long term
profit. The population dynamics were then observed for the different discounting
rate policies and compared by simulating the system using the same realization of
the Brownian motion. From the simulation, it could be observed that the simulated
biomass with the policy using the higher discount rate has is generally lower than the
simulated biomass using the policy with a lower discount rate. All this corresponds to
the more laborious harvesting effort in the policies with an increasing discount rate.
The stochastic Rosenzweig-MacArthur model is considered when the optimal har-
vesting policy is developed for a predator-prey system. The development of a simulta-
neous harvesting policy follows the same principals as in the single species model. The
Hamilton-Jacobi-Bellman equation for the stochastic Rosenzweig-MacArthur model
was transformed into advection diffusion form and then converted to weak form to
satisfy the solver used. It was then solved for the value function by backstepping
as the terminal value function is known, V (T) = 0. At each time step, the FEniCS
solver solved a linear variational problem where the bilinear form, derived from the
modified Hamilton-Jacobi-Bellman equation is equal to the linear form of the same
equation. Stability problems arose when the problem was being solved for the value
function, a workaround method that smoothed the value function after each solving
iteration was developed to remove the ripples in the value function, making it con-
vergent. When the value function has converged, it is said to be at its steady state,
and it was then used to compute the optimal harvesting policy.
With a defined method on how to derive an optimal harvesting policy, a total of six
cases were conducted to test how the policy would perform in a stochastic Rosenzweig-
MacArthur system for a variety of different conditions. These cases generated some
interesting results, in all cases but one the prey has a good odd of surviving while the
predator is not as lucky. If the base case is excluded the only cases where the predator
gets a chance to grow and become a profitable resource is where the main focus is on
the predator, that is in the cases where only the predator can be harvested and the
case where the price of the predator is increased. The uncertain stock assessment also
shows that the policy does indeed allow some room for uncertainty, the prey manages
to survive most of the time even though the biomass is overestimated by 80%, this is
harder to say for the predator. It heavily relies on the biomass of the prey, and if it
goes to low, it translates to a hard time for the predator. Therefore, it does not fare
well for the prey in this case where the prey is overestimated by such a large margin.
It is also observable when the optimal harvesting policies are compared between cases
that they often increase or decrease the harvesting effort in some states. This tells
that the surplus that is being produced not fully harvested, meaning the policy is
thinking about the noise in the system. This can also be seen in the policies that are
generated that at some states where the system should produce a positive surplus
in the deterministic system, the policy dictates that there should be no or minimal
harvest.

assumed that the predator grows at the same rate as they hunt prey, which is not

APPENDIX D
Smoothing the Value
Function Derived from
the HJB Equation
In this case, the value function should be monotonically increasing and bounded. This
would make the value function convergent. The value function derived in section 4.3
does however, not converge. The reason for this is how the equations are discritized,
the matrix that corresponds to the bilinear part in the linear variational problem
(eq.4.17) should have the eigenvalues in the closed left half of the complex plane.
That is however not the case so the finite element method is not stable. This causes
ripples when the value function is being solved and causes not to converge. There is
a way to remove these ripples and that is to smooth the value function after it has
been computed by solving the HJB equation at each time step.
The smoothing that takes place can be described by the following equation:
VR = (1 −k∆)VS
(D.1)
In the equation VR is the ”rough” function with the ripples that will be smoothed, k is
smoothing parameter, ∆is the Laplacian and VS is the new smoothed value function.
To demonstrate that this work an example is conducted. The rough function VR is
set to VR = sin λx and then it is possible to say VS = a sin λx, hwere a is a ”tuning”
multiplier to modifiy the smoothed value function. Inserting this into equation D.1
will yield:
sin λx = (1 −k∆)a sin λx
Which will then turn into:
sin λx = a sin λx + kλ2a sin λx
Then it is simplified by dividing with sin λx on both sides.
1 = a(1 + kλ2)
==PAGE:82==
74
D Smoothing the Value Function Derived from the HJB Equation
Then it is solved for a:
a =
1
1 + kλ2
It is seen that for a small k, the multiplier a will be very close to 1, so it is a minimal
change from the rough solution. This will do the trick and remove the ripples from the
solution. As stated before this will be implemented in the FEniCS code that solves
the HJB equation and generates the value function. Immediately after it has been
generated it will go through the smoothing process and the smoothed value function
will be the final value function generated in the time step. It will be used then as an
”old” value function in the next time step for the HJB equation and the process of
generating a new value function start again.
To be able to solve this in FEniCS the smoothing method needs to be written on a
advection diffusion form, which is done be following the procedures describe in section
A. After it has been written on an advection diffusion form it is then converted to
weak form, like it was done with the HJB equation in section 4.3. Finally, to solve
it in FEniCS, at has to be set up as a linear vatational problem where the bilinear
form is equal to the linear form like it was defined in section 4.3.2. With all theses
steps completed, the final equation will be:
wVS + K∇Vsw = wVR
(D.2)
Here, w is the test function. As said before, FEniCS will solve this form for the newly
solved value function generated from the HJB equation.
Note-This method of smoothing the value function was developed by the supervisor
of this thesis, Uffe Høgsbro Thygesen in one of the many meetings throughout the
project.

APPENDIX F
Codes
The codes that where used to compute value functions, the optimal policies and the
results can accessed at: .
There are two folders at the link, one that contains the single species related codes and
one that contains codes that are used for the predator-prey system. Only the code for
the base case and the adjusted noise case are published. The reason being that the
cases that where conducted (noise model included) use the base model code but with
some minor adjustment that should be relatively easy to replicate. The noise model
is included on the website to show how the value function and the optimal policies
from the base model is compared their counterparts in the adjusted versions.
