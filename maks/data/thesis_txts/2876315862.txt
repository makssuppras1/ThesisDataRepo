Performance Evaluation of a PET/MR Scanner
Abstract
Hybrid positron emission tomography/magnetic resonance (PET/MR) systems have be-
come indispensable in preclinical research, enabling simultaneous functional and anatom-
ical imaging with improved soft-tissue contrast. However, quantitative accuracy and spa-
tial resolution in small-animal PET remain strongly dependent on system performance,
reconstruction algorithms, and physical radionuclide properties. This work presents a
comprehensive performance evaluation of the MRS*PET/MR 3024 preclinical scanner,
following the NEMA NU 4–2008 standards and extended phantom-based tests using 18F
and 68Ga. The study assessed quantitative stability, recovery coefficients (RCs), spill-over
ratios (SORs), noise characteristics, and spatial resolution across iterative reconstruc-
tions, with additional analysis of Gaussian filtering, Jaszczak-based micro-resolution, and
high-activity behavior.
For 18F, mean uniformity remained stable (U ≈0.176 MBq/mL) while image noise in-
creased from 10 % to 29 % across iterations, reflecting the expected resolution–noise
trade-off. RCs for 4–5 mm rods approached 0.8–0.81, whereas RCs for smaller rods re-
mained limited by partial volume effects. Corresponding SORs decreased below 0.16 (air)
and 0.10 (water). For 68Ga, RCs were lower (≤0.71 for large rods) and SORs higher,
consistent with its longer positron range, yet the uniformity and noise trends were similar.
Iterations 6–10 provided the optimal balance between contrast recovery and noise for both
radionuclides.
Gaussian post-filtering confirmed that mild smoothing (σ ≈1 px) can reduce noise mod-
estly but at the cost of small-structure contrast, while heavier smoothing compromises
quantitative integrity. Micro-resolution phantom measurements demonstrated a resolv-
able rod size of approximately 0.9 mm for 18F. High-activity experiments verified excellent
quantitative linearity, with measured half-lives matching theoretical values within 1% and
no evidence of saturation or non-linear behavior.
Overall, the MRS*PET/MR 3024 exhibits stable calibration, quantitative reliability, and
sub-millimeter resolution comparable to other SiPM-based preclinical PET/MR systems.
The recommended operating regime (iterations 6–10, optional σ ≈1 px filtering) provides
a robust balance between accuracy and image quality. These findings establish a validated
baseline for subsequent preclinical imaging studies using 18F and 68Ga and contribute to
ongoing optimization of quantitative hybrid PET/MR methodologies.

Resumé
Hybrid positron-emissions-tomografi/magnetisk resonans (PET/MR) er blevet uundværlig
i præklinisk forskning, idet teknologien muliggør samtidig funktionel og anatomisk billed-
dannelse med forbedret bløddelskontrast. Den kvantitative nøjagtighed og rumlige opløs-
ning i smådyrs-PET afhænger dog fortsat stærkt af systemets ydeevne, rekonstruktion-
salgoritmer og de fysiske egenskaber ved den anvendte radionuklid.
Dette arbejde præsenterer en omfattende præstationsvurdering af den prækliniske MRS*
PET/MR 3024-scanner i overensstemmelse med NEMA NU 4–2008-standarden samt ud-
videde fantombaserede tests med 18F og 68Ga. Undersøgelsen evaluerede kvantitativ sta-
bilitet, recovery coefficients (RC), spill-over ratios (SOR), støjkarakteristika og rumlig
opløsning på tværs af iterative rekonstruktioner, suppleret af analyser af Gauss-filtrering,
Jaszczak-baseret mikroopløsning og høj-aktivitetsadfærd.
For 18F forblev middeluniformiteten stabil (U ≈0,176 MBq/mL), mens billedstøjen steg
fra 10 % til 29 % på tværs af iterationer, hvilket afspejler den forventede trade-off mellem
opløsning og støj. RC-værdierne for 4–5 mm stænger nærmede sig 0,80–0,81, mens mindre
stænger fortsat var begrænset af partial-volume-effekter.
De tilsvarende SOR-værdier
faldt til under 0,16 (luft) og 0,10 (vand). For 68Ga var RC-værdierne lavere (≤0,71 for
de største stænger) og SOR-værdierne højere, hvilket stemmer overens med den længere
positronrækkevidde, mens uniformitet og støjudvikling var tilsvarende. Iterationerne 6–10
gav den bedste balance mellem kontrastgenvinding og støj for begge radionuklider.
Gaussisk efterfiltrering bekræftede, at mild udjævning (σ ≈1 px) kan reducere støj i
beskedent omfang, men på bekostning af kontrasten i små strukturer, mens kraftigere
udjævning kompromitterer den kvantitative integritet. Mikroopløsningsfantom-målinger
viste en opløselig stangstørrelse på ca. 0,9 mm for 18F. Højaktivitetseksperimenter bekræft-
ede fremragende kvantitativ linearitet, hvor de målte halveringstider stemte overens med
de teoretiske værdier inden for 1 % og uden tegn på mætning eller ikke-lineær adfærd.
Samlet set udviser MRS*PET/MR 3024 stabil kalibrering, kvantitativ pålidelighed og
submillimeteropløsning, der kan sammenlignes med andre SiPM-baserede prækliniske
PET/MR-systemer.
Det anbefalede driftsregime (iterationer 6–10, valgfri σ ≈1 px-
filtrering) giver en robust balance mellem nøjagtighed og billedkvalitet. Disse fund etablerer
en valideret baseline for efterfølgende prækliniske billeddannelsesundersøgelser ved hjælp
af 18F og 68Ga og bidrager til den løbende optimering af kvantitative hybride PET/MR-
metoder.

Introduction
Translational research is fundamentally transforming modern medicine by creating a
bridge between basic molecular discoveries and clinical implementation for diagnostics
and therapeutics. This process requires rigorous preclinical validation, as the behavior of
a molecule observed in vitro often fails to predict its function in vivo. Advanced preclini-
cal research using living systems is therefore an indispensable step, mitigating the ethical
and logistical limitations of direct human experimentation while significantly enhancing
the translational value of scientific findings.
A central component of this pipeline is molecular imaging, which utilizes high-resolution
in vivo technologies to study disease processes non-invasively, quantitatively, and lon-
gitudinally. These tools allow researchers to repeatedly monitor disease progression or
therapeutic response in the same animal subject, thereby improving statistical power,
shortening research timelines, and enhancing the cost-effectiveness of biomedical innova-
tion. Preclinical imaging is now a core part of drug development, used across oncology,
neuroscience, and cardiology, to assess biodistribution, pharmacokinetics, and target en-
gagement prior to human studies [1–3].
Among these tools, Positron Emission Tomography (PET) has emerged as a gold stan-
dard for noninvasive quantification, capable of measuring metabolic, receptor-binding,
and pharmacokinetic processes with sub-picomolar sensitivity [3]. Despite its molecu-
lar power, PET inherently lacks the necessary morphological and soft-tissue contrast to
precisely localize functional changes. This limitation is effectively addressed by the in-
tegration of PET with Computed Tomography (CT) or Magnetic Resonance Imaging
(MRI) [4]. Both CT and MRI provide high spatial resolution, excellent soft-tissue con-
trast and key functional information such as diffusion and perfusion, making the hybrid
PET/MR system an ideal platform for comprehensive preclinical studies [5,6].
The realization of simultaneous PET/MR, however, required overcoming significant tech-
nical hurdles, primarily the incompatibility of conventional PET detectors with strong
magnetic fields. This challenge was resolved by transitioning to solid-state detectors, such
as Silicon Photomultipliers (SiPMs), which are insensitive to magnetic fields [4,7]. These
detector advances, combined with the adoption of standardized performance assessment
through the National Electrical Manufacturers Association( NEMA) NU 4-2008 protocol,
have driven major improvements in the sensitivity, spatial resolution, and quantitative
reliability of small-animal PET/MR systems [8,9].

1.1
Study Overview
In this thesis, three complementary experimental studies were performed to comprehen-
sively assess the performance and quantitative reliability of a newly installed preclinical
3T PET/MR scanner from MR Solutions (Surrey, UK). All experiments followed the
National Electrical Manufacturers Association (NEMA) NU 4-2008 framework [8], using
standardized phantoms and Radionuclides to ensure reproducibility and comparability
with previous system evaluations.
First, a performance evaluation study characterized the scanner using key NEMA met-
rics—spatial resolution, sensitivity, scatter fraction, count-rate performance, and image
quality [8–10]. These tests employed both Fluorine-18 (18F) and Gallium-68 (68Ga) la-
beled radionuclides with the NEMA NU 4-2008 phantom and a dedicated micro-resolution
phantom (often referred to as a mini-Derenzo or Jaszczak phantom) [11,12]. The NEMA
phantom quantified parameters such as uniformity, recovery coefficients, and spill-over
ratios, while the micro-resolution phantom assessed the scanner’s spatial resolution and
reconstruction performance across varying rod sizes. To optimize quantitative accuracy,
reconstruction was performed using iterative OSEM algorithms with and without mild
Gaussian post-filtering, a common procedure to balance resolution and image noise in
PET imaging [10, 13, 14]. Together, these tests established a baseline of system perfor-
mance and compliance with international standards [8,10,15].
Finally, a high-activity dynamic study investigated quantitative behavior at elevated count
rates to evaluate linearity, dead-time effects, and count-rate stability. A custom-designed
uniformly filled cylindrical phantom (“tube phantom”) was used to assess the system’s
response under high-activity conditions [8, 9, 16], providing insight into quantitative ro-
bustness for fast-kinetic or short-lived isotope studies such as those using 68Ga [17, 18].
These acquisitions tested the scanner’s electronic throughput and its ability to maintain
linear quantification without saturation [10,19].
Collectively, these experiments provide a comprehensive performance characterization of
the MR Solutions PET/MR scanner, validating its suitability for preclinical translational
imaging and ensuring adherence to NEMA NU 4-2008 quantitative standards. A detailed
review of the supporting literature is provided in Appendix B.

2
Theoretic Background
This chapter summarizes the physical and quantitative concepts that underpin the exper-
iments performed in this thesis and the evaluation of the MRS*PET/MR 3024 system.
The discussion begins with the fundamentals of radioactivity and positron emission, in-
cluding the impact of positron range and radionuclide selection on spatial resolution. It
then covers detector physics, tomographic image formation and quantitative calibration,
before introducing key image-quality, and performance metrics, micro-resolution phan-
toms and high-activity behaviour. Finally, specific aspects of PET/MR hybrid imaging
relevant to preclinical applications are outlined, providing the theoretical framework for
the subsequent methods and results.
2.1
Radioactivity and Positron Emission
Certain atomic nuclei are unstable because the balance between nuclear binding energy
and electrostatic repulsion cannot be maintained [20]. These nuclei spontaneously trans-
form into more stable configurations by emitting particles or electromagnetic radiation, a
process known as radioactive decay.
The decay rate follows an exponential law:
A = A0e−λt
where A is the activity (disintegrations per second), A0 the initial activity and λ the decay
constant. The T1/2 = ln 2
λ defines the time required for half of the nuclei to decay and is
an essential parameter in radiopharmaceutical production and imaging scheduling.
In beta-plus (β+) decay, a proton is transformed into a neutron with the emission of a
positron (e+) and an electron neutrino (νe):
p →n + e+ + νe
After traveling a few millimeters in tissue, the emitted positron encounters an electron
and the two annihilate, converting their rest mass into two 511 keV gamma photons emit-
ted almost exactly 180◦apart [2]. This phenomenon, a direct consequence of Einstein’s
mass–energy equivalence, E = mc2, provides the physical basis for PET imaging, as
illustrated in Figure 2.1 [20].

2.2
Positron Range: The Intrinsic Resolution Limit
A fundamental physical phenomenon that limits the achievable spatial resolution in (pre-
clinical) PET is the positron range (Rpos). This is the distance the positron travels between
its emission and the annihilation event. Since the PET scanner images the location of
annihilation rather than the point of emission, this travel distance introduces an inherent
blurring artifact. The range is proportional to the positron energy and inversely propor-
tional to the electron density of the medium it travels through. Rpos makes a significant
contribution to the overall spatial resolution in preclinical systems [18,22].
2.3
Radionuclides
The advancement of PET as a key diagnostic imaging tool is intrinsically linked to the
availability of suitable radionuclides and the radiochemistry to incorporate them into
biologically relevant molecules. Among the plethora of positron emitters, two radionu-
clides stand out for their distinct physical properties and clinical applications:
18F and
68Ga. The selection of a specific radionuclide is a critical decision, dictated by its half-life,
production method and chemical versatility, which together determine its suitability for
targeting different biological processes.
Fluorine-18 (18F) is rightly considered the workhorse of clinical PET. Its dominance is
primarily due to its favorable 109.8-minute half-life, which permits complex multi-step
radiosynthesis and allows for distribution from a central production facility. It is effi-
ciently produced in a medical cyclotron via the nuclear reaction 18O(p, n)18F, yielding
large quantities of fluoride in enriched water. As the archetype 18F-labeled tracer, 2-
[18F]Fluoro-2-deoxy-glucose ([18F]FDG) is widely used. Critically, its relatively low Emax

each addressing a unique niche based on its production logistics, chemical handling and
biological targeting.
2.4
Detection and Image Formation
PET scanners consist of multiple rings of scintillation detectors, often made from lutetium
oxyorthosilicate (LSO) or lutetium–yttrium oxyorthosilicate (LYSO) crystals coupled to
photomultiplier tubes or silicon photodiodes. When a 511 keV photon interacts with the
scintillator, its energy is converted into visible light, which is subsequently amplified and
recorded as an electrical pulse.
When two detectors register photons within a narrow coincidence timing window (a few
nanoseconds), the event is classified as a true coincidence, defining a line of response
(LOR) between the detectors. By collecting millions of coincidences from different angles,
a three-dimensional distribution of radioactivity can be reconstructed using algorithms
such as filtered back projection or, more commonly, iterative reconstruction methods like
OSEM. [13,18,20].
2.5
Quantification and Calibration
A defining feature of PET is its quantitative capability; the detected signal is directly
proportional to the local concentration of radioactivity, expressed in megabecquerels per
milliliter (MBq/mL). Accurate quantification requires careful calibration and correction
for several physical effects:
• Attenuation: Loss of photons traveling through tissue.

2.8
Micro-resolution Phantom
While NEMA NU-4 image-quality metrics quantify contrast recovery, spill-over, and uni-
formity, they do not directly characterize the smallest resolvable structure that the imag-
ing system can distinguish. For this reason, preclinical PET studies often complement

NEMA testing with a micro-resolution analysis, which provides an intuitive and quanti-
tative measure of spatial resolution.
The micro-resolution phantom consists of multiple groups of closely spaced hot rods ar-
ranged in hexagonal clusters, with diameters typically ranging from 0.6 mm to 1.2 mm in
preclinical configurations. The compartments of the phantom are filled with a positron-
emitting radionuclide, while the interstitial spaces remain cold. The phantom therefore
creates a high-contrast periodic pattern that is highly sensitive to system resolution,
positron range, and reconstruction parameters [9,11,16].
Quantitative evaluation of rod separability is based on the Valley-to-Peak Ratio (VPR)
method [15]. For each rod group, a line profile is drawn through adjacent hot rods and the
VPR is defined as the ratio between the minimum intensity in the valley and the mean
intensity of the two neighboring peaks:
VPR =
Ivalley
(Ipeak1 + Ipeak2)/2.
A smaller VPR indicates better visual separability. Following the convention established
in optical and PET resolution studies, a rod group is considered resolved when its mean
VPR is below approximately 0.735, the so-called Rayleigh criterion proxy [8, 10]. This
threshold corresponds to the point where the valley intensity drops to about 73.5% of the
adjacent peaks, beyond which individual rods can be visually distinguished with minimal
ambiguity.
The micro-resolution analysis thus complements NEMA NU-4 metrics by providing a
direct estimate of the system’s practical spatial resolution under realistic reconstruction
conditions. Because spatial resolution in PET depends not only on detector geometry and
crystal size but also on positron range, reconstruction algorithm and smoothing kernel,
the VPR-based micro-resolution study offers a valuable bridge between theoretical sys-
tem limits and empirically observed performance. Together with NEMA image-quality
and decay linearity analyses, it provides a comprehensive characterization of contrast,
resolution and quantitative stability for the PET/MR 3024 system.
2.9
High-Activity and Dead-Time Performance
Accurate quantification is particularly challenging in preclinical PET, where the small de-
tector volume and high injected activity concentrations often push the system close to its
physical limits [9]. At elevated count rates, detector and electronic subsystems experience
saturation, leading to dead time, pile-up and an increase in random coincidences. These
effects distort the linear relationship between measured and true activity and ultimately
degrade quantitative accuracy and image quality. A robust theoretical framework describ-

count rates and to ensure the reliability of decay curve and kinetic model analyses [18].
2.9.1
Dead-Time Modeling and Noise Equivalent Count Rate
At high event rates, each detected interaction requires a finite processing interval during
which the system is unable to register new events. This dead time leads to a nonlinear
response that can be described mathematically using either the non-paralyzable or par-
alyzable model [8, 9, 16, 24, 25]. These simplified models are commonly used to estimate
the system’s dead-time constant and to design correction factors that restore quantitative
linearity.
Non-paralyzable Model
In the non-paralyzable model, each registered event produces a fixed dead-time window
τ, any events arriving within this interval are ignored but do not prolong the dead time:
Ameas =
Atrue
1 + c Atrue
,
where c denotes the effective dead-time constant. This model is representative of modern
SiPM-based PET systems equipped with fast digital readout electronics and efficient
buffering, such as the MRS*PET/MR 3024 [11,26].
Paralyzable Model
In contrast, the paralyzable model assumes that an event arriving during an existing
dead-time interval extends the busy period, resulting in an exponential saturation of the
measured count rate:
Ameas = Atrue e−k Atrue,
where k is the paralyzable dead-time constant. This formulation is more characteristic
of analog detection chains or slower scintillator systems where pulse pile-up accumulates
and further suppresses detection at very high activities. In practice, most PET scanners
exhibit intermediate behavior, and hybrid empirical fits are applied to describe the system
accurately across the full activity range [9,23].
2.9.2
Noise Equivalent Count Rate
To standardize the evaluation of high-activity performance, the NEMA NU 4-2008 pro-
tocol defines the Noise Equivalent Count Rate (NECR) as a quantitative measure of
image SNR that accounts for the degrading effects of scatter and random coincidences [8].
NECR represents the true coincidence rate that would yield the same statistical noise in

NECR =

where T, S and R are the true, scatter and random coincidence rates, respectively.
At low activities, T dominates and NECR rises approximately linearly with activity. As
the system approaches saturation, the increase in randoms and dead-time losses causes
NECR to reach a maximum, the peak NECR, beyond which image quality and quanti-

operating range in terms of injected activity and acquisition timing. Clinical and preclini-
cal studies alike have shown a strong correlation between NECR and reconstructed image
SNR, validating NECR as a robust indicator of overall imaging performance [19]. For
example, Carlier et al. demonstrated that optimizing injected activity and scan duration
according to NECR maximizes image quality while minimizing radiation dose in 18F-FDG
PET.
2.9.3
Empirical Evolution and Implications for Imaging
Historically, early preclinical systems such as the microPET Focus F120 exhibited NECR
saturation near 0.8 Mcps at 88 MBq, indicating the onset of non-linearity at relatively
modest activity levels [27]. With the introduction of faster scintillators and improved
electronics, the Siemens Inveon extended this range to a peak around 130 MBq [28]. Fur-
ther developments in digital SiPM (dSiPM) technology and application-specific integrated
circuits (ASICs) significantly enhanced temporal resolution and count-rate handling [16].
Recent SiPM-based PET/MR systems have pushed this boundary even further.
The
SAFIR-II scanner achieved a coincidence timing resolution of 221 ps and sustained NECR
above 1.1 Mcps at 451 MBq with less than 9% degradation in MR SNR during si-
multaneous operation [24].
Similarly, Gonzalez-Montoro et al.
validated quantitative
stability in the ScintoTube PET insert by monitoring a uniform phantom throughout
the full decay range, reporting deviations below ±5% across multiple half-lives [29].
These advances demonstrate that contemporary SiPM-based architectures, including the
MRS*PET/MR 3024, maintain quantitative integrity and SNR stability even at activity
concentrations that would have caused severe saturation in previous-generation analog
systems.
Overall, the combined use of dead-time modeling and NECR analysis provides a compre-
hensive framework for evaluating quantitative stability in both static and dynamic pre-
clinical PET studies. By defining the activity range below the NECR peak, researchers
can ensure optimal data quality and reproducibility for high-activity and short-lived iso-
tope protocols, thereby maximizing the translational reliability of preclinical PET/MR

2.10
PET/MR Hybrid Imaging
Hybrid PET/MR systems integrate the molecular sensitivity of PET with the excellent
soft-tissue contrast and functional capability of MRI. This combination offers advantages
such as improved anatomical localization. A key distinction from PET/CT (Computed
Tomography) is the challenge of attenuation correction (AC): MRI does not provide the
direct electron density map obtained from CT. PET/MR systems rely on MR-based
attenuation correction (MRAC) methods that infer tissue attenuation coefficients from
MR images using techniques like segmentation, atlas-based models, or deep learning ap-
proaches [3,30].

DICOM Ingestion and Quantitative Scaling
Conversion from stored integers to physical activity concentration relies on key DICOM
metadata. The RescaleSlope tag provides the multiplicative factor applied to the stored

activity concentration expressed in becquerels per milliliteration(Bq/mL) is obtained as:
IBq/mL = PixelValue × RescaleSlope + RescaleIntercept.
In the datasets, RescaleIntercept was 0 and Units=BQML, indicating that the recon-
structed images are already calibrated in Bq/mL. (For reporting in MBq/mL, values were
divided by 106.)

5
Discussion
5.1
Overview
The aim of this work was to characterize the quantitative image quality, spatial res-
olution and high-activity performance of the MRS*PET/MR 3024 system following the
NEMA NU 4-2008 framework [8] and to identify robust reconstruction settings for routine
preclinical imaging with 18F and 68Ga. The Results chapter provided a detailed quantita-
tive description of uniformity, recovery coefficients (RCs), spill-over ratios (SORs), noise
behavior, spatial resolution and high-activity performance for both radionuclides.
In this chapter, these findings are interpreted in the context of established preclinical
PET and PET/MR literature, with particular emphasis on standardized performance
evaluation [9, 10], advances in hybrid PET/MR instrumentation [4–6] and recent high-
performance preclinical systems and inserts [11,12,16,23–25,28,29,32–34].
5.2
Image Quality Metrics and Reconstruction Trade-offs
5.2.1
Uniformity and Calibration Stability
The first key observation is the stability of the uniform-compartment mean activity for
both radionuclides across all investigated iterations. For 18F, U ≈0.176 MBq/mL and for
68Ga, U ≈0.255 MBq/mL, with only minor variation between iteration 1 and iteration 20.
This behavior is consistent with a well-calibrated acquisition and reconstruction chain and
indicates that changes in iteration number primarily modulate noise and resolution rather
than absolute quantification [9,10].
Such stability aligns with expectations from modern preclinical PET systems, where
NEMA NU 4-2008 image-quality phantoms typically show small deviations in uniformity
across reconstruction settings when proper normalization, attenuation and dead-time cor-
rections are applied [11,23,28]. Within this context, the MRS*PET/MR 3024 shows stable
global scaling, allowing reconstruction choices to be tuned for task-specific optimization
without systematically biasing global activity estimates.
5.2.2
RC, SOR and the Resolution-Noise Balance
The evolution of RCs and SORs with iteration follows the classical convergence and regu-
larization behavior of OSEM-type algorithms [13]. For 18F, RCs for 4-5 mm rods approach
∼0.8, SORs decrease below 0.16 (air) and 0.10 (water) and noise increases from about
10% to 29% across iteration 1-20. For 68Ga, large-rod RCs reach ∼0.7 with higher resid-
ual SORs and slightly lower noise at matched iterations, reflecting the stronger blurring

imposed by its longer positron range [18, 22].
These observations are fully consistent
with partial volume theory and established reports on the size dependence of RCs in
small-animal PET [9,11,23,27].
The correlation analysis between RC, SOR and noise highlights the intrinsic trade-offs
that are often qualitatively acknowledged but less frequently quantified.
For 18F, the
strong positive correlation between RC5 and noise confirms that recovery of fine detail
is inseparable from increased statistical uncertainty. For 68Ga, the particularly strong
inverse correlation between RC5 and SORair shows that improvements in recovery are
tightly linked to reductions in spill-over, emphasizing the sensitivity of this radionuclide
to reconstruction choices. These findings support the selection of an intermediate iteration
range (6-10) as a practical compromise. This recommended window is in line with previous
system evaluations that combine NEMA NU 4-2008 image-quality analysis with task-
based considerations [9,23,28] and provides a transparent, data-driven justification rather
than an arbitrary protocol choice.
5.3
Radionuclide-Dependent Performance and Physical
Interpretation
A central goal was to compare 18F and 68Ga performance within the same hardware and re-
construction framework. The systematically lower RCs and higher SORs for 68Ga at all it-
erations are in agreement with its longer mean positron range, which introduces additional
blurring before annihilation [18]. The fact that these differences are clearly resolved in the
NEMA NU 4-2008 phantom data corroborates the sensitivity of the MRS*PET/MR 3024
system to underlying radionuclide physics, rather than reflecting reconstruction artefacts.
From a practical perspective, this implies that: (i) caution is required when directly trans-
lating reconstruction protocols optimized for 18F to 68Ga without considering resolution
and PVE penalties, (ii) quantitative studies with 68Ga-labelled radionuclide may bene-
fit from partial-volume correction strategies or optimized ROI definitions adapted to the
achievable spatial resolution, as suggested by prior work in both preclinical and clinical
PET [22].
The present results thus not only benchmark the system but also provide radionuclide-
specific guidance, which is essential for multi-tracer platforms and translational workflows.

and Micro-resolution Analyses
The combination of NEMA NU 4-2008 rod RCs and micro-resolution VPR analysis
provides a coherent picture of the system’s resolving capability. The micro-resolution

Using the predefined

5.5
Impact of Gaussian Post-Filtering
The systematic evaluation of Gaussian post-filtering shows a predictable, but important,
behavior: mild smoothing (σ ≈1 px) can reduce apparent noise with only modest im-
pact on RCs for larger rods, whereas stronger kernels rapidly suppress small-rod RCs
and increase SORs, reflecting edge blurring. This is consistent with previous work on
post-reconstruction filtering in both cardiac and whole-body PET, where Gaussian filters
were shown to trade quantitative accuracy for visual smoothness and can bias parametric
estimates if not carefully controlled [14,22].
In the context of NEMA NU 4-2008 reporting, the present results support the recommen-
dation to: (i) base formal performance metrics on unfiltered images, in line with guidance
from Hallen et al. [10] and (ii) reserve mild filtering for specific visualization needs, clearly
documented in protocols rather than applied by default. For the MRS*PET/MR 3024,
this implies that quantitative phantom and kinetic studies should be reconstructed with-
out heavy post-smoothing, particularly when targeting sub-2 mm structures or subtle
contrasts.
5.6
High-Activity Behavior and Count-Rate Considerations
The high-activity phantom experiment demonstrated excellent quantitative linearity for
both radionuclides across the evaluated activity range: fitted half-lives agreed with refer-
ence values within < 1% and the recovery factor remained close to unity without signs of
saturation or compression [9,11,23].
While a full NECR characterization and dead-time modeling were beyond the scope of
this work, the absence of detectable non-linearities at typical and moderately high preclin-
ical activities positions the MRS*PET/MR 3024 as a robust platform for dynamic and
longitudinal studies. Compared with emerging ultra-high-rate systems such as SAFIR

5.7
Integration into the PET/MR and Preclinical Imaging
Landscape
The development of integrated and insert-based PET/MR systems has been driven by the
need for truly simultaneous functional and anatomical imaging, motion and attenuation
correction strategies and improved soft-tissue contrast [4–6]. Preclinical implementations,
including clip-on inserts, dedicated PET rings and benchtop systems, have demonstrated
that hybrid configurations can meet or exceed the performance of standalone PET while
maintaining MR compatibility [12,33,34].
Within this landscape, the MRS*PET/MR 3024 results presented here show:
• quantitatively stable performance in line with NEMA NU 4-2008 benchmarks [8],
• spatial resolution and RC/SOR characteristics suitable for submillimetre to low-
millimetre applications [16],
• predictable reconstruction trade-offs enabling protocol standardization.
These attributes make the system a solid platform for molecular imaging workflows de-
scribed by Cunha et al. [1], Rovera et al. [3] and James & Gambhir [2], including oncology,
neurology and drug development studies where reproducible quantification is crucial.
5.8
Limitations
Several limitations should be acknowledged:
1. Partial NEMA NU 4 coverage. The present work focused on image-quality, spa-
tial resolution and high-activity behavior. A complete NEMA performance report
(including absolute sensitivity, scatter fraction, NECR and energy/timing resolu-
tion) was not implemented, which restricts direct one-to-one comparison with all
published systems [9,10].
2. Radionuclide and protocol scope.
Only 18F and 68Ga were studied.
Other
isotopes with different energies or prompt gammas (e.g. 124I, 89Zr) may reveal further
limitations or require protocol modifications [18,31].
3. Phantom-based evaluation. All conclusions are based on standardized phan-
toms. While this is appropriate for system characterization, in vivo validation (e.g.

4. Limited modeling of advanced corrections. Attenuation, scatter and motion
corrections were applied according to the standard workflow of the system but not
systematically optimized or compared to state-of-the-art PET/MR-specific meth-
ods [30,35]. Further refinement may yield additional gains in accuracy.
Recognizing these limitations is important both to appropriately bound the present con-

5.9
Future Directions
Building on the present characterization, several concrete extensions are recommended:
• Perform a full NEMA NU 4-2008 performance evaluation, including sensitivity, scat-
ter fraction and NECR, to enable direct comparison with recent high-performance
PET/MR systems and inserts [8–10,24,29]. Expand with in vivo data.
• Investigate advanced reconstruction strategies (e.g. PSF modeling, resolution recov-
ery, Bayesian or penalized likelihood approaches) that may further improve RCs for
small rods without unacceptable noise inflation [13].
• Explore partial-volume correction and standardized VOI protocols for 68Ga and
other challenging radionuclides, following current recommendations for quantitative
PET and dosimetry [22,31].
• Evaluate MRI-based attenuation and motion correction schemes tailored to this
platform [30,35], particularly for brain and thoracic imaging.
• Validate the recommended reconstruction settings in representative in vivo studies
(e.g. oncology, neurology, cardiac), ensuring that the phantom-based optimization
translates into biologically relevant improvements [1].
• Include additional positron-emitting radionuclides with distinct positron range en-
ergies, such as 64Cu, 89Zr, or 124I, to systematically investigate the influence of
positron range on spatial resolution, recovery coefficients and spill-over behavior.
Such comparisons would extend the system’s quantitative characterization across a
broader spectrum of clinically and preclinically relevant isotopes [18].
These steps would further consolidate the role of the MRS*PET/MR 3024 as a quantita-
tively reliable and versatile instrument for state-of-the-art preclinical research.

Conclusions and Future Perspectives
This thesis presented a comprehensive quantitative evaluation of the MRS*PET/MR 3024
preclinical imaging system based on the NEMA NU 4-2008 framework, micro-resolution
analysis and high-activity decay testing using 18F and 68Ga. The results demonstrate that
the system maintains stable calibration and quantitative accuracy across all reconstruc-
tion iterations, with uniform-region activity varying by less than two percent. Recovery
coefficients, spill-over ratios and image noise exhibited the expected iteration-dependent
trends, enabling the identification of a practical reconstruction range between iterations
six and ten that offers an optimal balance between contrast recovery and precision.
Radionuclide-specific analysis confirmed that 18F consistently achieved higher recovery
and lower spill-over than 68Ga, consistent with the shorter positron range of fluorine-18.
Nevertheless, both radionuclides showed stable and reproducible quantitative performance
within the optimized reconstruction window. Combined NEMA and Jaszczak assessments
revealed an effective spatial resolution of approximately 1.0 mm for 18F, placing the sys-
tem within the performance range of modern SiPM-based preclinical PET/MR scanners.
Gaussian post-filtering was shown to reduce apparent image noise but to degrade quanti-
tative accuracy, particularly for small structures, indicating that unfiltered or only mildly
filtered data should be used for quantitative analyses.
High-activity decay experiments confirmed linear quantitative response and consistent
half-life estimates within one percent of theoretical values, with no evidence of detec-
tor saturation or non-linear count loss. Collectively, these results demonstrate that the
MRS*PET/MR 3024 provides stable calibration, predictable quantitative behavior and
sub-millimeter spatial resolution for both 18F and 68Ga. The system can therefore be
regarded as a quantitatively reliable and well-characterized platform suitable for longitu-
dinal, multi-tracer and high-count preclinical imaging studies, providing a robust opera-
tional framework for reproducible molecular imaging research.

B
Literature Review
Preclinical molecular imaging lies at the core of translational biosciences because it en-
ables non-invasive, quantitative, and longitudinal measurements in vivo that connect
molecular hypotheses with physiological and pathological outcomes. Among the available
modalities, small-animal positron emission tomography (PET) and hybrid PET/mag-
netic resonance (PET/MR) are especially powerful: PET provides extraordinary sensi-
tivity to picomolar level molecular processes, while MRI contributes superior soft-tissue
contrast, multi-parametric tissue characterization and spectroscopy without ionizing ra-
diation. This union supports high-value studies in oncology, neuroscience and cardiology
and it allows the same animal to be scanned repeatedly, improving statistical power and
reducing cohort sizes. [1,2].
B.1
Performance Evaluation of Preclinical Systems
Positron Emission Tomography (PET) has become one of the most powerful molecular
imaging tools, providing quantitative insight into biological and pharmacokinetic processes
at picomolar-level sensitivity. The introduction of hybrid imaging, most notably PET
combined with Magnetic Resonance Imaging (MRI), expanded the scope of preclinical
research by uniting PET’s molecular specificity with MRI’s anatomical and functional
detail [4].
The concept of PET/MR dates back several decades, but its realization was delayed due
to the incompatibility of conventional photomultiplier tubes (PMTs) with magnetic fields
and the complex electromagnetic design required for concurrent operation. Early “hybrid”
configurations were sequential, involving bed transfers between PET and MR systems or
tri-modal PET/CT/MR setups, which introduced temporal misalignment and physiolog-
ical variability. The first truly simultaneous preclinical PET/MR scanner, demonstrated
by Judenhofer et al., used an avalanche photodiode (APD)-based PET insert inside a 7 T
MRI. The system achieved 1.9 mm spatial resolution, 0.23% sensitivity, and less than 6%
loss in MR signal-to-noise ratio (SNR) [4], proving the feasibility of simultaneous acqui-
sition and inspiring the transition to silicon photomultipliers (SiPMs) and digital SiPMs
(dSiPMs), which are magnetically insensitive and offer superior timing resolution [5,7].
NEMA NU-4 2008 Framework and Performance Metrics
To standardize performance evaluation across small-animal PET systems, the National
Electrical Manufacturers Association (NEMA) introduced the NU 4-2008 protocol [8].
This standard defines uniform methodologies to measure spatial resolution, absolute sen-
sitivity, scatter fraction (SF), count-rate performance, and image quality (IQ) using a
41

dedicated phantom.
Goertzen et al. [9] compared eleven preclinical PET systems us-
ing this framework, demonstrating systematic improvement in resolution and sensitiviy,
establishing NEMA NU-4 2008 as the reference standard for quantitative scanner com-
parison.
The image-quality phantom provides three key metrics: (i) uniformity, as the percent
standard deviation in a central uniform region, (ii) recovery coefficients (RCs), derived
from 1–5 mm hot rods to assess partial-volume effects, and (iii) spill-over ratios (SORs) in
air and water inserts, indicating scatter and background contamination. Together, these
measures define the quantitative fidelity of small-animal PET systems. Energy resolution
(FWHM at 511 keV) and coincidence timing resolution (CTR) are also routinely reported,
for instance, SAFIR-II achieved CTR ≈221 ps, demonstrating progress toward time-of-
flight (TOF)-capable preclinical systems [24].
Technological Evolution and Hybrid Integration
Advances in detector technology have driven PET/MR performance improvements. SiPMs
replaced APDs and PSPMTs due to their compactness, high gain, and immunity to mag-
netic fields, while dSiPMs further enhanced timing precision and reduced afterpulsing.
Several representative systems illustrate this evolution: the “clip-on” PET insert evalu-
ated by Vrigneaud et al. achieved ∼1.3 mm resolution and 4.7% sensitivity under NU-4
standards [34], Courteau et al. integrated a dual-ring SiPM PET inside a 7 T MRI, reach-
ing 1.7–2.4 mm resolution and ∼8% sensitivity with minimal MR interference [12], and
Gonzalez-Montoro et al. developed the ScintoTube “edgeless” continuous-crystal PET in-
sert, achieving near-isotropic ∼1 mm resolution and 3–5% sensitivity with less than 5%
degradation of PET energy resolution under simultaneous MR operation [29]. Systems
such as SimPET (0.7–1.45 mm resolution, 4.2% sensitivity, peak NECR 151 kcps) [26]
and Bruker’s 7 T PET/MR (0.8 mm resolution, peak NECR 774 kcps) [11] confirm that
modern hybrid systems can deliver sub-millimeter resolution and high sensitivity at field
strengths up to 9.4 T.
Quantitative reliability in hybrid systems depends on accurate corrections for attenua-
tion, scatter, randoms, normalization, and dead time. MR-based attenuation correction
(MRAC) methods have evolved from Dixon segmentation to ultrashort echo-time (UTE)
imaging and machine-learning pseudo-CT generation, reducing soft-tissue bias reduced to
a few percent when validated when properly validated [6,30]. Furthermore, motion correc-
tion using MR navigators or data-driven gating and MR-guided PET reconstruction (e.g.,
anatomical priors in OSEM) enhance contrast recovery while preserving resolution [35].

Applications and Quantitative Integrity
Simultaneous PET/MR enables synchronized molecular, structural, and functional infor-
mation from the same subject under identical physiology. In oncology, FDG- and FLT-
based PET combined with diffusion or perfusion MRI allows precise mapping of viable
tumor subregions and therapeutic response [4]. In neuroscience, PET tracers for amyloid,
tau, and neurotransmission co-registered with structural MRI or fMRI improve disease
staging and longitudinal monitoring [5]. These combined approaches reduce biological
variability and increase statistical efficiency by allowing each subject to serve as its own
control.
Despite these advantages, PET/MR integration presents technical challenges, including
MRAC accuracy for bone and hardware, cross-modality interference, and residual partial-
volume effects.
Modern designs address these issues through non-magnetic materials,
RF shielding, DOI-enabled detectors, and continuous crystals.
Reported performance
confirms minimal MR degradation: compact 7 T systems preserve B0/B1 homogeneity
[12], and continuous LYSO inserts maintain PET energy and timing stability within ±5%
[29,36].
In summary, PET/MR has evolved from APD-based prototypes to robust SiPM-driven
systems capable of sub-millimeter resolution, high sensitivity, and full electromagnetic
compatibility.
The NEMA NU-4 2008 standard provides a common quantitative lan-
guage, while modern correction and reconstruction techniques ensure reliable quantifica-
tion. These advances define the methodological foundation for the present study, which
evaluates both baseline performance and high-activity behavior of a preclinical PET/MR
platform.
B.1.1
Evidence on Gaussian Post-filtering from PET Literature
Post-reconstruction Gaussian smoothing is widely used to regularize iterative reconstruc-
tions. In cardiac PET with 13N-ammonia, Kim et al. systematically compared filtered
backprojection and OSEM (2-6 iterations) with Gaussian kernels of 5, 10, and 15 mm.
A 10 mm filter improved visual image quality and increased SNR relative to non-filtered
images, while coronary flow reserve estimates did not change significantly across filter-
ing choices, overly aggressive smoothing (15 mm) tended to underestimate rest and stress
flow compared with reference reconstructions [14]. These findings illustrate the general
resolution-noise trade-off: moderate smoothing can stabilize noise without unacceptable
loss of quantitative fidelity for certain endpoints, whereas excessive smoothing degrades
contrast recovery.
Although the quantitative targets and scales differ in small-animal
PET/MR, the same principles motivate reporting and benchmarking of filter widths to-
gether with RC, SOR, and uniformity metrics.

for quantitative PET, particularly in small-animal imaging where injected activities can
reach hundreds of MBq in small volumes. At elevated count rates, dead time, pile-up,
and random coincidences distort the proportionality between measured and true activity,
affecting quantitative accuracy, dynamic modeling, and radionuclide kinetics [9,27].

try to production quality control—and is especially important in preclinical imaging,
where small system volumes amplify the effects of dead time [29]. Two models describe
detector response under high load: the non-paralyzable and paralyzable models, repre-

Empirical Studies and System Evolution
Early scanners, such as the microPET Focus F120, exhibited NECR saturation near
809 kcps at 88 MBq [27], illustrating the onset of non-linearity at high activity.
Sub-

higher optimal count rates [28]. The adoption of SiPM and dSiPM detectors, together
with custom Application-Specific Integrated Circuits (ASICs), further increased count-
rate capability and reduced dead-time losses [16].

Digital PET/MR systems like SAFIR-II achieved coincidence timing resolution of 221 ps

Linearity, Stability, and Correction Models
Maintaining quantitative linearity over time is essential for reliable longitudinal and dy-
namic PET imaging. Courteau et al. [12] conducted 24-hour decay tests using a uniform
syringe phantom on a 7 T PET/MR system, confirming that measured activity closely
followed theoretical decay with negligible drift.
Emvalomenos et al. [23] applied non-
paralyzable dead-time models (effective constant ∼122 ns) to a PET insert operated both
in standalone and simultaneous MR mode, achieving < 5% residual deviation.
Dead-time correction may be implemented analytically, using the equations above with
fitted system constants, or empirically through lookup tables from experimental count-loss
curves. When properly validated, both approaches restore quantitative linearity across
the system’s dynamic range [16,24,28].
Implications for Preclinical Imaging
Modern preclinical PET/MR systems can sustain high-activity acquisitions without com-
promising accuracy, enabling short-frame dynamic imaging and high-dose studies. Im-
provements in detector response, readout speed, and correction modeling have expanded
the linear dynamic range by nearly an order of magnitude compared with early designs.
Current SiPM-based scanners routinely maintain linearity up to 400–500 MBq with NECR
peaks above 700 kcps [11,24,29].
Collectively, these developments demonstrate that high-activity PET/MR is now both
technically feasible and quantitatively reliable. The adoption of standardized NU 4 pro-
cedures ensures inter-system comparability and enables robust benchmarking for new hy-
brid designs. These principles form the methodological foundation for the experimental
high-activity evaluation presented in this thesis.
Interpreting NECR and Designing for the Optimal Operating Point
The NECR curve encapsulates how a system trades signal and contamination as activity
increases: at low activity, T rises faster than S and R, so NECR improves, near the peak,
incremental gains in T are offset by randoms and dead-time losses, beyond the peak,
NECR falls as randoms and lost events dominate. In practice, the “useful” operating

window is the activity range bracketing the NECR peak in which bias (from dead time
and randoms) remains small and precision (SNR) is near-maximal [8, 9, 27, 28]. Early
systems reached their peak at tens of MBq in mouse/rat phantoms (e.g., Focus F120
at ∼88 MBq) [27], whereas modern SiPM/dSiPM platforms move the peak substantially
higher and flatten the descending limb through faster electronics and improved timing
(e.g., SAFIR-II >1.1 Mcps NECR at ∼451 MBq) [24].
Several acquisition and system parameters shift the NECR curve. Narrowing the lower
energy threshold reduces scatter and randoms but may lower sensitivity, tightening the
coincidence timing window suppresses randoms at the cost of occasional true losses if the
CTR is modest, extending axial coverage raises sensitivity (and scatter) and can increase
both NECR peak height and the activity at which the peak occurs [9,28]. Empirical data
confirm these trends: the Inveon study quantified how energy and timing choices move the
peak and the saturation onset [28], while dSiPM-based PET/MR inserts sustain higher
instantaneous rates with minimal dead-time penalties due to short shaping times and per-
channel processing [16, 24]. System geometry and shielding further modulate randoms:
larger detector solid angle increases singles and therefore randoms, but superior timing
discrimination and optimized delayed-window subtraction temper their impact [8,9].
Dead Time in Practice: Channel vs. System, Paralyzable vs. Non-paralyzable
Real scanners exhibit a mixture of paralyzable and non-paralyzable behavior because dead
time accrues from multiple stages (detector pixel, ASIC, DAQ buffers). Pixel-level dead
time in SiPM/ASIC front-ends tends to be closer to non-paralyzable, whereas shared re-
sources (e.g., coincidence processors) can exhibit paralyzable characteristics under bursty
loads. Accordingly, practical calibration fits both models to measured prompt/true curves
and selects the model (or effective composite) that minimizes residuals across the intended
activity range [23,24]. Systems with highly parallel readout and short shaping constants
(e.g., SAFIR-II) show gentler roll-off and smaller model residuals at high activity, consis-
tent with shorter effective dead time and better pile-up rejection [24].

series for 18F or a shorter series for 68Ga. Several studies have implemented variants
of this design and report < 5% deviations from theoretical decay when corrections
are proper [12,23,29].

Isotope-Dependent and Sequence-Dependent Considerations
High-activity performance also depends on isotope choice and MR sequence load. Isotopes
with higher positron energy (e.g., 68Ga) exhibit longer positron range, which primarily
influences spatial resolution and RCs rather than dead-time per se, however, their shorter
half-life compresses the time window for decay-series sampling, increasing sensitivity to
timing metadata and frame definitions. Conversely, 18F affords longer validation windows
and typically lower prompt scatter, simplifying dead-time calibration [9]. In simultaneous
PET/MR, fast gradient or RF-intensive sequences can, if insufficiently shielded or syn-
chronized, introduce small disorders to PET timing or energy baselines. Modern systems
report minimal mutual interference when engineered correctly (e.g., B0/B1 preserved,
PET photopeak and CTR within ±5% under sequence load) [12,29,36].
Translating these findings into operating guidance, small-animal PET/MR experiments
should (i) plan injected activities to stay near the NECR peak during the time of interest,

Complete Python Implementation for NEMA IQ Analysis
This appendix contains the full Python source code developed for the systematic analysis
of NEMA IQ phantom data, which formed the methodological foundation for the image
quality assessments presented in this thesis. The software represents a complete imple-
mentation of the NEMA NU 2-2018 standard for PET image quality evaluation, with
extensions for multi-radionuclide support and automated batch processing.
Software Architecture and Key Features
The code is structured into several modular components:
• DICOM Processing Module: Handles reading of DICOM series with proper sort-
ing, rescaling, and quantitative calibration using dose calibration factors to convert
pixel values to MBq/mL
• ROI Generation Engine: Implements geometric cylinder creation for all NEMA
phantom regions with radionuclide-specific coordinate presets
• Metric Calculation Core: Computes recovery coefficients (RC) using the maxi-
mum pixel method, spill-over ratios (SOR) for cold regions, and uniformity statistics
with proper error propagation
• Visualization Toolkit: Provides quality control overlays to verify ROI placement
and data integrity
• Batch Processing Framework: Enables automated analysis of multiple recon-
struction algorithms and radionuclide datasets
Computational Environment
The analysis was performed using Python 3.8 with the following key libraries:
• pydicom: DICOM file parsing and metadata extraction
• numpy and pandas: Numerical computing and results organization
• scikit-image: Image processing and ROI geometry operations
• matplotlib: Quality assurance visualization
Implementation Notes
The code includes comprehensive error handling for DICOM parsing, robust sorting of
image slices using both InstanceNumber and ImagePositionPatient fallbacks, and proper
unit conversion for quantitative analysis. All NEMA metrics are computed with uncer-
tainty estimates following standard error propagation principles.
