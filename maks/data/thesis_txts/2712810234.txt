Abstract
When doing inference on the parameters and states of stochastic differential
equations (SDEs) then there are a number of different methods to apply. One
of these methods consists of fitting a statistical latent variable model, where
the hidden states are considered latent variables. The fitting of these models is
performed in TMB which performs an integral approximation using the Laplace
approximation. We also compare and validate the inferred parameters with the
continous time stochastic modelling (CTSM) method implemented in the ctsmr
package. This project focused on examining the issues that occur when approx-
imating the transition density with an Euler-Maruyama scheme. The SDEs we
consider are the Cox-Ingersoll-Ross process and Geometric Brownian motion.
These SDEs were examined as we know their exact transition density. Due to
its simplicity then Geometric Brownian motion was only used to verify approx-
imations of the transition density.
We considered how the transition density approximations depended on the SDE
transform and number of latent variables. We considered how the score statistic
bias depended on the true model parameters and quality of the approximation.
We also examined how the score statistic bias translated to the parameter bias.
From the score statistic then there is a theoretical result for the expected pa-
rameter bias, we compared this estimate with simulated parameter estimates.
Parameter estimation was also done with CTSM.
To achieve a good approximation of the transition density, then it was necessary
to compute the transition density of the Lamperti transformed SDE. This is be-
cause as the number of states increases then the state estimates will be chosen
such that they minimize the diffusion, which if state dependent will skew the
state estimates as the state number increases. It was also possible to apply the
Euler-Maruyama scheme to different transforms of the SDE while computing
the same transition density. In the case of Geometric Brownian motion then
this was important for a good approximation.
There was a bias in the score statistic for the CIR-process when using the
Euler-Maruyama approximation and increasing system noise caused this bias
to increase.
From this bias we would expect a negative parameter bias and
that the velocity of the CIR-process had the most negative bias. This was not
verifiable through simulations. We also discovered that a bad transition density
approximation will lead to a large negative parameter bias.
CTSM seemed to achieve similar parameter biases to TMB, but CTSM could
not estimate parameters for large enough system noise unlike TMB.

Introduction
Stochastic differential equations (SDEs) are a family of models useful for com-
bining theoretical knowledge and stochastic behaviour due to uncertainty. The
task of estimating the embedded parameters and states of an SDE can be seen
as finding the parameters Оё that maximize the likelihood L given a sequence of
N measurements YN = [Yt0, ..., YtN ] taken at times 0 вүӨt0 < ... < tN

known but can be approximated.
There are several methods for solving the problem of estimating transition den-
sities and finding optimal states and parameters. Among these methods are
hidden markov models [Thygesen, 2021, p. 201-202], the extended Kalman filter
combined with numeric optimization methods [Kristensen and Madsen, 2003].
Here focus on using statistical latent variable models fitted with the R package
TMB. This is because it is a relatively recent method. A SDE which we know
the analytical transition density of is the Cox-Ingersoll-Ross process (CIR). We
use this model for data generation, such that we can estimate parameters and
compare with the known values.

5.4
TMB method
5.4.1
TMB architecture
TMB is a R programming package useful for fitting statistical models with la-
tent variables. TMB requires an user defined C++ template that computes
the objective function. TMB integrates the C++ template with the packages
CppAD and Eigen. TMB computes the derivatives and objective function esti-
mates which R uses to evaluate the negative log likelihood. An overview from
the TMB documentation is shown in figure 1.
To compute the negative log likelihood then TMB needs to approximate an in-
tegral of the objective function over the latent variables. This is done with a
Laplace approximation in TMB, which requires computation of the hessian and
gradient of the objective function.
The CppAD package performs automatic differentiation.
This method com-
putes the derivatives of a function f by utilising that any function is a sequence
of arithmetic operations and elementary functions with known derivatives. It
can then use the chain rule to combine the intermediary derivatives [Kristensen
et al., 2016, p. 1-3].

7.4
Discussion and conclusion
As expected applying the Euler-Maruyama scheme to the Lamperti transformed
system seems to achieve the same probability estimates as the analytical prob-
ability. When the Euler-Maruyama scheme is applied to the system in natural
domain then the accuracy of the approximation is dependent on the number of
latent variables (see figure 2 and 5). Increasing the number of hidden states
makes the Euler-Maruyama scheme in the natural domain fit the analytical dis-
tribution better as the Euler-Maruyama scheme becomes more appropriate, but
the density estimate is still biased (see figure 3 and 5). We see that the relative
error for the Euler-Maruyama scheme with no Lamperti transform increases
as a function of the time between observations, indicating that increasing the
number of hidden states will not give a good approximation if the observation
time step is too large.
Geometric brownian motion is a relatively simple process as estimates with the
Lamperti transform and Euler-Maruyama scheme gives equivalent results to the
analytical probability and the Lamperti transformed system has no state depen-
dent drift. Next we will consider the CIR-process as it is a more complicated
SDE.

8.6
Discussion and conclusion
8.6.1
TMB using the likelihood of the log transformed variables
From figure 7 and 8 we see that there is a choice of hidden states that leads to
a decent approximation of the analytic density, but too many states leads to a
bad approximation as show in figure 10. We also see that TMB estimates most
of the hidden states to be above 1, but from 11 we would expect the mode to be
around or below 1 for X0.5. As the number of states increases then TMB will
prioritise the choice of states that minimizes the log diffusion of the transformed
systemвҖҷs SDE over minimizing the contribution of Brownian motion as discussed
in section 5.6. For p(log(Y0) вҶ’log(Y1), 0 вҶ’1) then we know the hidden states
will be skewed to higher values as the number of hidden states increases.
8.6.2
TMB using the likelihood of the Lamperti transformed vari-
ables
When we consider using the transition density in Lamperti space then the es-
timated hidden states are closer to the mode in the natural domain compared
to using the log-transform (see figure 13 and 11). Another important difference
is that the approximated transition densities do not diverge from the analyt-
ical result as the number of hidden states increases (see figure 14). There is
some difference between the used SDEs, but they both converge to the same
value. The Euler-Maruyama scheme applied to the Lamperti transformed SDE
gives slightly worse estimates according to figure 15, but both methods converge
to similar errors as the time between observation steps increases and we only
compare in one point. This could be because the distribution becomes more
uniform. Our choice of parameters makes the distribution of Y1 easy to approx-
imate, if we consider an unstable CIR-process with states close to the boundary
33
==PAGE:34==
0 then the approximations will not work as shown in appendix C.
8.6.3
Conclusion
TMB needs to compute a negative log likelihood function to estimate hidden
states and parameters. The experiments in this report suggests that the best
result are achieved by computing the likelihood with the transition density in
Lamperti domain. Not doing this risks TMB estimating the hidden states by
prioritising minimization of the diffusion which can give states that diverge as
the number of hidden states increases.
An issue arises when the estimated hidden states are close to 0 this is a boundary
point, so the Laplace approximation would not be ideal. We want to compute
the likelihood with transition densities in Lamperti domain, so we have to inte-
grate over the same domain.

12
Overall discussion
In the experiments described in section 7 and 8 we examined the issues with
approximating the transition density. Geometric Brownian motion is well ap-
proximated by both the Laplace approximation and Euler-Maruyama scheme
when applied the log transform, so using the Euler-Maruyama scheme on the
log transformed SDE and computing the log transformed SDE gave an efficient
approximation.
When approximating the CIR-process then it became apparent that if the dif-
fusion of the SDE depends on the latent variables then increasing the number of
latent variables will give a diverging estimate. This is because when performing
the Laplace approximation needed by TMB to compute the likelihood then we
estimate the latent variable that minimizes a sum. This sum will if the number
of hidden states increases become dominated by the logarithm of the SDEвҖҷs
diffusion.
From the Geometric Brownian motion experiment we see that it matters what
SDE we are applying the Euler-Maruyama scheme to, even though we are com-
puting the same likelihood.
This can also be seen when approximating the
CIR-process (figure 10 and 14). If the transition density is badly approximated
then we can suggest applying the Euler-Maruyama scheme to a different SDE
while still computing the transition density in the Lamperti domain using the
change of variable rule.
A source of error when approximating the transition density can occur when the
state estimates are close to the boundary. The approximation of an unstable
CIR-process where this is the case can be seen in appendix C.
The experiments on estimating the score statistic for 500 observations suggests
that the score is biased when applying an approximation of the transition density
(see section 9). The state estimates seemed acceptable for the approximations
and the results were very similar for the different transition densities we approx-
imated. To understand how the score bias depended on model parameters then
we tried to vary the system noise while keeping the fraction between system
noise and measurement noise constant (as described in section 10). The score
statistic bias could then be used to estimate the expected parameter bias, this
was done with a theoretic result. We then compared the expected parameter
bias to a simulated parameter bias (section 11). While there tended to be a neg-
ative bias for most parameters then the velocity parameter of the CIR-process
did not behave as we would expect from the theoretic result. This could be
because our sample size was not large enough, but it could also be due to the
existence of local optima. As such from this report then we can not expect the
parameter bias estimated from the score bias to explain the simulated bias.
We also tested the importance of varying the number of hidden states used in
the approximations. We found that a bad approximation had a worse negative
bias, a time step of 0.01 between hidden states seems like a good choice for the
approximation. Finally we compared our parameter estimates with CTSMвҖҷs
estimates which had a similar distribution to TMBвҖҷs estimates, but for a large
enough system noise CTSM had diverging estimates and could not produce re-
55
==PAGE:56==
sults. If there is a lot of system noise Оі then TMB would be able to produce
estimates, but there will be a considerable bias.
12.1
Options for future work
There might be other issues when doing these approximations that we have
not considered. The Laplace approximation assumes there is a unique choice of
latent variables that maximize the likelihood, the double well model is a SDE
with a stationary distribution with two modes for certain parameter values.
This type of distribution is ill-fitted with a Gaussian bell, causing a possible
error when using the Laplace approximation. Another important area is SDEs
in multiple dimensions where non-linear problems have to be approximated as
linear, but might not be necessary in TMB.
More comparative experiments between CTSM and TMB implementations would
help determine situations where one method is more appropriate than the other.
It would also be ideal to compare with the time complexities of the different
methods.
The box-cox transform is to make data resemble a normal distribution, a normal
transition density would be ideal for applying the Euler-Maruyama scheme. So
this could be an useful transform to apply difficult SDEs.
Another point of interest for future work is that we only considered the Euler-
Maruyama scheme for approximating the transition density.
Other schemes
include the Heun method or Milstein scheme. Which have better convergence
properties than the Euler-Maruyama scheme.
13
Conclusion
TMB can be used to estimate the embedded parameters of a SDE, but it re-
quires that the transition density is computed in the Lamperti domain. The
Euler-Maruyama scheme can be used to approximate the transition density, im-
provements can be made by applying the scheme to a different transform of the
SDE and using the change of variable rule to evaluate the transition density in
the Lamperti domain.
The theoretical result to estimate parameter bias from score bias achieved dif-
ferent results to simulated parameter estimation. An option for this could be a
small sample size or the existence of local optima that the numerical optimiza-
tion routine converges to.

Inference in nonlinear stochastic differential equations - Project plan
Main counsellor: Uffe HГёgsbro Thygesen
Co-counsellor: Jan Kloppenborg MГёller
Motivation
Stochastic differential equations are a popular class of models that combine
system-specific knowledge while allowing for model imperfections through
stochasticity. When building these types of models then it becomes necessary to infer
the parameters of the model from observed time series.
There are several methods and corresponding programming packages that can be used
for parameter estimation. This project will aim to establish one or more benchmarks for
comparing two different programming packages each employing different methods for
parameter estimation. We compare Template Model Builder (TMB) that fits statistical
latent variable models and Continuous Time Stochastic Modeling (CTSM) that utilizes
the extended Kalman filter.
The comparison will be done by simulating processes and using case studies to show
interesting aspects with the methodologies.
TMB performs a Laplace approximation and requires transition probabilities between
states to compute a likelihood function. The Laplace approximation, to be accurate,
requires the latent random variables to be normally distributed i.e. the difference
between adjacent states must be normally distributed. This is not always the case, in the
Cox-Ingersoll-Ross process then the transition probability is that of a non-central
chisquare distribution. This makes the Cox-Ingersoll-Ross process useful as an initial
benchmark since we can choose to simulate it with theoretical transition probabilities or
different discretization schemes. We can also test the TMB implementations using
approximate transition probabilities or the theoretical transition probability and test the
importance of choice of state space and temporal discretization schemes.
The TMB and CTSM programming packages are employed at DTU compute and as such
it is important to establish quantifiable properties for the estimators obtained with each
method, so the methods can be compared.
TMB and CTSM are both available as R packages, so experiments and parameter
estimations will be carried out in R.
==PAGE:69==
Prior work
The Cox-Ingersoll-Ross process is a well-researched process and there have been
experiments to infer its parameters using different methodologies such as the extended
Kalman filter [1].
Research questions
What are the main differences between estimating parameters with the extended
Kalman filter and latent variable model approach?
How do these differences translate into practice?
Are there situations where these differences are more significant than others?
How does the choice of state space and temporal discretization schemes affect the latent
variable model approach?
