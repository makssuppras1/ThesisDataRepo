Summary
The general push towards more accurate medical imaging methods has led to the devel-
opment of several three dimensional (3D) imaging techniques, based on MRI, CT and
Ultrasound. This last one, while having great potential, still has not been applied exten-
sively beyond research. The purpose of this thesis is to develop techniques to perform 3D
ultrasound imaging using a Row-Column array. This is a technically challenging system
that has the potential of providing real-time cost-effective 3D medical images.
Throughout this work, a complete Row-Column imaging system with Synthetic
Aperture has been designed, veriямБed and simulated. Hence, a detailed description of the
inner working of such systems is provided and their advantages as well as limitations are
explored. Imaging using a conventional delay and sum beamformer is compared against
synthetic aperture beamforming for Row-Column arrays considering line elements as
ultrasound sources. Moreover, a virtual array in transmission is successfully implemented
for this transducer to increase the signal to noise ratio of the imaged volume.
The simulations and analysis performed in this work show results from the ямБrst Row-
Column with Synthetic Aperture and Fixed Focusing obtained at the Center for Fast
Ultrasound.
keywords: medical ultrasound, 3D, row column, imaging, integrated apodization, syn-
thetic aperture, synthetic apodization, beamforming, virtual sources.

CHAPTER1
Introduction
1.1
Purpose
The purpose of this project is to investigate the Row Column Synthetic Aperture using
ямБxed focusing in transmission. A setup has been designed, implemented and simulated
using Field II and the row-column beamforming tool. The image quality is assessed by
the measurements of the resolution and contrast of the point spread functions obtained.
1.2
Motivation
Nowadays, 3D ultrasound is almost used only for obstetric ultrasonic, where the fetus is
surrounded by water. These conditions make it possible to acquire 3D ultrasound images.
For other medical applications, 3D imaging with high quality is still a technical challenge
(due to large, complex, wiring, the heating of the probe and the large data rate provided
by a fully-addressed array). The 2D Row Column Addressing, although it is still in an
early research stage, offers a solution to the high number of channels needed that aims to
be a protagonist in the future of 3D imaging.

CHAPTER2
Theory
This chapter provides a description of the theoretical principles and the techniques that
have been applied throughout this project. For a detailed description of other fundamental
concepts in ultrasound, the reader is referred to the literature [1].
To begin, section 2.1 presents a literature review of the main three-dimensional (3D)
ultrasound techniques to date. This leads to a detailed description of element addressing
in Row Column imaging (section 2.2) and 3D imaging using a Row-Column Array with
ямБxed focusing in transmission (section 2.3). An attempt to overcome some of the side
effects of Row-Column arrays is to use integrated apodization, explained in section 2.4.
Finally, the approach to beamforming with these arrays is explained in section 2.5).
2.1
State of the Art
Medical three-dimensional (3D) ultrasound has been in the spotlight for clinical imaging
research since the ямБrst commercial 3D scanner was released in 1989. Its effectiveness as a
method of obtaining medical images, has since been demonstrated in several applications,
however 3D imaging is only available in high-end scanners and there is still a long way
to achieve the full potential of this technology and making it cost effective. This last
characteristic is vital since one of the main advantages of ultrasound imaging compared
to other methods is its relative low cost, aside from being a non-invasive method.
In the last two decades, several authors have developed different approaches for 3-D
ultrasound imaging [2] and [3]. A simpliямБed description of these techniques is that 3D
ultrasound imaging can be obtained using a fully-addressed two dimensional (2D) array of
transducer elements or using a moving 1D array. 2D arrays present a lot of inconveniences
in both economical and technical aspects when building them. Thus, most of todayтАЩs
3D ultrasound imaging systems use 1D arrays to acquire the images. These techniques
go from freehand scanning a 1D array with an operator, which leads to distortions since
normally images are tracked without information of the position or the orientation of
the images, to more advanced mechanical 3D probes [4] that are based on a 1D array
transducer inside a hand-held instrument that has control of rotation, tilt or translation
to avoid distortions. This techniques are used in combination with 3D reconstruction
algorithms however, these algorithms are out of the scope of the main focus of the study
and will not be further elaborated.
2D array probes are considered the future to create a three-dimensional real-time

2.2
Row Column Arrays
A central concept in this thesis is to understand what a Row-Column Array is and how
3D Ultrasound Imaging is performed with such an array.
The driving principle behind row-column addressing is to transform a 2D array into
two orthogonal 1D arrays. This is accomplished by addressing an entire column or
row of elements instead of each individual element, illustrated in ямБg. 2.1. As stated in
section 2.1, this reduces the complexity of the wiring design and the power disippation
thermal effects of a fully addressed 2D array by reducing the number of channels from
N 2 to 2N. Therefore, a Row-Column array with 192 ├Ч 192 = 36, 864 sub-elements will
only have Ntot = 2 ├Ч 192 = 384 channels. Each single element in a row-column array is
a full row or a full column element that consists of Nsub sub-elements, meaning that all
the sub-elements from the same row or column will be excited with the same signal and
the same time delay.
Both rows and columns can be used interchangeably to transmit and receive but
throughout this study the convention "Row-Column тЖФTransmit-Receive" is followed.

2.3
Row Column Focusing
As previously described, every element in a Row Column (RC) array consists of Nsub
sub-elements that are emitting the same signal with the same time delay. This means that
the sources for the ultrasound ямБelds are line sources instead of point sources. This can
be seen in ямБg. 2.2, which shows how these elements have a line segment as focal point
instead of a single focal point like traditional ultrasound sources. Throughout this section,
concepts in focusing will be explained for a 1D array and then extended to 2D arrays.
Figure 2.2: Focal points with 1D linear array (left) and Row Column array (right).

Chapter 2. Theory
Focusing is a basic technique that permits the ultrasound ямБelds to converge in phase
at a certain focal point (1D array). This is achieved by introducing time proямБles in
transmission or reception. These proямБles are calculated at each of the emitting elements
so that the wave each one transmits is added coherently in the desired direction or at
certain depth and incoherently otherwise, as shown in ямБg. 2.3.
In a RC array, each of the rows in transmission is equivalent to each of the elements
of a 1D array. Therefore, since the delay proямБle is the same for all sub-elements in a
row, the focusing point in the elevation plane will be constant, and therefore a line, as
shown in ямБg. 2.2. Focusing in reception with the columns follows an analogous process
in an orthogonal direction, this is illustrated in ямБg. 2.4. The limitation of such an array
is consequently, that a focusing point cannot be obtained along the axis of the elements,
which severely limits the resolution in this direction.
Two basic methodologies can be used for focusing, Fixed Focusing and Dynamic
Focusing. Fixed Focusing only achieves focusing at a certain focal depth. To maintain
focus over the axial direction, dynamic focusing is needed. This is performed at the
beamforming stage (in reception) using a delay and sum beamformer or synthetic aperture
beamforming, which is explained in section 2.5.2.

2.4.2
Sliding Aperture and Electronic Apodization
A sliding aperture is a sub-aperture that is moved along the total aperture. When using
the Row-Column of N elements, an sliding aperture of Nact can be moved along the
transducer by choosing which elements are active or not. This is done using electronic
apodization. As opposed to integrated apodization, this one is not applied during the
fabrication process. It can be changed at any time, since it is applied electronically by
exciting elements with different voltage amplitudes weighting the inямВuence that each
single element has.
By controlling the active sub-aperture of the transducer, the acceptance region can be
moved along the aperture. Additionally, different windows can be applied to the active
aperture, controlling its size and shape. This has a direct impact in the radiation lobes,
as a larger aperture will produce a more directive main lobe. Additionally, different
weighting functions will affect the sidelobe level, which can be controlled using electronic
apodization windowing.

2.5.2
Synthetic Aperture
Synthetic Aperture (SA) is an ultrasound technique that aims to maintain a constant resolu-
tion along the axial dimension by combining multiple low resolution images produced by
single element transmissions and receiving with the whole aperture [22]. In Row Column
Synthetic Aperture Imaging (RC-SAI), for each emission a Low Resolution Volume
(LRV) is acquired. Several LRVs are then summed coherently into a High Resolution
Volume (HRV). This is the beamforming technique implemented in previous studies for
imaging with a Row Column array, and while emitting with single elements provides the
largest possible number of LRVs to coherently sum into a HRV, it has some drawbacks.
One the one hand, there is a low signal to noise ratio (SNR) that results from ямБring from
only one element and on the other hand the frame rate decreases since N ямБrings happen
between consecutive frames.
In order to solve this with 1D array transducers, Karaman et al [23] proposed a
solution combining several elements that emit de-focused wave. This creates a so-called
virtual source, which insoniямБes a region with higher power than a single element. This
approach is applied in this work, with the difference of using focused waves as opposed
to defocused ones. While the former generate a smaller acceptance region, if the ROI
is deeper than the focal depth (ямБg. 2.9), the acceptance region is large enough to use
several LRVs to beamform a certain region. This combination of multiple elements in
transmission into a ямБxed focused point follows the approach explained in section 2.3.
In this work, NLRV = N тИТNact + 1 LRVs are obtained by emitting with a ямБxed
focusing sliding aperture of Nact rows in transmission and receiving with N columns.

close to the focal line leads to being insoniямБed by less emissions that contribute to the
beamforming point since the acceptance region is narrower than at deeper points.
This is also combined with the fact that the point is located close to the edge of the
aperture at Y = ┬▒47. Similarly to the case of the scatterer at P(47╬╗, 0, 75╬╗), there is

combination of these leads to having a PSF with less energy than the central ones, and in
this case, even lower than the -60dB limit. This is more noticeable in the transmission
plane than in the receiving plane since it can be already detected at the PSFs located at

this lateral position. In the analysis of resolution, it can be seen that the FWHM gets
much higher from Y = 0 to Y = 47╬╗ meaning that the resolution is worse. In the case
of contrast, some problems were encountered when measuring the Cystic Resolution at

Project Conclusion and Future work
The initial aim of this MasterтАЩs Thesis was to implement a Dual Stage Beamformer with a
Row Column Array for 3D ultrasound imaging. Throughout the course of this project it
was seen how this goal was farther away than initially expected due to the peculiarities in
the nature of the Row Column Array. This array presents a steep learning curve from a
linear 1D array and there are a few non trivial ideas that need to be understood in detail
before proceeding to more advanced beamforming. However, much has been achieved
and the ямБnal results of this thesis are satisfactory.
Moreover, this project of exploratory nature has been the ямБrst study that successfully
implements a Sliding aperture Row Column array with virtual sources in transmission
at the Center for Fast Ultrasound. As already shown in section 4.3.1, this conямБguration
provides an optimal FWHM of FWHMXZ = 2.2╬╗ and FWHMY Z = 5.15╬╗.
To reach these ямБnal results a long processing time had to be taken into account, which
in the last stretch of the project proved to be critical. The current performance of the RC
Beamforming tool processes 250000 voxels (60 samples in each direction x, y an z) from
one emission in 11.4s in a standard PC [17], which for a large sampling resolution along
several planes in space, results in long waits before results can be obtained and analyzed.
During the implementation, a lot of time was dedicated to getting the expected results
from the RC beamforming tool since the initial ones did not agree with the expected ones.
Towards the end of the project, an undocumented bug was found in the RC beamforming
tool. It was found that the coordinates to be input for the points to beamform in the RC
beamforming tool were ямВipped, being (y, x, z). This caused unexpected errors when
applying Synthetic Dynamic Apodization.
Notwithstanding, the RC Beamforming tool, along with the research performed by
Rasmussen, et al. has been tremendously valuable in this project. It is my hope that this
project served to identify key points to improve in this tool and to provide unique insights
to keep advancing towards real time 3D ultrasound imaging with such a clever, albeit
complex array design.
To sum up, this project started with an implementation of Fixed Focusing at Tx and
Rx where the data is obtained plane by plane and without using the RC Beamforming
Tool so point sources instead of virtual line sources were assumed. These results were
improved with the implementation of the Row Column Synthetic Aperture with ямБxed
focusing in transmission. Here, the importance of the synthetic apodization was raised
since its inямВuence in the ямБnal imaging system is notable.
An additional study was performed and it demonstrated that the ghost echoes can be

The results show that this set up can be a good basis for future research. Row Column
could potentially be used for 3D imaging but there is still a long way to achieve high
quality imaging. The next steps to continue this project should be centered in analyzing
