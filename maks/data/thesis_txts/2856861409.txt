Abstract
In this project we will study two types of function fields over finite fields, namely the Giulietti-
Korchm´aros function field and certain Fermat function fields.
Both are examples of maximal
functions fields, which means that they are of particular interest in relation to coding theory. One
goal of this project is to prove that they are in fact maximal, and another goal is to compute
the Weierstrass semigroups associated with these function fields.
For the Giulietti-Korchm´aros
function field, we compute the semigroup of every place, while for the Fermat function field, we
compute Weierstrass semigroups for some places but not all. We conclude the project by presenting
a conjecture about the Weierstrass semigroups of the specific Fermat function fields and by pointing
out areas that could be interesting future work.

Chapter 1
Introduction
A maximal function field F over a finite field Fq is a function field that achieves the maximal
possible number of rational places subject to the Hasse-Weil bound,
|N(F) −(q + 1)|≤2gq1/2,
where N(F) denotes the number of rational places of F, and g is the genus of F. In this thesis, we
will present two examples of such function fields, namely the Giulietti-Korchm´aros (GK) function
field and certain Fermat function fields. We will first show that they are in fact maximal. For
the GK function field, this is achieved by constructing a series of Kummer extensions in order to
determine the genus of the function field and thereafter by directly counting the number of rational
places.
In the case of the Fermat function fields, we will show that they are subfields (up to
isomorphism) of the Hermitian function field over Fq2, a well-known maximal function field. This
inclusion then implies the maximality of the Fermat function fields.
Additionally, we compute the Weierstrass semigroups for certain places in these function fields.
The Weierstrass semigroup of a place P can be understood as the set of integers m for which there
exist functions in the field with P as their only pole and pole order m. By definition, the genus of a
numerical semigroup Λ is g(Λ) := |N0\Λ|, in particular if H(P) denotes the Weierstrass semigroup
of a place P and G(P) denotes the gaps of H(P), we have
g = |G(P)|= |N0\H(P)|.
We will determine the Weierstrass semigroups using two main approaches:
1. Constructing telescopic sequences contained in H(P) and comparing their genera to g.
2. Constructing functions with certain valuations to identify gaps in H(P) and then ensuring
that the number of found gaps matches the genus of the function field.
The Weierstrass semigroup is the same for all but a finite number of places, namely the Weierstrass
places, and for the GK function field we will also determine these. We will finish the thesis by
presenting a conjecture regarding the Weierstrass semigroups in a specific Fermat function field.
Before diving into it, we bring a small section on one of the applications of the theory that we
treat in the thesis, namely coding theory, followed by a brief discussion of research in this field.
Applications to Coding Theory
Coding theory is the study of detection and correction of errors that occur due to noise in the
transmission of information.
Maximal function fields are used in the construction of algebraic
geometry (AG) codes, an important and well-known class of error correcting codes introduced by
Goppa in 1982 (see [1]). An AG code is characterized by three key parameters:
 the length (n) which is the total amount of data transmitted,
1

 the dimension (k) which is the amount of information encoded, and
 the minimum distance (d) which says something about the number of errors the code can
detect and correct.
The dimension relative to the length, k/n, is called the rate of the code and is an indication of how
much data we actually need to send compared to how much useful information it encapsulates. In
general, a good and efficient AG code should have a large minimum distance, enabling it to correct
a lot of errors, as well as a large rate. However, the three parameters n, k and d are connected by
the Singleton bound,
k + d ≤n + 1,
meaning that the code needs to balance k against d. The best way to get large values for both
of these are using codes that allow for n to be very large, and this is where the maximal function
fields come into play. They are exactly the functions fields with the maximal number of rational
places, which directly determine the length of the codes derived from them. In fact, AG codes
are constructed by evaluating rational functions from a function field at a set of distinct rational
places, P1, . . . , Pn.
The collection of usable functions is called the Riemann-Roch space, L(G), and depends on
the choice of a divisor G. If we define D := P1 + · · · + Pn, then CL(D, G) denotes the code that
evaluates functions from L(G) at the places P1, . . . , Pn. Note that CL(D, G) has the dual code
CΩ(D, G) defined using differentials instead of functions, which is in fact also an AG code. The
designed distance of the code CL(D, G) is the integer n −deg G, while for the code CΩ(C, D) it is
deg G −(2g −2), and these values work as lower bounds for the minimum distance of CL(D, G)
resp. CΩ(D, G).
The dimension of L(G) is determined by the degree of G and the genus of the function field,
and this influences both the dimension of the code k and the minimum distance d. The divisor G
is chosen such that the places P1, . . . , Pn are not in the support of G; in the simplest case, G is
chosen as aQ for an a ∈N and a place Q distinct from Pi for i = 1, . . . , n. Knowing the Weierstrass
semigroup of any place of a function field helps in choosing a place Q that leads to high quality
codes by controlling which functions appear in L(G). In particular, G = aQ will obtain a better
value for k exactly when Q is a Weierstrass place ([2, Section 3.1.2]).
A Look at Weierstrass Semigroups for Multiple Places
In this thesis we will only compute Weierstrass semigroups for a single place. However, because of
its relevance to more recent research, we find it appropriate to bring a perspective to Weierstrass
semigroups for multiple places. The concept of Weierstrass semigroups was extended to multiple
places in 1990 by F. Delgado for algebraically closed fields (see [3]) and to finite fields in 2005 by P.
Beelen and N. Tutas (see [4]). For places Q1, . . . , Qm, the Weierstrass semigroup H(Q1, . . . , Qm)
is defined as the set of tuples (k1, . . . , km) such that there exists a function with pole divisor
exactly k1Q1 + · · · + kmQm. If the divisor G is chosen as the sum of two or more places, that is
G = a1Q1 + · · · + amQm for some m ∈N and with Qi ̸= Pj for any i = 1, . . . , m, j = 1, . . . , n, then
knowledge of H(Q1, . . . , Qm) is very useful when constructing efficient codes. A code arising from
such a divisor is called a multi-place code.1
1In the literature ”multi-point code” is the more commonly used term, but as we in this thesis use algebraic
function field theory rather than a geometric approach, we call it multi-place codes.
2

It turns out that choosing G as the sum of several places is beneficial for the error-correcting
properties of the associated code. For instance, G. Matthews proves in [5] that for a two-place
code with divisor G = a1Q1 + a2Q2 a specific choice of coefficients a1 and a2 constructed based on
gaps of H(Q1, Q2) can result in AG codes with a minimum distance greater than the usual lower
bound. In [6] Matthews investigates a quotient of the Hermitian function field and determines the
Weierstrass semigroup of any number of Fq2-rational points, which are used to construct AG codes
with minimum distance also exceeding the designed distance. These codes have better parameters
than any comparable one-place code over the same function field. Similar results were found by
M. Fujisawa and S. Sakata in [7] for the Hermitian function field. Specifically for the GK function
field, A. Castellanos and C. Tizziotti have shown in [8] that two-place codes have better parameters
than comparable one-place codes. In [9] D. Bartoli et al. have further explored multi-place codes
arising from the GK function field.
As mentioned, this thesis focuses on single-place Weierstrass semigroups. Although multi-place
codes seem to outperform single-place codes, computing the Weierstrass semigroups for multiple
places is more complex and falls outside the scope of this thesis. Nevertheless, studying single-place
Weierstrass semigroups provides a foundation for future research that can extend the analysis to
multiple places.

In this section, we again let K be a perfect field and F/K be a function field of one variable with

Definition 2.4.1. The genus of a numerical semigroup Λ is defined as g(Λ) := |N0\Λ|.
Now we introduce a specific semigroup that - as the title of the thesis indicates - we will work
much more with.
Definition 2.4.2 (Weierstrass semigroup). For a rational place P ∈PF , the Weierstrass semigroup
H(P) is the set of integers k such that there exist a function in F/K with pole divisor kP. The
integers that are not in H(P) are called gaps (or gap numbers) of H(P), and the set of gaps is
usually denoted by G(P).
The Weierstrass semigroup is the same for all but a finite number of places of the function
field. The places with a different Weierstrass semigroup than the general Weierstrass semigroup
are called Weierstrass places. The following two theorems will come in very handy when we wish
to determine the Weierstrass semigroup of a place.
Theorem 2.4.1 ([14, Corollary 14.2.5]). An integer n is a gap number of H(P) if and only if there
exists a holomorphic differential ω such that vP (ω) = n −1.
Theorem 2.4.2 (Weierstrass Gap Theorem [10, Theorem 1.6.8]). There are exactly g gaps of
H(P), i1, . . . , ig. Furthermore, for g > 0 we have i1 = 1, and ig ≤2g −1.

Chapter 4
The Fermat Function Fields
Let q be a power of a prime p. In this chapter we will explore Fermat function fields on a certain

Chapter 5
Wrapping up
In this project, we presented a thorough analysis of the Giulietti-Korchm´aros function field and
certain Fermat function fields. We computed the genus and proved the maximality of both types of
function fields, a property that establishes that they are indeed suitable for applications in coding
theory.
We mapped out the Weierstrass semigroups for all places of the Giulietti-Korchm´aros
function field, while for the Fermat function field, we found the Weierstrass semigroups for places
P ∈O and P ∈PFm \ O when m = q+1
2 . We made progress in determining the semigroups for
P ∈PFm \O when m = q+1
3
and believe that completing the proof of the final conjecture presented
would be manageable were it not for technicalities like the hand-in date for this thesis. A natural
continuation of this work is determining the Weierstrass semigroups for P ∈PFm \O when m = q+1
4 ,
and - potentially - obtain a general result for any divisor m.
Another natural and more applied extension of what we have investigated in this thesis could
be to construct AG codes from these function fields and analyse their parameters and performance.
We could also extend the Weierstrass semigroups to those of several places and investigate if these
give rise to better performing codes as suggested in [5, 6, 7, 8].
