Abstract
Hearing aids are able to restore audibility to hearing impaired (HI) listeners. However,
they are unable to distinguish between attended and unattended speakers in a multi-
speaker environment. Using electroencephalography (EEG), it becomes possible to
decode which speaker a listener is attending to, and selectively apply gain to that
speaker. This has been found to be possible using single-trial EEG in combination
with canonical correlation analysis (CCA), a technique that ﬁnds correlations between
speech features (envelopes) and neural responses. These correlations can be fed to a
classiﬁcation algorithm that is able to classify between an attended and unattended
stream. Studies have shown that single-trial EEG responses to audiovisual speech
provides an additional modality that eases the attention task compared to audio-only
speech. In this work, the integration of audiovisual speech in an existing real-time
Brain-computer-Interface (BCI), was explored. Experiments were performed on 20
normal-hearing (NH) and 3 HI listeners in order to assess the beneﬁts in attention
from the visual element in speech. Audiovisual speech has been found to generate
neural activity that are picked up by electrodes placed near the occipital channels,
which is not present when audio-only speech is presented. Previous studies reporting
a shortening in the latency of the EEG-to-speech response with audiovisual speech
were also conﬁrmed. Moreover, audiovisual speech was found to increase real-time
attention decoding accuracies. It also facilitated faster switching from one speaker to
another (4.5s faster switching). Attention decoding was also observed to be possible
with HI listeners. The visual element alone was also investigated by computing de-
coding models such as CCA, using visual features only instead of speech envelopes.
The models conﬁrm that attention can also be decoded using only visual inputs (72%
accuracy using CCA), oﬀering an additional possibility which can be used in hearing
aid devices. Integrating the speech envelope and visual features in a single CCA de-
coder yielded accuracies that are 2% better compared to using the speech-envelope
only (based on 6.5s decoding segments). The results presented in this work suggest
that visual elements play a key role in auditory attention decoding.

CHAPTER 1
Introduction
1.1
Motivation
Hearing loss impacts over 6% of the current world population, and this ﬁgure is
predicted to increase in the coming decades [WHO, 2018]. This further leads to a
decline in quality of life, and increased mental health issues. Currently, the most
commonly used remedy is a hearing aid, which is a device that partially restores
audibility.
However, such a device is not fully equipped to restore hearing in all
acoustic scenarios.
One such acoustic scenario is referred the "cocktail-party" problem. This is a well
known phenomena in hearing science which regards a listener’s ability to segregate an
attended speech stream whilst rejecting a number of competing unattended speech
sources. This ability to segregate several streams can be attributed to some ongoing
cortical processes [Sharma, 2013].
Hearing impairment degrades this decoding ability. In fact, hearing impaired lis-
teners have an extremely hard time in such environments, even when ﬁtted with
a hearing aid.
Current commercial hearing aids are able to provide ampliﬁcation
thereby restoring loss of sensitivity. However, in the cocktail-party problem, ampli-
fying both the attended (signal) and unattended streams (noise) is not particularly
useful, as this keeps the signal-to-noise ratio constant. The goal is to have hearing
aids that are able to apply selective gain based on attention.
Given that cortical processes are responsible for attention, it then becomes possi-
ble to extract cortical information and try and relate it with incoming auditory signals.
This is a neuroscience approach to solving the cocktail party problem, which was the
main focus of the Cognitive Control of a Hearing Aid (COCOHA) project. The pri-
mary aim was to build an attention decoder capable of classifying an attended stream
and unattended stream [COCOHA, 2018]. This alleviates the previously mentioned
limitation of a hearing aid, assuming the hearing aid is able to separate the diﬀerent
sound sources. It is also realizable, keeping in mind the practical hardware limitations
of a hearing aid. This shifts the domain of the hearing aid into the brain computer
interface (BCI) paradigm. With such a BCI application, speech intelligibility may
possibly be enhanced.
Attention is also driven by visual information. Research has shown that coherent
visual and auditory information represent an increase in overall intelligibility [Reis-
berg, McLean, and Goldﬁeld, 1987]. Visual information adds additional redundancy

in attention decoding. However, the integration of audiovisual features in an attention
decoding pipeline is still relatively unexplored.
1.2
Project Goals
The BCI solution to the cocktail party problem, along with audiovisual features raises
several interesting questions:
1. Does such an attention decoder work in a real-time BCI paradigm, where at-
tention is used to control auditory feedback? If so, how can performance be
quantiﬁed?
2. Is there an element of visual attention?
If so, can a combined audiovisual
modality provide an increase in a better attention decoder?
3. Are there any particular diﬀerences between hearing impaired (HI) and normal
hearing (NH) listeners? If so, can hearing impaired listeners beneﬁt from such
a system?
The main goals of this project were to answer these questions in a systematic
manner. The main recurring theme throughout this thesis was to evaluate the eﬀects
of the audiovisual element in attention decoding, with the study of the system with
respect to HI listeners being a secondary goal. Visual and hearing-impairment eﬀects
were investigated by the design of an experiment, where the two conditions can be
separated and studied in isolation. This was done by collecting electroencephalogra-
phy (EEG) responses to audio-only speech and audiovisual speech for subjects with
normal hearing. EEG responses to audio-only speech were also collected for a small
number of HI listeners.
Attention decoding was evaluated using diﬀerent models that are recurrent in
literature, more speciﬁcally, system identiﬁcation and correlation-based models. Per-
formance was mainly evaluated by computing how accurately the attention decoder
identiﬁed the attended speaker in a two-speaker cocktail-party-like environment. Ex-
amining if the models give similar results on the performance (decoding accuracy)
with respect to audio-only and audiovisual data, provides a more robust argument
whether or not there is an audiovisual aﬀect.
Real-time attention decoding was also evaluated experimentally in terms of how
well the system is able to identify the correct speaker, and also how quickly the system
able to adapt (if at all) when a listener switches attention to another speaker.
1.3
Structure of the Thesis
Chapter 2 sets the scene by introducing the background required to model attention
using neural responses. The focus is on how cortical responses to speech are gener-
ated, measured and modelled. Models using linear methods, such as the forward and

backward models, are explained. More recent techniques, such as canonical correla-
tion analysis (CCA), are also discussed in some detail. The chapter also introduces
recent ﬁndings and approaches in audiovisual attention decoding, as well as some BCI
concepts.
Chapter 3 describes the experimental procedure used to study the eﬀect of au-
diovisual stimulation on the neural response of NH listeners. The procedure used to
determine whether a real-time decoding BCI system works for NH and HI listeners
is also detailed. The hardware set-up, and diﬀerent experimental conditions are ex-
plained. Further to this, the chapter describes the diﬀerent models applied to the
data collected in the experiments.
Chapter 4 presents the results attained in the experiments. These are structured
in two parts. The ﬁrst part deals with evaluating how well the described models are
able to classify attention, and what the parameters related to attention are. Each
model is used to compare the diﬀerences between audio-only and audiovisual eﬀects.
They are also used to evaluate if there is an element of visual attention in the neural
response. The second part deals with real-time decoding performance. This is how
well the previously described model work in a real-time setting, which is slightly more
indicative of how a hearing aid would operate. Again, the results obtained for the
audio and audiovisual modalities as well as NH vs. HI listeners are presented.
Chapter 5 provides a summary and discussion of the results obtained in relation
to what has previously been found. This chapter also includes some of the current
system limitations.

CHAPTER 2
Theory
This chapter introduces some fundamental concepts and recent ﬁndings regarding
modeling of auditory evoked potentials, attention decoding, brain computer interfaces
and audiovisual integration. The chapter starts by brieﬂy describing the process of
the entrainment of a neural response from an auditory stimulus, how this response
can be modelled using various techniques, and how it can used in a brain computer
interface to decode attention.
2.1
Auditory Evoked Potentials
2.1.1
From pressure waves to electrical activity
In order to understand how one can measure a cortical representation of an auditory
stimulus, it is useful to consider the whole auditory chain from a functional view-
point. This is relatively important in this case, as it will motivate the preprocessing
procedure used to transform an auditory stimulus in an attention decoder.
The sense of hearing starts with the generation of sound. Whenever an object
vibrates, it causes movement of air molecules. This movement causes instantaneous
pressure diﬀerences (compressions and rarefactions) in time and space that propagate
outwards from a source.
The acoustic wave then reaches the ear. An illustration of the structure of the
human ear is shown in Figure 2.1.
The outer and middle ear act as a linear ﬁlter and do not change the pressure
waveform much. The outer ear has a complex geometrical shape which enhances and
attenuates certain frequencies. The middle ear represents an acoustical-mechanical
transducer which changes acoustic energy to mechanical vibrations via the ossicle
chain.
This mechanical motion in the middle ear chain then causes the oval and
round window to move due to the applied pressure. Inside the inner ear is a structure
called the cochlea that contains a membrane referred to as the Basilar Membrane
(BM). The mechanical properties of this membrane are responsible for frequency-
place transformation. Functionally it acts as a ﬁlterbank. The movements of the BM
then cause inner-hair cells (IHCs) and outer-hair cells (OHCs) to move. The electro-
chemical distribution of charges outside and inside of the IHCs cause the mechanical
vibrations to be converted into electrical signals. These electrical signals generate

2.1.2
Measuring Evoked Potentials
In order to decode auditory attention, cortical activity in response to speech stimuli
needs to be measured. Electrical activity generated in the brain due to external sound
stimulation are referred to as Auditory Evoked Potentials (AEPs). AEPs can
be measured with techniques such as EEG or magnetoencephalography (MEG). Of
particular interest are EEG recordings, given that they have high temporal resolution
and are highly practical in terms of cost and portability.
EEG readings usually require heavy preprocessing due to their noisy nature. Neu-
ral responses measured by EEG are capturing the far-ﬁeld potential generated by
action potentials travelling along axons (quadropol moments which last a few mil-
liseconds), and post-synaptic currents (dipole moments lasting around 10 millisec-
onds) travelling along the dendrites from synaptic connections. More speciﬁcally the
main contributor to the far-ﬁeld potential are the post-synaptic currents.
The ﬁeld potential read by EEG electrodes is capturing the culmination of several
processes happening inside the brain. Thus activity generated by some source in the
brain, is picked up by several electrodes, giving rise to very poor spatial resolution.
Formally, if there are ’S’ sources in the brain, electrode ’j’ will read the potential, bj(t),
corresponding to some linear combination of the ’S’ sources as outlined in Equation 2.1
where the weights, ajs, refer to the sensor-to-source mixing weights (sensor in this

external to the body such as the equipment itself. Baseline Noise is an example of
an extraphysiologic artifact that is associated with low frequency drifts. This comes
from poor electrode contact and drift from the EEG instrumentation ampliﬁers them-

Electrooculography (EOG) signals) are also a nuisance when trying to analyze EEG
data. As stated in [Remijn et al., 2014], EOG artifacts occur due to eye movements
and blinks. The cornea is positively charged, while the retina is negatively charged.
This generates an electrical dipole, and when there are eye movements, changes in
the electrical ﬁeld surrounding the eyes will distort the electric ﬁelds over the scalp.
In order to extract the relevant EEG sources, the EOG signal must be ﬁltered out.
Auditory attention requires ﬁnding some relationship between the speech stimulus
and the measured neural responses. Therefore removing such artifacts is an important
step to obtain any meaningful information between the two signals.
2.2
Modeling Neural Responses
There are several techniques that can be used to relate neural responses to auditory
stimuli. This section introduces the theoretical underpinnings behind these modelling
approaches, their interpretation as well as their limitations.
2.2.1
Averaging Event-related Potentials
Decoding auditory attention from brain signals is not a trivial task. Previously, the
main approach to study evoked potentials due to auditory stimuli was Event-related
potentials (ERPs). It is based on the principle that electroencephalography (EEG)

recordings are extremely noisy. The signal amplitude obtained by an EEG recording
is in the low µV range, while the noise amplitude is generally much higher. There-
fore, the signal-to-noise ratio is very poor. In order to alleviate this issue, averaging
techniques are applied.
If the signal of interest is S, and the noise is represented by N, then the measured
potential, R, by one electrode is given in Equation 2.2.
R = S + N
(2.2)
If the same stimulus is presented P times, the individual responses can be averaged
in order to improve the signal to noise ratio. The assumption is that the noise is
uncorrelated in each reading. If this is the case the SNR is improved by a factor ’P’.
This averaging technique is suitable for periodic stimuli. However, averaging is not
particularly useful when dealing with non-stationary stimuli such as speech since the
elicited neural response will also be non-stationary. Such limitations are alleviated by
system identiﬁcation techniques which use data-driven models that relate the neural
responses to speech.
2.2.2
System Identification Techniques
Relatively recent studies have employed system identiﬁcation techniques when trying
to relate the elicited neural responses to speech ([J. A. Osullivan et al., 2014], [D. D. E.
Wong et al., 2018], [Alickovic, Lunner, and Gustafsson, 2016], [Kerlin JR, 2010]) .
This approach treats the underlying processes as a black box system, which can be
described by its input/output characteristics.
System Identiﬁcation assumes that
the underlying process can be modelled as a linear system. This is mathematically
convenient as the process can be described as a convolution with an impulse response
which would characterize the system.
The problem at hand is to try and relate ’J’ channels of EEG data to a speech
signal. There are two main linear models which are commonly used, The Forward
Model and the Backward Model (Stimulus Reconstruction).
The forward model ﬁnds a system response, referred to as a temporal response
function (TRF), which when convolved with the envelope of the speech stimulus, pre-
dicts the neural response. If there are ’N’ speech stimuli, the ’N’ neural responses can
be estimated by convolving each of the stimuli envelopes with the TRF obtained from
the forward model. By calculating the correlation between the estimated responses
and the actual measured responses, it becomes possible to identify to which speaker
the subject is attending to.
Conversely, the stimulus reconstruction model, also known as the inverse model,
tries to reconstruct the speech envelope using the measured EEG responses. Similarly,
with the backward model, the neural response is convolved with the found impulse
response to estimate the speech envelope. The estimated speech envelope can then be
compared with each of the ’N’ speech stimuli envelopes, by computing the correlation,
and the one with the highest correlation would be the attended talker.

Regularization
The described system identiﬁcation methods, require the estimation of a large number
of parameters (the weights for the impulse response). The forward model requires
τ parameters to be estimated while the backward model requires τJ parameters.
In data-driven calculations, estimating a large number of parameters may lead to
overﬁtting. One common technique to limit overﬁtting, is regularization.
The forward and backward models are very similar computationally, in the sense
that they estimate an impulse response using linear regression. Regularized linear
regression (RLR) adds a regularization term, λ, to Equations 2.6 and 2.10.
This

Correlation-based Methods
System Identiﬁcation methods with diﬀerent regularization methods have been proven
to accurately decode attention, [D. D. E. Wong et al., 2018]. However, the correla-
tion scores obtained with stimulus reconstruction are in the order of 0.1 which are
relatively low and could lead to poor decoding accuracy in a real-time application. In
[Cheveigné, D. D. Wong, et al., 2018], canonical correlation analysis (CCA) was used
to identify its applicability for a real-time attention decoder.
CCA is a multivariate analysis technique used to explore correlations between
two data-sets, [Hardoon, Szedmak, and Shawe-Taylor, 2004]. In this case, the two

data-sets correspond to two matrices B and A, which represent the ﬁltered brain
signals and audio features. Therefore, CCA exploits both multiple EEG channels
(spatial ﬁltering) and multiple audio features at the same time. The two matrices,
A and B can be thought of as diﬀerent views of representing the same information,
[Stratos, 2019]. CCA tries to extract this common representation by ﬁnding a new
coordinate system for the two matrices such that in this new coordinate system they
are maximally correlated.
One way to exploit audio features would be to consider ﬁltered versions of the
audio envelope, i.e modulation ﬁltering. This can be done by ﬁltering the attend-
ed/unattended audio envelopes using F diﬀerent kernels. This means that the audio
envelope matrix A has dimensions T ×F. These F kernels can be thought of as a way
of potentially representing a modulation ﬁlter-bank, given that it takes the speech
envelope and resolves it into diﬀerent modulation frequency bands.
The matrix B is constructed by ﬁltering using the F diﬀerent kernels. That is,
each EEG channel will have, F, ﬁltered representations. For each ﬁlter kernel, the
64 channels are concatenated into one column vector such that B has dimensions
T × 64F where T represents the number of time samples in a particular time interval.

case each column captures the spectrotemporal aspect of the signal) then F diﬀerent
linear combinations may be deﬁned for the speech envelope. Similarly, 64F = Q linear
combinations may be written for the ﬁltered EEG signals. In this case F < Q, and
hence F linear combinations are selected to minimize the number of computations.

diﬀerent spectral bands of the speech envelope and EEG signals.
It is important to mention that such methods are prone to overﬁtting, since many
parameters need to be estimated from a limited amount of data. As with most sta-
tistical techniques, this limits the ability for the model to generalize well to ’unseen’
data. Therefore, CCA is most often preceded by principle component analysis (PCA).
PCA reduces the dimensions of the EEG and speech envelope matrices by throwing
out components that explain very little variance. Moreover, the number of CC com-
ponents that are passed in to a classiﬁer after CCA can be determined using K-fold
crossvalidation, which splits the data into multiple training and test sets. Here, the
CCA model is trained using the training set, and then the correlation scores are com-
puted using the test set. This gives an unbiased estimate of performance of the CCA
model.

2.3
Audiovisual Speech
In this section, some ﬁndings on the neural responses to audiovisual speech are pre-
sented. Most ﬁndings are based on the previously described system identiﬁcation
modelling techniques.
Attention decoding in multi-talker environments has been investigated using audio-
only speech. However, there is a visual element to speech. Audiovisual cues seem to
oﬀer an additional modality that can potentially aid attention decoding applications
when available. In fact, the addition of congruent visual stimuli to audio speech oﬀer
an additional redundancy that improves speech intelligibility. The redundancy comes
in the form of a temporal precedence. Using functional magnetic resonance imaging
(fMRI) techniques, [Arnal et al., 2009], have explained that a direct corticocortical
pathway transmits visual information to the auditory cortex. This signal acts as a
prediction of what the auditory stimulus will be. Thus one can think of the visual
stimulus, to provide some sort of speed-up in the neural response to speech.
[J Crosse and C Lalor, 2014] compared the neural response to congruent audiovi-
sual (AV) and audio only (A) speech using TRFs (forward model). The two TRFs
show a negative and positive peak which are prominent. Of particular importance
was the negative peak that occured at around 100ms. This negative peak appeared
11.4ms earlier for the audiovisual TRF. This further suggests that the auditory re-
sponse latency is reduced when congruent AV speech is presented.
[A. E. O’Sullivan et al., 2017] use TRFs that are modelled using visual stimuli
in addition to the speech envelope. This is done in order to explain whether there
is a neural response to a visual stimulus. They focus on two types of visual stimuli,
visemes (these correspond to mouth movements of phonemes), and the frame-to-frame
motion signal. The study shows that the TRF combines information from the visual
and auditory cues, eluding that attention is decoded better when audiovisual infor-
mation is used in a model rather than just using audio-only information.
Congruent AV speech has been shown to enhance the neural response of the speech
envelope in [Crosse, Butler, and E. C. Lalor, 2015]. This has enhancement was at-
tributed to neural generators which are not active when presenting just audio. Their
rates are in the region of 2-6Hz which coincide with syllabic rates. The study uses
stimulus reconstruction methods to correlate the estimated speech envelope with the
actual speech envelope when subjects are presented with audio and audiovisual speech.
The audiovisual case obtained higher correlations, meaning the speech envelope is
represented better in the neural response. This suggests potential attention decoding
strategies that incorporate visual features which could greatly enhance decoding ac-
curacy in multi-speaker environments. [J. A. O’Sullivan et al., 2013] also show that
decoding attention using audiovisual features promises to be a viable BCI application.
This being said, incongruent AV speech (when the speech stimuli and visual stimuli
do not match) does hinder speech perception. This could potentially lead to a worse
decoder if incongruent visual cues of unattended speakers interfere with the audio
cues of the attended stream. However if incongruent audiovisual speech is presented,
[A. E. Osullivan, Lim, and E. C. Lalor, 2019], have shown that it is still possible

2.4
BCI systems
The main application of this study is a BCI system. Therefore, in this subsection
some basic terminology typically used in general BCI systems are introduced.
A BCI can be deﬁned as an interface which maps brain signals to external actions.
BCI systems rely on the measurement of brain activity. Such measurements can be
done using two main approaches, Invasive and Non-Invasive methods. Invasive
methods are less common since they require surgery, to place measurement probes in-
side the subject’s scalp or brain. One commonly used method for invasive procedures
is the Electrocorticography (ECoG). In ECoG electrodes are placed on the cerebral
cortex, bypassing the skull. Non-invasive are more popular due to the fact that they
are cheaper and do not require surgery, since they monitor brain activity from outside
the skull. EEG and fMRI are two such methodologies. Particularly EEG is very com-
mon in BCI applications since it is more practical in terms of use, cost and superior
temporal resolution.
It is no surprise that non-invasive signal acquisition generally have poorer signal-to-
noise ratios compared to invasive methods. As a direct consequence, signal processing
is a crucial aspect in BCI systems, where the aim is to extract task-relevant features
from the acquired brain signals. In this particular work, the task is to try and classify
which is the attended speaker from a number of speakers.
This implies that the
signal processing goal is to extract features from the EEG signals that relate to the
speech envelopes of the various speakers. Once the right features are extracted, the
BCI system maps the information into some action. This action also relays feedback
(directly or indirectly) back to the brain. Hence, such systems are also called closed-
loop systems. For example in the BCI system used for this work, the action is to
amplify the attended speaker. If the attended speaker is louder, then the neural will
be able to track this speaker better. Hence the ’gain’ action indirectly changes the
neural response.
The general paradigm of a BCI system, used for a speaker classiﬁcation task,
is shown in Figure 2.3. It summarizes the main subsystems that comprise a BCI
application, these being signal acquisition, feature extraction (signal processing), and
some classiﬁer that actuates some device.

It is important to quantify with some metric the performance of a BCI. [Wolpaw
et al., 2002] describes numerous ways on how this can be achieved. There are two
levels of evaluation that try to describe performance from a user point of view, and
more generally from a system point of view.
User performance can be quantiﬁed by using the correlation of the neural response
with the control stimulus. The correlation ﬁgure captures user intent, that is, how
well the intended signal is represented from the extracted signal. In this particular
case, it will be a measure of how closely the attended speech envelope correlates with
the EEG stream.
System performance may be quantiﬁed by looking at the classiﬁcation accuracy
and decoding speed. For example, the classiﬁcation accuracy at a certain decoding
speed, would be the percentage of time the system correctly identiﬁes the attended
speaker. The decoding speed is then related to the time window the classiﬁer looks
at when making a decision.
In most cases a larger classiﬁcation accuracy can be
achieved at the expense of a reduction in decoding speed, and vice-versa.
The two parameters, accuracy and speed, can be combined by borrowing notions
commonly used in information theory.
These measures come about, as BCIs can
be considered to be communication systems. A communication system is one that
sends and receives signals through some noisy channel, and that ﬁts exactly with the
deﬁnition of a BCI. The goal is to send a message from the brain to an external device.

It is deﬁned as the amount of information that can be transmitted per unit time,
therefore having the units bits/trial. This is deﬁned mathematically as a bit rate, B,
by Equation 2.15 where ’N’ is the number of user choices, and ’P’ is the probability
of a given choice at a particular state.

This can be related to the system speed by dividing the ITR by the decoding

the probability of a choice increases. This means that for a larger decoding time
the ITR will start to drop. Thus the ITR as a function of decoding time will have
some maximum value and this dictates the decoding speed that is used in a BCI
system. This being said, another criteria for choosing the decoding speed is also based
on the intended application. For example, in a cocktail party scenario, choosing a
relatively large ’T’, will most likely result in correct classiﬁcation of the attended
speaker, however if the subject switches attention, the decoder will take a long time
to respond to such a switch.

Experimental setup
In this section, the experimental paradigm used to investigate the eﬀect of audiovisual

3.1.1
Subjects
Twenty normal-hearing (NH) and three hearing-impaired (HI) subjects (all right-
handed) with normal/corrected-to-normal vision were recruited for the experiment.
The task was to attend to a speech stream presented at a comfortable listening level
of approximately 60 dB SPL.

The speech material used in this experiment were of two known Danish speakers

was used in order to apply frequency-speciﬁc gain to compensate for loss of audibility.
This was done according to the Cambridge formula [W Peters et al., 2000].
All the speeches were normalized to the same rms value, after cutting out silent
parts longer than 0.5s. The male and female audio were then merged together with
their corresponding video, such that the male speaker appears on one side of the

3.1.3

Before describing the details of the experimental procedure, it is important to intro-

• Training Phase:
The training phase is used to collect data and train an
attention decoder based on the collected data.
• Testing Phase:
The testing phase is used to evaluate how well the trained
attention decoder works with data it has not seen before in a real-time setting.
The experiment consisted of two main parts, the training phase and the testing

In both phases, the subjects were instructed to attend to one of the two

During recordings, participants were instructed to keep movements and eye blinks to
a minimum.

classiﬁer would then output some probability measure (for each class) that indicates
whether a person was attending to the left or right speaker.

3.2.1.2
Backward Model
For the backward model, a similar approach to the forward model implementation
was used. In this particular case, diﬀerent models (λ values) on ’N-1’ trials. The ’N-1’
decoders are then averaged for each λ, and then tested on the remaining trial using
LOOCV. The test metric used in this case was the correlation of the reconstructed
speech envelope and the actual attended speech envelope. The one that results in the
highest λ was then chosen as the optimal model.
In the backward model a simple classiﬁer was also constructed, that computes
which speaker was the attended one. This was done by using the averaged attended
decoder obtained by LOOCV. Both the unattended and attended stimulii were recon-
structed. The reconstructions were then compared with the actual speech stimulii
using the Pearson correlation coeﬃcient. The one having the highest correlation was
selected as the attended speaker.

3.2.3
Addition of visual features
Another important facet that was explored were the addition of visual features to the
described system identiﬁcation and correlation methods. When a subject attends to
a speaker, they also look at a speaker, i.e the target talker is in the central part of
their visual ﬁeld. This means that the visual features in their central vision may have
a diﬀerent neural representation from that of the unattended speaker. If this is the
case, the neural representation could be used to improve attention decoding.
Face detection was done using an algorithm called Histogram of Gradients
(HOG) coupled with an SVM. HOG works by scanning all pixels in an image and for
each pixel, the intensity gradient (how dark the pixel is compared to it’s neighbour
pixels). The SVM is trained with these gradients for several images with faces. This
was performed using a Python wrapper for a C++ library, "dlib" [King, 2009]. The
library contains face detection algorithms as well as facial landmarks detection. Land-
mark detection in "dlib" is performed using an ensemble of regression trees [Kazemi
and Sullivan, 2014]. These were essential to extract several features such as the mouth,
eyes, and jaw movements. These movements, in particular mouth and lip motion, are
correlated with the speech envelope.
To extract the visual features, the two-talker videos were downsampled from 30 fps
to 20 fps and their resolution was brought down from 1920 × 1080 to 800 × 600. This
was done in order to lower the computation time needed to perform face detection
and landmark extraction. "dlib" was then used on the training audiovisual material
to extract 68 diﬀerent facial landmarks. These landmarks are shown in Figure 3.8.

that they are centered at a mean of 0 and having variance σ2 = 1. This makes the
landmarks robust to face location in the frame. Moreover, to make the landmarks

stimuli (passed through the modulation ﬁlterbank) and the EEG responses. This

neural responses such that it is maximally correlated with the visual input. BROT
represents the rotated visual stimulus, such that it is maximally correlated with the
neural responses.
The second CCA was constructed by maximizing the correlation between the
speech envelope and the EEG responses. As described above, this gave two rotated
matrices CROT and DROT, with CROT representing the rotated EEG matrix and
DROT representing the rotated speech envelope matrix.
Essentially the two CCA models can be thought of independently capturing the
visual and speech attention mechanisms. The two models were then combined by
stacking the rotated matrices [AROT CROT] and [BROT DROT]. This procedure is
summarized in Figure 3.11, where all the relevant matrix dimensions are shown. In the

• The optical ﬂow and speech envelope were passed through the same ﬁlterbank

CHAPTER 4
Results
In this chapter, the results obtained from oﬄine data-analysis and the real-time de-
coder described in the previous chapter are presented. These results are aimed to
answer the following questions:
• Is it possible to decode attention in real-time from the EEG in a neurofeedback
loop that controls the relative gain of the attended talker?
• What aﬀects the neural responses when subjects are presented with audiovisual
vs. audio-only stimuli? How can one model such eﬀects? Does the ’AV’ modality
improve auditory attention decoding and performance of the real-time system?
• Are the decoders able to exploit visual information for attention decoding?
• Are HI listeners able to beneﬁt from such a BCI system?

CCA Model
CCA yields an additional advantage over stimulus reconstruction, as it enables to
combine multiple electrode channels and stimulus features (in this case envelope
modulations or visual modulations). CCA yielded better classiﬁcation accuracy, al-
lowing shorter decoding segments of 6.5s, as opposed to the 49s window considered

in stimulus reconstruction. Here, results from three diﬀerent CCA models which use
speech-envelopes, visual features and audiovisual features are presented.
4.1.3.1
Envelope-based CCA
Given that stimulus reconstruction with speech envelopes showed diﬀerences between
the ’AV’ and ’A’ modality it is important to showcase whether this is also the case with
CCA. If the same patterns are observed than the argument that the ’AV’ modality
has an aﬀect on attention is reinforced. Moreover, the real-time decoder is based
on CCA, and hence in order to anticipate real-time performance it is important to
quantify how well the model works on the collected training data.
The results presented here reﬂect a CCA classiﬁer operating on window lengths of
6.5s. To justify this decoding segment length, consider Figure 4.8, which illustrates
the accuracy and ITR (averaged across ’A’ and ’AV’ subjects) as a function of the
decoding segment. For subjects trained with ’AV’, the classiﬁcation accuracy and ITR
is greater for all decoding lengths than subjects trained with ’A’ stimuli (this will be
discussed later in reference to one decoding length).
The ITR suggests decoding
lengths of 1s can still be used, however the decoding accuracy is quite low (63%) for
this window length (even though it is still above chance). On the other hand, long
decoding segments give high accuracy, at the expense of a slow system (highlighted
by the ITR). Thus a trade-oﬀbetween the two was chosen to be 6.5s. This gives
relatively high ITR and also an average decoding accuracy of 80%. It can also be
noted that the ’AV’ modality seems to be a better training method than ’A’ training
for an attention decoder as it can signiﬁcantly improve the decoding speed.
For
example, a participant trained with ’A’ stimuli, will achieve an accuracy of 80%
at a decoding length of 10s, while a participant trained with ’AV’ stimuli, will
achieve an accuracy of 80% at a decoding length of 6.5s.

follow one speaker and then switch attention to the other speaker in each trial. This
was done in order to observe whether the real-time decoder would follow the attended
speaker. The results here present:
• The eﬀect of acoustic feedback on the decoder response (closed-loop vs. open-
loop)

• The eﬀect of ’A’ and ’AV’ training on the real-time decoder. Can the decoder
pick up visual-related brain responses (for subjects trained using ’AV’ stimuli)
to help make decoding better?
• The eﬀect of presenting ’A’ and ’AV’ stimuli during closed-loop testing. Does
a decoder trained on auditory stimuli perform similarly when visual input is

• The decoder performance for NH vs. HI subjects.

4.2.1
Decoder Performance
In the testing phase, a CCA-based decoder is used to evaluate two main criteria.
These are switching performance and overall accuracy of the classiﬁer.
The decoding accuracy can be quantiﬁed by considering the time spent by the
classiﬁer output attending to the correct speaker. Note that the state tracker output
can be considered as a smoothened version of the classiﬁer output.
The average decoding accuracy, for a subject was obtained by averaging the time
spent correctly identifying the attended talker across all trials. The classiﬁer operated
on 6.5s decoding segments, hence 100% accuracy can only be achieved if the switch
in attention was instantaneous which is not a realistic scenario. Thus the accuracies
obtained here, will be lower than the ones shown for oﬄine data analysis. Figure 4.13
shows a box-plot of the accuracy for each testing modality (CLAV, CLA and OL).
The closed-loop results (CLAV - audiovisual stimulation, CLA - audio stimulation)
give an indication if applying neurofeedback helps or reduces decoding accuracy, by
direct comparison to the open-loop results (1). The results are grouped into two to
illustrate the eﬀects of the training modalities, audiovisual (AV) and audio (A), on
the testing performance. It should be noted that the decoder was not trained on any
visual features such as mouth movements, but rather only on the speech envelopes of
the two speakers.
In general there is a trend that ’AV’ training modality leads to a slightly higher
decoding accuracy compared to the ’A’ training modality. This is not statistically
signiﬁcant due to the high variation across subjects. A useful comparison would be
to distinguish if there is a signiﬁcant diﬀerence between the ’CLAV’ case with ’AV’
training to the ’CLA’ case with ’A’ training.
In fact, the ’CLAV’ case with ’AV’
training is signiﬁcantly better (P = 0.0257, Wilcoxon rank sum test). This conﬁrms
the initial hypothesis that audiovisual features are useful for decoding attention, at
least in terms of accuracy.

The other important metric is the switching time. In a practical scenario a hearing
aid must be able to decipher an attention switch within a few seconds or even less.
This is currently limited by the decoding length considered. Switching time is also
related to the accuracy. A shorter switching time indicates a higher accuracy over
time.
The results obtained were constructed using the average output (across all trials)
of the two state trackers. Let one state tracker output represent Speaker 1 (SPK1)
and the other represent Speaker 2 (SPK2). The test consists of attending to SPK1
and at t = 25s, the subject should switch attention to SPK2. Therefore attended
state, ATT, can be deﬁned as in Equation 4.1.
ATT(t) = SPK1(t) −SPK2(t)
(4.1)
Given that SPKx, x ∈[1, 2] ranges from [0, 1], the attended speaker would be
x = 1 if ATT > 0, and x = 2 if ATT < 0. The switching time may be deﬁned as the
time, Tswitch, where SPK1(t) = SPK2(t) s.t. ATT(t) = 0.
Any eﬀects in lateralization (i.e. spatial preference) are averaged out unless explic-
itly stated. This is because SPK1 and SPK2 are averaged across all trials irrespective
of whether the subject was ﬁrst asked to attend to the right/left speaker and then to
switch to the left/right speaker.

the switch impossible. This suggests a ceiling gain, whereby the diﬀerence between
the attended and unattended speech streams must be tuned to some minimum level
diﬀerence, such that the ability to switch attention is not impaired.

CHAPTER 5
Discussion
5.1
Audiovisual Effects
The training results for all the tested models showed that audiovisual cues improve
attention decoding performance.
The forward model served to discern any possible diﬀerences in the neural repre-
sentation of the speech envelope when subjects are also presented with the visual face
as opposed to hearing only the audio. A TRF with prominent negative and positive
peaks was observed at the frontocentral sites (Figure 4.1). The negative peak for the
audiovisual TRF occurs on average, 9.4ms, earlier than the audio TRF. This conﬁrms
the study conducted in [J Crosse and C Lalor, 2014], where the same eﬀect has been
observed.
This earlier representation may reﬂect a speed-up of speech processing
in the auditory cortex. In fact, studies on macaque monkeys have shown that the
addition of visual features speed up the latency of auditory-cortical spiking activity,
[Chandrasekaran, Lemus, and Ghazanfar, 2013]. The scalp maps in Figure 4.2 indi-
cate that the negative component in the audiovisual case, shows additional activity
(not present in the audio-only case) in the occipital region.
The multi-channel stimulus reconstruction and CCA models yielded higher clas-
siﬁcation accuracies when subjects were presented with audiovisual cues. However,
the correlation between the reconstruction of the attended envelope and the actual
envelope did not improve signiﬁcantly between the audiovisual and the audio-only
case (Figure 4.5). What did improve was how well the unattended envelope was ’re-
jected’ in the audiovisual case. The unattended correlations were signiﬁcantly lower
when subjects had visual information.
As a direct consequence, the classiﬁcation
accuracy which is based on the diﬀerence between the attended and the unattended
correlations increased. This was also, to a lesser degree, observed in the CCA model
(Figure 4.9). The classiﬁcation accuracies in the stimulus reconstruction model are
not directly comparable to the ones in the CCA model, given that the CCA classiﬁer
is operating on 6.5s windows (used to predict how accurate real-time decoding will
be), while the backward model uses 49 long windows (hence the higher accuracy).
A visual TRF based on the optical-ﬂow was also obtained, highlighting brain activ-
ity on the posterior channels. The TRF can also be related to (although not directly
comparable) to visually-evoked-potentials (VEPs). VEPs show a similar pattern to
the obtained TRF and the peaks in the VEP have been known to be aﬀected by spa-
tial attention [Russo, Teder-Sälejärvi, and Hillyard, 2003]. This activity illustrates
==PAGE:74==
58
5 Discussion
an eﬀect of visual attention giving rise to the possibility of a visual-based decoding
approach.
The stimulus reconstruction and CCA based models show that such a
decoding strategy is possible given the above-chance decoding accuracies obtained in
Figures 4.7, 4.11.
It was also evident that classiﬁcation accuracies obtained with the speech en-
velopes were in general higher. However the combination of the two modalities should
yield a better decoder, which is what was observed in Figure 4.12. In [A. E. O’Sullivan
et al., 2017], a motion signal (similar to optical ﬂow) was used to construct a visual
decoder (this decoder was based on the forward model, unlike what was done in this
project). They obtained above-chance correlations (i.e. the correlation scores were
better than those obtained when running the forward model with a random signal).
However, in contrast to what was observed here, the attended correlations (recon-
structed vs.
actual signals) were higher for the motion signal than those for the
speech envelope. This is not a directly comparable result, as these models are com-
puted on the forward model, which computes correlations on a channel-by-channel
basis, whereas in CCA and stimulus reconstruction, the attended correlations repre-
sent a spatio-temporal averaged score.
5.2
Real-time System Performance
Real-time system performance is one of the main motivations in ﬁnding out whether
attention decoding is worth exploring for hearing-aid applications. Moreover evaluat-
ing the eﬀect of ’A’ and ’AV’ training on the results is also important in the scope of
this work. The real-time results in Chapter 4, show that attention decoding is indeed
possible in a real-time setting for both NH and HI participants, although this was
not entirely convincing for all the subjects tested.
5.2.1
NH Participants
The decoding performance in terms of accuracy (Figure 4.13) and switching time
(Figure 4.15), demonstrated that the system does indeed work for NH participants.
The average decoding accuracy was above chance level in all test conditions, both for
’A’ and ’AV’ training.
For ’A’ subjects, it was interesting to note that the system worked for the ’CLAV’
test condition. This means that the neural responses used to train the decoder did
not include visual information. Moreover, it also slightly improved the average switch-
ing time. This further supports the notion of audio-visual integration speeding up
auditory attention in the neural responses. Cutting out the feedback also resulted in
adequate decoding performance evidenced by the ’OL’ switching time and accuracy.
For ’AV’ subjects, the decoder performed better on average (in terms of accuracy
and switching time) in the ’CLAV’ test case than those for ’A’ subjects. This makes
sense, given that the CCA decoder was trained on neural responses to ’AV’ speech.
==PAGE:75==
5.2 Real-time System Performance
59
From this, one can extrapolate the best decoding strategy. This would be one,
where the CCA model is trained on ’AV’ speech, and then tested on ’AV’ speech as
well. This yields an improvement over all the other combinations. Access to a visual
stimulus is not always possible. The fact that the decoder trained on ’AV’ worked for
the ’CLA’ test case shows that in such a scenario attention decoding is still possible,
with a slight decrease in performance.
5.2.2
Effect of Hearing Impairment
An important aspect of this project was to evaluate whether the system is able to work
for HI participants. The ultimate aim is to solve the cocktail-party problem for HI
listeners. Interestingly, there was no evident diﬀerence between NH and HI listeners in
terms of real-time system performance. The system did not work for one subject, but
this was also observed in some NH subjects. For the other two, the system performed
adequately (Figure 4.17. A particularly interesting diﬀerence that was observed for
one particular subject was the open-loop performance compared to closed-loop in
terms of switching attention.
For high levels of closed-loop gain, the unattended
speaker becomes close to inaudible. If this is the case, switching attention to the
unattended speaker will be diﬃcult or impossible. This sets a precedence regarding
the maximum level diﬀerence between competing speakers for the system to allow
switching. The attended speaker should be ampliﬁed in such a way, as to allow the
other speaker to be heard. This is yet another parameter that requires further tuning.
The decoders used for HI participants to evaluate real-time performance were
based on the same procedure for NH participants.
Thus they are not optimally
ﬁt for the HI case. In fact, for better performance, the compressive eﬀect on the
envelope should be less pronounced. The main goal is to represent to transform the
stimulus into a form closest to the neural representation such that the correlation
between them increases. As of now, cochlear ﬁltering is also not implemented, as the
broadband speech envelope is estimated. Ideally, a more robust preprocessing should
include a cochlear ﬁltering stage that can also be tuned for HI subjects (broader
ﬁlters).
A gammatone ﬁlterbank implementation is the de-facto standard and its
implementation details are explained in [Jepsen, Ewert, and Dau, 2008b].
While
this will better system performance, it poses a computational challenge on real-time
processing of the ﬁltering stage.
5.2.3
BCI Illiteracy
It is important to note that both for a select number of NH and HI subjects, the
decoder performed at chance-level. In the case of NH subjects, this happened both
with ’A’ and ’AV’ training. In fact, with BCI systems, this is a very common problem,
where for some participants the system does not work. This problem is referred to
BCI Illiteracy. [Sannelli et al., 2019] report that around 40% of their subjects could
==PAGE:76==
60
5 Discussion
not achieve BCI control for a Sensorimotor Rhythm based BCI system. This can be
attributed to some uniqueness in brain activity.
Small diﬀerences between the attended and unattended correlation scores indi-
cated whether the real-time decoding system would be perform at chance level. These
set of results pose the following questions:
• What are the diﬀerences in the neural mechanisms between subjects who were
able to use the BCI system?
• Given this problem, is there a solution that can restore system functionality for
this population?
A more robust experimental design to what has been used in this project, might
have used some behavioural test (for example presenting a multiple-choice question
after each trial). This would allow one to distinguish better whether the results are
indicative of BCI illiteracy or poor attention in the training session.
5.3
Practical Issues
In this study, there are several practical issues that need to be overcome before such
a system can be employed in a hearing-aid. Below is a list explaining some of the
most evident challenges, that were not tackled in this project.
1. Real-time extraction of visual features: In this project, visual features
were integrated in an ’oﬄine’ system. For a real-time implementation, addi-
tional sensors to the hearing aid must be used to extract visual information
for the user’s ﬁeld of view. The main challenge here is to do face-detection
in real-time. Once the visual features are extracted, it is relatively simple to
integrate them in the pipeline. There are other questions that arise such as:
What happens when there are no detectable faces?
2. Access to clean speech signals: In this project, it was assumed that the clean
attended and unattended speech signals were available. In a practical scenario,
the signal received at the microphone input of a hearing aid would be a mixture
of the two with additional eﬀects such as reverberation and noise. Therefore,
source separation must be performed, prior to any attention decoding. High
decoding accuracies were still observed when using noisy and reverberant speech
in [Fuglsang, Dau, and Hjortkjær, 2017]. It was found that the neural tracking
of a speech stream in a multi-speaker environment was robust to reverberation,
noise and distortion.
3. EEG Acquisition: In this project, 64 channels of EEG were used to capture
the neural response of the participants. This it not so practical in a real-life
scenario.
A viable alternative is to use ear-EEG. [Fiedler et al., 2017] have
==PAGE:77==
5.3 Practical Issues
61
shown that auditory attention could be decoded using single-trial EEG with
ear-EEG.
4. Decoding Time: Real-time decoding was based on EEG and speech windows
of 6.5s. Pushing this to a smaller window length, gives a faster response time,
however with lower accuracy. This being said, it has been observed that audio-
visual cues (even when the decoder does not include visual features) enhance
the decoding accuracy. This gives some more room, to further decrease this
time-window.

CHAPTER 6
Conclusion
The major focus of this project was to study the eﬀect of audiovisual cues on attention
decoding, and whether this can be used to steer hearing aids. These eﬀects were
investigated by collecting EEG data from 20 NH and 3 HI subjects. Ten subjects
were presented with two-talker audio-only, while the remaining ten were presented
with two-talker video.
Audiovisual cues were found to improve performance in attention decoding us-
ing single-trial EEG data. Using data modelling techniques, subjects presented with
audiovisual cues showed a shortening in the negative peak of the TRF. This is hy-
pothesized to be related to some speed-up in the attention mechanism, whereby the
congruent visual stimuli relay information about the speech signal.
Further data
modelling using stimulus reconstruction and CCA, yielded an increase in classiﬁca-
tion accuracy when audiovisual information was presented. This increase was 3.8%
when using stimulus reconstruction and 5% when using CCA.
The real-time decoder simulates the ability of a hearing-aid performing attention
decoding tasks. This has proven to be possible with 6.5s decoding segments. The
collected data has shown that audiovisual presentation facilitates attention switching,
by a shortening in the switching time of 4.5s. This being said, there were cases where
with some participants the attention decoder did not function correctly. This can be
partly attributed to a well-known phenomena referred to as BCI illiteracy.
Real-time decoding for hearing impaired listeners was also investigated. This was
important, to evaluate if such a system would be feasible in a hearing-aid. This was
done with three participants, so any statistical inference can not be made. Two of
the three participants were able to steer the attention decoder. Moreover, further
consideration needs to be taken when applying gain to the attended speaker. The
gain must not make the unattended streams inaudible as this would hinder attention
switching, as was observed.
Lastly, the integration of visual features in an attention decoder were touched
upon. Mouth area was highly correlated with the speech envelope and hence did not
give any ’new’ information about the visual elements in the neural responses. Optical
ﬂow was a better visual feature, compared to mouth area. Mouth area was highly
correlated with the speech envelope, and hence did not provide visual information into
the models. In contrast, using optical ﬂow as a visual feature enabled the extraction of
visual aspects in the measured EEG responses. This was indicated by the respective
TRFs, which portrayed activity near the occipital electrodes. Moreover, above chance

decoding accuracies were also obtained with the backward model and CCA. The audio
and visual modalities were also combined in a CCA decoder, whereby a 2% increase
in decoding accuracy was observed.
6.1
Future Work
In this project, audiovisual cues were found to add some performance beneﬁt. This
leads to other interesting questions that follow, and that can be explored in future
work.
One aspect would be to try and ﬁnd other visual features, such as luminosity that
could convey other information the optical ﬂow is not measuring. The optical ﬂow
was passed through a ﬁlterbank in this project. It may be interesting to try and relate
the preprocessing stage to the mechanisms of visual processing in the visual system,
similar to what has been done for the speech envelope. The implementation of the
real-time visual feature extraction is also another yardstick that needs to be tackled.
The real-time decoder was based on CCA. However, auto-encoder methods have
been proven to outperform CCA-based methods [Bhatt, Jha, and Raman, 2017].
These auto-encoders are known as CorrNets. Similar to CCA the CorrNet is able to
extract the common elements from two diﬀerent views of the data (neural responses
and visual/auditory features). Applying CorrNets seems to be a viable option to try
and squeeze out more performance from the real-time decoder or perhaps operate it
at shorter decoding segments.
Another interesting facet to explore would be Multiway Canonical Correlation
(mCCA). This technique can be used to extract the ’common’ elements of the neural
responses from subjects trained with the same stimuli, which is the case in this work.
The idea is that subjects trained with the same stimuli should elicit activity from
common neural generators and some ’noise’ which is subject dependent. [Cheveigné,
Di Liberto, et al., 2018] have applied this method to obtain a ’denoising’ ﬁlter that
can be used in a preprocessing stage before ﬁtting the CCA model. This could yield
higher accuracy.
Functionally, the system has proven to work for both normal-hearing and hearing-
impaired listeners. This alone is not enough. Future studies should try to link the
increase in SNR provided by this system to perceptual beneﬁts. One way to do so is
to try and measure speech intelligibility as a function of the SNR, while performing
real-time decoding.
