Abstract
In coding theory, codes are used to represent information. Codewords and the error that
can occur when the codeword passes through the channel can be seen as matrices.
Consequently, instead of the standard type code and metric e.g. Hamming metric, we can
use the rank as a metric and then define Gabidulin codes which are error correcting codes
over a rank-metric. Gabidulin codes over finite fields are well studied and has already
application as in network coding but they also have been generalized to infinite fields
like number fields using Ore polynomials also called skew polynomials. Furthermore,
decoding Gabidulin codes while correcting errors is equivalent to find a row reduced basis
of a free left module over Ore polynomials. It boils down to compute the Popov form of a
matrix over Ore polynomials which is a row reduced form with minimal degrees. Moreover,
there exists an algorithm with coefficient growth control to compute this Popov form in a
polynomial time. The complexity of decoding Gabidulin codes has been well studied over
finite fields. Here, we analyse the bit complexity of decoding Gabidulin codes over number
fields which has not been done yet. We state the bit-complexity of decoding a codeword
using two ways of computing the Popov form of a matrix. The resulting bit-complexity
is polynomial on several variables related to the number field, the automorphism, the
representation used to represent the elements of the number field, the construction of the
Gabidulin code, and the magnitude of the error that can occur during transmission.

Introduction
Coding theory is the study of the properties of codes and how they can be used and opti-
mized in different applications. A code is a system of rules that is used to represent and
convert information. Codes are built to ensure efficient data transmission. This thesis
focuses on particular coding techniques that correct errors due to noise that appears dur-
ing transmission. In simple cases, this error could be, for example, dust on a CD or a bit
change when sending a message online.
Figure 1.1: Transmission of a message which is encoded and sent by the sender. Then,
noise is introduced by the channel and the receiver decodes the received message that
can contain errors.
The code elements, codewords, can be seen as matrices over a field hence they have a
rank. We can build rank-metric codes that ensure that the distance between codewords is
maximal where the distance is a rank-metric. Gabidulin codes are such rank-metric codes
that ensures that the distance between codewords is maximal. The error induced by the
channel is also seen as a matrix that is added to the sent codeword. If the rank of the
error is not too large, the received message can be decoded and the original message is
retrieved. Gabidulin codes have been extensively studied over finite fields [Del78; Gab85;
Rot91] and using linearized polynomials. It can be applied in cryptography [GPT91] or
for building space-time codes [GBL00]. Gabidulin codes also attracted a lot of attention
because Silva, Kschischang and K√∂tter [SKK08] used lifted Gabidulin codes to create
an almost optimal subspace codes for random linear network coding. However, some
applications like space-time can benefit from rank-metric codes over infinite fields [Rob15].
Therefore, for the last decade, Ore polynomials have received a lot of attention. They can
be considered as the general case over any field of Linearized polynomial over finite fields.
With Ore polynomials, Gabidulin codes can be defined over field extensions and espe-
cially, number fields as in [ALR13]. However, as for the decoding algorithm in [ALR18],
the coefficients might grow exponentially during computations. This is not a problem in
finite fields where the size of coefficients is bounded by some prime number or a multiple
of a prime number.
Moreover, decoding Gabidulin codes can be linked with computing the normal form of a
matrix over Ore polynomials. The Popov form is a matrix in which rows form a reduced
basis with many nice properties. There exists an algorithm to compute the Popov form
of a matrix over Ore rings with a polynomial complexity as described in [KRS17]. Fur-
thermore, the algorithm uses a linearization to make the computation of the algorithm in

the field where the coefficients of the Ore polynomials lie. This allows a better control
of the coefficient growth. Nevertheless, the bit-complexity of such procedure cannot yet
be found in the literature. This report gives a bit-complexity of the decoding algorithm of
Gabidulin codes over a number field using the algorithms for computing Popov form of
matrices over Ore rings.
In Chapter 2, we recall basic definitions and concepts that are needed to understand the
rest of the thesis. In particular, the first part focuses on field extensions while the second
part narrows down to the particular case of number fields. This chapter also defines a
property on the field, the field extension, and the automorphism that is a prerequisite in
several lemmas and theorems in the thesis.
Chapter 3 is dedicated to Ore rings and its algebra [Ore33]. First, the well-known lin-
earized polynomials and their properties are exposed. We similarly recall the definition
and several properties of Ore rings. Then, linearized polynomials and Ore polynomials
are compared and linked. Finally, as in [ALR18], we define annihilator and interpolating
polynomials which are important polynomials used in the decoding algorithm.
In Chapter 4, we use the results in [ALR18] to define a generalized Gabidulin code on Ore
polynomials. Consequently, we first define a rank metric that we use to define Gabidulin
codes. Second, a key equation, a shift register problem, and Popov form of matrices over
Ore rings are linked together as in [M√ľe+16] to build a decoding algorithm.
Chapter 5 highlights known properties of matrices over Ore rings and points out the prop-
erties of the Popov form. Then, based on [KRS17], we explain the algorithm to compute
the Popov form via linearization.
In Chapter 6, we choose and describe a representation for the elements of a number field
and define a size function to evaluate the bit complexity of the decoding algorithm. Then,
we present some results related to coefficient growth.
Finally, in Chapter 7, the algorithms from Chapter 4 and Chapter 5 are adjusted to be
computed using the chosen representation. The main result of this thesis is presented: a
bound on the bit complexity of the decoding algorithm. Two bit-complexities are presented
depending on the algorithm used to compute the Popov form. The results are discussed
in the last section before concluding this master thesis.

3
Ore Polynomials
This chapter focuses on Ore Polynomials also called skew polynomials. Traditionally,
Gabidulin codes are defined based on a particular case of Ore polynomials : the linearized
polynomials on a finite field with the Frobenius automorphism and without derivation. It
has been shown, originally in [Del78; Gab85; Rot91], that linearized polynomials are well
suited when it comes to maximizing the distance between codewords when the distance
is the rank metric. In this thesis, we generalize Gabidulin codes by defining them over Ore
polynomials and infinite fields as in [ALR18] and [M√ľe+16]. This chapter aims at defining
objects, notations and basic algebra of Ore polynomials.
First, we recall the properties of linearized polynomials. Secondly, we will go through a
similar process but for Ore polynomials using the results of [Ore33]. Then, we present
the link between Ore polynomials and linearized polynomials. Finally, we present the
annihilator polynomial and the interpolating polynomial over Ore ring.

7.3
Discussion
Theorem 7.2.7 and 7.2.8 state two bit-complexities for the decoding algorithm in the case
of a straight-forward computation of the Popov form and in the case of a refined algorithm,
respectively. The improvement is significant on the size s of the input matrix. However,
in our case where s = 2, we do not gain in complexity on the size of the support vector
n. Consequently, using the best known algorithm to compute Popov form as presented in
[KRS17] is an efficient strategy if we deal with matrices of arbitrary size. Especially, as we
discuss at the end of Section 5.2, it might be useful to introduce an orthogonal defect as
in [KRS17]. The complexity would depend on this orthogonal defect and then we might
gain a factor s or n in the complexity.
Also, we see that the contribution of the polynomial associated with the sent codeword, the
support vector and the error on the magnitude of the coefficient is asymptotically similar.
Which means that we can choose the support vector g and the polynomial f associated
to the codeword such the S(f), S(g) ‚Č§S(e) ‚Č§ő≤ where ő≤ is a bound on the size of the
coefficients of the error. Then, we can bound our coefficient growth with the variable

Conclusion
In this thesis, we recall several Ore polynomial properties that can be found in the original
paper [Ore33]. Left-modules over those polynomials are free and matrices over Ore rings
can be defined. We are especially interested in the Popov form of a matrix since it is a
basis of the row span of the original matrix but with minimality on the degree of the entries.
The Popov form can be used to decode Gabidulin codes and we present the algorithm of
[KRS17] that computes the Popov form in a polynomial time via linearization. Besides, we
narrow our thesis to ŌÉ-polynomial that is to say Ore rings without derivation. We use those
polynomials and their properties to define generalized Gabidulin codes as in [ALR18]. As
in [M√ľe+16], we recall the key equation and the shift register problem and how they can
be used to decode Gabidulin codes. Finally, we present a decoding algorithm based on
the previous results. In the last part of the thesis, we analyse the bit complexity of the
decoding algorithm. To that end, we consider a number field and a representation of the
coefficients of this number field. To improve the bound on the size of the coefficient, we
do as much computation as possible in a Z-module. We evaluate the coefficient growth
during the algorithm and then we evaluate the complexity of it. The result is polynomial in
the size of the codes. However, it is based on several variables depending on the number
field itself, but also the automorphism, the chosen representation, the sent word and the
error induced.
Moreover, the result obtained in this thesis is based on very few assumptions on the
number field and this result can constitute a base work to evaluate the bit-complexity of
Gabidulin code for specific applications. Indeed, on a number field the coefficient growth
can be a burden to the complexity of the decoding algorithm. The maximum time allowed
to decode Gabidulin codes is specific to the application and the actual time to decode is
also specific to the application. There are no other bit-complexity of decoding Gabidulin
codes over number fields and thus it is difficult to compare with other works. However,
several improvements can be done on the result we give as discussed in Section 7.3 and it
would be interesting as future work to apply the decoding algorithm on a specific example
and implement it. We could compare it with the implementation done in [ALR18].
Finally, in space-coding, a set of matrices over the complex numbers is used for wireless
communications via multiple antennas [TSC98]. There are several constructions based
on taking a Gabidulin codes over a finite field and mapping the codewords to a complex
alphabet while preserving the rank metric as [LGB03]. Instead of considering a mapping,
we can directly use a Gabidulin code over a number field that is contained in the complex
field. In [Rob15], Robert Gwezheneg proposes a new encoding scheme for spacetime
coding using Gabiduling codes in infinite fields. However, more analysis are necessary
on the subject and the decoding algorithm presented in this thesis is a possible decoding
algorithm that could be used for space time coding. If implemented, it could be compared
in performance with the current Gabidulin codes over finite fields that are used in space-
coding.
