Summary
English
Current wind turbine simulators uses aero-elastic modiÔ¨Åed version of Blade
Element Momentum theory (BEM ). These models rely on full-polar 2D air-
foil data. But doing experiments for airfoils when designing and optimizing
is expensive and time consuming. A reliable and fast full-polar simulation
tool is therefore need. In this report a solver that is able to simulate an
airfoil in a separated Ô¨Çow is developed. The solver is an improvement of
a solver described in [Maskew and Dvorak, 1978]. The improved solver is
named Double Wake Model (DWM ). Both models are iterative and the im-
provement is in the way which a wake is changed doing the iteration. DWM
was found to improve the results for small wakes. A relationship between
the DWM -parameters and Angle of Attack (Œ±) was also developed. The re-
lation was called the Wake Angle Relation (WAR). The WAR-parameters
was calibrated by experiential data and tested. The test showed that WAR
was able to give relatively good predictions just after separation. But the
test was limited by quality of the data. The hope is to couple WAR with
a viscid-inviscid interaction solver and make a reliable and fast full-polar
solver.

The Double Wake Model
In this section the main model for this report is presented. The model is
named the Double Wake Model (DWM ). It is an improvement of a model
introduced by [Maskew and Dvorak, 1978]. The diÔ¨Äerence between the two
models is in the way the wake is moved. This is presented under the Moving
Wake subsection in section 3.2.
The chapter is structured by Ô¨Årst introducing the idea behind the model.
After a short description of things like the model parameters and the ini-
tial wake is given. Then the Numerical Formulation is presented where the
changes from the Panel Method (PM ) is shown. Next the process of Solving
for a steady wake. This is in principle a presentation of the code structure.
The last section is Validation where the solver parameters are adjusted to Ô¨Åt
experimental data. Hereby showing that the model is implemented correctly.
The DWM is an inviscid model to simulate separation using panels. It
assumes that at a point on the suction side, called the Separation Point
(SEP), the vorticity arriving here leaves the airfoil and is then convected
with Ô¨Çow. At the same time vorticity arriving at the TE also leaves the
airfoil and is converted with the Ô¨Çow. This is the two wakes and hence the
name the Double Wake Model. At a steady state there can not be any net
vorticity in the Ô¨Çow. This implies that the vorticity leaving at SEP and TE
has to have the same magnitude but with opposite sign. This is the Kutta
condition for DWM .
Model parameters
The model has two parameters, namely: Wake Separation point (WS) and
Wake Length (WL). Their geometrical interpretation can be seen in Ô¨Ågure
3.1. The parameters are not the same as in [Maskew and Dvorak, 1978]. The
change was made because a new method for moving the wakes was developed.

Further Work
The aim for DWM is to be part of a full polar solver. The solver should be
able to give reasonable results in a time scale useful for design and optimiza-
tion.
This could be done by combining it with an viscid-inviscid interaction
solver like Q3UIC ([Garc¬¥ƒ±a et al., 2014]). The Q3UIC solver is able to give
good results up to separation. From this point DWM is expected to take
over and if WAR is used, Œ±SEP could then become an input from Q3UIC. At
an initial stage keeping the other WAR-parameters Ô¨Åxed.
If more data was available the other WAR-parameters could be be related
to the airfoil shape and hereby give a more accurate result. If more data is
not available CFD simulation could be used to investigate the airfoil shape
inÔ¨Çuence on the WAR-parameters.
DWM also needs to be investigated for its dependence after LE SEP. Some
initial investigations has already been done and a switch to a separation-
parameter that is measured in the y-direction instead of along the chord
seams to be a good idea. This is because SEP can move to the other side of
LE, and this is not possible to capture with the current DWM -parameters.
Maybe also another model for WL is needed since WAR was found to overes-
timate it when the Ô¨Çow had LE SEP. But more data is needed to investigate
this.

Conclusion
The solutions for a potential Ô¨Çow around a panel was found using complex
number. The complex number description was found to have a simpler form
compared to the description with two dimensional real numbers.
The potential attach Ô¨Çow solution for a general airfoil geometry was found
using complex number Panel Method (PM ).
The model was then extended to describe the potential separated Ô¨Çow
for a general airfoil shape.
The model was named Double Wake Model
(DWM ). It takes two input parameters, namely: Wake Separation point
(WS) and Wake Length (WL). The model is inspired by a similar solver
found in [Maskew and Dvorak, 1978]. The model was improved by the intro-
duction of the so called Dynamical Wake Panel Length (‚àÜPW), which allows
the wake panels to vary their length. This insures a consistent WL, even af-
ter wake has been moved. The old model was found to be inconsistent if the
WS to WL ratio was grater than one. This was solved by the new method.
DWM was validated using data for the FFA-W3-211 airfoil. The model
was able to Ô¨Åt the data well, especially the pressure peak was in all cases
possible to Ô¨Åt. There was a big agreement between the data and the model
for Angle of Attack (Œ±) larger than 20‚ó¶. DWM is therefore expected to be
implemented correctly.
An empirical relation for the DWM -parameters: WS and WL was de-
veloped. The relation was named Wake Angle Relation (WAR). The WAR-
parameters is: Œ±SEP, Œ±LE, c and WL90. The WAR-parameters were cali-
brated using results where WS and WL was adjusted to Ô¨Åt experimental
data.
At last WAR was tested for 6 diÔ¨Äerent airfoils. The model was found
to give relatively good results for a limited region just after separation. 5
of the 6 airfoils was measured in the VELUX wind tunnel and indications
that the data might not be valid for strongly separated Ô¨Çow could explain

measured in the VELUX wind tunnel was found to give good result, also
in strong separation. More data from diÔ¨Äerent wind tunnels are needed to
make a more reliable test of the model.
It seems reasonable that combining WAR with an viscid-inviscid inter-
action solver like Q3UIC could be a reliable and fast solver for full-polar
