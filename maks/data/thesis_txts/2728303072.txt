1
Introduction
1.1
Background
The scheduled opening of an extension to Copenhagen’s metro line M4 is set for early
2024. This extension will connect the Sydhavn area to the Nordhavn area, as well as
integrate with the broader metro system in Copenhagen. The extension plan includes the
construction of five new underground stations: Fisketorvet, Enghave Brygge, Sluseholmen,
Mozarts Plads, and an end station at the existing Ny Ellebjerg station. The development
of the extension at Ny Ellebjerg involves building a concourse hall that requires excavation.
The tunnels for the Sydhavn M4 extension are already in place and have been closely
monitored for displacements in the tunnel lining. Given the generally assumed low stiffness
of the limestone formation beneath Copenhagen, a certain degree of displacement was
expected and accounted for in the design. However, the observed displacements in the
tunnel lining of the Sydhavn extension show a discrepancy compared to the predictions
made by the model.
Interestingly, the monitoring data suggests that the tunnels’ actual displacement is less
than initially anticipated. This unexpected behavior indicates that the limestone forma-
tion, in which the tunnels are built, is stiffer than previously assumed and is of particular
interest.
1.2
Aim
This miss match of the anticipated displacements of the model and the measured displace-
ment provides an opportunity of research. The goal of this thesis is to research the stiffness
of the limestone surrounding the tunnels and provide the model with an updated Young’s
modulus. The Young’s modulus of limestone can be calculated using various methods. A
fairly common method of deriving the Young’s modulus is simply by analysing the stress
strain curve.
This allows us to see the Young’s modulus for a certain segment of the
curve. The Young’s modulus can however be derived using elastic waves. In this report
the Young’s modulus will be derived via the elastic waves which has not been done for the
limestone that the metro tunnels are built in. The elastic wave will first derive the bulk
modulus and from that the Young’s modulus will be derived. The expectation is that the
limestone is stiffer than anticipated in the model and therefore the measurements do not
match the initial model.
The discrepancy between the predicted displacement based on the model and the actual
measured displacement necessitates this research to gain a better understanding of the
stiffness of the limestone surrounding the tunnels. The aim of this thesis is to investigate
the stiffness of the limestone and provide an updated Young’s modulus for the model.
Various methods can be employed to calculate the Young’s modulus of limestone. One
common approach is analyzing the stress-strain curve, which reveals the Young’s modulus
for a specific section of the curve. Alternatively, the Young’s modulus can be determined
using elastic waves. In this report, the Young’s modulus will be derived using elastic waves,
a method that has not yet been applied to the limestone in which the metro tunnels are
constructed. The elastic wave analysis will first derive the bulk modulus, which will then
be used to calculate the Young’s modulus. It is anticipated that the limestone is stiffer
than originally estimated in the model, which would explain the discrepancies between the
measured values and the model predictions.

1.3
Method
To be able to derive a new Young´s modulus based on elastic waves then we require data
that contains P-wave and shear-wave velocities. Fortunately, we have received P-wave
and shear-wave velocities from the engineering consultancy Geo which is a privately owned
industrial foundation based in Denmark. This data was initially created for Metroselskabet
by the owners of the metro system in Copenhagen.
This data is derived from three
limestone samples that were each subjugated to a triaxial test and during the test, P-
wave velocities and shear-wave velocities were measured. Unfortunately, the data from
Geo does not contain usable Shear-wave velocities.
Therefore, in this report the bulk
modulus cannot be derived directly from the P-wave and shear-wave velocity but the bulk
modulus has to be modelled. There were two models created that derive the bulk modulus
only using the P-waves. From these modelled bulk modulus a Young’s modulus can finally
be derived that can be put into model that calculates the deformation in the tunnel lining.
Furthermore, Geo has already carried out an analysis of the triaxial test and the Young´s
modulus from their report will presented and compared to the new Young´s modulus
derived within this report.
The stress-strain curves that can be derived from the triaxial data of the limestone sample
is also interesting to analyse and compare to the P-wave velocity. This is because it is in-
teresting to compare the axial stress point of which shear failure occurs on the stress-strain
curve compared to when the maximum bulk modulus is reached. This is an interesting
comparison because when the stiffness decreases, the limestone sample could already be
considered to be failing due losing its stiffness.

2
Geology of the Copenhagen Area
The geology of Copenhagen has been thoroughly studied and the ground formations are
well understood. What is of importance for this report is that the reader is aware of the
to main limestone formations in Copenhagen and that it has been affected by uplift. It
is important for the reader to be aware that the limestone is of the same characteristics
because different formations could have different stiffness parameters which would affect
the derivation of the Young’s modulus. Another aspect that affects the stiffness of the
limestone is that the Copenhagen Limestone has been subjugated to uplift which would
cause micro fractures within the limestone and the micro fractures would be plastic de-
formations. This would cause the Young’s modulus that is derived from the stress-strain
curve to be lower than the true Young’s modulus.
2.1
The Limestone Within the Copenhagen Area
The limestone in the Copenhagen area is called the Danian limestone. The Danian lime-
stone within Copenhagen is split into two categories, the Copenhagen limestone and the
Bryozoan limestone. The Copenhagen limestone usually lays on top of the older Bryozoan
limestone. The Copenhagen limestone is divided into a lower, middle and upper unit [1].
The Ny Elbjerg station is located in Bryozoan limestone.
2.2
Uplift Within The Limestone
Copenhagen is located on the easternmost Danish-Norwegian Basin and it is trending to-
wards the WNW-ESE [2]. The Danish-Norwegian Basin is constricted by the Sorgenfrei-
Tornquist fault zone [2].
From the Sorgenfrei-Tornquist zone is the NW-SE trending
Carlsberg fault [3]. The limestone west of the Carlsberg fault has been uplifted by up to
90m [3]. The uplift could result in fracture in the sandstone. This is because if it reaches
the surface there will be no compacting stress acting on the sandstone and therefore it will
expand. The expansion of the limestone could cause micro fractures with in the limestone.

The ground and tunnel structure deformations were simulated using the Plaxis model.
The software used was Plaxis 2D a finite-element software widely utilised in geotechnical
engineering and is often used to model excavations and tunnels. This chapter explains the
Plaxis model, the location of the building area which the model describes, and outlines
the process of updating it with the derived variables from this report.

The model provided by Ramboll
Danish consulting engineering group Ramboll used the Plaxis model to analyse displace-

5.3
Gassman’s Fluid Substitution
The saturated sample in the triaxial test has a measured pore pressure of zero and there-
fore, the test is said to be drained. However, this is not quite true. Even though the pore
pressure is measured to be zero throughout the entire test there is pore pressure within
the sample because the elastic waves being sent throughout the sample cause vibrations
throughout the sample. The vibrations cause the water within the sample to vibrate and
therefore water pressure acts within the sample. To mitigate the effects of pore pressure
within the sample caused by the elastic waves, the Gassman’s fluid substitution equation
will be applied.
The Gassmans’s fluid substitution equation is expressed with a bulk modulus K but it can
be approximated so that it can be used to express a shear-wave velocity Vs using a method

7
Conclusion
The goal of this report was to analyse the limestone surrounding Ny Elbjerg station and
update the stiffness values used within the model with a new updated way on deriving
the Young´s modulus. The new method was supposed to use elastic waves. However,
there were diﬀiculties on acquiring data where the shear wave velocity was readable. To
compensate the lack of having a measured S-wave velocity the bulk modulus was modeled
and from the bulk modulus other elastic parameters of the limestone surrounding the Ny
Elbjerg station could be derived. The desired parameter for the Plaxis model was the
Young´s modulus. In this conclusion
The results that were derived would indicate that the limestone is stiffer at Ny Elbjerg
then initially assumed. The updated plaxis . Firstly because the modelled elastic wave
Young moduli shown in Table 6.10 are larger for the corresponding labs when compared
to the static analysis in Table 6.11 but this requires that we assume the static modulus
can not be drastically altered by picking different segments when deriving it. Secondly,
because the value picked in Plaxis model is very low or around 0.875GPa. The lowest value
of the modelled Young’s modulus based on the elastic wave is 1.27GPa for lab no. 39.2
Table 6.10 and his based of the Hashin & Shtrikman boundary which is a questionable
method because it derives a negative Poisson’s ratio. The elastic method derives a value
of 4.48GPa and is more accurate then the Hashin & Shtrikman because the values derived
from the bulk modulus derived from the elastic method, discussed in section 6.10 are more
realistic then the values derived from the Hashin & Shtrikman.
The problem in the research for this report was that there was no data on the shear wave
velocities.
If there is ever done a triaxial test on bryozoan limestone with the elastic
wave velocity measurements it would be interesting to calculate the Young´s Moduslus
direcctly. Other interesting research could be to find another model that can model the
bulk modulus from the P-wave velocity.
