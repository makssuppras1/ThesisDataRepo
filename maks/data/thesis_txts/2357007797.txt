Abstract
Over the last few years, wavelengthagnostic has been receiving a lot of interest as low-cost solution
for Wavelength Division Multiplexing - Passive Optical Networks. This solution is based on the fact
that the Optical Network Unit does not know the transmitted wavelength from the Optical Network
Unit to the Optical Line Terminal. For this reason, dedicated wavelengths are needed. Dedicated
wavelengths imply high costs and this is an important disadvantage. In order to reduce the cost, the
idea of using tunable lasers as transmitters has come up recently.
Some advantages of tunable lasers are their high bit-rate, a large reach and better tolerance to reﬂec-
tions. These features with the fact of reducing costs and complexity make of tunable lasers the best
option for Optical Network Unit. There are several ways to tune a laser, but generally tunability can
be achieved by electronic tuning (applying an electrical ﬁeld or an electric current), thermal tuning
(changing the work temperature) or mechanical tuning (moving the mirrors).
The idea of designing a controller relies on tuning the laser automatically instead of manually. The
aim of this thesis is to create a controller capable of transmitting in certain wavelengths without any
calibration, that is, the user can select a wavelength and the controller is the one who has to ﬁx some
values (for this project, currents and temperature) in order to transmit in that wavelength. To achieve
this, a database with the relation between wavelengths and currents and temperature values is needed.

Chapter1
Introduction
1.1
Problem Statement
The new optical ﬁber access networks are based on Passive Optical Networks (PON) technology.
Two main PON versions can be distinguished: GPON (Gigabit Passive Optical Network) and EPON
(Ethernet Passive Optical Network). This project is focusing on GPON as it is the preferred option
in Europe, North America, India and Latin America because it oﬀers more management features
of the Quality of Service (QoS). Addressing GPON, diﬀerent architectures such as NG-PON1, NG-
PON2, WDM-PON and DWDM-PON can be diﬀerentiated. NG-PON1 uses TDM (Time Division
Multiplexing) for the Point to Multipoint architecture. NG-PON2 uses TWDM, that is, the multiple
XG-PON1 using WDM. These architectures will be explained in the Theoretical Background chapter.
WDM and DWDM will be further described in more detail since they are the GPON architecture that
are going to be used in this project. Both of them are based on WDMA (Wavelength Division Multiple
Access). In this context, the term ”wavelength-agnostic” appears, what means that the wavelength
from the Optical Network Unit (ONU) to the Optical Line Terminal (OLT) is not known by the ONU
a priori [1]. Because of that, dedicated wavelengths are needed, so, in order to reduce costs, tunable
lasers have to be used. The main diﬀerence between WDM and DWDM is the channel spacing, that
in DWDM is lower than in WDM. This allows having more channels and for this reason DWDM will
be the one considered in this project.
In order to understand how a laser can be tuned, a brief description of each semiconductor laser and
how to tune them are included in the annex Tunability of semiconductor lasers.
A v-cavity laser is going to be used in this project. This laser becomes essential for next generation
networks, like WDM-PON. This laser requires three electrodes for controlling the wavelength tuning.
To access to all the channels, it is necessary to control these three electrodes. For this reason, it is
very diﬃcult to tune it. So, a new structure was designed, V-coupled cavity, where an only electrode
is needed to control the tuning wavelength. It provides a tuning range of 40 nm and it covers C and L
band [2]. That is why a coupled v-cavity laser ﬁts really good the project aim.Therefore, the objective
of this project is to create a controller capable of controlling a tunable laser when the user wants to
transmit at a speciﬁc wavelength.
1.2
Contributions
This work is focused on developing a controller to manage the laser tuning. This controller must be
able to send packets automatically when the user chooses a speciﬁc wavelength. These wavelengths
have been selected according to the ITU-T recommendation . A Raspeberry Pi is the hardware part
of the controller that is going to be used in this project, whose features ﬁt well the project idea as it
will be described later on.
1

Figure 1.1: Set-Up scheme.
Figure 1.1 shows the block diagram of the set-up, which is the main contribution of this thesis. So, as it
is shown in the ﬁgure, a user interface has been created, which allows the user to choose the wavelength

1.3
Methodology
This is an experimental project in which, using the software provided by the laser company in order to
change the electrodes and temperature values, the relationship between wavelengths and the electrodes
currents will be obtained. Then, a software capable of processing this data and of creating the packet
that is going to be sent to the evaluation board has been created. There is no limitation as far as
temperature is concerned, that is, working at room temperature is possible.
The ﬁrst stage of this work is introducing the theoretical background required. During the Theoretical
Background chapter, a brief introduction to the access networks will be made, as well as its develop-
ment along the history. This project is focused on the PON architecture. In this kind of architecture,
diﬀerent standards can be found such as GPON, EPON, NG-PON1, NG-PON2, WDM-PON and
DWDM-PON. DWDM-PON network is the point of interest of this thesis, because it is the current
technology and it provides such features as high security, transparent protocol and other more advan-
tages that are described in the next chapter. Apart from the access network, the hardware controller
that is going to be used is also explained, clarifying the diﬀerences between an Arduino Uno and a
Raspberry Pi 3. This chapter will be concluded describing the USB protocol as such information is
going to be used to connect the Raspberry to the laser evaluation board.

The next stage describes the experiments that have been done. For this aim, a simpliﬁed scheme of the

1.4
State of the art
As it has been described in the Problem Statement section, wavelength-agnostic is getting a lot of
interest in WDM-PON and DWDM-PON systems. One of the biggest challenges to cope with is the
need of colorless operation of the ONU to get low operational costs. Colorless ONUs means that all
of them are the same and each ONU should have a speciﬁc wavelength. With these requirements, the
best ONU transmitter is a tunable laser since they reduce the cost and complexity of the network.
Because of the increasing demands of the users, the deployment of optical access networks is one of
the most important telecommunication challenge now. However, this is not a local objective, it is
an international aim. In 2009, the United States have activated the ambitious program Connecting
America: The National Broadband Plan to provide 100 Mb/s downstream and 50 Mb/s upstream con-
nections to one hundred millions American households and 1 Gb/s connections to local communities
by 2020 [3]. In order to supply this necessity, several access network approaches were developed like
FTTx solutions. The most important of these solutions is FTTH as well as the most expensive where
around the 70% of the cost is due to the trenching. However, there are options to reduce these costs
like infrastructure sharing.
PON networks are actually considered the most cost-eﬀective way to deploy FTTH networks [3]. These
networks are based on P2MP topology without any active devices. There are other solutions like Active
Optical Networks, that also use a P2MP topology but they require optical-electrical conversion. Some
of the most highlighted features of PON networks are their high reliability and low power consumption
(since there are not active elements), as well as simpler maintenance.
Another option to develop FTTH is to use reach-extended PONs which are considered passive despite
they use optical ampliﬁers. This is due to the absence of optical-electrical conversion.
Since the greatest advantage of PONs is the infrastructure sharing, signals for and from each user
must be combined at the physical level through multiplexing techniques in downstream and multiple
access techniques in upstream [3]. Two main multiplexing options can be distinguished. The ﬁrst one
is based on time division (TDM) and the other is based on wavelength division WDM. TDM-PONs
were the only PON model standardized by ITU-T in 2014.
Most of the basis of ITU-T standards are related to the FSAN (Full Service Access Network) speciﬁca-
tions. BPON (Broadband Passive Optical Network) was the ﬁrst standard developed by this company.
It provides around 900 Mbps downstream rate and 400 Mbps upstream bit rate and a 20 km reach. As
an alternative,IEEE developed EPON which is based in the Ethernet protocol. This standard provides
a symmetric bit rate of 1.25 Gbps. EPON is commonly used in Asia. FSAN rapidly answer to this
standard by developing GPON. It provides a downstream rate of 2488 Mbps and in the upstream
channel a bit rate of 1244 Mbps. This standard can reach 30 km and it has come across a great success
3
==PAGE:15==
in America and Europe. GPON evolution into XG-PON, which provides 10 Gbps in downstream and
2.5-10 Gbps in upstream.
A generic PON is constituted by an OLT placed in the CO (Central Oﬃce) connected to N ONUs.
This is possible thanks to splitters [3]. In both sides (OLT and ONU), a laser diode is commonly used.
However, for WDM-PON laser diodes are still expensive as dedicated wavelengths are needed.
According to DWDM multiplexion, since it is the one that is going to be considered in this project, its
nodes started as ﬁxed add/drop ring structure without the ability to dynamically route wavelengths
[4]. These structures where the wavelength is ﬁxed are usually named as colored. As these structures
can be an issue, there is a large necessity of removing restrictions wavelength assignment, that is, get
resources with dynamic assignment of the wavelength.
Over time, networks have evolved to support higher bit rates with the expectation that upgrading
existing networks would require minimal changes [4]. There are some limitations that have been kept
in mind such as chromatic and polarization mode dispersion, but they can be reduced just by using
digital signal processing like using FPGA. As we have mentioned, DWDM has experienced a reduce
in channel spacing. Moreover, a 50 GHz spacing is becoming common in new systems.
DWDM was used as a point-to-point topology between nodes, where, the only needed function was
wavelength multiplexing and demultiplexing at each node [4]. However, DWDM became more ﬂexible
when multinode conﬁguration was developed.
So taking into account the necessity of colorless ONUs, the idea of using tunable lasers as transmit-
ters appeared. The state of the art of tunable of semiconductor laser is covered in the Tunability of
semiconductor lasers annex.

Chapter2
Theoretical Background
2.1
Introduction to optical ﬁber access networks
As a pointing start, the meaning of an access network should be pointed out. Considering its function,
two types of optical networks can be distinguished:
• Transport network: it does the routing, processing and traﬃc transport tasks.
• Access network: it is the part of the network between the ﬁnal user and the CO.
Figure 2.1 shows the diﬀerences between these two types of optical networks. As it can be seen in this
ﬁgure, the central oﬃce is the union point between the Transport and the Access networks.
Figure 2.1: Transport and Access Networks [5].
2.2
Current Access Networks technologies
• xDSL (Digital Subscriber Line) Technology: this kind of technology uses the local telephony
cable to provide a digital connexion. So, there are 2 diﬀerent channels, one for downstream and
upstream and the other for voice. It can transmit multimedia information with a high bit rate.
There are diﬀerent versions such as ADSL, ADSL2, VDSL, etc.
• HFC(Hybrid Fiber-Coaxial): it combines optical ﬁber and coaxial cable. Figure 2.2 shows the
basic scheme of this technology. The users bandwidth is really small but the bit rate can reach
57 Mbps in downstream and 9.2 Mbps in upstream for 25 km.
5
==PAGE:17==
Figure 2.2: HFC scheme [5].
• FTTx(Fiber To The x) technologies: it allows each user to connect with the local oﬃce using
optical ﬁber. It integrates voice, data and video in the same network. It has a large bandwidth,
is simple, cheap and ﬂexible.
2.3
Passive Optical Network
PON were installed in order to provide broadband services worldwide[6]. A PON network features a
point-to-multi-point (P2MP) architecture to provide broadband access. The P2MP architecture has
become the most popular solution for FTTx deployment among operators[7]. Figure 2.3 shows the
typical architecture.
Figure 2.3: PON Architecture [8].
2.3.1
PON History
In 1995, FSAN was created. It was formed by 7 big operators and their objective was to deﬁne the
PON standards. In 1999 BPON/APON, based on ATM, appeared and in 2003 a new PON standard
was created: GPON, with a 1 Gbps rate, high speed and eﬃciency. This chapter is focusing on this
standard. In 2001 EFM (Ethernet in the First Mile) was founded and its objective is to standardize a
1.25 Gbps symmetrical system: EPON.
2.3.2
EPON
EPON oﬀers a higher average bandwidth for the client than GPON. It can support up to 1.25 Gbps
in downstream (1510 nm) and upstream (1310 nm), so it is symmetrical. EPON can give service to
32 clients. The maximum reach is 20 km [9]. The EPON frame length is variable.
6
==PAGE:18==
2.3.3
BPON and GPON
BPON is ATM cell-based and for that reason, it was ﬁrstly named APON (ATM-PON). But then,
the name was changed to BPON to emphasize that they were not limited to ATM traﬃc [9]. BPON
can support speeds of 622 Mb/s in downstream and 155 Mb/s in upstream, so it is much lower than
EPON. With the aim of increasing the nominal rate, FSAN and ITU developed the GPON standard,
which provides a rate of 2.4 Gb/s downstream and 1.2 Gb/s upstream. GPON can support 64 or even
128 clients and the maximum reach is 25 km.

2.4
New PON architectures
After developing the GPON standard, FSAN continued studying NG-PONs (Next Generation PON)
and divided it into two phases, NG-PON1 and NG-PON2. These NG-PONs were developed in order
to obtain transmission rates of 10 Gb/s or higher.
2.4.1
NG-PON1
NG-PON1 can be considered as a mid-term upgrade from GPON, and its deployment was immediate
because it conserved the ODN (Optical Distribution Networks), so it can use the optical ﬁber and
the splitters of GPON. It enhanced TDM from GPON, with a higher transmission rate. It inherits
the point to multipoint (P2MP) architecture of GPON. There is coexistence with GPON, the passive
infrastructure is the same, but the Tx and Rx work with higher rate. The new wavelengths are shown in
Figure 2.4, where it can also be appreciated the NG-PON2(TWDM-PON) wavelengths. The speciﬁed
NG-PON1 system is called XG-PON1. FSAN decided to deﬁne XG-PON1 as an asymmetric system,
where the downstream rate is 10 Gbps and the upstream is 2.5 Gbps so this means that it has improve
four times the downstream bandwidth of GPON and twice in the upstream case.
Figure 2.4: XG-PON and TWDM-PON wavelengths [10].

2.4.2
NG-PON2
This standard started on 2011 by FSAN. Its main objective is to achieve a transmission rate higher
than 10 Gbps. They planned these requisites: get a 40 Gbps transmission rate in downstream and
upstream, cover a distance of 40 km and use a splitter ratio of 1:64. They had diﬀerent alternatives:
• XLG-PON: 40 Gbps TDM-PON
• OFDM-PON: Quadrature amplitude modulation (FT)
• TWDM-PON: multiples XG-PON1 using WDM. They adopted this alternative in 2012.
Its main characteristics are: multiplex 4 XG-PON using 4 pair of wavelength, so a 40 Gbps rate in
downstream and a 10 Gbps rate in upstream can be achieved. As it can be seen in Figure 2.4, there
is coexistence with the other PON.
2.4.3
WDM-PON
This is a new technology, based on WDMA(Wavelength Division Multiple Access). The main problems
of this technology are its relativies high cost and the diﬃculty to redistribute the user non-used capacity.
The physic layer objectives are: minimizing the costs, increasing the security and getting independency
of the physics parameters. To achieve these, colorless ONUs must be used, what means that there are
no lasers in the user dependencies, all the ONUs are the same and dedicated wavelengths are needed.
For this purpose, a cost-eﬀective tunable laser might be used in next generation access networks [11].
However, wavelength-agnostic WDM-PON has received a lot of interest as a low-cost solution for
metro-area networking.
Wavelength-agnostic means that the ONU does not know the wavelength
from this ONU to the OLT and for that reason, tunable lasers are needed. This is the main point of
this thesis and it will be developed in the next chapters. In WDM-PON case, the ODN consists of
wavelength selective elements like Arrayed Wave-guide Gratings (AWGs) [1].
A WDM-PON is a promising solution for access networks since it provides almost unlimited bandwidth,
high security and protocol transparent. Other advantages of this architecture are the scalability in
total capacity and the lower insertion loss [12].For residential access, wavelength-sharing is a must but
each client must have a bit rate of, at least, 500 Mb/s [13].
2.4.4
DWDM-PON
Dense Wavelength Division Multiplexing is deﬁned in ITU-T G.671. This WDM technology is char-
acterized by a narrow channel spacing. There are several variety of channel spacings: 12.5 GHz, 25
GHz, 50 GHz, 100 GHz, etc [14]. For this project, a channel spacing of 100 GHz has been chosen.
With this technology, up to 44 dedicate and secure channels operating at 10 Gbps can be reached.
Each channel is protocol transparent, that means, it is independent of other channels and can accept
any protocol. The requirements of multi-service support, high bit rates and increased reach lead to
the use of DWDM in the next generation access systems [12].
Tunable lasers have been developed and widely used for dense WDM systems in core/metro networks
[15].

Figure 2.5 illustrates an example of DWDM-PON. The OLT is formed by diﬀerent transmitters and
receivers that are connected to an AWG. These transmitters and receivers are set at the wavelength
band of the port of the multiplexing to which is connected. When the network load is low, the array
of transmitter can be replaced by tunable lasers so they can shift from one wavelength to another.
By doing this, the number of transmitters is reduced and it will be less OLT transmittes than the
number of ONUs. After the multiplexer, that in this case is an AWG, there is ﬁber in order to connect
both AWG. Related to the ONUs, each ONU is formed by a transmitter, that in this case must be a
tunable laser because all ONUs are going to be the same, and a receiver. As the ONU is connected to
the AWG and only a speciﬁc wavelength will arrive to the ONU, it is not necessary to have a tunable
receiver [9].

2.6.4
USB Protocol
Universal Serial Bus (USB) was developed by Compaq, Intel, Microsoft and NEC. The aim was to ﬁnd
a solution to the variety of interconnection methods. It should be plug and play. It was developed to
avoid DMA (Direct Memory Access) use.
USB speciﬁcation deﬁnes three diﬀerent speeds [21]:
• Low speed: it is thought to use with low speed devices such as mouse. In this case, the cable is
thinner and more ﬂexible.
• Full speed: in order to use the rest of devices.
• High speed: it was introduced with USB 2.0. If a device is 2.0, that doesnt mean it will use high
speed.
The current USB speciﬁcation is USB 3.0. The used topology is tired star topology. A master is able to
control 127 slave devices. The host controller is connected to a hub and this allows the computer to have
various USB ports. All the communications are started by the host. There cannot be communication
between the devices. These devices need to be asked by the host to talk. Example of data ﬂux: In a
speciﬁc instant, only the host or a device can transmit. When the host is transmitting, the packets go
downstream until they reach the device. Only the addressed device will take the data [21]. Only one
device is able to transmit data to the host and the hubs will repeat the packets on the upstream way.
13
==PAGE:25==
A packet is the smallest element in a transmission. Features:
• The least Signiﬁcant Bit is always transmitted ﬁrst. In multi-byte data the Least Signiﬁcant
Byte is transmitted ﬁrst.
• Each packet implies a certain number of bytes at the transmission speed. Before and after the
packet is sent, the bus is in idle state.
• The packets start with a synchronization patron, then the data and the EOP (End Of packet).
• The data codiﬁcation is NRZI (Non Return To Zero, Invert on zeroes).
A transaction is a three packet sequence that transmit in a safe way the data. There are three types
of transaction:
• OUT Transaction: it sends data from the host to the device.
• IN Transaction: is sends data from the device to the host. The host will start this transaction
• SETUP Transaction: it sends conﬁguration data form the host to the device.
4 diﬀerent forms to transmit data in a USB bus can be distinguished. Each one has its own purpose
and characteristics as it can be seen in Table 2.3:

When an USB device is plugged, the host realizes that a device has been plugged and it sends a reset

Tunability of semiconductor lasers
Figure I.1 shows a simpliﬁed scheme of how tunable lasers works. The optical power P should only
depends on the laser current whereas the laser wavelength should only depends on the tuning current.
However, this is the ideal mechanism and it cannot be obtained with tunable laser diodes. This is
due to the fact that the tuning mechanism also aﬀects to the optical losses, so the optical power
will change also with the tuning current. On the other hand, the laser current will also aﬀect the
wavelength because it increases the temperature.
Generally, a laser can be tuned by these three
methods: Electronic tuning, Thermal tuning and Mechanical tuning. Electronic tuning refers to the
application of an electric current which modiﬁes the refractive index. Mode number m and wavelength
λ of a laser diode depend on the eﬀective refractive index n and the eﬀective length L of the cavity:
mλ/2 = nL[12].
Figure I.1: Scheme of electronically wavelength-tunable laser.
The tuning range represents the most speciﬁc parameter for a tunable laser. It was found that the
spectral linewidth is strongly aﬀected by the tuning functions.

Distributed Feedback (DFB) Lasers
This kind of laser is typically used for WDM systems. The feedback is distributed throughout the
cavity length. To achieve this, there is an internal grating that makes that a periodic variation of the
mode Index-Feedback happens. This occurs thanks to Bragg diﬀraction, a phenomenon that couples
the waves in the forward and backward directions. But this coupling only happens if the wavelength
satisﬁes this equation: λ = m(λB/2¯n). In this equation, λ represents the grating period, m (integer)
is the order of the Bragg diﬀraction and ¯n is the average index. If m = 1, that means that the ﬁrst
order of the Bragg diﬀraction, the couple between forward and backward waves will be maximum [23].
The semiconductors that use the DFB mechanism can be classiﬁed in two types: (1) DFB lasers; (2)
Distributed Bragg Reﬂector (DBR). The main diﬀerence between both of them is that in a DFB laser,
as we have mentioned before, the feedback occurs throughout the cavity length whereas in a DBR
there is a mirror in each end part of the laser which reﬂectivity is maximum at λB. Figure I.2 shows
this structure.
Figure I.2: DFB laser structure [24].
The gain of these lasers is not enough if direct modulate is required. For this aim, a phase shifted DFB
lasers is used, where the grating is shifted λ/4. This shifted produces a phase diﬀerence of π/2, and
because of that, these lasers provide a higher gain. The periodic variation of the thickness of the active
layer implies a periodic variation of the mode index all along the cavity length. This fact leads the
forward and backward waves being coupled [23]. Modern WDM systems require single mode and they
need a wavelength that remains ﬁxed all along the system, so lasers must be as narrow-linewidth as
possible. The large number of DFB lasers that are used inside a WDM transmitter make this system
expensive. This problem can be solved using semiconductor lasers whose wavelength can be tuned.
With this aim, multi-section DFB and DBR were developed. A typical laser structure consists of three
regions: the active section, the phase-control section and the Bragg section. We can bias each section
independently. The injected current in the Bragg section determines the Bragg wavelength and the
one injected into the phase control is used to change the phase of the feedback. The laser wavelength
can be tuned over the range 10 15 nm by controlling these currents [22].
Distributed Bragg Reﬂector (DBR)
First of all, we must note that the reﬂection coeﬃcient for a periodic structure depends on the wave-
length. If this wavelength is close to the Bragg wavelength λB, the reﬂection of the grating will be in
phase. If the wavelength is far from λB, the reﬂection will not add up in phase. Taking into account
this principle, we can conclude that it will be a dependence between the wavelength and the cavity
gain, what is the main idea behind DBR [22]. The basic principle of a DBR is that there is a grating
at one or both ends of the laser structure. The structure of this kind of laser is more complex than a
conventional laser because the active layer is only in part of the laser and not all along it. In Figure
I.3, we can see what we have just described.

It is important to ensure a good power distribution from the active part to the grating because any
unbalance will lead to some losses. If the power distribution is good enough that guarantees a perfect
coupling.
Coupled-Cavity Semiconductor Laser
In this kind of laser, single mode operation is obtained just by coupling the laser cavity to an external
cavity. The in-phase feedback only occurs for those laser modes which wavelength is almost the same
as one of the longitudinal modes of the external cavity. There are several schemes of coupled cavity
lasers. We are going to describe two of them.
In the scheme shown in Figure I.4, it can be checked that the light from a semiconductor laser is
coupled with an external grating. In order to get a stronger coupling, the reﬂectivity of the cleaved
facet must be reduced. This kind of laser is very important for their tunable capacity. That means
that a wavelength over a range of 50 nm can be selected.
The problem of this laser is that it is
non-monolithic, what makes diﬃcult the stability of the system [24].

between the two sections. The range of tunability of this laser is over 20 nm by modifying the injected
current. The problem of these lasers is that this tuning is not continuous.

Vertical-Cavity Surface-Emitting Lasers (VCSELs)
This kind of laser suﬀered a rapid development in the late 1980 and now is one of the most important
laser types for short-reach data communication. Actually, VCSEL laser based on GaAs emitting at
850 nm has become the dominant optical source in multi-mode transmission. In addition, this kind of
laser is also highly attractive in transmission in the 1300-1600 nm range. They are very important for
applications such as communications between wireless base stations, computer communications, etc.
The output beam is perpendicular to the plane of the active layer and the emitted light is a circular
beam. Bragg reﬂectors consisting of layers with alternating high and low refractive indices provide the
optical feedback. Due to the short cavity length, very high reﬂective index (>99.5%) is required, so
the reﬂectors have between 20 and 40 layers [24]. Also for this reason, the space between longitudinal
modes is large compared to the width of the gain curve. Therefore, if the resonant wavelength is near
to the gain peak, mono-mode operation will be achieved. Other advantages of this laser include lower
production price, eﬃcient ﬁber coupling because of the circular output beam. Due to its continuous and
mode-hop free tuning, it is important in other applications like WDM-PON because the infrastructure
of this system requires a speciﬁc wavelength for each ONU, so widely tunable VCSEL can be used for
this aim [25].
Most of VCSELs wavelength is usually about 1 m. One disadvantage of these lasers is that it is more
diﬃcult to fabricate them for long wavelengths (in 2nd and 3rd window). Another disadvantage is that
the output power is very small (few milliwatts) because of the small active layer. VCSEL lasers are
well suited for WDM applications because they can be tuned over a wide range using the micro-electro-
mechanical system (MEMS). A VCSEL laser can also be tuned by using the temperature dependence
or the refractive index of the Bragg reﬂector. Because of the large space between longitudinal modes,
the tuning is very continuous and only one control current is needed.
There are some problems with temperature tuning such as the large increase of temperature is required.

can be achieved by using an external mirror near the laser surface and adjusting the distance between
both of them.
A good VCSEL tuning (about 102nm) can be obtained using surface micromachining technique. This
technique uses DBR mirrors which leads better alignment and it has a better cost-eﬀective ratio.

V-cavity semiconductor laser
Tunable lasers have become essential for next generation networks. Multiple electrodes with complex
control algorithms are usually required for wavelength tuning. Some schemes for reducing fabrication
complexity have been developed such as using a structure formed by two ring resonators. Nevertheless,
three electrodes for controlling the wavelength tuning still been required. To access to all the channels,
it is necessary to control three electrodes.
This fact makes the wavelength tuning very diﬃcult.
However, a new tunable laser structure was designed, V-coupled cavity. In this device, an only electrode
is needed to control the tuning wavelength. This kind of laser is attractive for short reach applications.
It provides a tuning range of 40 nm and it covers C and L band [2].

V-cavity laser consists of two Fabry-Perot cavities with slightly diﬀerent optical path lengths. As it
can seen in Figure I.8, there are three electrodes on the surface. The ﬁxed gain electrode and the
common electrode provide stable gain and the channel selector electrode is used for select the channel
wavelength. The wavelength is ﬁxed by using current injection that alters the refractive index through
thermo-optic eﬀect. The laser is fabricated on InGaAsP/InP multiple quantum wells structure whose
gain peak is on about 1520nm.
A V-cavity laser can achieve a tunability range of 40 nm by combining the electrodes current and the
temperature. The SMSR is about 35 dB. This device presents some advantages such as compactness
and easy wavelength control.

II. TOSA Tuning Characteristics 
TOSA tuning characteristics are specified in number of channels at 100GHz or in tuning 
wavelength range in nanometers. Reference parameter settings are provided for injection 
current into each electrode and TEC temperature in the data file that comes with the TOSA 
delivery. A typical tuning curve is given, as shown in Fig. 2. 
 
Fig.2 Typical TOSA tuning characteristics. 
 
For each channel, a set of current setting values for each electrode are provided for reference 
purpose. These parameters give a guideline about how to set the laser to specified 
wavelength. Customer may adjust the parameters to fine tune the wavelength or to change 
the output power. 
 
To tune the wavelength channel, change the injection current into Pin 7 (Electrode 2). The 
wavelength changes according to the tuning curve as shown in Fig. 2. The side mode 
suppression ratio (SMSR) will change somewhat within the current range for each channel. 
One should set the operating point that provides high enough SMSR.  
 
To fine tune the wavelength, the following three approaches may be used:  
(1) Change the injection current on Pin 4 (Electrode 1). This is not to be used when the laser 
is directly modulated with RF signal applied on Pin 4, since it will affect the modulation 
extinction ratio.  
(2) Change the injection current into Pin 7 (Electrode 2) and Pin 8 (Electrode 3) 
simultaneously. Increase the currents on both pins by approximately the same amount, 
while maintaining the good SMSR.  
(3) Change the TEC temperature by changing the current between PIN 1 and 2, while 
monitoring the temperature through the on-board thermistor (Pin 9).
