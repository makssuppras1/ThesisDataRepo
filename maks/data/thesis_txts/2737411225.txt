Abstract
The metabolism in cancer cells is a reasonable hallmark for the detection of cancer and can
be investigated with Magnetic Resonance Imaging (MRI) of hyperpolarized Carbon-13 (13C)
labeled substrates. This method is however limited by time as the polarization of the substrates
decreases in a short time and different methods of accelerating MRI are of interest. In this
project, it is the aim to gain experience with an insert gradient able to create an oscillating
gradient. This oscillating gradient will potentially accelerate MRI when working simultaneously
with the gradients of the MRI scanner as data in k-space can be acquired in lanes instead of lines.
Experience with an oscillating gradient has been gained in three ways. 1: An insert coil has been
connected to an audio amplifier while placed inside the scanner and a resonance circuit has been
tested and simulated to see how eddy currents will affect the current in the insert coil. It was
seen that the eddy currents had a considerable effect on the images and further investigation on
how to improve the circuit is needed to continue. 2: MRI data acquired while using an oscillating
gradient of the scanner has been analyzed and compared with the results of simulated data. The
oscillating gradient in the scanner has been managed by the MNS research pack and the results
show a close resemblance with the results of the simulated data, making it a decent method
for characterizing the parameters of an oscillating gradient. 3: The oscillating gradient of the
scanner has been used to acquire data for magnetic resonance spectroscopy imaging (MRSI) with
water, ethanol, and acetone. The data has been reconstructed with two different methods: the
non uniform fast fourier transform (NUFFT) and by applying the fourier shift theorem in using
the fast fourier transform (FFT). The different methods used when working with an oscillating
gradient in MR scanning have been used to gain experience and the different observations are
documented in this report. This includes considerations in regards to the Point Spread Function
(PSF) when reducing the number of phase encoding steps introduces aliasing artifacts, visually
comparing the different imaging methods, and how the scaling of the oscillating gradient strength
influences the reconstructions.

Introduction
In the world today, cancer is a major cause of death being the origin of nearly one out of six[1] and
generally causing a lot of issues for the people living with the disease and their relatives. Cancer
can develop in many different ways throughout the body and spread by creating metastases.
The cancer cells are different from normal cells in several ways, including resisting cell death,
inducing angiogenesis, enabling replicative immortality, activating invasion, evading growth sup-
pressors, and sustaining proliferative signaling[2]. The formation of a tumor requires a mutation
in the normal cells which transforms into cancer cells. This transformation depends on different
changes in the metabolism that facilitate the different characteristics by supplying additional
nutrients necessary for proliferation[3]. A specific hallmark in mature and malignant cancers is
known as the Warburg effect where the cancer cells favor the so-called ”aerobic glycolysis”. In
aerobic glycolysis, the cancer cells will produce larger amounts of lactate than normal cells using
either oxidative phosphorylation or anaerobic glycolysis regardless of the presence of oxygen[4].
The reprogramming of the metabolism in cancer cells can be detected and possibly quantified
using this hallmark where increased conversion of pyruvate into lactate can be observed. This
can be done by MRI of hyperpolarized 13C labeled substrates[5]. However when working with
hyperpolarized substrates the polarization decreases due to nuclear interaction, why the method
is limited by time, and methods that can accelerate MRI are of high interest.
MRI is a non-invasive imaging technique that can show anatomical and physiological features by
using the magnetic properties of the nuclei in the body. Atoms with an odd number of nuclear
particles have a nuclear spin. The spin is random when not exposed to a magnetic field, but
when placed inside a magnetic field as in an MRI scanner the net magnetization aligns with
the main field. When placed in an MRI scanner the nuclei will spin at the larmor frequency,
which allows exciting the spins out of equilibrium with a radiofrequency field to obtain a signal
called the free induction decay (FID). To gain spatial localization, gradient fields are applied
in addition to the main field by gradient coils, so that the precession frequency changes with
position. MRI has many applications where the focus in this project will be on gaining experi-
ence with the method introduced by Versteeg et al in [6], where the scan time is 5.3 fold shorter
without causing Peripheral Nerve Stimulation (PNS) or introducing any artifacts. The shorter
scan time is achieved by introducing an insert coil driven by an audio amplifier which creates a
fast switching gradient that can be combined with the conventional imaging methods to enable
sampling in k-space in lanes instead of lines.
Initial experience with creating an oscillating gradient using an insert coil was gained in a
previous project where the behavior of the hardware setup was described and attempted charac-
terized. The objective of this project has been to gain experience with an insert coil used as an
oscillating gradient. This has been attempted by exploring how an insert coil connected with an
amplifier behaves inside an MR scanner. When using an insert coil as an oscillating gradient it
is relevant to be able to characterize the gradient by its frequency and amplitude, why a model
for characterizing the oscillating gradient using signal processing has been suggested. Finally
when introducing an oscillating gradient the sampling trajectory in k-space will be non-uniform
causing a problem with the conventional reconstruction methods, why two different methods
have been used in reconstruction MRSI.

Theory
In this section, the theoretical background of this project is presented. This includes:
• Background on the relation between current, frequency, and magnetic field in an insert
coil.
• A model for simulating the signal acquired by the scanner and how this can be used to
characterize the gradient strength of the insert coil.
• Description of k-space, how the oscillating gradient influences the sampling trajectory, and
how a PSF can be used to investigate the sampling density.
• Background on how MRI and spectroscopy can be combined as MRSI and the expectations
related to the results in MRSI.
• Fourier theory used in the reconstruction of non-uniform MR data.
2.1
Resonant circuit
When working with an insert coil in MRI it is relevant to investigate the different features of
the coil such as the magnetic field generated by the coil, the inductance of the coil, and their
relation to the components of the coil driving circuit. The magnetic field strength (B) generated
by the current (I) in a coil is described by the law of Biot-Savart. For a Helmholtz coil, B along
the center line of the coil at a position (z0) is given by the relation in equation 2.1.
B =

In equation 2.1, µ0 is the vacuum permeability, N is the number of turns and r is the radius of
the coil. In addition to the magnetic field strength generated by a current in the insert coil, it
should be considered how magnetic fields from the scanner can induce a current in the coil called
eddy currents. This can be described by Faraday’s law of induction which states that a change
in magnetic field will induce a current in a nearby conductor such as a coil. The direction of this
current can further be described by Lenz’ law stating that the direction of the current opposes
the change in magnetic flux. The change in magnetic flux is given as: dΦ/dt = B/dt · A cos(θ),
where B/dt is the change in magnetic field, A is the area of the surface of which the field is
changing and θ is the angle between the direction of the magnetic field and the surface.
When working with an insert gradient in an MR scanner it can be an advantage to use a resonant
circuit to further amplify the current in the coil[7]. In a circuit with an inductor and a capacitor
(LC circuit) in series the impedance will be close to zero when the frequency is in resonance
and relatively high outside of resonance. The resonance frequency (ω) is given by equation 2.2,

As seen in equation 2.3, the voltage rating will increase with an increase in current, which will

2.5
Magnetic resonance spectroscopy imaging
In MRSI it is the goal to combine methods of MRI and spectroscopy to gain both spatial and
frequency-dependent information about an object which enables the characterization of metabo-
lites in a tissue. In spectroscopy, it is possible to distinguish different nuclei in a compound as
the different positions in the chemical structure will have different electron structure shielding
the nuclei. The different nuclei will have a different resonant frequency described by equation
2.10, where σ is a shielding constant[14].
ω = γB0(1 −σ)
(2.10)
In spectroscopy the signal strength is proportional to the number of nuclei of interest and σ
depends on both the overall chemical structure and the position of the nuclei[15]. This means
that a compound can show multiple peaks if the nuclei have different chemical environments.
As the precession frequency depends on B0 the results in spectroscopy are shown on a relative
axis either relative to a substance that the compound can be mixed with as the reference or
with an offset that can reference the ppm scale to a frequency axis in Hz.

to increase sampling density in k-space. The increased sampling density can possibly reduce

and acetone, where the results can be predicted due to prior knowledge. Generally, water is

the NUFFT. The NUFFT is an approximation of the non uniform discrete fourier transform

3.2
Characterization of oscillating gradient
When using an oscillating gradient it is relevant to ensure correct values of the amplitude and
frequency of the oscillations in order to know the sampled locations in k-space. The characteri-
zation can be done with signal processing as described in section 2.2, where it is assumed that
the AUP of the magnitude spectrum of the first and second derivative of the phase of the FID
is proportional to the amplitude of the oscillating gradient. Additionally, it is assumed that the
location of the peak in the magnitude spectrum of the first and second derivative of the phase of
the FID corresponds to the frequency of the oscillating gradient. When using the magnitude of
the first derivative of the phase of the FID to compute the AUP it will be referred to as model
1 and when using the second derivative it will be referred to as model 2. Before analysis or
reconstruction can be done from the data a few steps of preprocessing must be done, which are
described in appendix D.
Characterization of an FID acquired while using an oscillating gradient was done by first com-
puting the phase of the FID using the MATLAB functions angle and unwrap. Then the first
derivative was found by finding the difference between each point of the phase and dividing by
the corresponding difference in time. For model 1 the filter with a hanning window was applied
here by multiplying the first derivative with a normalized hanning window of the same length.
As the derivative of the phase does not oscillate around zero an offset was added before the
filter and then subtracted again after the applied filter. The second derivative of the phase was
found by finding the difference between each point of the first derivative and dividing by the
corresponding difference in time. For model 2 the filter with a hanning window was applied here
by multiplying the second derivative with a normalized hanning window of the same length.
The magnitude spectrum was then found by computing the absolute value of the FT which was
computed by the build-in MATLAB function fft and fftshift. The AUP was then computed
by summing the values of the peak of interest in the magnitude spectrum.

5.1
Insert gradient
The insert gradient has been tested by running an imaging sequence for different scenarios as
described in section 3.1. MRI were obtained while the coil was placed in the scanner and the
results are shown in figure 9 and 10. The coil was tested while not connected to the audio
amplifier, while connected with an audio amplifier, and when connected with an audio amplifier
and a capacitor. When the coil was connected in a circuit it was seen, that eddy currents from
the gradient coils of the scanner caused the signal to only show a phantom in the center of
the image. The reason for this has not been quantified but may be explained by Lenz’s law
which states that the current induced in the coil will have a direction that opposes the change
in magnetic flux. This means, that when the gradient of the scanner changes for the different
positions along the vertical axis, then the eddy current induced in the insert coil will create a
magnetic field that will attenuate the change in field from the gradient.
To understand the relation between current and frequency, the two circuits have been modeled
in LTspice where the amplifier is modeled as a resistor denoted R1, and the small resistors are
removed. The coil is modeled as an inductor with L=162 mH and the capacitor with C=100 nF,
which gives a resonant frequency of 1250 Hz when using equation 2.2. The voltage induced in the
insert coil has been modeled as a voltage source with an amplitude of 10 V. An AC simulation
shows the amplitude of the current in the inductor for different frequencies. A possible solution
to the issue is adding a large resistance in the circuit attenuating the current in the frequency
range of the gradient of the scanner.
The frequency of the eddy currents is uncertain but
assumed to be relatively higher than the resonant frequency. This assumption is based on the
relation that the change in frequency can be found as the product of the gyromagnetic ratio,
the distance, and the gradient strength and is commonly in the range of 1000-2000 Hz in the
direction of the slice selection gradient[22]. The modeled circuits and the AC simulation are
seen in figure 30 for circuit a on the left and circuit b in the middle. On the right is simulated
a circuit as circuit b, but with the resistor increased to 1000 Ω.

5.2
Characterization
In this section the characterization shown in section 4.2 will be described. The characterization
is made with two models where model 1 will compute the area under the peak (AUP) of the
magnitude spectrum of the first derivative of the phase of the FID and model 2 will compute the
AUP of the magnitude spectrum of the second derivative of the phase of the FID. It is assumed
that the AUP of model 1 and 2 is proportional to the gradient strength Gosc. Furthermore, it
is assumed that a peak in the magnitude spectrum will be located at the frequency that the
corresponding signal component is oscillating with. When the gradient strength is constant and
the frequency is varying it is assumed that the AUP of model 1 is constant and the AUP of
model 2 is proportional to the frequency.
When looking at figure 11 on the left it is seen that the measured FID follows the shape of the
expected FID as described in section 2.2. In figure 11 on the right is seen the phase of the FID,
where both low-frequency oscillations and higher-frequency oscillations can be observed. The
higher oscillations are expected due to the oscillating gradient, while the low-frequency oscil-
lations may be a sign that the small inhomogeneities described by Boffset are not constant as
assumed. The signal loss of the FID may be described by both Boffset and nuclear interactions
causing dephasing. The dephasing may also explain the increase in noise seen towards the end
of the first and second derivative of the phase of the FID as shown in figure 12 in the upper row.
When looking at figure 12 in the lower left the magnitude spectrum of the first derivative is seen
with the two large peaks located at 1270 Hz and not at 1300 Hz as expected which is also the
case for the magnitude spectrum of the second derivative seen in figure 12 on the lower right.
When applying the highpass filter in computing the second derivative it suppresses the peak
of the low frequencies, but also increases the higher frequencies, which may include unwanted
noise. In this case, it is however still possible to designate the range of frequencies where the
AUP should be calculated.

quencies and shown in figure 13 and 14 respectively. When varying the gradient strength the

suggesting an additional factor should be included in the model. In this model, the thickness of

5.3
Reconstruction
5.3.1
Considerations on method choice
When applying the fourier shift theorem it is assumed from equation 3.6 that the phase roll can
be seen when looking at the phase of the ratio of the FT along kz given as: ϕ(s1(z, t1)/sn(z, tn)) =
2πκ(tn). When looking at figure 16 the computed ϕ(s1(z, t1)/sn(z, tn)) for four different tn are
shown before the applied shift on the left and after the applied shift on the right. Here the
slopes are approximately linear in both figures and the slope is sufficiently decreased after the
applied shift. This is however an approximation as the phase is not completely linear and the
slope is not completely 0, which may introduce errors in the reconstruction. Furthermore, it is
assumed that the absolute value of the FT along kz will decrease over time due to general signal
loss when nuclear interaction causes dephasing, which can be seen when looking at figure 16
on the lower left. This is also seen when comparing the expected phase roll with the measured
phase roll in figure 16 on the lower right.
An oscillating gradient can be used to reduce scan time by reducing the number of phase encod-
ing steps without introducing aliasing artifacts. However, when the amplitude of the oscillating
gradient is relatively small compared to the distance between the lanes in k-space reducing the
number of phase encoding steps will have a similar result as when the oscillating gradient is not
applied. This may be the case when using a k-space trajectory as shown in figure 17 on the right.
When looking at figure 18 it is seen that the PSF is the same for the MRSI with an oscillating
gradient and without an oscillating gradient, when the number of phase encoding steps is not
reduced or when reduced by half. When reducing the phase encoding steps by half aliasing is
expected to be introduced at the edges of FOV, but when using the reconstruction with the
fourier shift theorem they are attenuated by the applied filter. When removing additional phase
encoding steps figure 18 shows that aliasing artifacts are introduced and a difference between
using the cartesian grid shown in blue and the oscillating grid examples in red and green can
be seen. The PSF shown in red has been computed with the k-space trajectory shown in figure
17 on the left. Only a small difference in the PSF can be observed even though the lane width
is larger than the distance between each lane. A larger difference in PSF is seen for the PSF
shown in green which has been computed with twice the gradient strength of the PSF in red
thereby increasing the lane width. The lane width can also be increased by decreasing the os-
cillating frequency. When considering whether the difference in PSF is enough to avoid aliasing

5.3.2
Influence on scaling of gradients
The influence on the signal strength in MRSI of only scaling the gradient strength of the oscil-
lating gradient can be seen in figure 22 on the left where only a relatively small difference in
signal strength can be observed. When scaling the oscillating and the linear gradient as seen in
figure 22 on the right the profile in blue has a higher signal strength than the profile in red even
though the profile in red has the larger oscillating gradient strength. This can be explained by
the change in FOV and resolution as the MRSI with the higher linear gradient strength has a
smaller FOV and a higher resolution. As the sum of the signal strength of all the sample points
is proportional to the concentration of nuclei the signal strength of a single sample point must
decrease when more sample points are shown as the resolution increases. This is also seen in
figure 23 where the three different profiles are created from reconstruction with three different
linear gradient strengths but with the same gradient strength for the oscillating gradient. Here
it is seen that the profile with the higher linear gradient strength also has the highest number of
sampling points available to show the slice, while the profile with lower linear gradient strength
has fewer sampling points and the contour of the slice is therefore less smooth. The FWHM is
shown in cross symbols at the expected edge of the slice in both figure 22 and figure 23 and is
by visual inspection close to but not on the line between sample points in all profiles regardless
of the gradient size. A general trend can be observed as the line of the profile is seen to the right
of the marked crosses in all cases, why the reason may be linked to the reconstruction procedure.

Conclusion
In this project experience with using an oscillating gradient in MR scanning has been gained.
It has been investigated how an insert coil driven by an audio amplifier behaves inside an MR
scanner and how the eddy currents induced in the coil can affect MRI. The effect was seen in an
MRI where eddy currents from the gradient coils of the scanner induced a current in the insert
coil. The current induced in the insert coil then generates a gradient that attenuates the gradient
of the scanner and thereby affects the resulting image in the spatial direction of the gradient of
the insert coil. The coil has been tested in a resonant circuit which limited the possible range of
frequencies but enabled an increase in current through the coil which can increase the strength
of the oscillating gradient. The change in current through the coil when changing the frequency
has been simulated in LT spice, but further considerations must be made to gain meaningful
data with an insert coil as an oscillating gradient.
To characterize the gradient strength and frequency of an oscillating gradient two models were
proposed. It was assumed that the AUP of the magnitude spectrum of the first and second
derivative of the phase of the FID was proportional to the amplitude of the oscillating gradient.
This was seen for a range of gradient strengths of measured FID and compared with the results
of a simulated FID. A clear proportionality was seen for both measured and simulated data
where the simulated deviated slightly from the measured results indicating that an additional
factor should be included in the model used for simulating the FID. When characterizing the
frequency of the oscillating gradient it was assumed that the AUP of the second derivative of
the magnitude spectrum of the phase of the FID was proportional to the frequency. Here the
AUP of the simulated data showed a linear relation, while the AUP of the measured data de-
viated slightly. It was also assumed that the frequency could be determined from the location
of the peak in the magnitude spectrum of the first derivative of the phase of the signal, where
a deviation of 3% was seen.
Two different methods for reconstructing data with an oscillating gradient were used for the
reconstruction of MRSI along a spatial dimension z and the frequency dimension. The recon-
struction can be done by using the fourier shift theorem that enables the use of the fast fourier
transform or it can be done by using the non-uniform fast fourier transform which is an approx-
imation of the non-uniform discrete fourier transform. A point spread function has been used
to visualize the consequences of reducing the number of phase encoding steps, which introduces
aliasing artifacts. It was seen that the intensity of the aliasing artifacts depends on the relation
between the width of the lane of the oscillating gradient and the distance between each lane.
A further investigation for an optimal choice for lane width and distance between lanes, while
considering the aliasing artifacts and the limits of the scanner could be of interest for future
research. Aliasing artifacts introduced when reducing the number of steps in the phase encoding
direction were compared for the NUFFT and the reconstruction with the fourier shift theorem
at the center profile showing similar results for the two methods. However, when looking at the
2D reconstruction using NUFFT it was seen, that the aliasing artifacts are also notable at other
frequencies. When reconstructing MRSI the gradient strength may also influence the image
intensity and FOV, which has been visually examined for different gradient strengths. Here
it was seen that the oscillating gradient had a small influence on the intensity while changing
the linear gradient will change the FOV. Additional parameters such as slice thickness and the
number of slices have been used in data acquisition and the corresponding reconstructions show
resemblance with the expected spatial dimensions. However, when utilizing a larger part of
FOV a disadvantage of using the fourier shift theorem was found. This was seen when using
Page 34 of 37
==PAGE:43==
6
the filter that was implemented to reduce ringing artifacts, attenuated the intensity at the edges
of the reconstruction. 2D MRSI with different compounds has been reconstructed and visually
analyzed. A general improvement in this project would be an optimal method for validating
the different results and thereby being able to quantitatively evaluate the results. Many obser-
vations and experiences have been gained in the different aspects of working with an oscillating
gradient, where many challenges are still yet to be solved.

Additional info on running waveforms
The conventional parameters are chosen in the experiment setup on the scanner control system
for chemical shift imaging (CSI) together with control variable cv3 for a waveform slot that
matches a created waveform file connected by symbolic links and cv4 for number of excitations.
The waveform file is created by a routine in MATLAB where a function from MNS called
write ak wav converts a given waveform to the units of the scanner. write ak wav takes the
inputs: name, gradient waveforms, full sampling bandwidth (bw), FOV and description. The
waveform files in this project are created with FOV=0.2 m, bw=10 kHz and varying gradient
waveform. The gradient waveform has three dimensions, where the first is points in time, the
second is interleaves and the third is number of groups, where the number of groups in this
project is chosen t be 3.
When creating the waveform file in MATLAB the function write ak wav will provide addi-
tional information about the waveform including maximum gradient strength, maximum slew
rate, worst trajectories, filesize, name of file, time resolution in µs for gradient and samples
in k-space and time duration in µs for gradient and samples in k-space.
For all waveforms
used in this project, the time duration for both gradient and samples i k-space was the same at
102400µs, but the time resolution for the gradient was 4 µs, but 100 µs for the samples in k-space.

D
Preprocessing
The data acquired from the MR scanner using the MNS research pack must be converted from .7
files to .h5 files for accessing the information in MATLAB. The information in the .h5 file is then
extracted by looking at the header cell and data cell respectively. The data cell contains a cell
for each interleave with a number of datapoints given by the number of samples times two times
the number of channels, which needs to be combined to a single FID. This can be done by first
extracting a FID for each channel by assuming that every second point is the imaginary part of
the signal and then allocating each sample in time as the sum of the real and imaginary part.
ii
==PAGE:48==
6
Then the data must be divided into one array for each channel and combined by computing a
weighted sum of the all channels. To get the highest SNR the FID should be created from a
sum of the FID from all the different channels weighted by the conjugate of the first point in
the FID of each channel as this point can represent the coil sensitivity.
