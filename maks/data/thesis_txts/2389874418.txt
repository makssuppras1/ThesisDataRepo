Abstract
In this project application relevant for optical quantum information protocols are demonstrated,
which includes non-Gaussian state and qubit generation at Furusawa & Yoshikawa Laboratory,
together with continuous variable entanglement generation from a single squeezing source using
optical switch and delay line in optical ﬁber at the DTU QPIT group.
By tapping and detecting a single photon from a Gaussian squeezed vacuum state, the non-
Gaussian photon subtracted squeezed state is generated, which is a good approximation to the
small amplitude odd Schr¨odinger’s cat state. Here tools for data analysis based on quadrature
autocovariance are introduced, which includes estimation of the temporal mode function us-
ing principle component analysis (PCA). With diﬀerent mode functions estimated with PCA,
the non-physical eﬀect of using quadrature dependent mode functions are demonstrated and
discussed. By quantum tomography the generated non-Gaussian state is characterized with a
Wigner function negativity of −0.1042 ± 0.0001 for ℏ= 1, and the state is found to be aﬀected
by losses.
With the generated non-Gaussian state, the state is encoded in the time-bin qubit. Extending
the autocovariance based tools to include orthogonal measure quadratures in a complex plane
using heterodyne detection, the qubit phase is extracted directly from the autocovariance using
PCA. To characterize the qubit generation scheme, qubits of single photons are measured and
analyzed with two-mode quantum tomography. Here, besides loss, temporal mode orthogonality
aﬀected by ﬁltering is discussed.
Finally, a setup for EPR-state generation from a single squeezing source is constructed:
An optical switch and delay line are used to dividing the temporal modes of the squeezing
source into two diﬀerent spatial modes, which we generate the EPR-state. This is all done in
optical ﬁbers, and the concept of using optical switches and delay lines has potential in cluster
state generation and measurement based quantum computation. Unfortunately, the setup for
EPR-state characterization is not ﬁnished in this work. Instead the squeezing in the ﬁber is
characterized just before the EPR-state is generated in a ﬁber coupler. Here squeezing up to
−3.66 ± 0.13 dB is measured, and possible improvements are discussed.

Chapter 1
Introduction
In quantum information theory, information is encoded into a state of a quantum system in
the form of a qubit. Unlike classical bits, which has to be in either of two orthogonal states (0
or 1), a qubit allows superposition of both states at the same time (0 and 1) – a fundamental
property of quantum mechanics. As bits are transmitted from place to place and processed using
algorithms in a classical computers, qubits can be manipulated using quantum information
processing (QIP). This allows some problems to be solved in a polynomial time, which on a
classical computer appears to be a class non-deterministic polynomial problem. Example hereof
is Shor’s factoring algorithm where the task is to eﬃciently ﬁnd the prime factors to some integer
[1].
A quantum computer for such quantum algorithm as Shor’s factoring algorithm is not straight
forward to implement in practice. Several competing technologies exist for the task of hosting a
quantum system for quantum computation, for instance trapped ions, cavity quantum electrody-
namics and superconducting electronic circuits. More likely is a universal quantum computer as
a mixture of diﬀerent technologies with diﬀerent advantages. In this work the focus is on optics,
which has the advantage of no photon-photon interaction, together with the quantized electro-
magnetic ﬁeld being one of the simplest quantum mechanical systems; the harmonic oscillator.
Furthermore, optics is proven to be a great information carrier over long distances for classical
information, exactly due to no photon-photon interactions – properties which we hope to exploit
in quantum information networks with long quantum channels for quantum communication.
So far encoding of quantum information has mainly followed two diﬀerent direction: Discrete
variable (DV) and continuous variable (CV) where the observables have discrete and continuous
eigenvalues respectively. With quantum optics based on harmonic oscillators, it has observables
in the CV regime including Gaussian and non-Gaussian states, deﬁned as having a Gaussian
and non-Gaussian Wigner function. Gaussian states, including coherent and squeezed states,
are relatively easy to produce and manipulate using optics, which enables linear transformation
mapping Gaussian states onto other Gaussian states. However, it is shown that in a universal
quantum computer some non-Gaussianity is needed [2], which can be included by hybridiz-
ing DV and CV technologies [3]. This can be realized by some non-Gaussian transformation
resulting in non-Gaussian state, and leads to the title ’Quantum Information Protocols with
Non-Gaussian States of Light’ of this report. Here this non-Gaussian transformation is realized
as a probabilistic measurement-induced interaction performed on a Gaussian state by photon
counting.
1.1
Subject of this thesis
In this work the Gaussian squeezed vacuum state is prepared, from which a single photon is sub-
tracted and detected, leading to the non-Gaussian photon subtracted squeezed vacuum (PSSqV)
state, which is a well approximation to the Schr¨odinger’s cat state with small amplitudes. This

process is realized optically by Neergaard-Nielsen in [4], and it is the same procedure adopted
here. Schr¨odinger’s cat state has been shown to be a possible resource for universal quantum
computing [5], and a useful resource for fault-tolerant QIP [6]. Furthermore, it serves as a good
example of non-Gaussian transformation for producing non-Gaussian states.
As a further hybridization of DV and CV technologies, the generated non-Gaussian state
is brought into two-mode time-bin qubits where the PSSqV is distributed over two temporal
modes, and an arbitrary phase factor between the two modes is realized. This is then compared
to the more simple time-bin qubits generated from single photons in the DV-regime.
Finally, generation of entangled two-mode squeezed state, or EPR-state, is generated from a
single source of squeezed vacuum using a ﬁber network including an optical switch and a delay
line. This is in principle less resource-intensive compared to the more general approach, where
two vacuum states squeezed in diﬀerent quadratures are superimposed on a beamsplitter. More
important, it is a proof-of-concept of using ﬁber networks for generation of various multi-mode
quantum states, for instance cluster states in multiplexed in time domain as in [7], depending on
the conﬁguration of optical switches and delay lines. Such cluster states can be the resource for
measurement-based quantum computation [8], which together with non-Gaussian measurement
constitute a universal quantum computer [9].
Finally, Gaussian entangled EPR-states serve
as a good resource for quantum communication channels, which for long distances are conve-
nient to have in a ﬁber network. Non-Gaussian transformation can then be applied locally for
entanglement distillation [10].
1.2
Overview of this report
The work in this thesis has been divided into two parts: The ﬁrst part of this thesis is carried
out in Furusawa & Yoshikawa Laboratory using an already constructed free-space optical setup,
which is originally made for generation and teleportation of qubits and non-Gaussian states.
Thus my work at Furusawa & Yoshikawa Laboratory has mostly been data acquisition and
analysis of PSSqV states and qubits hereof, described in chapter 3 and 4, where single and two-
mode quantum tomography are performed on the measured data sampling the Husimi function of
the generated states using heterodyne detection. Here an analysis based on the autocovariance
of quadratures measured in time domain is carried out, where principal component analysis
(PCA) is used to estimate temporal mode functions, while complex PCA is used to extract the
phase between the two modes of time-bin qubits. With the experience gained from Furusawa
& Yoshikawa Laboratory, the second part of this thesis is carried out at Technical University
of Denmark (DTU), where the focus has been at constructing a setup for two-mode squeezed
state generation from a single squeezing source using an optical switch and delay line based on
optical ﬁber, and is described in chapter 5.
Chapter 3, 4 and 5 represent the main work of this thesis, while chapter 2 works as a reference.
In chapter 6 the work in Furusawa & Yoshikawa Laboratories and at DTU is summarized and
related in a discussion where a ﬁnal conclusion is given. Below are listed each chapter with short
descriptions.
Chapter 2 - Quantum optics theory: Theory of squeezed light, two-mode squeezed light,
photon subtraction from squeezed light and qubit are introduced. The chapter works as a
reference for the following chapters.
Chapter 3 - Non-Gaussian state preparation: A setup for generation of PSSqV state is

mating the temporal mode and quantum tomography.
Chapter 4 - Generation of qubits with non-Gaussian states: With the PSSqV state

chapter 3 to include the two quadratures in a complex plane, and using two-mode quantum

2.2.1
Optical parametric oscillator
Due to low higher order susceptibility in most materials, the squeezing from a single path of
pump light through a non-linear material in the continuous wave (CW) regime leads to very low
squeezing. This can be improved by using pulses of pump light, where very high pump power
can be obtained. However, since short pulses of light has a broad spectrum, the non-linear
crystal must then be short in order to avoid eﬀects due to dispersion. Typical squeezing levels
obtained in the pulsed scheme results in a variance, or noise level, reduction around 3 dB as in
[11].
The CW regime is greatly improved by placing the crystal of non-linear material inside a
cavity – an arrangement called an optical parametric oscillator (OPO). The cavity is kept on
resonance with the down-converted ﬁeld of frequency ω0, but not the pump ﬁeld of frequency 2ω0.
This is done by letting the mirrors of the cavity reﬂect ω0, but being highly transparent to 2ω0.
In the non-linear crystal a pump photon is converted to signal photons in spontaneous parametric
down-conversion, followed by parametric ampliﬁcation as signal photons can circulates in the
cavity until coupled out. As a result squeezing levels up to 12.7 dB has been achieved in [12]
and recently 15 dB in [13].

coupler leads to a decay rate γ1 of the cavity. Furthermore, the losses L in the cavity is modeled

The result is two entangled frequency mode, which we will use in this project to herald single
photon for qubit generation in Fock basis as described in section 2.5.

2.4.4
Temporal mode function
The concept of a mode function to assign a temporal mode which a quantum state occupies has
been discussed in details for photon-subtracted state generation in [14] and [17]. Applying a
mode function can be done in the setup in real-time by shaping the local oscillator (LO) power
used for homodyne detection. However, more easy is it to apply in the post processing, where
the mode-function can be modeled freely.
The choice of mode function is important in the
quantum tomography of the measured quadrature in time, ˆQ(t), where the a temporal mode is
selected as
ˆQs =

fs(t) ˆQ(t) dt .
(2.19)
If the mode function used does not overlap perfectly with the temporal mode containing the
conditioned state, we will include part of the unconditioned state in the tomography – in our
case the non-Gaussian squeezed vacuum.
Intuitively it would make sense, that the mode function of the conditional state is similar to
the autocovariance function in (2.9) around the detection time tc, since this describes entangle-
ment at diﬀerent times around tc: Only in the time range of correlation the signal is aﬀected
by the photon subtraction. Indeed for low pump power of the OPO it is shown in [21], that the
optimal mode function is a double-sided exponential function with the same decay rate as the
OPO bandwidth.
To avoid clicks in the trigger channel from non-degenerate photons generated in the OPO, the
trigger line is ﬁltered using ﬁlter cavities in resonance with only the degenerate frequency Ω= 0,
leading to the optimal mode function being convoluted with the cavities response functions. The
result is a smoothened out double-exponential function. Furthermore, other eﬀects in the non-
ideal experimental setup alter the mode function, and ﬁnally does the electrical ﬁlters applied
on the homodyne current result in the optimal mode function for the post processing being
convoluted by the corresponding electrical transfer function. As a result, estimating a mode
function analytically often results in a mode function far from the optimal mode function, and
numerical methods are often applied.
The most common method for estimating a temporal mode functions analytically has been
to vary the mode function in order to optimize parameters after the quantum tomography as
ﬁdelity, or for non-Gaussian states like the 1-PSSqV states, the negativity of the Wigner function
W(0, 0). However, this is a time-consuming process, where a good guess of the mode function
shape is needed, and even though one is not guaranteed to reach the optimal mode function.
In this project a rather new approach based on eigenfunction expansion of the autocovariance
function [22] is introduced in chapter 3 and 4 to estimate the optimal mode function directly from
the experimental data acquired using heterodyne detection. This is a powerful technique which
works well when the conditioned state has a higher variance than the background unconditioned
state.

2.5.2
Qubits and qutrits in Fock basis
The method described above for generation of time-bin qubits is not limited to the non-Gaussian
1-PSSqV states, but can be used in all cases where the generated state is heralded by a single
photon. This is demonstrated experimentally in chapter 4 with single photon generation where
a pair of non-degenerate photons of diﬀerent frequencies are generated in the OPO. The OPO
output is then frequency ﬁltered such that one photon continues in the signal line, and the other
photon is led through the interferometer in the trigger line, resulting in the c0 |1, 0⟩+ c1 |0, 1⟩

This scheme can be further extended by adding another detector in the trigger line: When
both detectors click we know that we have two photons in the signal line. If the second detector

the direct or delayed path such that the conditioned state in the signal line is another qubit,
c02 |0, 2⟩s +c20 |2, 0⟩s. One can also take the delay into account and look for delayed coincidence
clicks in the detectors in order to herald the |11⟩s state.

the trigger line as sketched in ﬁgure 2.6. When the two detectors click we know that we have

Non-Gaussian State Generation
In this chapter the generation and characterization of non-Gaussian states are described, based
on the generation of photon subtracted squeezed vacuum (PSSqV) states in Furusawa & Yoshikawa
Laboratory. The main focus is not on the experimental setup, as designing the setup has not
been part of my thesis, but rather on the analysis of the acquired data, and especially on the
estimation of diﬀerent optimal mode functions in ˆX and ˆP directly from the data using the
autocovariance and principal component analysis.
In section 3.1 the experimental setup is summarized, where the biggest diﬀerence from typ-
ical setups of PSSqV state generation is the use of heterodyne detection instead of homodyne
detection, together with the locking scheme for locking the phase of the prepared state to the
local oscillator. In section 3.2 the acquired data for analysis is presented and the useful autoco-
variance is analyzed. In section 3.3 principal component analysis is introduced for determining
an optimal mode function, which is then used for quantum tomography in section 3.4. Finally,
the result is discussed in section 3.5.
3.1
Experimental Setup
The experimental realization of the photon subtraction scheme for m-PSSqV states generation,
described in section 2.4, is sketched in ﬁgure 3. The setup is part of a larger setup for contin-
uous variable teleportation of non-Gaussian state, and is developed since 2009 at Furusawa &
Yoshikawa Laboratory. The setup is described and discussed in detail in the Ph.D. thesis of H.
Benichi [17] and master thesis of S. De Leseleuc [24]. Here I will summarize the current state
of the setup used in this work, while the full schematics of the overall setup can be found in
appendix B.
Light source
The source of light for the setup is a Ti:Sapphire laser (M Squared SolsTiS-SRX ) pumped by
a Coherent Verdi 10W laser.
The result is Gaussian TEM00 proﬁle of intensity 1.9 W at a
wavelength of 860 nm. The laser light is led through and isolator to improve laser stability, and
through an electro-optical modulator (EOM) modulating at 13.6 MHz for locks in the setup. To
stabilize the power the light also goes through an automated rotating wave plate and polarized
beamsplitter where a detector gives feedback to the rotating wave plate. After the EOM the
light is split into three paths: One for local oscillator (LO), one for probe in the signal line,
and one for second-harmonic generation (SHG). About 1 W is used to pump the SHG cavity
with a KNbO3 second order nonlinear crystal, locked using Pound-Drever-Hall (PDH) locking
technique in reﬂection [25]. The up-converted output of the SHG cavity at 430 nm is used for
pumping an optical parametric oscillator (OPO) for generating squeezing.

Probe switch
The signal line is locked to the LO at the state characterization by sending a coherent probe
beam through the signal line. Since the generated non-Gaussian 1-PSSqV state is delocalized
over the spectrum of the OPO, it is not possible to use side band locking without harming
the fragile non-Gaussian state. Instead the ”sample/hold”-scheme is used where the probe is
turned on and oﬀin periods of 500 µs.
When the probe is on (sample, 180 µs) all cavities
and interferometers are locked, while when the probe is oﬀ(hold, 320 µs) all feedback controls
for locks are kept constant while the generated state is characterized. This is possible if the
hold-period is short enough for us to neglect any drift of locks.
The probe switch, switching on and oﬀthe probe during sample/hold, consist of two acousto-
optical modulators (AOM). In an AOM standing sound waves is used to diﬀract incoming light
in an angle depending on the modulation amplitude, and approximately 80% of the light is
diﬀracted into the ﬁrst order. In this way we can direct the light in and out of the optical
path when we turn on and oﬀthe modulation respectively. This switching technique has the
advantage, that when the modulation is oﬀ, absolutely no probe light goes into the optical
path during the measurement phase. In order to compensate the frequency shift induced in
the diﬀracted light by the AOM, we use two AOMs with exactly opposite frequency shift, done
using the same modulator for the two AOMs.
A ﬂip-mirror is installed before the probe switch, and is ﬂipped up during manually pre-
alignment of the setup before initializing the sample/hold-scheme.
Optical parametric oscillator
For generating squeezing an optical parametric oscillator (OPO) in a bow-tie cavity conﬁguration
with a periodically poled potassium titanyl phosphate (PPKTP) second order non-linear crystal
is used. The OPO is pumped by the up-converted light from the SHG cavity, producing type-I
parametric down-conversion, and the output coupling mirror is of transmittance T = 0.12. The
OPO is operated below threshold resulting in squeezed vacuum at the OPO output as explained
in section 2.2.1.
The OPO cavity length is 500 mm resulting in 590 MHz free spectral range (FSR). The

the pump phase is locked to the probe phase at a maximum attenuation of the probe beam, i.e.
a π phase diﬀerence.
To measure the internal cavity losses, ﬂip-mirrors are placed after the OPO in order to probe
the OPO in opposite direction through output coupler – for no losses in the cavity all probe light
going into the cavity should come out again in the reﬂection path. The loss is then calculated
from the measured reﬂectance before acquiring data.
Trigger line
Photons are subtracted from the squeezed vacua in the signal line using a tapping mirror of
T = 0.97 transmittance. The subtracted photons are guided into the trigger line, where it is
frequency ﬁltered using two Fabry-P´erot cavities, FC1 (FSR: 169.4 GHz, FWHM: 110 MHz) and
FC2 (FSR: 58.7 GHz, FWHM: 36 MHz), in order to only get triggering from the degenerate
down-converted light from the OPO. They are designed with a very short cavity lengths of
0.88 mm and 2.55 mm to have high FSR, and FC1 and FC2 together only have one frequency
mode in common with the OPO, namely the degenerate frequency mode. Furthermore, they
are designed with a large FWHM to allow the full spectrum of the OPO’s degenerate frequency
mode through. The ﬁlter cavities are locked with tilt locking scheme [26], where the input beam
is slightly misaligned to the TEM01 mode of the cavity, such that spacial interference of the
dominating TEM00 and TEM01 mode appears. This is used to obtain an error signal using a
split photodiode detecting the reﬂected beam.
The photons going through both ﬁlter cavities are detected using an avalanche photodiode
(APD) (Perkin-Helmer SPCM-AQRH-14) with a dead time of 32 ns and quantum eﬃciency of
45 ± 5% at 860 nm. Before the APD is an AOM to protect the APD when the probe beam is
on, and during the measurement phase the AOM direct light into the ﬁber to the APD. Since
the APD detects in a very broad spectrum and is ”frequency blind”, we do not need another
AOM to compensate for the frequency shift induced in the AOM protecting the APD.
Finally, we use the click from the APD to herald a 1-PSSqV state in the signal line, and the
trigger click is send to a data acquisition system.
Heterodyne detection
To characterize the heralded states in the signal line, we use heterodyne detection bases on
two homodyne detectors detecting ˆX- and ˆP-quadrature at the same time. The expenses of
measuring ˆX and ˆP at the same time is additional vacuum noise introduced when splitting the
signal to the two homodyne detectors on a beamsplitter. Thus it is not exactly ˆX and ˆP of the
conditioned state we measure; it is the state’s Husimi function we are sampling. Heterodyne
detection has shown to beneﬁt over homodyne detection in some cases, especially for Gaussian
states [27, 28]. For non-Gaussian states it has not proven beneﬁcial, and only requires extra
equipment compared to homodyne detection. The reason for using heterodyne detection is to
be able to characterize the cross-covariance between the ˆX- and ˆP-quadrature. For the non-
Gaussian states generated here the cross-covariance will be zero, but that is not the case for the
qubits generated in chapter 4, where we use the same setup but with a modiﬁed trigger line.

The local oscillator (LO) for the heterodyne detection is led through a mode cleaning cavity

homodyne detectors, HD-x and HD-p, the LO and signal line is locked to the probe from the

with a π/2 phase diﬀerence between the two arms, we deﬁne the coherent probe beam in one
arm to be along the ˆX-quadrature, and so the other along the ˆP-quadrature, and in this way
HD-x and HD-p are locked to the ˆX and ˆP quadratures. Finally, the probe in the signal line
is locked to the ˆP-quadrature using a tapping mirror of 0.99 reﬂectance. Since the pump in
the OPO is locked to attenuate the probe, we will see squeezing in the ˆP-quadrature at the
heterodyne detector.
If we lock HD-x and HD-p using the same modulation frequency fx or fp for both, we will
measure ˆX or ˆP in both detectors. We can then obtain the original ˆX or ˆP quadrature of
the state in the signal line before additional vacua are introduced in the heterodyne detection
scheme, by subtracting the two homodyne signals such that the additional vacua are canceled
out. Throughout the chapter we will refer to this as homodyne detection. This is convenient
for measuring the squeezing of the vacuum state used for generating 1-PSSqV states. However,
when we gain exact measure of the ˆX or ˆP quadrature, we lose all information of the other
quadrature, but this is okay for characterizing the squeezed vacuum state since we know it is
squeezed along ˆP.
Instead of using an additional probe signal from the probe-MZI to lock the homodyne detec-
tors, one could simply lock the LOs to the ˆX- and ˆP-quadrature of the probe in the signal line.
However, in the original teleportation setup, the probe-MZI with an additional OPO in each
arm serves as a resource of EPR-states for the teleportation. In that case it is important that
the homodyne detectors used for the Bell’s state measurement are locked along the quadrature
deﬁned by the EPR-state, and is the reason for the slightly complicated locking scheme used
here for the non-Gaussian state characterization.
Data acquisition
The AOMs for the sample/hold-scheme is controlled using a ﬁeld-programmable gate array
(FPGA). When being in the measurement phase of the sample/hold-scheme, 1.5 µs long time
frames with 1 ns resolution are recorded from the HD-x and HD-p when the APD clicks. This is
done by using an analog to digital converter (ADC), and stored for analysis. Each time frame
represents one quadrature measurement after applying a mode-function to the recorded time
frame as explained in section 2.4.4. One full state characterization measurement consist of 20000
recorded time frames from each the HD-x and HD-p.
Fast power ﬂuctuation and some unavoidable electronic noise at low frequencies results in an
increase of noise in the squeezed spectrum below 100 kHz. In order to achieve proper quadrature
measurements from the time frames, this low frequency noise is ﬁltered out electronically using
a high-pass ﬁlter (HPF) on the signal from HD-x and HD-p before the ADC.
All locks described above in cavities and interferometers are achieved by using mirrors glued
to piezoelectric transducers (PZT) such that cavity lengths and phases can be scanned and
locked. For the feedback to the PZTs an error signal is detected and led through an analog
PID-controller, or together with the demodulation mechanism for the error signal: A lock-box.
The lock-box is specially designed for the setup to re-lock when the lock is lost in order to
support long time stability. Furthermore, the lock-boxes keep the signal to the PZTs constant
during the measurement phase of the sample/hold-scheme using a logic circuit and a TTL input
from the FPGA controlling the AOMs.

We conclude the generated 1-PSSqV state to be highly aﬀected by loss, which is not surprising
since the used setup is build for photonic qubit generation and teleportation, and not pure PSSqV
state generation. Some of the sources of loss in the current setup are considered in [24]: Finite
quantum eﬃciency of the photo diodes of 0.99 in the homodyne detectors, together with mode-
matching between the signal line and the LO with a typical visibility around 0.95-0.97, leading to
a minimum eﬃciency of the homodyne detectors of ηhom. = 0.94; Noise from electronic circuits
with a signal-to-noise ratio leading to an eﬃciency estimated to be ηcirc. = 0.98; Propagation
ineﬃciency from the OPO to the homodyne detectors due to non-unity reﬂectance of mirrors and
unwanted reﬂection and scattering on optical components, and an eﬃciency of 0.98 is measured.
Furthermore, a tapping mirror of 0.01 reﬂectivity is used in the signal line to lock the probe
from the probe-MZI to the probe from the OPO before the heterodyne detection, and as a result
we expect a total propagation eﬃciency of ηprop. = 0.97; Finite escape eﬃciency of the OPO
due to loss L inside the OPO. This leads to an escape eﬃciency ηesc. = T/(T + L), where T is
the transmittance of the OPO output coupling mirror. With internal cavity loss measured to
be 0.004 and T = 0.12 we get ηesc. = 0.96. The total expected eﬃciency is then
η = ηhom.ηcirc.ηprop.ηesc. = 0.86 ,
which is still better than the loss we expect by maximizing the ﬁdelity to a 1-PSSqV state
subjected to loss. Compensating for the loss expected from the experimental setup, η = 0.86
in the MaxLik iterations, we get a density matrix leading to a Wigner function negativity of
W(0, 0) = −0.1718 ± 0.0002.
An ineﬃciency, which we cannot measure directly in the setup, is the mode mismatch between
the used mode function, and the actual temporal mode. Even though the mode function used
here is estimated directly from the measured data, we cannot be sure to have perfect overlap with
the actual temporal mode. For instance, one could imagine impurities in the setup changing
random in time could aﬀect the temporal mode somehow, leading to a mode function which
vary for each measured time frame. The estimated mode function using PCA will then be an
average of the temporal mode in all the measured time frames. As a result, depending on how
much the actual mode function in a given time frame diﬀers from this estimated average mode
function, we will include some of the unconditioned squeezed vacuum state in the extraction
of the quadrature measurement, which alters the purity of the MaxLik estimated state. To
estimate change or uncertainty of the mode function, we can record several sets of 20000 time
frames, and analyze each set. However, it will then be diﬃcult to assure, that any change in the
mode function estimated from each measure sets are not due to change in the setup. Measuring
each sets takes around 30 s, but in between it takes few minutes to initializing the setup, and
we need the setup to be very stable.
Finally, we also have dark counts in the APD and scattered photons from the environment
triggering the data acquisition, which leads to detection of the unconditioned squeezed vacuum
state. When combined with measurement of the conditioned state from real trigger events, this
looks like loss of the conditioned state, as the squeezed vacuum state and the 1-PSSqV state
only diﬀers by a single photon. A typical rate of fake counts in the APD is 200 s−1, while the
total count rate is around 2000 s−1 leading to a ratio of true clicks round 90%, which brings us
closer to the optimal loss compensation.
3.5.3
Quadrature dependent mode function
So far, for extracting measured quadratures of the generated states, we have used mode functions
estimated using PCA, leading to mode functions of diﬀerent temporal extensions in the ˆX- and
ˆP-quadrature. However, as a quantum state occupies a speciﬁc mode in frequency, time, space,
polarization, etc., regardless of the measured observable, it does not make sense to use diﬀerent
temporal mode deﬁnition in the ˆX and ˆP. Thus, using diﬀerent mode functions in ˆX and ˆP is
nonphysical, and referring to them as ’mode functions’ is somewhat misleading.

With the 1-PSSqV state generated and described in chapter 3, in this chapter we continue
to put this state into a superposition of two diﬀerent temporal mode in order to resemble a
time-bin qubit of this non-Gaussian state and the the unconditioned Gaussian squeezed vacuum
state. This is part of a larger project in Furusawa & Yoshikawa Laboratory with continuous
variable teleportation of qubits with non-Gaussian states, though here we only focus on the
state generation and characterization. In order to simplify the characterization of the qubit
generation, we continue to generate qubits of single photons and vacuum resulting in a simpler
analysis.
The qubit state generation setup is simply a small extension on the experimental setup
used in chapter 3, and is described in section 4.1 where the state generation of single photons
are described as well. In section 4.2 the qubit with 1-PSSqV is characterized using the tools
introduced in chapter 3 based on autocovariance, while in section 4.3 we get a better insight in
the time-bin qubit generation scheme by characterizing the generated qubits of single photons
using two-mode quantum tomography.
4.1
Experimental Setup
In order to bring the generated 1-PSSqV state, described in chapter 3, into a qubit in the signal
line, only the trigger line is needed modiﬁed as described in section 2.5: In between the last ﬁlter
cavity and the triggering APD is a Mach-Zehnder interferometer (MZI) with one path delayed.
The result is, that when a photon arrives at the APD, it could have taken both the direct and
delayed path, and we have no way to tell if the 1-PSSqV state in the signal line is late (direct
trigger path) or early (delayed trigger path) in time; i.e. a time-bin qubit.
Sketch of the experimental setup trigger line is shown in ﬁgure 4.1 – everything else is
the same as described in section 3.1. After the ﬁlter cavities, the polarization is made linear
in between s- and p-polarization before it is impinged onto a polarizing beam splitter (PBS),
where the s- and p-polarized part of the beam is transmitted and reﬂected into the delayed and
direct interferometer paths respectively. Before the beamsplitter is a half-wave plate (HWP)
to ﬁne tune the splitting ratio to 50:50. In the delayed path the beam is coupled into a 50 m
long ﬁber, which delays the beam by
∼250 ns before the beams again are joined together on

s- and p- polarized light from the delayed and direct path is guided to the APD, and in this way
adjust the ratio of clicks between photons from the two paths. This splitting ratio can also be
adjusted using the ﬁrst HWP before the interferometer, but one then has to take into account

4.1.1
Qubit and qutrit in Fock basis
The setup is easy changed to heralded single photon generation by replacing the tapping mirror
in the signal line with a frequency splitting cavity (FSC), and using the non-degenerate output of
the OPO. This is sketched in ﬁgure 4.2 together with two MZIs and APDs for qutrit generation.
In order to operate the OPO such that a non-degenerate frequency mode is on resonance
with the ﬁltering cavities in the trigger line at ω, the OPO is pumped with frequency 2ω + ∆ω,
where ∆ω is the OPO free spectral range. The OPO then generates correlated photons in the
frequency modes (ω + j∆ω, ω + (j + 1)∆ω), j ∈N, and for j = 0 we split up the generated
photon pairs using the FSC which is on resonance with ω while the ω + ∆ω is reﬂected. The
FSC is of half the size of the OPO and thus doubled FSR, and all frequency modes of ω +2j∆ω
are transmitted, but only the ω-frequency mode is on resonance with both ﬁlter cavities FC1
and FC2 in the trigger line. As a result, when having a photon in the trigger line after FC2, we
know a photon of frequency ω + ∆ω is in the signal line, and this is the correlated photon pairs
that we will use for qubit generation of single photon.

Temporal mode orthogonality
Finally, we investigate the inﬂuence of the HPFs before the ADC. The HPF used is of 2nd
order with a cutoﬀfrequency at 100 kHz in order to ﬁlter out low frequency circuit and laser
noise. In ﬁgure 3.7(right) the result is seen for the generated 1-PSSqV state, where the principal
component is negative after detecting the conditioned state. This is due to the causal response

at the conditioned state as seen in ﬁgure 3.4. As a result, the temporal mode is extended in
positive time depending on the response of the HPF, and two temporal modes 250 ns apart will
overlap and thus not be orthogonal.
This mode overlap due to HPF is tested by preparing the |0, 1⟩state, which arrives early at
the heterodyne detector: Quadrature at the temporal mode of the |0, 1⟩state and a temporal
mode 250 ns later, which should only contain vacuum, is then estimated with the same double
exponential decaying mode function used in the tomography of the generated qubits, and the
result of the MaxLik iterations are seen in ﬁgure 4.7(e). For comparison a similar estimated
density matrix of the same data, but with the second temporal mode being 250 ns before the
temporal mode containing the single photon, is shown in ﬁgure 4.7(f). In (e) ρ10,01 is non-zero
indicating correlation between between the two temporal modes: When a photon is detected in
the ﬁrst mode we get some anti-correlated signal in the second temporal mode. This is not the
case in (f) where the second temporal mode is before the ﬁrst temporal mode, and only includes
vacuum noise. The same is the case for the two-photon subspace of ˆρ, and we conclude non-zero
overlap between the two temporal modes due to the high-pass ﬁltering.
For generation of real qubits of orthogonal logic states, the overlap between the two states
making up the qubit needs to be eliminated. The overlap can reduced by a longer delay path
in the MZI of the trigger line, or by optimizing the HPFs used in order to shorten its response
functions. Part of future work on this setup is to minimize low frequency noise in the setup
in order to take out the HPFs. Alternatively, an optical HPF could be implemented in the
trigger line as in [30]: Here the reﬂection on a cavity with small bandwidth and on resonance
with the OPO is used in the trigger line, leading to the conditioned state being encoded in the
OPO bandwidth except in bandwidth center within the bandwidth of the optical HPF cavity.
Finally, one could look into more exotic mode functions in order to solve the problem in the
post processing as in [7], where the mode function is designed with a negative part next to the
temporal mode in order to subtract the the response from a previous temporal mode included
into the current temporal mode.

Chapter 5
EPR-state generation with one
squeezing source
In this chapter we work on generation of two-mode squeezed entangled state, or EPR-states
named after the Einstein–Podolsky–Rosen paradox, using two temporal modes of a single
squeezed vacuum resource. Most often two optical parametric ampliﬁers (OPOs) are used for
generating squeezing in two spatial modes, which are then mixed on a beamsplitter with a π/2
phase diﬀerence resulting in a two-mode squeezed state as described in section 2.2.2. Here the
idea is to use a single OPO and an optical ﬁber switch to divide the OPO temporal output
modes into spatial modes, which with a delay ﬁber is overlapped on a ﬁber coupler resulting in
a two-mode squeezed state at the coupler output half of the time. This has the advantage of only
using one OPO, but more it is a proof-of-concept of using optical switches and delay lines in an
optical ﬁber network with quantum states. This can be used for cluster state generation, where
for instance ultra large cluster state of up to one million temporal modes has been generated
in Furusawa & Yoshikawa Laboratory using a single delay line and two OPOs [7, 31]. With
several switches and delay lines, one can in principle generate various diﬀerent kind of cluster
states with diﬀerent temporal and spatial dimensions, which can be used for measurement-based
quantum computation [8].
The work presented here is carried out at Technical University of Denmark, and consist of
designing and constructing an experimental setup for EPR-state generation in ﬁber from a single
external squeezing source. In section 5.1 the design of the ideal setup is discussed, and this is
the goal to realize experimentally. In section 5.2 the process of building the setup is described,
while in section 5.3 various characterizations of the setup is presented. Unfortunately, within the
limited time of this thesis, the full setup is not completed: The setup for EPR-state generation
is complete, but the system for characterizing is not, and thus a full characterization of the
generated EPR-states is not given here. Finally, an outlook to the ﬁnished setup is given in
section 5.4.
5.1
Concept and setup design
In ﬁgure 5.1 the concept for generating two-mode squeezed states, or EPR-states, using a sin-
gle squeezed light source together with an optical ﬁber switch and ﬁber delay line is sketched.
Squeezed vacuum from a squeezed light source is coupled into ﬁber, where a ﬁber switch alter-

trolling the phase between path-1 and path-2 to be π/2 out of phase after the coupler, we get
a two-mode squeezed state at the coupler output mode-a and -b, or an EPR-state. The setup
is similar to a Mach–Zehnder interferometer (MZI), but with the ﬁrst beamsplitter replaced by

5.1.1
State generation
The squeezing source is an optical parametric oscillator (OPO) in free space. The squeezed
vacuum state is coupled from the OPO into the ﬁber switch, where lenses are used to shape the
free-space beam to optimize the spacial mode overlap to the ﬁber guided mode. By controlling
the ﬁber switching time, we know when an EPR-state is present at the ﬁber coupler output.
The ﬁber available is standard single mode ﬁber which does not maintain the polarization, and
thus a polarization controller in one of the paths is needed to ensure identical polarization from
the two paths such that the two modes overlap in the coupler.

controller in path-2. To generate the feedback signal coherent probe light is send through the
switch leading to interference at the ﬁber coupler output half of the time (while the other half

second ﬁber tapping-coupler with splitting ration 1:99, and the interference is detected with a

φ phase diﬀerence between the two paths, the interference is proportional to cos φ, and we can

DC-locking scheme.

5.2

In the time of my master thesis at DTU the state generation part of the setup was completed.
This includes achieving high free-space to ﬁber coupling eﬃciency, designing and testing low
loss phase controller, building a sample/hold DC-locking scheme using digital electronics, and

5.2.1
Squeezed vacuum source
The light source for the setup is a ﬁber laser lasing at 1550 nm, which is optimal for silica based
optical ﬁbers. The setup of the OPO and its pump is not part of this thesis, but is described in
the master thesis of Casper R. Breum (2017) [32]. The OPO has a periodically poled potassium
titanyl phosphate (PPKTP) second-order nonlinear crystal in a bow-tie conﬁgured cavity of
three high reﬂective mirrors and one coupling mirror of 0.1 transmittance. The OPO cavity
is designed with a free spectral range of 967 MHz and 8.0 MHz HWHM bandwidth. The OPO
is pumped by light up-converted with second harmonic generation in a similar cavity with a
PPKTP crystal. The result is a parametric gain of G = 9.6 when pumping the OPO by 381 mW
up-converted light, while the OPO is designed to have a threshold around 800 mW. The amount
of squeezing we can generate in the OPO is not yet measured.
A probe beam is sent through the OPO for phase locking of interferometers in the state
generation and characterization.
To avoid a large coherent amplitude when measuring the
generated state in time, the probe is turned on and oﬀusing a chopping wheel before the OPO.
With a chopping wheel, the on/oﬀ-times of the probe cannot be controlled, and we need a
tapping mirror and detector to know when is the probe on/oﬀ. When the probe is on, the pump
phase is locked to the probe by detecting the probe ampliﬁcation after the OPO, or gain. This
is done using a tapping mirror of 1% transmittance after the OPO, and the transmitted probe
is detected and used as an error signal for feedback to a PZT-mounted mirror for phase control
of the pump before the OPO. The locked phase deﬁnes the quadrature of squeezing.
5.2.2
Fiber coupling
For an ideal entangled EPR-state with perfect quadrature correlation in the mode-a and -
b, inﬁnite squeezing is needed. However, such squeezing is impossible as it requires inﬁnite
energy, and a realistic EPR-states has entanglement depending on the level of squeezing, and
for no squeezing the entanglement vanish.
For this reason it is important to maintain the
level of squeezing of the generated squeezed vacuum states, which decrease when the state is
subjected to loss. This can be pictured in the Wigner function formalism where loss corresponds
to a convolution with a Gaussian function of width increasing with the losses, resulting in a
broadening of the squeezed state’s Wigner function. In conclusion, all losses in the setup need
to be minimized, and thus one condition for proper EPR-state generation in ﬁber is that we can
eﬃciently couple the squeezed light into ﬁber.

5.2.3
Optical switch
With the light coupled into ﬁber, we use an optical ﬁber switch to to alternately guide the light
into path-1 and -2. In path-1 a 200 m ﬁber delay line leads to a delay of τ = 200 m · neﬀ/c =
0.966 µs, where c is the speed of light and neﬀ= 1.449 the eﬀective refractive index of the
guided mode in SMF-28 single mode ﬁber. Thus to overlap the signal in path-1 and -2 after the
delay line, we need a switching frequency of 1/(2τ) = 518 kHz. To realize such switching speed
we use the Nanona solid-state ﬁber-optic switch from Boston Applied Technologies Inc. The
Nanona switch is based on the electro-optic eﬀect in the ceramic material lanthanum-modiﬁed
lead magnesium niobate titanate, where the refractive index is changed when an external electric
ﬁeld is applied. This is utilized to switch the guided light into the two paths, and with no moving
parts, switching frequencies up to 1 MHz is possible.

phase change is observed when scanning the actuator from 35 V to 125 V.

oﬀ. To do this the PID-control unit in the FPGA is modiﬁed with an if-then-statement setting
the error signal to zero when a TTL signal is low. We cannot use the signal from the probe

the chopping wheel clipping the probe. To solve this problem, the detector signal is connected
to a digital input of the FPGA, where it is used to generate a TTL signal synchronized with
the chopping wheel: When the detector signal goes above the threshold for logical high of the
digital input, a delay of 4.1 ms is programmed into FPGA before the TTL to the PID-control
unit is set to high. After 8.4 ms the TTL is set to low again before the chopping wheel starts
clipping the probe beam. The logic programmed in the FPGA is sketched in ﬁgure 5.6, and the

Chapter 6
Discussion and conclusion
Here implementations of non-Gaussian state generation in chapter 3 and 4 and optical switching
and delay lines as used in chapter 5 in quantum information processing is shortly discussed.
Finally, the results in this thesis are summarized, and a conclusion is given.
In chapter 5 it is illustrated how optical switching can be used to divide temporal modes
into spatial modes: Here squeezing in diﬀerent temporal modes of a single squeezing source is
divided into diﬀerent spatial modes by using a switch and optical delay line, which are then used
for EPR-state generation. Such technology has several applications within quantum information
processing (QIP). A simple application is to divide a time-bin qubit, encoded in diﬀerent tempo-
ral modes, into a dual-rail qubit. Or the other way; transforming dual-rail qubits into time-bin
qubits. More general, optical switching and delay lines make it possible to combine temporal
and spatial encoding of information, and convenient in QIP schemes with time multiplexing if
we want to access part of the time-encoded information.
Time multiplexing are useful in measurements based quantum computation (MBQC), where
a cluster state is ﬁrst prepared as an entangled resource state, on which qubits measurements
are performed together with an arbitrary input state, and the result is feed forward through
the cluster state for conditional measurement. The length of the quantum computation then
depends on the size of the entangled resource state, and with time multiplexing cluster state
of arbitrary sizes can be generated without having to scale the experimental implementation as
well [7]. With optical switches and delay lines, we can generate cluster states time-multiplexed
in diﬀerent ways. For instance, if we generate two one-dimensional time-multiplexed CV cluster
states as in [31], each of total temporal length T, and then we somehow delay one cluster states
by T using optical switching and delay lines before mixing it with the other cluster state, we
can generate some sort of a two-dimensional time-multiplexed cluster states.
Another example of using optical switching and delay lines is given in [34]. Here a MBQC
scheme is implemented using a so called quad-rail lattice as entangled resource state, and
measurement-based linear optics is used to implement large virtual interferometers with this
states without equally scaling the experimental implementation. Here optical switching is used
to input states into the quad-rail lattice for computation, and equally reading out the ﬁnal state.
However, one requirement for using switching and delay lines in QIP, is that we can obtain high
eﬃciency in the experimental realization, such that information is not lost after few computation
steps.
Finally, for universal continuous variable (CV) quantum computation, some sort of non-
Gaussianity is necessary [3]. In MBQC this can be obtained by either adding ancillary non-
Gaussian states, or performing non-Gaussian measurements.
However, the CV regime only
consist of Gaussian states and transformation, and to achieve non-Gaussianity, we need to
hybridize the CV and discrete variable (DV) regimes. This is illustrated in chapter 3, where non-
Gaussian transformation in the form of single photon detection, transforms a Gaussian squeezed
state into the non-Gaussian photon subtracted squeezed vacuum state.
Implementing non-
Gaussian DV projective measurements the cubic phase gate can be realized [35], which together

6.1
Conclusion
In this project applications relevant QIP has been demonstrated. This includes non-Gaussian
state generation, qubit generation and EPR-state generation from a single squeezing source
using an optical switch and delay line.
In chapter 3 non-Gaussian projective measurement by single photon detection is utilized
for generating non-Gaussian photon subtracted squeezed vacuum (PSSqV) states, which closely
approximate to Schr¨odinger’s odd cat state of small amplitude. This is done using an already
built setup, and the focus has been on analysis of the acquired data.
Here tools based on
the quadrature covariance function estimated from 20000 quadrature measurements in time
are introduced, including principle component analysis (PCA) for estimating a temporal mode
function directly from the acquired data without any prior knowledge of the generated state.
With estimated mode functions, quantum tomography is performed on the PSSqV state using
heterodyne detection sampling the state’s Husimi function, instead of the more classical approach
a single homodyne detector is used. The result is a PSSqV state with Wigner function negativity
of −0.1042 ± 0.0001 for ℏ= 1, and ﬁdelity of 0.65 to the odd Schr¨odinger’s cat state of 0.99
amplitude.
The low ﬁdelity and Wigner function negativity is concluded to be due to low
eﬃciency in the experimental setup, where loss of 32% is estimated.
Finally, with diﬀerent
mode functions estimated with PCA, the non-physical eﬀect of using quadrature dependent
mode functions are demonstrated and discussed.
With the generated non-Gaussian PSSqV state, the state is encoded in a time-bin qubit in
chapter 4. Here the autocovariance tools introduced in chapter 3 extended to complex variable
including orthogonal measured quadratures in a complex plane, possible with the heterodyne
detection scheme. As a result the qubit phase is extracted directly from the quadrature autoco-
variance function using PCA. To analyze the qubit generation scheme, qubits of single photons
are generated, measured and analyzed using two-mode quantum tomography. Again high losses
are seen, leading to non-orthogonality between the two modes in the time-bin qubit. Further-
more, the temporal mode orthogonality is found to be aﬀected by electronic ﬁltering of the
signal from the heterodyne detection, leading to an increase in the temporal mode extension.
Suggestions on improvement of this mode orthogonality are given, though this is concluded to
be less of a problem than the loss eﬀecting the generated qubit.
Finally in chapter 5, a setup for generating entangled EPR-state from a single squeezing
source is realized, using an optical switch and delay line for dividing temporal output of the
squeezing source into two diﬀerent spatial modes. This is all done in optical ﬁber, where in the
beginning of the setup squeezed light is coupled into ﬁber, and the setup works as a proof-of-
concept for using optical switches and delay lines in ﬁber based quantum networks applicable
for QIP. In the process high free-space to ﬁber coupling eﬃciency of 97% is obtained, a ﬁber
stretcher for phase modulation with low loss is designed and built, and a sample/hold-locking
scheme is realized using fast and tunable FPGA, which is convenient for later optimization.
Unfortunately, within the limited time of this thesis, the setup for state characterization is
not ﬁnished. However, with a single homodyne detector available, the squeezing through the
setup for state generation is measured to be up to −3.66 ± 0.13 dB. The eﬃciency of the two
modes for EPR-state generation is found to be 0.591 and 0.506. This includes losses in the
squeezing source and the homodyne detector. Considering possible improvements of the setup,
we conclude an eﬃciency improvement up to 0.79 is possible with the current ﬁber optic switch,
where EPR-state of up to −6.8 dB squeezing can be generated in ﬁber.
