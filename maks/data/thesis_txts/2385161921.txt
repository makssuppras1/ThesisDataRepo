Summary
In this master thesis, we mainly concern with statistical methods for replicated sensory
discrimination test data an their implementations in R. The sensory discrimination test
have been applied to solve problems from sensory and consumer science. Currently,
we have some method to do sensory discrimination test with normal binomial data set.
However, the method to deal with replicated data set is not so much, especially, power
function and sample size function for this kind of data set.
Generally, beta-binomial model and chance corrected beta-binomial model are used
to ﬁt replicated data set in sensory discrimination test. These has been studied for many
years and the well-know analysis tool is called sensR in R package. sensR is a powerful
package for sensory discrimination test, which include the application of beta-binomial
model and chance corrected beta-binomial model. The current estimation method for
parameters is maximum log-likelihood function and the conﬁdence interval is based on
Wald conﬁdence interval. In this thesis, we will provide another method to estimate the
parameters, proﬁle likelihood estimation and corresponding conﬁdence interval.
Another attribution of this thesis is we will introduce power function and sample size
function for replicated sensory discrimination test. This will ﬁll the gap in the sensR
package for the replicated sensory test.

Chapter 1
Basic theory we need in this thesis
The major task of sensory discrimination test is to compare two products in certain as-
pects(the speciﬁed sensory attribute), which we often refer to as A and B. The basic
methods of sensory difference (or similarity) analysis are to provide reliable sensory
measurements and to conduct valid tests.
1.1
Sensory discrimination test protocols
The sensory discrimination methods are widely used for trained sensory assessors or
sensory scientist in different ﬁelds, like quality control, product development, food in-
dustry and so on. The major task of discrimination tests is to compare two products,
which we often refer to as A and B. The most basic and mostly used sensory difference
tests are the following seven tests: Duo-Trio, Triangle, 2-AFC, 3-AFC (AFC = Alternat-
ive Forced Choice), tetrad, A-not A and same-different. In this thesis, we mainly focus
on the ﬁrst ﬁve methods.
1.1.1
Triangle test
In the Triangle test, three samples are shown to the assessors including both sample types
at the same time, two samples are from the same product and anther one is from the
different product.e.g. AAB, ABA or BAA. Each assessor is asked to exam the three
sample in order to indicate either which sample is different with the other two samples or
which two samples are most similar.
1.1.2
Duo-Trio test
The Duo-Trio test is an alternative to the triangle test and a more complex test. In the
Duo-Trio test the assessor (consumer) is also shown with three samples of three types,
e.g.AAB, ABA or BAA. The difference from triangle test is that one of the sample is
considered as a control sample and this sample is similar to one of the other two samples.
1
==PAGE:9==
Hence, the task of this test is to select which of the remaining two samples is most similar
to the control sample.
1.1.3
Tetrad test
The tetrad test is another sensory discrimination test that is used for four samples: two
samples from one product and two from a different one. The main task of this method
is to form the samples into two groups based on their similarity, which is characterized
as unspeciﬁed tetrad. There are six possible types to be shown to the assessor from
the two products, e.g. AABB, BBAA, ABAB, BABA, ABBA and BAAB. Both AABB
and BBAA are correct answer. The other tetrad method is called speciﬁed tetrad, which
means the only correct answer is AABB. In this thesis, we only consider the unspeciﬁed
tetrad.
In the Triangle test, Duo-Trio test and Tetrad test, the sensory scientist will only know
the result that the samples are different but not which part(attribute) is different. However,
in practical it is always important which part to know which attribute is different. For
example, if we have two different food product, it is always interested in that which
speciﬁc attribute is different, e.g. sweetness, bitterness and so on. Hence, the above three
methods are not good choice to distinguish the difference.
1.1.4
2-AFC test
The two-alternative forced choice(2-AFC) test is one of the simplest discrimination test
protocols where two different samples are presented to the assessor and the assessor is
asked to determine if a speciﬁc difference exists between two samples, especially which
speciﬁc attribute is identiﬁed to be affected. The results of 2-AFC test presents the dif-
ference of a speciﬁc attribute between the two products. The examiner must be sure that
the two samples only differ in the single speciﬁed sensory dimension.
1.1.5
3-AFC test
The three-alternative forced choice (3-AFC) method is similar to triangle test where the
assessors receive three samples and are asked to indicate which sample(s) are higher or
lower in a speciﬁed sensory attribute. In any speciﬁc 3-AFC study there are six possible
serving types (AAB, ABA, BAA or BBA, BAB, ABB) that should be randomly shown
2
==PAGE:10==
across all assessors. The major task of the three-alternative forced choice test is to ask
the sensory scientist to determine if there exist difference of speciﬁed attribute between
the two products or which sample has the strongest (or the weakest) difference in the
speciﬁed attribute. As we know in 2-AFC, the assessor only knows that the two samples
differ in the single speciﬁed sensory dimension.
These ﬁve methods have the same characteristic that each assessor is presented with
two different samples and asked to distinguish the difference between the samples. How-
ever, this is different in the case of the A-not A and same-different tests where each
panelist is shown with only one of the two samples. In the A-not A case, assessors are
presented with both A and not-A samples before the test. Then in this test the assessor
is presented with only one sample and is asked if the sample is ‘A’ or ‘not A’. And in the
same-different test each assessor is presented with one of the possible pairs of samples,
AA, AB (or BA) or BB, and is asked if the sample pair is the ‘same’ or ‘different’. For
all of test methods the panelist is forced to make a choice/give an answer.

1.3
Analysis of Data from Basic Sensory Discrimination
Tests
The result from any one of these test methods is either correct or incorrect, hence the dis-
tribution of these answers is therefore a binomial distribution. In the following analysis,
we mainly focus on binomial distribution.
For the duo-trio and 2-AFC protocols we have the guessing probability, pg = 1/2.
This means that if there is no difference between the products, then the probability of a
correct answer, pc is one half. Similarly for the triangle, tetrad and 3-AFC protocols we
have the guessing probability, pg = 1/3. The observed proportion of correct answers
cannot be directly used to estimate the proportion of discrimination, pd.
The relation between the probability of a correct answer, pc, and the probability of
discrimination, pd, is
pc = pg + pd(1 −pg),
(1.4)
where the guessing probability, pg is 1/2 for the duo-trio and 2-AFC protocols and 1/3

for the triangle, tetrad and 3-AFC protocols. The reverse relation is
pd = (pc + pg)/(1 −pg).
(1.5)

correct probability, pc = 0.7, and we choose discrimination test method, tetrad method,
which guessing probability is 1/3. Then the discrimination probability is 0.55.

1.5
The Power of the Statistical Test
Discrimination tests are applied to present if two different samples are different or not.
The power of the discrimination test is an important measure for this test. The power of
a discrimination test is deﬁned as the probability of ﬁnding a difference if the difference
actually exists. In other words, it is the probability of making the correct decision that
the two samples are different.
Statistically, there are two types of errors that the assessor of any sensory protocols
can make when testing the null hypothesis, H0. The null hypothesis states that the two
products are the same; the alternative hypotheses, H1, states that there is a difference
between the two products. A Type I error (α) occurs when the assessor rejects the null
hypothesis (H0) when it is actually true. When making a Type I error in a discrimination
test, we would conclude that the two products are different but they are actually not
different. The Type I error (α) is decided by the assessor. Usually, α is chosen to be very
low (0.05, 0.01, or 0.001) which means that there is a 1 in 20, 1 in 100, and 1 in 1,000
chance, respectively, of making a Type I error(Type I deﬁnition, Wikipedia). A Type II
(β) error occurs when the assessor accepts the null hypothesis (H0) when the alternative
7
==PAGE:15==
hypothesis (H1) is actually false. When making a Type II error in a discrimination test
we would conclude that the two products are the same, but they are actually different.
The Type II error is the probability of the risk of not ﬁnding a true difference when one
actually exists(Type II error deﬁnition, Wikipedia). This can be illustrated in Table 1.2.

1.6
Statistical power and simple size
In discrimination test, it is important to ﬁnd the relationship between power and sample
size. For example, if we would like to test a difference of two products under a power,
say 0.8, then we should know how much samples we need in this test in order to be time
and money economical. In this section, we will consider calculation of power and sample
size based on the binomial distribution.
1.6.1
Critical value
The critical value, xc, is the smallest integer number that satisﬁes the following equation
given a signiﬁcant level (α),
P(X ≥xc) ≤α
where
X ∼binom(pc0, n)
(1.12)
where pc0 is the probability of correct answer under null hypothesis, 1/3 for triangle or
3-AFC test and 1/2 for duo-trio and 2-AFC test. In sensR package, we can obtain the
critical value, xc, by findcr function.
Example 1.1: consider the situation that we have a 40 trails in a 3-AFC test. We have
the signiﬁcant level,α = 0.05,and pc0 = 1/3.Then we can obtain the critical value for
this test is 19,

1.6.3
Sample size determination
The sample size calculation is how to ﬁnd the sample size such that we can obtain the
target power in a sensory difference test. It is easy to say, however the calculation is
challenging. Hence, in this section, we will talk about a simple way to calculate the
sample size.
Formally, the expected sample size, n⋆for a sensory difference test, is the smallest
integer number that satisﬁes
P(X ≥xc) ≥target-power
where
X ∼binom(pc, n⋆)
(1.15)
and where P(X ≥xc) is the actual power of the test. This deﬁnition (1.15) is easy to
understand and we have already performed this calculation in sensR package.

choose in a difference test using the triangle protocol, where the null hypothesis is no
difference and target power is 0.8. We say the actual difference is d′ = 1.5 under the
alternative hypothesis and we choose a signiﬁcant level, α = 0.05. By using package
sensR, we may get the sample size for the exact binomial test with

number of consumers or assessors, for instance, it is time and money consuming to ﬁnd

record the number of correct answers for each: x1, ..., xN. For example, the ﬁrst assessor
have 4 correct answers out from 10 replicates, which is x1 = 4.

3.1
Replicated data
As we mentioned in section 2.1, replicated sensory discrimination test is often used in
real life. Currently, we can analysis this kind of data by betabin in R package, sensR.
By using this function, we can use the beta-binomial model and the chance-corrected

sample size with desired statistical power for replicated binomial data.
3.2
Power function
The power of a test is the probability that the test correctly rejects the null hypothesis (H0)
when the alternative hypothesis (H1) is true. In our case, it can be equivalently thought of
as the probability of detecting a difference while the hypothesis is no difference. In this
part, we will mainly consider the power based on corrected beta binomial model, which
is used to ﬁt replicated experiment data set.
As we mentioned above, we can calculate the p-value by betabin in sensR package
based on given replicated experiments This p-value represents the probability to reject
the the null hypothesis (H0) for independent experiment, which is similar to power. In
this paper, we will calculate the power based on simulations.
Recall that power is the probability of getting a signiﬁcant result when there is in fact
a difference, thus in the simulation method it is the proportion of signiﬁcant results to the

total number of tests:

no.test
(3.1)
We can use R to generate random data from the model under the alternative hypothesis
and then perform the signiﬁcance test. We can even do that many many times and record
the p-values allowing us to calculate the power via (3.1). In next step, we will analyze
this power function step by step.
Firstly, we simulate the outcome of replicated sensory difference tests, this can be
done by discrimSim in sensR package. Then we can calculate the p-value of this
simulated data set by betabin or discrim in sensR. We ﬁrst try to use betabin to ﬁt the
replicated data set, if the result is not empty, we can simply use function summary in
sensR to ﬁnd the p-value. Otherwise, we use discrim to obtain the p-value. Finally we
can combine the simulation function into this step to obtain a function which can be used
to calculate p-value based on a simulated replicated data set.

powerCI <- function(N, power){
#N is the number of simulation for power calculation
#power is the objective power
p <- power
Nsim <- N
se <- round(sqrt(p*(1-p)/Nsim), 3)
a <- 1 - 0.05/2
ci <- round(p + se %o% qnorm(c(a, 1 - a)),3)
ci[ci < 0] <- 0
ci[ci > 1] <- 1
table <- array(NA, dim = c(1, 4))
rownames(table) <- c("power")
colnames(table) <- c("Estimate", "Std. Error", "Lower",
"Upper")
table[1,1] <- p
table[1,2] <- se
table[1,3] <- (ci[2])
table[1,4] <- (ci[1])
return(table)
}

3.4.2
Sample size function with bisection method
In this section, we will introduce a simple algorithm, bisection method, which is a root-
ﬁnding method that repeatedly bisects an interval and then selects a sub-interval in which
a root must lie for further processing. It is a very simple and robust method, but it is also
relatively slow.
In the sample size function, we ﬁrstly choose a number as assumed sample size,
N, and start bisection method between interval (1,N). The ﬁrst step we calculate the
power based on sample size, 1+N
2 . If this power is larger than target power, then the
second step we calculate power with sample size which is the middle value between
interval (1, 1+N
2 ),otherwise the middle value between interval (1+N
2 ,N). We repeat this
procedure until we obtain the power between (target power-0.01,target power+0.01), then
the corresponding sample size is we expected.
