Abstract
In recent years, object detection has become a more and more popular topic. The target
of the project is finding a suitable way to detect the aim red ball. In the old days, hough
circle transform and edge detection are both useful methods in this kind of task. On
the other hand, neural networks and deep learning gradually replaced traditional image
detection algorithms and became the mainstream methods of target detection. A series
of target detection algorithms based on deep learning could be roughly divided into two
major types: two- stage algorithms and one-stage algorithm. In two-stage algorithm,
candidate regions were generated and then performed CNN classification (RCNN). In
one-stage algorithm, the input image was directly applied into the algorithm and get
the category and corresponding positioning at the output (YOLO). The main idea of
this thesis is comparing different methods then decide which model is the most suitable
model to apply.

Motivation
In every three years, Mohamed Bin Zayed International Robotics Challenge (MBZIRC)
holds in Abu Dhabi. The aim of this competition is inspiring the future of robotics
through innovative solutions and exceptional technologies. Robotics is going to have a
transformative impact in a various areas and human social aspects, such as in disaster
response, healthcare, domestic tasks, transport and construction. However, there is a
gap between current reality in robotic capabilities and the requirements of potential
applications.
MBZIRC 2020 competition is motivated by pushing technological and
application boundaries in robotics. There are three challenges in total. In challenge
1, a group of drones will autonomously track a set of objects (such as intruder UAVs)
following 3D 8 shape trajectories inside arena.

Traditional Method
The tracked target, soft red ball in the video looked like a red circle in the image. The
sphere could be detected by searching the edges in the image that create the circle.

2.1.2
FindContours
FindContours is one of the contour extraction functions in OpenCV. It takes 8-bit
signle channel image as input. Non-zero pixels are arranges as 1’s and zero values still
0’s. The search mode of the contour could be defined as external contour or all contours.
The element in the output vector holds a set of points composed of continuous points.
Each point set is a contour.

2.2
Deep Learning Method
The idea of using convolutional neural network to do object recognition was first pro-
posed by Yann Lecun in 1998 [Lec+98]. Figure 2.2 is the architecture of LeNet-5. It
could be seen as an example for CNN. Input a picture into the network, and then pass
through multiple convolution and pooling layers. The category of the target given at
the output. Because of the limit of computer hardware technique, it was too expensive
to be used in daily life.
Figure 2.2: Lenet.
With the development of hardware and software technology, deep learning become
more and more popular since 2011. In 2011, a team from Canada won the ImageNet
LSVRC competition. They put AlexNet forward in the next year. Compare with LeNet,
AlexNet has more layers, adds max pooling, drop out technique to avoid overfitting and
use ReLU as active function [KSH12]. The state of art researches on object detection
could be generally classified into two kinds; two-stage (RCNN) and one stage (SSD,
YOLO) [Liu+20].
In 2014, Ross started to use Region Convolutional Neural Network (RCNN) on tar-

get detection [Gir+14]. Comparing with the traditional sliding convolution window to
determine the possible regions of the target, R-CNN uses the selective search method to
pre-extract some candidate regions that are more likely to be the target. This method
greatly improves the speed and reduces the computational cost.
2.2.1
Faster RCNN
Ross proposed a new Faster RCNN in 2016 [Ren+16]. In terms of structure, Faster
RCNN has integrated feature extraction, proposal extraction, bounding box regression,
and classification into one network. This change let the comprehensive performance
been highly improved, especially in detection speed. Figure 2.3 gives four main concept
of Faster RCNN.
Figure 2.3: Faster RCNN.
• Conv layers: Faster RCNN first uses a set of basic conv+relu+pooling layers to
extract image feature maps. The feature maps are shared to the subsequent RPN
layer and fully connected layer.
• Region Proposal Network: The RPN network is used to generate region proposals.
This layer uses Softmax to determine whether the anchors are positive or negative,
and then applies bounding box regression to correct the anchors to obtain accurate
proposals.
==PAGE:18==
8

• RoI Pooling: This layer collects the input feature maps and proposals, then ex-
tracts proposal feature maps after combining these information, and sends them
to the subsequent fully connected layer to determine the target category.
• Classifier: Using the proposal feature maps to calculate the category of the pro-
posal, and at the same time using bounding box regression again to obtain the
final precise position of the detection box.
2.2.2
YOLO
YOLO is the short for ’You Only Look Once’.
It was proposed by Joseph in 2016
[Red+16]. YOLO solves the object detection as a regression problem. After the input
image passes through an inference, the position of all objects in the image, their category
and the corresponding confidence probability could be obtained in a short time. But
YOLO detector was insensitive to the crowed objects and small target. As improvements,
in YOLOv2 Joseph used batch normalization to speed up the convergence progress and
based on the offset of anchor to predict the coordinates [RF17]. After that, the author
made some update on YOLOv2 called YOLOv3 in 2018 [RF18]. The feature extractor of
YOLOv3 is a residual model. It contains 53 convolutional layers, so it is called Darknet-
53. FPN architecture (Feature Pyramid Networks for Object Detection) has been used
to achieve multi-scale detection.

Network Structure

easily noticed that, there are 6 different kinds of layers: convolutional layer, shortcut

layer, maxpool layer, route layer, upsample layer and yolo layer.

• Convolutional layer: Defined the number and the size of the kernel, using LeakyReLU
as activation function.
• Shortcut layer: ResNet shortcut branch, add the feature matrix on the input
branch to the output matrix obtained after passing through two convolutions on
the main line.
• Maxpool layer: Applied in spp model. In order not to change the shape of the
input feature layer, using padding method. So that after doing the max pooling
down sampling, the shape of the feature map remains the same.
• Route layer: If there is only one parameter in route layer, it points to the output
of a certain layer. When there are multiple parameters exist, the output of those
layers will concatenate in the depth direction.
• Upsampling layer: Enlarge the height and width of the input feature matrix as
twice as the original, appeared twice in spp model.
• Yolo layer￿Connected with each predictor and generating a series of anchors.

CHAPTER 5
Conclusion
In this thesis, four different kinds of object detectors , FasterRCNN and Y OLOv3_spp
has been performed and investigated.
In the experiment of Houghtransform and
FindContours, the procedure was given step by step. After that, the experiment of
