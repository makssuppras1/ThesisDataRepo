Abstract
A loudspeaker model using fractional order derivatives has recently showed promising re-
sults in describing both the creep eﬀect in the suspension and eddy currents in the motor
system. Other models exist for modeling the creep eﬀect, and does it quite well in the
frequency domain. The fractional order model has a clear advantage, as it is well-deﬁned
in the time-domain. However, the discrete time-domain solution is computational expen-
sive as the entire history is used to calculate the gradient, that is used to predict the next
sample.
In this work an eﬃcient method for calculating the discrete solution is developed using
multi-stage processing. The computational complexity is greatly reduced, but with the
cost of accuracy. The multi-stage solution is an approximation, hence some error is intro-
duced. The algorithm is designed with two tuning parameters, which makes it possible to
ﬁnd the best compromise between accuracy and eﬃciency.
Four loudspeakers with diﬀerent surrounds are used to validate the model. The transfer
functions of the loudspeakers are found using frequency analysis, and the fractional order
model is ﬁtted to get the parameters. The parameters are used for simulation in the time-
domain using the new multi-stage solution, which are then compared to measurements.
A good agreement between the fractional order model and the displacement of the loud-
speaker is found.
As the ultimate goal is to use the fractional order models for feed-forward compensation it
is discussed how the non-linearities of the loudspeaker could be included in the simulation.
Furthermore, measurements results shows that the loudspeaker model is not yet complete,
as the displacement transfer function is level dependent.

Chapter 1
Introduction and Motivation
1.1
Motivation
The electro-dynamic loudspeaker is a complex electro-acoustic transducer and is by na-
ture, unfortunately, a very non-linear device. The general task of a sound system, i.e.
playback device, ampliﬁer and loudspeaker is to reproduce exactly the sound that is saved
on the playback device. Today’s playback devices are vast and range from analog record
players to highly advanced digital players. Most of today’s playback devices do a very
decent job of converting the recorded material to a voltage that is sent to the ampliﬁer.
Audio ampliﬁers also exist in a vast range of classes and designs, most of which do an
excellent job of amplifying the signal to the loudspeaker.
A loudspeaker, which is almost always a moving-coil type, often changes the signal quite
signiﬁcantly, compared to the ampliﬁer and playback device. As the moving-coil loud-
speaker, by nature, is a non-linear device, no matter how it is designed, it will never be
linear. Which means that other methods must be considered in order to complete the task
in hand: reproducing the sound correctly.
Many methods exist for reducing the inﬂuence of the non-linearities in the loudspeaker, but
one of the more successful is feed-forward compensation using a model of the loudspeaker.
The idea is, that with an exact model of the loudspeaker, the output of the loudspeaker
can be predicted and compared to the input. The distortion added by the non-linearities
is sought to be removed. By comparing the input to the predicted output, thus ﬁnding
the diﬀerence, the output can be pre-distorted, in such a manner, that a distortion-less
output is achieved.
But, there is a problem. The model that is used for the pre-compensation has to be very
accurate, or else more distortion will be added. The lumped parameter model that has
existed for years, is a suﬃcient model for most cases, but the model deviates from the real
loudspeaker, below and above the resonance frequency. The integer order model, which is
commonly used, predicts a constant displacement below the resonance of the loudspeaker,
but in reality, the displacement of the diaphragm is rising towards lower frequencies for
the same input level. This known as the creep eﬀect. Another problem of the standard
model, is that it predicts the inductance to be pure, and does not consider any losses. As
many authors have shown, the eddy current losses do have a quite signiﬁcant importance
in the impedance at higher frequencies.
Modeling the lossy inductance is done in various ways, and one of them is the Leach model,
which uses a fractional order inductance, and has been shown to model the behavior quite
well. Recently a fractional order model for the viscoelastic rubber material, which is caus-
ing the creep, has been developed, which likewise shows good results. The advantage of
the fractional order models, is that they are quite simple to calculate in the frequency do-
main. Another advantage is that the fractional order model has a direct translation to the
==PAGE:13==
1.2 Goal and Approach
3
the time-domain which is usefull for non-linear modeling. The disadvantage is, that it is
very tedious to calculate the solution in the time-domain. The feed-forward compensation
is usually done in the discrete time-domain, which means that time-domain solution must
be used.
The problem with the discrete time-domain solution of the fractional order derivative
is that for each sample all the previous samples must be used in order to calculate the
next sample. This results in a quadratic rising calculation time and can not be used in
any real-time application. The goal of this thesis is to ﬁnd a method for reducing the
calculation time, and the complexity of the calculation, so that it might be possible to use
the fractional order derivative models in real-time.
1.2
Goal and Approach
The main goal of this project is to investigate the possibility of making a more eﬃcient
algorithm for calculating the fractional order derivatives. By studying algorithms for other
types of problems, several ideas of approximating the solution has inspired an algorithm
for use with loudspeakers.
The performance of the algorithm is investigated, ﬁrst using a simple mechanical system,
and later the model for the loudspeaker. To investigate if the time-domain solution cor-
responds with reality, measurements have been performed on several loudspeakers, which
are then compared to the simulations.
As the feed-forward compensation of the loudspeaker is performed using digital signal
processing, the approximation of the solution is investigated with theory from signal pro-
cessing. Using standard techniques, it is sought to simplify the calculation. Some error is
introduced by approximating the solution, thus a compromise between accuracy and eﬃ-
ciency must be chosen. By changing the parameters the error is introduced and compared
to the reduced calculation time.

Chapter 2
Fractional Order Derivatives
The content in this in this chapter is, if not stated otherwise, a summary of [15].
The ﬁrst mentioning of the idea of fractional order derivatives was in 1695, in a letter from
Leibnitz to L’Hospital, where he discusses the notion of a half-order derivative. Since then,
the theory of fractional, or non-integer, calculus has been investigated theoretically. In
recent years it has been found that using fractional calculus can greatly improve control
systems, both by making fractional order controllers and creating better models for sys-
tems. Fractional order components has the advantage of not being either e.g. a resistance
or a capacitor, but can be something in between. This has been found useful for modeling
real world problems, which can not be described by e.g. the simple capacitor. In diﬀerent
scientiﬁc areas fractional calculus has been applied to model systems, especially those that
shows a ”memory” behavior, which is seen in electrochemistry, biological systems, mass
transport, diﬀusion and viscoelasticity.

2.2
State-Space Models
To model a non-linear system, e.g. a loudspeaker, the time-domain solution using state-
space representation is often used. The state-space representation consists of matrices
containing the parameters of the loudspeaker. In the linear model, these parameters are
constant, and do not depend on the state of the loudspeaker. In contrast, the parameters
in the non-linear model can be state dependent, and can thus be changed in regards to their
displacement or current. In the following, the state-space solution for the commensurate
order system will be introduced, and afterwards expanded to the non-commensurate order.
The loudspeaker model is non-commensurate as the fractional coeﬃcients of the suspension
and the inductance do not have any relation, and are thus not integer multiples of each
other.

Stability analysis of non-commensurate order
The stability criteria of non-commensurate order system has, to the authors knowledge,
not been deﬁned analytically, so one must settle with numerical analysis [14]. Finding the
stability criteria for each of the diﬀerent fractional orders can be done, but since there
is interaction between the variables, due to the system equations, the solution could be
stable at a lower frequency, than what the criteria predicts. So calculating the criteria
can give rise to a higher sampling frequency, than is actually needed, thus requiring an
unnecessary amount of calculations.
To ﬁnd a sampling frequency where the solution is asymptotically stable, numerical cal-
culations is performed in Matlab. This is an ad-hoc method developed for this project,
and the validity is not tested on other systems. It is however useful for ﬁnding a suitable
guess for a stable sampling frequency.
The system is excited with an impulse, and a number of samples is calculated, e.g. the
ﬁrst 100 samples. It is then checked if the amplitude of all the state variables are less than
a given threshold. An appropriate threshold is chosen for the system, and was in this case
the value 0.5 for all variables, which was found empirically. If the variables are higher than
the threshold, it indicates that the system is unstable, as the impulse response is expected
to decay. If the value over the threshold, then the sampling frequency is increased, the
samples are calculated again and the check is performed, etc. When all the values are be-
neath the threshold, the loop is stopped and the sampling frequency, at which the system
is stable, is found. As stated, this method is not well tested, so for some systems it might
not work. However, it was found to yield decent results when used with the loudspeakers
in this project.

Chapter 3
Multi Stage Processing
As shown in the last chapter, the discrete fractional order solution relies on the entire
history, in order to calculate the next sample. The solution has a high computational
complexity, which makes it unusable in any real-time application. To make the fractional
order model attractive to use in non-linear compensation, the computation must be sim-
pliﬁed, so that it is more eﬃcient. In this chapter a brief overview of some other authors
suggestions for eﬃcient algorithms are given. After this, a new algorithm developed in
this project is described. As the implementation is a multi-rate process, some digital ﬁlter
theory is brieﬂy discussed. The consequence of diﬀerent design parameters are discussed,
and ﬁnally the computational complexity of the proposed solution is compared to that of
the full solution.
3.1
Multi Step Methods
The discrete time-domain solution is tedious to calculate as the next state in the depends
on the entire history, so the full solution is not that attractive to use in simulations. Many
authors have suggested how to reduce the computational load, by approximating the so-
lution or by using matrix systems that simplify the calculations.
The most basic solution to the problem, is to limit the memory, known as the ”Short-
Memory” principle [17]. Instead of using the entire history, only a given length of the
recent past is used. The result is that the sum is limited and thus the calculation eﬀort
is greatly reduced. With some systems, this approximation is ﬁne, but with systems that
have a long memory, the error coming from this approximation is quite signiﬁcant.
Other approximations look at the characteristic of the weighting function and uses a log-
arithmic memory principle. The idea is that the history is logarithmically spaced time
instances, instead of linear. This principle has been described in multiple publications, and
some of them are brieﬂy discussed here. In [6] the principle is called ”nested mesh” and
in the publication it is described how the weighting function can be approximated with
a logarithmic increasing step size. This approach is also, to some extent, used in this thesis.
The logarithmic memory approach is also used in [12], but with the goal of using as lit-
tle memory as possible. Only the most important samples are kept and the memory is
constantly cleansed of history that is no longer important. In both of the cases above,
the decimation is performed with a ﬁxed factor. This means that the second step size is
always twice as large as the ﬁrst, the third is four times, etc. In [8] a variable step size is
used depending on the fractional order and the system. As the weighting functions of the
diﬀerent fractional orders have diﬀerent slopes, the approximation is improved by taking
this into account. As it will be shown in this chapter, a ﬁxed step size was chosen in this
project, due to choice of implementation.
If a simulation of a system is performed, where the entire input is known, it is possible

length in order to save some computation time [18]. An equidistant grid is discretized
with the same length between the points on the surface. The non-equidistant grid is more
useful when calculating, e.g. the diﬀusion of heat on a surface, where the surface can then
be divided into non-equal sized areas. However as the goal with this thesis is to calculate
the fractional order derivative in real-time, it is not possible to know the entire input in
advance.
An approach that resembles the idea in this thesis is found in [22]. Here a multi-rate sys-
tem is used to estimate the fractional order derivative of a real-time measurement. In the
paper a multi-rate system is used on a measurement input, which is low-pass ﬁltered and
decimated multiple times, before a FIR diﬀerentiator outputs the estimated derivative.
As this is a measurement and not a simulation, the actual fractional order state-space
solution is not calculated, which is done in this thesis.
In this thesis ideas from the above has been collected into a multi-stage system, with
logarithmic memory, that is calculated sample-by-sample, so it could be implemented as a
real-time process. Many other eﬃcient ways of calculating the fractional order derivatives
can be found in [26].
3.2
Approximating the Weighting Function
The weighting function, which is multiplied to the past states of the system, see Equation
2.22, has, as shown in Figure 2.1, an ever decaying long tail, meaning that the gradient
of the fractional order system has an everlasting memory. When calculating the numer-
ical solution to the fractional order system, this requires an inﬁnite amount of memory,
a luxury not found in any computer.
Furthermore an quadratic increasing amount of
calculations are needed in order to calculate the next sample, since the history is getting
longer and longer. Both of these factors make a time domain simulation of the FO model
of a loudspeaker unattractive to use in real-time applications or long simulations.
With the weighting function plotted is in the double logarithmic, it is approximately a
straight line, Figure 2.1. Thus the function is close to a power function. To investigate
the curvature of the weighting function, the acceleration is found, by taking the second
derivative. The lower the acceleration the more the adjacent samples are alike and can
be approximated to be the same. The acceleration of the weighting functions for diﬀerent
fractional orders are shown in Figure 3.1. When the acceleration is closer to zero, a larger
amount of the function can be assumed to be linear. Thus the weighting function is piece-
wise linear and for higher sample numbers a broader piece of the function can be assumed
to be linear. For higher sample number, i.e. earlier history, the weighting function is
more ﬂat and adjacent samples are approximately the same. Since it resembles a power
function it is fair to assume that the step size can be increased logarithmically, which is
also suggested in [6].
To control the order of approximation a variable η is introduced. This is the number of
samples that are used in the summation before approximating the weighting function. As
the acceleration of the weighting function is decaying with a power function, it is fair to
assume that the approximation can be done with a power relationship. In this project a

3.3
Multi Stage Processing
From the time-frequency analysis in the previous section, it was analyzed that the fre-
quency content of the weighting function is lower for history that is far in the past. Thus
the high frequency content of distant history is not as important as the low frequencies,
making it is possible to save some calculations, by only using the frequency content of the
history that matters.
Multiple eﬃciency methods are available, as mentioned in the introduction to this chapter.
In this thesis, the idea of multi-stage processing has been used, in order to achieve the
goal of a more eﬃcient algorithm. Multi-stage processing is also used in wavelet packet
analysis and the time-frequency analysis of such is shown in Figure 3.6 and the binary
tree is shown in Figure 3.7 [25].
Figure 3.6: Time-frequency spectrum of wavelet transform.1
Gray area: Only one branch.
The wavelet transform takes the input, x[n], and splits the frequency content into two,
using a low and a high pass ﬁlter, H0(z) and H1(z) respectively. Both ﬁlters have a cut-oﬀ
frequency at 0.5 normalized, i.e. half the frequency band. After the ﬁltering, the signal is
decimated with a factor of two. Each sample now corresponds to twice the time period,
as the sampling frequency is halved. This results in what is seen in the time-frequency
plot in Figure 3.6 where the ﬁrst part of the response has a broad frequency range and a
short time range. After the ﬁltering and decimation, the frequency range is halved and
the time is doubled, as the sampling sampling frequency is halved. By doing the ﬁltering
and decimation multiple times, the result is a narrow frequency band with a long time
representation.

3.3.1
Digital Filtering
When decimating the signal in the multi-stage solution, the ﬁlter before the decimation,
H(z), ensures that a minimum of aliasing occurs. The Nyquist theorem states, that in or-
der to represent a signal with a frequency bandwidth of Fmax = B the sampling frequency
must be at least Fs > 2Fmax = 2B [19].
Removing half the samples from the history corresponds to a decimation by a factor two,
resulting in a new sampling frequency of Fs/2. If no ﬁltering is done prior to the dec-
imation, the new signal, y(n), will be an aliased version of the original, x(n).
If the
spectrum X(ω) only contains frequency content in the region 0 < f < Fs/4, it would be
possible to decimate the signal without ﬁltering, but if there is frequency content from
Fs/4 < f < Fs/2 it will be mirrored back around the new Nyquist frequency of the deci-
mated signal, Fs/4.
In order to ensure that there is a minimum of aliasing, the signal, x(n), is ﬁltered with a
low pass ﬁlter that attenuates frequencies above Fs/4. Digital ﬁlters are designed using
normalized frequencies, where the highest frequency the ﬁlter can process, which is the
Nyquist frequency Fs/2, is set to 1. This means that the ﬁlter should attenuate normalized
frequencies above 0.5 and have the response shown in Figure 3.9.

5.1

One way of modeling the loudspeaker is to use the lumped parameter model [11]. The
model collects the complicated system into a few components that describes the system.

and the part of the suspension that is moving together with it, is modeled as a single
mass. This way a simple second order damped mechanical system is achieved, which is
well deﬁned by analytical solutions. The mechanical system is driven by the force that is
generated by the current running through the coil. The coil is placed in the magnetic ﬁeld

Investigations of the design of the ﬁlter were done, with some interesting ﬁndings. At
ﬁrst, the assumption was that the ﬁlter needed to have high stop band attenuation, in

was always the same. It was found that the error was lowest when the DC component of

that due to the negative gain at DC this negative gain accumulates with the multiple ﬁl-
tering in the multi-stage processing. As a digital ﬁlter is a normalized ﬁlter, the eﬀect on
the actual frequencies is depending on the sampling frequency. As the sampling frequency
is halved for each downsampling the frequencies, of which the normalized frequency corre-
sponds to, is changed. Except for the DC component, f = 0, which will always be 0 Hz, no
matter the sampling frequency. So when the ﬁlter is used multiple times in the multi-stage
solution, the ripple in the rest of the passband will aﬀect new frequencies, but the DC
component will accumulate. Thus having a negative gain at DC will increasingly changes
the DC oﬀset of the approximated history, which in turns, will result in error in the output.
However, it seems like this is not the full explanation. Filter 2 just has a positive gain
at DC, and with a larger magnitude, this accumulates in a even bigger error, when used
multiple times. So from the arguments above there should also be an error when using
ﬁlter 2. But using ﬁlter 2 results in a signiﬁcant lower error. Note that the simple average
has 0 dB at DC. So from this it seems like a positive gain at DC is beneﬁcial, but the
exact reason for it, has not been found.

6.8 Discussion of the Evaluation
55
All in all there are two parameters that one can control when using the multi-stage solution
for loudspeakers. The sampling frequency is not counted as a control parameter, as this
is often set in order to comply with the Nyquist theorem.
As shown, the number of
samples in each stage, η, controls how well an approximation the multi-stage solution is.
The higher the number the lower the error, with the cost of extra calculation time. The
number of stages is controlling how long a period of time the system remembers, and
depending on the simulation, this may be important or not. However, when adding an
extra stage the time is doubled, so for example in the case described above, adding a stage
would double the time memory to 1342 seconds without adding signiﬁcant calculation
time. The multi-stage solution can thus be tuned for accuracy or eﬃciency, depends on
which is most important for the application.
6.8
Discussion of the Evaluation
The error deﬁned in the beginning of this chapter does not give much detail about what
kind of error is present. A constant oﬀset would result in a quite signiﬁcant error, as the
the diﬀerence between the full and the multi-stage solution would be constantly diﬀerent.
The error for the limited memory was not calculated, as this is not a constant error, and
was better represented with the time plot. But the relative error coming from the constant
deviation would, depending on the simulation time, accumulate a signiﬁcant error, due to
the constant deviation.
This is the drawback of using only a single value for the error, but in order to compare
the diﬀerent parameters it was deemed the best choice. The error is relative which should
make the error independent of excitation signal, but as the system is frequency dependent,
this is not deemed to be the case. Furthermore the aliasing is a non-linear eﬀect, which
makes it diﬃcult to model. Thus the error for the diﬀerent excitation signals will not yield
the same result for the same system, but can only be used in reference to each other.
Another way of describing the error was not investigated, as the relative error gave a
good indication of how the diﬀerent parameters inﬂuenced the approximation. Of course
the single value, does not tell where the error comes from, if it is noise or an oﬀset, it
simply tells how alike the approximation is to the full solution. Thus it is hard to tune
the model using the relative value, as there is no indication of what is actually improved,
when tuning the parameters. It could be the frequency response, with lower ’aliasing’, or
the time response, with a lower oﬀset. Thus for future work it would be interesting to
look at more fulﬁlling ways of describing the error.
6.9
Chapter Summary
In this chapter the harmonic oscillator and loudspeaker models have been used for eval-
uating the performance of the new algorithm. Using the harmonic oscillator, it has been
found that the error of the approximation depends on a number of parameters. The system
it self has an inﬂuence on how well the multi-stage solution performs, as it is a function
of resonance frequency, the fractional order coeﬃcient and the sampling frequency. The
approximation can be improved by increasing the number of samples in each stage and
also the number of stages. Both improvements come with the cost of higher computational
eﬀort, but can be varied to match either high accuracy or faster calculation, so the task
is to ﬁnd the best compromise between the two, for a given application.

7.3
Frequency Domain Measurements
The four loudspeakers are measured and analyzed both in the frequency and time domain.
In the frequency domain the transfer functions of the loudspeakers are used to show the
creep eﬀect of the loudspeaker towards lower frequencies.
The transfer functions of the loudspeakers are measured using the voltage, current and
displacement. The test signal is a pseudorandom noise signal, consisting of logarithmically
spaced sinusoids with random phase. The sines are all played at the same time, thus
exciting the loudspeaker with a broad band of frequencies. As the creep eﬀect is present
at frequencies below the resonance, and with the resonances of the loudspeakers being in
the range of 100-150 Hz, the measurement must go down to very low frequencies, to get
several decades below the resonance. It was chosen to measure from 1 mHz to 18 kHz, as
this range shows both the creep and the eﬀect of the lossy inductance. In order to get a
frequency resolution of 1 mHz, the measurement time is 1000 seconds. Furthermore the
multitone (pseudorandom noise) was played for 30 seconds, before the Fourier transform
is made. This is done to avoid some of the transient from the loudspeaker, but the system
can not be assumed to be completely stationary for all frequencies. More information of
the calculation of the test signal can be found in [4]. The measurements were performed
using a sampling frequency of 50 kHz.

with the parameter not changing signiﬁcantly. However, sometimes the moving mass is
seen to increase at higher levels, as more of the suspension is moving together with the
diaphragm, thus a larger mass. But as these measurements where performed at very low
level, it is deemed that this is not the case. The change in the mass parameter, is deemed
to be the ﬁtting algorithm trying to compensate for the softening suspension at higher
levels, in order to get the resonance frequency right.
As the level increases the compliance, C0 also increases. At the same time the viscous
damping, RMS, decreases. For PSA04 that is the most damped suspension, the damping
is more depending on the level, and changes quite signiﬁcantly. For PSA03, that is the
least damped, the damping also changes, but the starting value is much lower, and the
order of the change is much lower than the change in PSA04. Furthermore, the order
of the fractional compliance, α, rises with higher levels, and the viscoelastic coeﬃcient,
Rα, is decreasing for higher levels. The cause for these changes is deemed to have some
connection to the dry friction in the rubber, which will only be described brieﬂy, as this
has not been investigated thoroughly.

When the rubber is produced, the manufacturer can mix the plastic material, which is used
for the synthetic rubber, with some ﬁlling, called carbon-black. The carbon-black consist
of small particles of carbon, which is mixed into the rubber base before vulcanization.
In Figure 7.10 it is shown how the polymer chains of the elastomer (rubber) and the
carbon-black are side by side, not chemically joined, but connected with cross-links [3].
PSA04, which has the most damped suspension, has a higher amount of carbon-black,
than the PSA03. At low driving levels, the cross-links between the carbon-black and the
polymers are still active, and dry friction is present. At higher levels the cross-links are
softened and the friction is reduced. As there is no friction element in the model used, it
is deemed that the higher damping, RMS, at lower levels is because of the dry friction. At
higher levels the fractional order, α, rises, thus changing the fractional order compliance
to have a higher damping. Recall that the closer α is to zero the more it acts like a spring,
and closer to one it acts like a damper. Because the parameter RMS basically collects
all the damping that can not be explained by the rest of the model, the frictional part
of the rubber is modeled by this.
It is deemed that the viscoelastic damping is more
dominant at the higher levels, as it is the polymers in the rubber that are acting, so the
viscoelastic coeﬃcient, α, rises and the viscous damping, RMS, decreases. A way to model
this frictional behavior is mentioned in Section 8.1.
7.5
Time Domain Measurements
Other viscoelastic models derived for the loudspeaker, e.g. the LOG-model, mentioned in
the introduction, has no direct translation from the frequency domain to the time domain.
This is the advantage of the fractional order model, as the Laplace transformation of the
fractional order derivative exists, Equation 2.6. With the lower computational complexity
of the multi-stage solution, the FO model for the loudspeaker can be calculated faster and
is thus interesting for simulation purposes.
The parameters derived from ﬁtting the model to the frequency response, are used in the
state-space representation, as described in Section 5.1.2, and the discrete-time solution
is calculated using the multi-stage approach. In order to validate that the model is cor-
responding with the actual movement of the loudspeaker, and to compare the model to
the integer order model, a square pulse is applied to the loudspeaker. The square pulse
was chosen as this resembles the DC-shift of the loudspeaker when it is driven into the
non-linear domain.

may have changed the stiﬀness of the suspension. It could also come from the fact that the
loudspeaker, before the measurement, was placed horizontally on a table, and the mea-
surement was performed with the loudspeaker mounted vertically. When the diaphragm
is placed horizontally the gravitation of Earth pulls the diaphragm into the loudspeaker,
with a constant force. This is similar to having a constant DC-oﬀset on the terminals,
which cause a creeping motion of the diaphragm. When the loudspeaker is placed verti-
cally and excited with a signal, the diaphragm moves back to its rest-position. The eﬀect
is seen in the transfer function for the PSA01 in Figure A.1a where some huge deviation
is present at the lower frequencies.
Thus the measurement is aﬀected by the unknown state that the suspension is in before
the measurement. This is aﬀected both by short- and long-term changes. The short-term
changes are softening of the suspension due to high amplitudes. This is often seen as a
change in compliance after the loudspeaker has been excited with a large signal for several
minutes. After a short time, i.e. the same hour, the compliance has often returned to the
value before the large signal was applied. Thus the short-term changes are a reversible
eﬀect. The long-term changes are related to the degradation of the rubber suspension. As
years pass, the suspension can change quite signiﬁcantly, and compared to the short-term
changes, the long-term is often not reversible. Thus the compliance is changed for good.
The biggest inﬂuence on these measurements is the short-term changes. If the measure-
ments are performed before and after a large signal is applied, two very diﬀerent results
could show.
All the measurements are performed only once. Thus no average is performed of the mea-
surements, which could increase the accuracy. For the very low frequencies, i.e. 1-10 mHz,
there are only 1-10 periods of the signal to make the FFT. If there is a slight disturbance in
the 1000 second signal, it will show as deviation in the frequency response. For the higher
frequencies, above 1 Hz, there are lots of periods, which in principle acts like averages. So
even if there is a slight disturbance at one of the periods, all the other periods are alike,
so the disturbance will be suppressed. For better measurements multiple measurements
should be performed at the same level, and averages should be made.
To avoid some of the transients in the loudspeaker, from the start of the measurement,
the signal is present for 30 seconds before the FFT is made. In order to get better transfer
functions it should be considered to run the signal for e.g. a 1000 seconds, so the eﬀect of
the transients is suppressed as much as possible. Another way could be to slowly fade in
the signal, so the beginning of the test signal had no rapid changes.
The wire connecting the resistor and loudspeaker was fairly long, in the range of 2.5 me-
ters. As what was measured as the voltage across the loudspeaker, was actually measured
at the resistor, there were a signiﬁcant diﬀerence between two. The resistance in the wire
was measured to be around 0.3 Ω, which is approximately 10% of the DC-resistance in
the loudspeakers. It was a problem when using the measured voltage signal as input to
the simulation, as the voltage level at the resistor was higher than at the actual voltage at
the terminals. A shorter cable between the resistor and the loudspeaker should be used,
or the voltage should be measured at the terminals, so that the measured voltage actually
corresponds to the voltage delivered to the loudspeaker.

Chapter 8
Conclusion
The main goal of this work was to investigate the possibility of making an eﬃcient al-
gorithm for use with fractional order models for loudspeakers.
An eﬃcient method is
needed in order to use fractional order models in real-time applications and simulations,
as the normal solution has a calculation time that is depending on the number of samples
squared, O(k2). An eﬃcient algorithm based on standard signal processing methods has
been implemented in Matlab. The algorithm is developed around a multi-stage structure,
as this was found to be beneﬁcial due to the characteristic of the fractional order solution.
The eﬃciency is gained from approximating the solution, which means that errors are
introduced, compared to the full solution. The error introduced is described based on the
tuning parameters of the algorithm, but the error was found to be dependent on the given
system.
The multi-stage solution has been designed, so that there are two parameters that control
the accuracy of the approximation. The ﬁrst parameter controls the order of the approx-
imation, and the other controls the amount of time that the system remembers. With a
better approximation and a longer time memory, a low error is achieved, but the calcu-
lation time is also higher. A more coarse approximation and a short memory lowers the
calculation time, but increases the error. Depending on application the two parameters
can be tuned, so that the best compromise between calculation time and error is achieved.
The error from the multi-stage solution was at ﬁrst quite signiﬁcant, as what appeared as
aliasing, was present. It was discovered that this was due to the fact that the decimation
was only performed at certain time stamps. Since the approximation uses multiple stages
at the same time, this gave a ’skewed’ history approximation. By interpolating between
the points, the history is approximated better, as the sample, at which the weighting
function is evaluated, is corresponding better to the value of the history. A signiﬁcant
decrease in error was gained when using interpolation, and is recommended, even though
it has a slightly higher computational complexity.
As part of the multi-stage solution, a ﬁlter is used before decimating the signal. This is
done in order to reduce aliasing. Some comparison of diﬀerent ﬁlter designs has been done,
and it was found that some ﬁlters performed signiﬁcantly better than others. Especially
it was found that the odd order ﬁlter had a lower error than the even, for this application.
The response of the ﬁlter, in terms of ripple in the passband, also has an inﬂuence on
the error. While not completely understood, it seems like having a positive gain at DC,
compared to a negative, results in a lower error. More research needs to been done to fully
conclude if, and why, this is the case.
Four almost identical loudspeakers were supplied by PointSource Acoustics, where the only
diﬀerence between them was the material used for the surround. The loudspeakers were
measured and analyzed. Firstly, they were analyzed in the frequency domain, where their
transfer functions were used to get the parameters for the fractional order model. Here-
==PAGE:85==

75
after, the parameters derived from the frequency analysis were used in the time-domain
using the newly developed algorithm. Using a square pulse, it was possible to evaluate
that the fractional order model predicted the displacement of the loudspeaker better than
the integer order model. However, when the measurements are performed at diﬀerent
levels, the transfer functions change. This is deemed to be due to the fact that the friction
in the rubber is not modeled. Fitting the model to the diﬀerent transfer functions resulted
in changing parameters. Thus the parameters only described the loudspeaker response
at a certain level. This yielded a signiﬁcant diﬀerence when the state-space solution was
compared to the measurements.
The algorithm is developed so that it is possible to include non-linearities, as a state-
space representation is used. The matrices of the system can be updated depending on
the state of the loudspeaker, and can thus include displacement and current dependent
non-linearities. The model that is used to estimate the non-linearities should also be used
to simulate them. As e.g. the Klippel R&D system uses an integer order model to ﬁt
the parameters to the measurements, this can not be directly translated to the fractional
order model. Thus a method for estimating the non-linearities using the fractional order
model should be developed ﬁrst.
8.1
Future Work
The work presented in this thesis is a step towards using fractional order models in loud-
speaker modeling and compensation. Below are some ideas for further development of the
model and the algorithm.
 The model for the rubber suspension should be expanded to be amplitude dependent,
in order to fully model the loudspeaker. A possibility is adding a number of frictional
elements as suggested in [3].
 The non-linearities of the loudspeaker needs to be updated with the fractional order
model, in order to properly estimate the parameters, which could be used for feed-
forward compensation.
 The algorithm is currently only implemented in Matlab.
In order to see if it is
possible to actually run in real-time it should be implemented on a microprocessor.
As the sum is consisting of a large number of independent multiplications, it might
be beneﬁcial to implement it on a GPU with multiple cores.
 The interpolation that is used in the multi-stage solution is of ﬁrst order. It could
be investigated if a higher order would decrease the error signiﬁcantly compared to
the added complexity.
 The decimation is always performed with a factor of two. It might be beneﬁcial to
have higher orders of decimation for history that is further into the past. It could
also be that for certain systems a general higher decimation factor could be used, or
else an adaptive factor, compared to e.g. the shape of the weighting function.
 The inﬂuence of the ﬁlter is not fully understood in terms of the frequency response.
A more thorough investigation is needed to fully understand this. Other types of
ﬁlters could also be investigated.

Appendix B
MATLAB Code
In this appendix the implemented Matlab code is shown. The measurement script is left
out, as it is the same as found in [4]. The procedure is
 Measure the loudspeaker using a multi-tone signal and derive the transfer functions.
 Load the transfer functions into the ﬁtting algorithm script, where the fmincon will
minimize the error between the model and the transfer functions.
 Used the derived parameters from the ﬁtting algorithm to set up the system matrices
in the script.
 Use the multi-stage solver to calculate the state-space solution.
Tip: When the desired number of stages, number of samples have been found and the
ﬁlter have been found, it can decrease the computation time, quite signiﬁcantly if a MEX
ﬁle is compiled by Matlab. Instead of deﬁning eta, fRes, fs and b outside the function,
they can be included in the function. When the MEX ﬁle is compiled, an example of
the improvement is what took 22 seconds with the function, took 2.8 seconds with the
compiled function. This was a 3 second signal with 50 kHz sample rate.
