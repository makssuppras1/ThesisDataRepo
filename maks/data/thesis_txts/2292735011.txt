Abstract
DTU Fotonik: Department of Photonics Engineering

MsC in Telecommunication
Heterogeneous Optical Fiber Interfaces for Short Range Data
Communication
by Peter Madsen
This thesis evolves around the problematics of limited capacity in Multimode Fiber
(MMF) links within data centers.
When upgrading an optical link from 10Gbps to
40Gbps or higher, it eventually requires an overhaul in the ﬁber structure. An over-
haul which is costly for the network managers. The thesis conducts an investigation of
available commercial products which potentially solves this problem by using transceiver
modules which allows for the use both MMF and Single Mode Fiber (SMF) ﬁber link
connections. This gives a smooth transition from 10Gbps to 40Gbps and lowers the cost
of future ﬁber upgrades. The objective of the thesis is to provide a simpliﬁed solution
by interconnecting MMF and SMF without the use of a specialized commercial product.
Thus allowing the network managers to upgrade the transceiver modules to ones that
are optimized for SMF ﬁber connections, while still running the transmission on MMF
ﬁber links. We se through the experimental results that, it is indeed possible to intercon-
nect MMF and SMF by using standard Ferruler Connector / Physical Contact (FC/PC)
connectors and reach an error free transmission with a Bit Error Rate (BER) of 10−12.
But stability becomes an issue at lower levels of receiver power. This instability can be
limited by reaching Equilibrium Modal Distribution (EMD) before the interconnection.

Chapter 1
Introduction
1.1
Problem Statement
For many years, the basic ﬁber structure in most applications have been consisting pri-
marily of MMF. The reason for this, is that the MMF was the ﬁrst ﬁber type ever
produced and historically MMF transceiver links has been cheaper than SMF due to
the more relaxed manufacturing constrains and light sources for 850nm have been em-
ployed in these links for more years. The increased ﬁber bandwidth and transmission
distance gained from using SMF have made this the preferred ﬁber type for many newer
applications[1].
Figure 1.1: IP traﬃc forecast made by Cisco [2]
1
==PAGE:12==

The Internet Protocol (IP) traﬃc forecast seen in ﬁgure 1.1 states that by 2018 the
total IP traﬃc will be as high as 135.5EB per month, and growing to 168EB by 2019.
This puts an extreme pressure on network managers around the world, to make sure
that their networks can handle this kind of traﬃc. In many cases, optical links in the
network that consists of MMF will be limiting the performance of the network. Thus
network managers are forced to take actions to increase the capacity of the network links
if they want to cope with the increased traﬃc.
Considering the ﬁber structure in datacenters, changing the ﬁber link between servers
can be an almost impossible challenge. When a datacenter is faced with the dilemma of
upgrading internal transmission speeds from 10Gbps to 40Gbps in order to overcome the
massive traﬃc growth, they are at the same time facing an expensive task. Not only do
they have to upgrade the interface of the servers to 40Gbps compatible pluggable optical
transceivers like the Cisco 40GBASE Module[3] but they also have to consider the ﬁber
link between the servers. Does the MMF between the servers has enough bandwidth for
the given distance between the servers? If not, simple solutions to this problem are ei-
ther to shorten the distance between the servers, add multiple MMF ﬁber links between
servers or the use a SMF link instead. Changing the ﬁber type used in the link between
servers also means using SMF compatible 40Gbps pluggable optical transceivers at the
servers interface. Either way, both solutions requires changes to the ﬁber structure and
increased costs. Sometimes this ﬁber link expenditure is inevitable like many other costs
in a datacenter, but unlike other upgrades to the datacenter, which can often be divided
into multiple cost expenditures, the ﬁber link have to be considered if the current ﬁber
link is not supported by the 40GBASE standard[4]. For example if the MMF ﬁber band-
width length product is enough to support a 40Gbps transmission, usually the server
interfaces are upgraded at diﬀerent times to lower the overall expenditure at the given
point[5]. If the ﬁber link does not support a 40Gbps transmission, the ﬁber link have
to be upgraded along with the server interface, thus the increased cost. Now if a homo-
geneous optical ﬁber interface could be used between MMF and SMF, the expenditure
of upgrading the ﬁber link can be divided from the expenditure of upgrading the server
interface or, in a best case scenario, it could be delayed for some years. This is why this
thesis is devoted at investigating the feasibility of a homogeneous optical ﬁber interface
for short range data communication.

1.2
Methodology
The methodology of this thesis is mainly based upon experimental validation, because
the theory for ﬁber propagation provided by Maxwell’s equations is depending heavily
on physics rather than telecommunication. Thus providing an answer to how each mode
propagates through a speciﬁc piece of ﬁber requires a deep background into physics.
The experiments simulate a single transmission link in a Quad Small Form-factor Plug-
gable (QSFP) 40Gbps transceiver. The main objective is to investigates the feasibility of
coupling from MMF to SMF, by the use of standard FC/PC connection interface as well
as a prototype mode coupler and a mode ﬁlter. The evaluation is done on a system level
by analyzing the overall link performance, for diﬀerent lengths and types of ﬁbers, by
using BER measurements. The experiments also include an investigation of the FC/PC
coupling performance while using PRBS sequences with length of 7 and 31.
1.3
State of the Art
The problematics involved in upgrading datacenters have seen huge interest, from equip-
ment suppliers such as Arista, Corning, Finisar and Mellanox.
These companies in
particular have been working with the problem of upgrading the ﬁber structure in dat-
acenters. This interest have resulted in two commercial products, one from Finisar[6]
and one from a collaboration between Corning and Arista [7]. Both products claims to
be 40G QSFP universal transeiver modules in the way, that they support transmissions
over both MMF and SMF up to a certain length. Both modules uses Wavelength Di-
vision Multiplexing (WDM) transmission, where the modulated output from four Light
Ampliﬁcation by Stimulated Emission of Radiation (LASER)s, at four diﬀerent wave-
lengths in the 1310nm region, are multiplexed into one ﬁber. The receiver QSFP module
demultiplexes the received optical signal into four Photo Diode (PD)s with matching
wavelengths.

Chapter 2
Theory
2.1
Fibers
The optical ﬁber is one of the central parts in a modern datacenter. There is a constant
development in diﬀerent ﬁber types, always trying to improve the ﬁber characteristics
for speciﬁc applications and systems.
But if one boils the optical ﬁber down to its
essentials, all ﬁbers can then be divided into two main categories. Fiber with multiple
optical modes and ﬁbers with only a single optical mode, namely MMF and SMF ﬁbers.
The general concept of optical ﬁbers is most easily described through what is called the
geometric optics.
In general all ﬁbers consist of a core and a cladding made out of silica. The core and the
cladding is structured such that they have diﬀerent refractive indexes. This is needed in
order to ”trap” or guide light inside the ﬁber.

2.1.1
Multimode Fiber (MMF)
MMF is a ﬁber type which has a relatively large core diameter (50-62.5µm) compared
to the overall diameter of the ﬁber (125µm). This allows for an easy coupling of light
into the ﬁber, which means that MMF is often used together with simple light sources
such as an Light Emitting Diode (LED) or transmitters where the coupling angle and
alignment is less critical. On the other hand, due to this larger core size the MMF allows
for multiple optical modes to propagate in the ﬁber. This brings the downside of MMF,
which is the limitation in data rates and transmission distance due to modal dispersion,
a phenomenon caused by these multiple optical modes.
2.1.1.1
Optical Modes
A mode inside a ﬁber corresponds to speciﬁc solutions to the wave equation which is
derived from Maxwell’s equations.
Diﬀerent ﬁber modes can have diﬀerent possible
propagation paths inside the ﬁber.

2.1.2
Single Mode Fiber (SMF)
SMF is a ﬁber type which is limited to only guide one optical mode, the fundamental
mode HE11 (also known as mode 01 in ﬁgure 2.2). When the ﬁber only allows one mode
there is no modal dispersion in the ﬁber and this increases the bandwidth of the ﬁber. If
a ﬁber needs to only supports the fundamental mode, it forces a constraint on the value
of the normalized frequency V < 2.405. Looking at equation 2.1, it is seen that the
constraint can be met, by either changing the radius of the ﬁber core or the wavelength.
The refractive index of the ﬁber core is optimized for clear silica which minimizes the
degradation of light in the ﬁber. This in turn limits the refractive index change at the
core-cladding interface. The refractive index change ∆has to be as small as possible in
order to make a ﬁber which only supports the fundamental mode (see section 2.8).
There also exists a limit to how small the core can be made before the margin of errors
in the production becomes unattainable. The smallest core radius that have been pro-
duced is about 3µm but they are normally kept at 4−4.5µm to allow some imperfection
in production [8].
In order to keep the V < 2.405 the wavelength has to be suﬃciently long. The usual
SMF will become single mode around a wavelength of 1200nm and above.

Optical systems are evaluated relative to their power-/loss budget. The loss budget is
calculated from the combined loss from couplings, splices or in general any optical con-

2.2.2
Intrinsic Loss
Intrinsic loss comes from manufacturing properties inside the ﬁber. Such as mismatch
in ﬁber core diameter and mismatch in ﬁber index proﬁle.
2.2.2.1
Mismatch in Fiber Core Diameter
It is known from the basic ﬁber theory, that the single mode conditions of a ﬁber is
based on both core diameter and wavelength of the source (see section 2.1.2). Thus
changing one of these two conditions will break the single mode condition and more
modes may be able to propagate through the ﬁber.
This makes it clear that when
coupling ﬁber, a mismatch in the core diameter of the two ﬁbers when coupling them,
will allow for either fewer or more modes. Depending on the mode types the modes may
be attenuated, other modes may cause dispersion in the ﬁber. All leads to a form of
performance loss in connection.
2.2.2.2
Mismatch in Fiber Index Proﬁle
As with mismatch in the ﬁber core diameter, a mismatch in the ﬁber index-proﬁle will
allow for either fewer or more modes. This is due to the allowed propagations way of
the light inside the ﬁber.

2.2.5.1
MPD, EMD and Launch Conditions
MPD aﬀects the insertion loss and bandwidth measured in couplings and connections
in MMF transmission systems. MPD, Launch conditions and ﬁber length are heavily
interconnected. At the initial state of a MMF transmission system, the launch conditions
determines the initial MPD in the ﬁber. The launch condition is characterized as either
being under-ﬁlled or overﬁlled. The names refer to how the optical power is ﬁlling the
core of the ﬁber, and is dependent on the optical source and the intrinsic- and extrinsic
factors between the optical source and the ﬁber.
(a) Underﬁlled launch condition
(b) Overﬁlled launch condition
Figure 2.11: Common launch conditions for MMF[20]
As seen in ﬁgure 2.11a if the source has a small spot size (such as a LASER or VCSEL)
and the angular distribution is smaller than the ﬁber core diameter, the launch condition
is called under-ﬁlled. If the source has a large spot size (such as an LED) as seen in
ﬁgure 2.11b and the angular distribution is larger than the ﬁber core diameter, the
launch condition is called overﬁlled. An under-ﬁlled ﬁber has more power in the lower
order modes than higher order modes. Lower order modes travels closest to the core
where the propagation path is shorter. Higher order mode travel closest to the core-
cladding interface and have a longer propagation path than lower order modes. Due to
this, higher order modes have a greater risk of being scattered or absorbed. An overﬁlled
ﬁber will have more power in all order of modes throughout transmission, but at the
cost of launch power lost due to mismatches between ﬁber and source [17]. An overﬁlled
ﬁber will have an even MPD in the core and is often called uniform.
When measuring the insertion loss of either under-ﬁlled or overﬁlled ﬁbers, the loss will

QSFP is a development from the Small Form-factor Pluggable (SFP) in the sense that it

Lucent Connector (LC) connector or an MPO connector. This determined by the link
capacity and reach (LR or SR).

2.5
Wavelength Division Multiplexing (WDM)
WDM is a transmission method used in optical systems to better utilize the bandwidth
of the optical bands seen in ﬁgure 2.4. Each band covers a set of wavelengths, much like
frequency bands in Radio Frequency (RF) communication. An optical system is speciﬁed
to operate within a certain band, dependent on the wavelength of optical transmitter.
To make the transmission system work, the receiver needs to be within wavelength range
of the transmitter. The ﬁber used in optical systems has almost the same behavior for
multiple wavelengths within the optical band. Thus multiple transmitters can transmit
to multiple receivers through only one ﬁber as seen in ﬁgure 2.16.
Figure 2.16: System with multiple transmitters and receivers using WMD
In systems using WDM each transmitter and receiver pair must use diﬀerent wave-
lengths. The number of transceiver pairs a WDM systems can have, is limited by the
system’s ability to eliminate noise from crosstalk and channel interference.
With as
little as 50GHz channel spacing it is possible to provide a WDM system where up to 64
transceivers, transfers data at 10Gbps, up to a distances of 20km on a single SSMF[25].
To compare this to the channel spacing of the QSFP module. If we use wavelength of
1271nm from table 1.1 and a 50Ghz spacing, the channels will be at 1271nm, 1270.7nm,
1270.5nm and 1270.2nm. This is a much lower channel spacing then the 20nm spacing
used QSFP modules.

5.1
Conclusion
The experiments conducted as part of the thesis show that, with a standard FC/PC
connector we are able to provide a possible solution for the problem statement1.1. This
solution copes with the state of the art, which is the universal QSFP transceiver that
transmits 40Gbps PRBS31 over four WDM channels. It does so error free with a receiver
sensitivity of −10dBm over a distance of 160m with OM4 and 1km with SSMF. The
experiments show that obtaining EMD before coupling into SMF adds extra stability
meaning better receiver sensitivity of plus 2dB for a BER at 2 on the −log(BER) scale.
Due to limited knowledge about the prototype mode coupler, we cannot conclude if a
ﬁber tapering is a possible solution to coupling between MMF and SMF. Though we
are able to conclude that the performance of the prototype mode coupler is similar to
that of a mandrel mode ﬁlter for receiver powers below −20dBm, and equal to the
performance of standard FC/PC connectors at receiver powers higher than −17dBm for
the number of bits transmitted. We have shown that with the tested solutions, we are
able to transmit a 10Gbps PRBS31 up to 2 times the distance of that of the state of
the art with OM4 and over 5 times the distance with SSMF. By using WDM of four
channels we are able to provide a similar 40GBASE QSFP transmitter as the state of
the art, but with added extra transmission distance, which allows for larger scale use
and an expansion of the data center. An expansion of the data center is not possible
with the current state of the art, as it only allows 160m OM4 and 1km SSMF.
Any instability from the extra modes from WDM with wavelengths below 1299nm can

5.2
Future Work
Although the thesis is able to bring solutions that are able to comply with the state of
the art, there are still some things to be investigated in the future, before the solution
can be truly implemented.
In the experiments, only FC/PC and FC/APC connectors was tested, but many of the
other connector types may be used in data centers, which is why a test of possible con-
nectors should be done.
The instability of the system seen in the experiments is clearly an eﬀect of coupling
between MMF and SMF, but this could maybe be avoided by using a larger PD with
MMF pigtail.
In terms of range extension between servers in a data center. It could be worth inves-
tigating the possibility of adding extra SMF on-top of an existing MMF link using the
solutions provided by the thesis, to see if there is any performance changes.
Lastly this thesis evolves around the 40GBASE standard, and already IEEE has ap-
proved the 100GBASE standard with even higher link capacity. Therefor is would be
interesting to upgrade the DFB so it can be modulated at 25Gbps, and see if the solutions
provided still holds.
