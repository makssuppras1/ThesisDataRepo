Introduction
Biological networks is a way of representing relationships and interactions between different
biological entities.[3] When modellers work with biological systems the first available informa-
tion they have of the networks is their directed graphs. This graph representation contains a
set of nodes that are connected by a set of edges and captures the most reliable information
available about the biological system. These directed graphs are constructed empirically, how-
ever techniques for their automatic inference through data does also exist. Based on this. they
further construct explicit models to get mechanistic details; famous models are ordinary dif-
ferential equations (ODEs), Boolean networks (BNs), chemical reaction networks (CRNs) etc.
Among these, BNs are particularly interesting due to their simplicity in representing biological
networks[4].
A BN consists of a set of Boolean variables that denotes ON/OFF behaviour
between nodes, with each Boolean variable being updated according to an update function
encoded as a logical rule.
However BNs suffer from an issue that affects most Biological network representations,
namely the state space explosion. This happens since there are exponentially many states with
respect to the number of Boolean variables. This causes issues with practical analysis of BNs as
the state space explosion can lead to situations where graph analysis is too resource and time
costly to perform. There are methods to combat the issue of state space explosion, particularly
reduction algorithms to reduce the state space of BNs is useful here.
One such reduction
method is BBE reduction[5], a reduction method that identifies maximal subsets of variables
which retain the same value, whenever they obtain the same value (or initialised equally).
As covered a modeller will start with the biological networks represented as a directed graph,
and from there transform it into a more compact representation such as BNs. In this study
we will reverse-engineer this process. As we collect the qualitative models from the GINsim[1]
and BIOmodels[2] repositories, the qualitative models will be transformed back to their graph
representation. This is done for a multitude of reasons, firstly is that we wish to compute
structural graph metrics on the networks, which is best done on their state transition graphs.
We also wish to see how the metrics correlate with the reduction ratio of BBE and how they
vary after BBE reduction.
In Section 2, we give the preliminaries of Boolean networks and how they can be represented
as directed graphs. In Section 3, we present the toolchain that will be used to perform the
investigations needed in the paper, and we also give preliminaries for the computations. In
Section 4 a graph theoretic study of the qualitative models found on the GINsim and BIOmodels
repository will be performed. In Section 5, a study of the correlation between the graph metrics
of the Boolean networks and the reduction ratios of the BBE reductions will be carried out.
In Section 6, a study on the effects the reduction has had on the network structures will be
conducted. In Section 7, the paper will be concluded and potential future works will be briefly
discussed.

Boolean Networks
There is a variety of models and equations that biologists use to represent biological networks.
When working with a biological network the first information a modeller has about the system is
what is known as the interaction graph. This graph consists of nodes and directed edges where
the nodes denote proteins, genes, molecules etc. while the directed edges denote an effect such
as activation, promotion, deactivation, inhibition or repression. Formally an interaction graph
is defined as follows:
Definition 2.1. An interaction graph G is a pair (V,E), that consists of a set of vertices
V = {v1, ..., vn} and a set of edges E ⊆V × V .
However, working with a interaction graph in this format does not allow a modeller to look
into the underlying system of the biological network. Yet, by expanding on the concept of the
interaction graph it is possible for modellers to build more precise models, such as systems of
differential equations, difference equations, chemical reaction networks, stochastic equations,
etc. In recent years the representation know as Boolean Networks(BNs)[4] have gotten interest
due to its simplicity.
A BN consists of a set of Boolean variables meant to denote ON/OFF behaviour between
nodes. Each BN variable is updated according to an update function encoded as a logical rule.
Formally a BN is defined as follows:
Definition 2.2. A BN is a pair (X, F) with X = {x1, x2, . . . , xn} being a set of variables and
F = {f1, f2, . . . , fn} being a set of update functions.
To expand on BNs, an example will be defined. We define a Boolean as the pair (X, F)
with X = {A, B, C} being the set of variables, and F = {A′, B′, C′} being the set of update
functions. The update functions defined as follows:
A′ = B ∧C
B′ = A
C′ = ¬A
Where the update function for A’ and B’ represents activation, and the update function for C’
represents deactivation. To represent the graph there are a few options, an informal represen-
tation could be used to better visualize the update functions. Such as seen in figure 1.

State Space Explosion.
The total possible size of a State Transition Graph created from
a BN is 2n with n being the number of variables in the BN. The size of the STG grows ex-
ponentially with the number of BN variables. This makes BNs vulnerable to a concept known
as the State Space Explosion. The State Space Explosion occurs when the state space grows
at a fast rate . This leads to a situation, where exploring the entire state space is essentially
impossible with limited time and memory resources. The issue occurs in all types of networks
where the state space grows exponentially such as BNs. To present this growth we can again
consider the example BN defined in section 2. The BN contained the 3 variables X = {A, B, C}
so the size of the network will be 2n = 23 = 8. Now pretend that instead of 3 variables, the
BN contained 10 variables. The size would now be defined as 2n = 210 = 1024. This should
shine a light on why the issue of State Space Explosion is a problem when seeking to analyze a
state space. Even though this concept is troubling, there are methods to mitigate it. One such
method is to reduce the size of the state space to make analysis on larger networks more feasible.

Reduction algorithm
Several reduction techniques have been proposed in the literature [6, 5, 7] to attack the state
space explosion. Here, we focus on Boolean Backward Equivalence; a reduction method that
identifies maximal subsets of variables which retain the same value, whenever they obtain the
same value (or initialised equally).

Format Converter.
Firstly the Boolean networks are converted to a .BNET format. The .BNET format is a BoolNet
format that stores a list of functions in a text file. These functions are composed of the variables
and the update rules. If we consider the example of a Boolean network proposed in section 2.
Here we had 3 variables with 3 update functions, one of these being the variable A with the
update function A = B ∧C. This would be represented in a .BNET file as the line: A, A &
B. Each variable and update function pair has their own line in the file. The format converter
is based on the COLOMOTO Notebook [8]; a unified environment to edit, execute, share, and
reproduce analyses of qualitative models of biological networks. The models stored in a .BNET
format are now ready to be reduced by the ERODE [5] tool.
3.3
ERODE.
The ERODE[5] tool will reduce the models with both maximal BBE reduction and ID BBE
reduction. ERODE[5] permits the reduction of a large amount of models at once, and stores in
a .csv file the reduction magnitude of each model.

Freeman centralization.
The centrality measure is used in graph theory and network anal-
ysis as a way to indicate the importance of a specific node to the network as a whole. The
higher the value of centrality of a node the more influence it has on the network. On top of

this, there are many different methods and variations of calculating a nodes centrality such
as degree centrality, closeness centrality and betweenness centrality. One issue with centrality
measures is that they are calculated on individual nodes in the network. The Freeman central-

value is then divided by the theoretically largest such sum of differences in any network of the
same size. The theoretically largest sum, can then be considered as the maximal value that the
centrality of the networks is compared to.

Degree centralization.
We first examine degree centralization [12], which conceptually is
the simplest of the centralization measures. Degree centrality is defined as the number of ties a
node has, i.e. how many edges the node has connections to. Concerning directed graphs such
as BN, degree centrality is split up into 2 separate measures of degree centrality, these being

Correlation.
A correlation coefficient measures the strength of the relationship between 2
different variables.
The correlation is an indication that when one of these variables see a
change in value, the other variable will generally see a change in a specific direction. Knowing
the correlation coefficient between two variables is very important as it allows one to make
predictions about one variable based on the other. Two types of correlation that is often used
is Pearson[16] and Spearman correlation[17].
Pearson.
Pearson correlation is a linear correlation and, as such, it is computed as the ratio
between covariance of two variables and the product of their standard deviations. The corre-
lation coefficient will always be between -1 and 1. -1 denotes a strong negative relationship
and 1 denotes a strong positive relationship. To further expand, a correlation coefficient of -1
indicates that for every positive change in one variable, there is a negative change of a fixed
proportion in the other variable. For a correlation of 1 the opposite holds, with a positive
change leading to a positive change of fixed proportion. A correlation coefficient of 0 means
that when one variable changes there is no change in the other, as such they have no relationship.
Spearman.
Spearman is the non-linear correlation, where instead of describing the rela-
tionship according to a line it is done using a monotonic function. The Spearman coefficient
measures correlation in the same manner, spanning from -1, a negative relationship, to 1 a
positive relationship.
P-value.
When working with any type of correlation it is important to consider if the cor-
relation is a coincidence rather than an actual correlation. This is done by proposing a null
hypothesis, that says there is no statistical significance between 2 variables. From which the p-
value is computed as the probability of obtaining results that supports the null hypothesis.[18]
Therefore when computing the correlation coefficient between two variables, it is also necessary
to compute the p-value between them. Since the p-value is used to compute the probability
of the null hypothesis being correct, a lower p-value will show greater statistical significance
between 2 variables. A p-value of less than 0.05 is generally the standard of being considered
statistically significant, as there is less than 5% chance of the null hypothesis being correct.
However different standards for p-value thresholds exists, therefore a p-value of less than 0.1
can in some cases be considered statistically significant as well.

From here we define the potential alternative hypothesis. Namely that there is a correlation
between some of the metrics and the size reduction of the BNs following a reduction.

Interpretation
We observe that the density has seen a general increase and that the larger the reduction, the
larger the increase in the density. Degree centralization[12, 11] has seen both an increase after
max reduction and a decrease, after ID reduction. For ID reduced graphs this means that edge
connections between nodes will be more evenly spread out than before reduction. As there
is less overall direct connection into and out of the central node of the networks. For max
reduced graphs the opposite is the case, where the edge connections now will be more focused
around connecting to a central node as compared to before the models was reduced. Closeness
centralization[13, 11] has seen a general decline in value after both reduction methods. This
decline however is expected, because as density increases, the closeness centrality of the central
node starts reducing while the closeness centrality of the non-central nodes starts increasing.
As such a general decline in closeness centralization is an indication that nodes in general are
closer to each other. Betweenness centralization[14, 11] has seen an increase in value after both
reductions. So the flow of the network is now more focused through a central node than was
previously the case. Average clustering[15] across the network has also seen a general increase,
this increase was however expected. As a higher average clustering means that the networks
has a more modular structure, and that they have a smaller possible average distance among
the different nodes. As such an increase in average clustering is an indication that the network
has become more dense and therefore forms more into clusters than previously. Some of the
metrics were more affected by the reduction than others. But a common factor across all the
reductions is that the smaller the reduction ratio is, the larger the change in metric value is
after the reduction.

7
Conclusion
In this thesis, the concept of Boolean networks[4] has been discussed, and basic representations
of Boolean networks have been shown. Furthermore, the issues of Boolean networks have been
presented and methods to solve these through reduction algorithms have been referenced. We
implemented a prototype toolchain to analyze and investigate the effect that the graph struc-
tures have on the reduction algorithms. This toolchain allows for users to convert the models to
formats for reduction and graph analysis. By exploiting the toolchain, we performed statistical
analysis has been proposed to present the graph and reduction data obtained from previous
parts of the toolchain in a manner that allows for discussions of graph structures, correlations
and comparisons.
Through usage of this toolchain a graph theoretical study has been performed on the qualitative
models obtained from the GINsim[1] and BIOmodels[2] repositories. This study has shown that
the qualitative models are small in size, and decently centralized with the nodes being located
close to each other making traversal from one node to any other in the interaction graph fast.
An investigation of the correlation between the graph measures and the BBE[5] reductions has
also been performed. This has shown that there is a clear correlation between the betweenness
centralization and the reduction ratios for both ID and max reduction. It has further shown
a weaker correlation between both degree centralization, and closeness centralization with the
reduction ratios.
Furthermore, an analysis of the effects on reduction on the network structure has been per-
formed. This investigation has shown a big increase in density across the BNs after reduction,
while betweenness centralization, clustering, and degree centralization has shown a smaller in-
crease. The only metric to see a general decrease in value is closeness centralization.
