Abstract
Maximal function fields are important to coding theory as they naturally allow to
construct the longest possible AG codes over a finite field Fq. The Giulietti-Korchmáros
(GK) function field was the first example of a maximal function field over a finite field
proven not to be isomorphic to a subfield of the Hermitian function field over the same base
field. This sparked interest in determining other examples of such function fields. In this
thesis, we study two known generalizations of the GK function field, one generalization by
Garcia, Güneri and Stichtenoth (GGS), and another by Beelen and Montanucci (BM). The
covering result by Serre manifest a direct technique in determining new maximal function
fields, by constructing subfields of known maximal function fields.
Two such function
fields, Yn,s and Xa,b,n,s, subfields of the GGS function fields were studied by Tafazolian,
Teherán-Herrera and Torres. We study the analogs of Yn,s and Xa,b,n,s for the BM function
fields, denoted ˜Yn,s and ˜
Xa,b,n,s, and show analogous results. We prove that the function
fields ˜Yn,s (resp.
˜
Xa,b,n,s) are not isomorphic to subfields of the Hermitian function field
for n = 3 and q > s(s + 1) (resp. n = 3 and s = 1 and not Galois subfield for s = 1). We
determine a subgroup of both Aut( ˜Yn,s) and Aut( ˜
Xa,b,n,s). In the case of ˜Yn,s, we use this
subgroup to prove that Yn,s and ˜Yn,s are not isomorphic. This shows that the difference
between the GGS function fields and the BM function fields also holds at the level of the
subfields Yn,s and ˜Yn,s.

Introduction
A function field F/Fq over the finite field Fq is of interest in coding theory due to the
possibility of explicitly constructing linear codes by evaluating functions of F at rational
places. The number of rational places N(F) of F is bounded by the Hasse-Weil bound [1].
1 + q −2g(F)√q ≤N(F) ≤1 + q + 2g(F)√q.
A function field that attains the upper Hasse-Weil bound is called maximal, for which the
prime power q is necessarily a square unless g(F) = 0. A well-known maximal function
field is the Hermitian function field Hq.
Hq = Fq2(x, y) :
yq+1 = xq + x.
The theory of function field extensions is a powerful tool in the study of function fields.
A result commonly attributed to Serre states that if a function field F/Fq is a subfield of
the function field F ′/Fq, then F is itself maximal [2]. It was speculated that all maximal
function fields F/Fq2 are isomorphic to subfields of the Hermitian function field over the
same base field Fq2. This conjecture was proven to be false by Giulietti and Korchmáros
in 2009 when they constructed a function field C3 over Fq6, called the GK function field,
and proved C3 is not isomorphic to a subfield of the Hermitian function field over Fq6 [3].
C3 = Fq6(x, y, z) :
yq+1 = xq + x,
zq2−q+1 = yq2 −y.
The GK function field has been generalized in two ways to two different families of
maximal function fields over Fq2n for n ≥3 odd. One generalization by Garcia, Güneri
and Stichtenoth, called the GGS function fields [4], and another generalization by Beelen
and Montanucci, called the BM function fields [5]. In this thesis we formally introduce
all above mentioned function fields and show the maximality of the GK function field and
the BM function fields.
An often used technique to construct new maximal function fields is to consider sub-
fields of known maximal function fields. These can be Galois subfields arising as the fixed
fields of subgroups of the automorphism group of the known maximal function field. Thus,
determining the automorphism group of a maximal function field is an useful advantage,
also in answering if a function field is isomorphic to other known function fields. In the
thesis we explicitly construct the full automorphism groups of the GGS function fields.
Two families of subfields, denoted Yn,s and Xa,b,n,s, of the GGS function fields were
studied in a paper by Tafazolian, Teherán-Herrera and Torres [6]. They arise as the fixed
fields of subgroups of the automorphism groups of the GGS function fields. The aim of this
thesis is to come up with analogous subfields, ˜Yn,s and ˜
Xa,b,n,s, for the BM function fields.
We determine subgroups of both Aut( ˜Yn,s) and Aut( ˜
Xa,b,n,s), the former by ’lifting’ a
maximal subgroup Mℓof Aut(Hq) and the latter by computing the normalizer of the fixing
group of ˜
Xa,b,n,s in a subgroup of Aut( ˜Yn,s). For the subgroup of Aut( ˜Yn,s) we conjecture
that we in fact have constructed the full automorphism group Aut( ˜Yn,s). Furthermore, we
show Yn,s and ˜Yn,s are not isomorphic for most choices of parameters n and s. In fact, we
show Yn,s and ˜Yn,s are isomorphic if and only if 3 | n and (qn +1)/(s(q +1)) | (q2 −q +1).

2.2
Function fields
Let K denote a field. We will usually suppose K is a finite field, but for now we let K
denote an arbitrary perfect field.
Definition 2.2.1. An algebraic function field F/K of one variable over K is a field
extension F ⊇K such that F is a finite extension of K(x) where x ∈F is some element
transcendental over K.
In this section we outline fundamental properties of function fields and extension of
function fields. An introductory text on function fields is found in [1], from which the
author started studying the subject. The above definition and the following results can
all be found in [1].

Maximal function fields
Algebraic function fields being interesting in their own right, are also appealing due
to their application in coding theory [9]. Chapter 2 in [1] gives a brief introduction to
how codes can be constructed from a function field F/K, the key idea being evaluating
a certain vector-space of functions at chosen rational places. Finite fields in general are
interesting to coding theory due to the discrete nature of communication over computer
networks. So when the base field K is assumed to be a finite field Fq, we obtain a suitable
framework for constructing linear error-correcting codes, namely algebraic-geometry codes
(AG codes). Some of the interesting properties of AG codes are the ability to explicitly
construct AG codes, decode the codes eﬀiciently and the codes admit good bounds on
their parameters [10].
A major idea in constructing AG codes is to evaluate functions at chosen rational
places. A famous result known as the Hasse-Weil bound states an upper and lower bound
on the number of rational places of a function field over a finite field. This bound is given
by the size of the finite field and the genus of the function field. A function field that
attains the upper Hasse-Weil bound is called maximal.
In this section, we present the Hasse-Weil Theorem and formally define maximal func-
tion fields. We then show the maximality of a few known maximal function fields, namely
the Hermitian function fields, the GK function field and the family of BM function fields.
3.1
The Hasse-Weil bound
A first step in presenting an explicit bound on the number of rational places of a
function field F/Fq is showing there simply must be a finite number of rational places.
More generally, there is a finite number of places of a certain degree ℓ.
Lemma 3.1.1. Let F/Fq be a function field over a finite field. For any ℓ≥1, there is
only a finite amount of places of degree ℓ.
Proof. It is suﬀicient to prove S = {P ∈PF | deg P ≤ℓ} is a finite set. Let x ∈F \ Fq
be a transcendental element over Fq and consider the rational function field Fq(x)/Fq and
the set S0 = {P0 ∈PFq(x) | deg P0 ≤ℓ}. For any P ∈S let P0 = P ∩Fq(x). Because F
and Fq(x) are function fields over the same constant field Fq we have deg P0 ≤deg P ≤ℓ.
Hence P0 ∈S0. There can only be a finite number of places lying over P0 and any place
P ∈S must lie over some P0 ∈S0. Thus it follows |S| is finite if we can show |S0| is finite.
Except from the pole of x, the places of Fq(x) are in one-to-one correspondence with the
monic irreducible polynomials over Fq. The degree of the place corresponds to the degree
of the irreducible polynomial. As {f(T) ∈Fq[T] | deg f ≤ℓ} is a finite set, it follows S0 is
a finite set.
A divisor A ∈Div(F) is called effective if all its coeﬀicients are non-negative and
denoted A ≥0. As a corollary of Lemma 3.1.1, we can tell there is a finite amount of
effective divisors of degree ℓ≥1. Define
An := |{A ∈Div(F) | A ≥0 and deg A = n}| < ∞.

Definition 3.1.8. A function field F/Fq is called maximal if N(F) attains the upper
Hasse-Weil bound, i.e. if N(F) = q + 1 + 2g√q.
Notice that for a maximal function field F/Fq, the size of the finite field, q, must

the upper Hasse-Weil bound becomes q +1, and we are always able to count q +1 rational
places of any rational function field Fq(x)/Fq. We have the q places (x = α) for α ∈Fq
and the place at infinity (x = ∞).

Closing comments
We summarize the key results shown in this thesis and reflect on further work related
to ˜Yn,s and ˜
Xa,b,n,s.
We introduced the GK function field and gave a proof of its maximality based on theory
of algebraic function fields. The two generalizations, the GGS function fields and the BM
function fields, were also introduced, primarily with a focus on the BM function fields. A
thorough proof of the maximality of the BM function fields was given, based on showing
the maximality of certain subfields and the splitting of rational places in these subfields.
We outlined the function fields Yn,s and Xa,b,n,s arising as subfields of the GGS function
fields, and introduced their analogs of the BM function fields, namely ˜Yn,s and ˜
Xa,b,n,s. We
showed that g(Yn,s) = g( ˜Yn,s) and g(Xa,b,n,s) = g( ˜
Xa,b,n,s). Furthermore, we constructed
a subgroup ˜G of Aut( ˜Yn,s) and used this result to show under which conditions Yn,s and
˜Yn,s are isomorphic. We could conclude for most choices of parameters n and s, ˜Yn,s is
different from Yn,s, only when 3 | n and M | (q2 −q + 1) are Yn,s and ˜Yn,s isomorphic.
Also, for q > s(s + 1), we showed the function fields ˜Y3,s are not isomorphic to subfields
of Hq3. We determined a subgroup of Aut( ˜
Xa,b,n,s) which was the normalizer of the fixing
group of ˜
Xa,b,n,s. We showed the function fields ˜
Xa,b,3,1 are not isomorphic to subfields of
Hqn and further ˜
Xa,b,n,1 are not isomorphic to a Galois subfield of Hqn.
On the other hand, there are still questions left unanswered for the function fields
˜Yn,s and ˜
Xa,b,n,s. The first question one could try answering is to fully determine the
automorphism groups Aut( ˜Yn,s) and Aut( ˜
Xa,b,n,s).
For the BM function field Xn, the
automorphism group Aut(Xn) was constructed in the same manner as in Theorem 4.2.2,
and it has been shown this is the full automorphism group for n ≥5 in [5]. We conjecture
that for 3 ∤n or M ∤(q2 −q + 1), the subgroup ˜G ⊆Aut( ˜Yn,s) in Theorem 4.2.2 is the
full automorphism group of ˜Yn,s. Similarly it would be interesting to determine the full
automorphism group Aut( ˜
Xa,b,n,s), as this could help decide if Xa,b,n,s is isomorphic to
˜
Xa,b,n,s. Lastly, the question about whether ˜Yn,s and ˜
Xa,b,n,s are subfields of Hqn has only
been partially answered in the thesis, see Theorem 4.2.5 and Theorem 4.4.3. Determining
so for further choices of parameters q, a, b, n, s is an appealing direction of study if one is
in search of more examples of maximal function fields not isomorphic to subfields of the
Hermitian function field.
