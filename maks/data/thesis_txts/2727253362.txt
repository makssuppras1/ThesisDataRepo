Abstract
Representing multiscattering media by path tracing light through microgeometry, is
only computationally feasible if some light transportation acceleration methods are
utilized. Rendering multiscattering media consisting of microgeometry is a method
of attaining micro details in macroscopic objects, thus relaying visual information
on a mesoscopic scale.
The aim of this thesis is to analyse light rays interaction with transparent microge-
ometry in order to examine the behaviour and find methods for ray path termination
criteria. The analysis of light rays in microgeometry is important to understand how
light behaves in different types of materials with different specific aspects, such as
absorption and refractive index.
Two path termination methods are presented, one based on a ray penetration depth
using discrete signed distance fields, and another base on the accumulated ray path
distance and probability of absorption. It was found that the termination of ray
paths is possible for accelerating rendering, but at the cost of rendering fidelity.

Introduction
Representing objects and material details, in computer graphics at mesoscopic scales
can be achieved using microgeometry or particles, modelled after real microscopic
structures, of different material and sizes. To relay microscopic details, the amount
of particles, that make up an object, should be very high and densely packed, making
light simulation through such objects computationally infeasible. It is necessary to
develop and evaluate other techniques of light transportion that will calculate the
simulated ray paths faster, but with similar results.
Analysing the simulated ray paths and light transportation through microgeometry
is the focus of this thesis, in order acquire new perspectives and to find solutions
that will attain the same fidelity, but render and converge faster. If it is possible to
achieve faster ray path tracing through microscopic geometry, using conventional
path tracing, it would less computationaly complexity when representing physical
objects and products, thus useful when creating digital twins1, and physical phe-
nomena involving light transportation, such as heat transfer. 2
In the following there will initially be a description of the related work on acceler-
ated ligth transportation3, after a section on background theory and methods used
in path tracing and microgeometry. These sections describes the ray path analysis
and the information gathered from ray path rendering experiments, along with all
the techniques used to simulate light paths within microgeometry. The analysis pro-
vides argumentation and information on finding ray path termination criteria and
examination of ray to surface distance, in the macroscopic volume. Which leads to
the resulting renderings using this ray termination criteria, comparing groundtruth
and path-termination renderings. Lastly the analysis and methods are to be dis-
cussed and related to possible improvements and other appropriate techniques in
the setting of accelerated light path tracing.

Related Work
The litterature regarding the work on rendering of media represented with mi-
crostructures and particles are all based on different implementations of acceler-
ated volumetric light transportation, as simulating convergent light paths in multi
scattering media is computationally infeasible.
[4] presents a shell transport approach, a technique that decreases the number of
scattering events needed, to render discrete random media1. By path tracing a cer-
tain distance within the media surface, then using shell transport to accelerate and
simulate the many scatterings within the volume, [4] achieves great speed up. Pre-
calculating sampled shell transportation function values in a course grid, to be used
as a representation of the path distribution within the medium, is done by tracing
rays from the center of the specific volume, thus making shell transportation at
render time, much faster. The considered scattering media in this paper is assumed
heterogeneous and homogeneously distributed, and non-absorptive.
The accelerated light transportation described in [3] is based on an aggregate of
different algorithms for faster convergence when rendering granular materials. The
procedure is a combination of path tracing, proxy path tracing, volume path trac-
ing, shell tracing and diffusion. Where conventional path tracing is used to simulate
the low order scattering within singular grains, proxy path tracing is an approxi-
mation of this, based on a novelle Grain Scattering Distribution Function. [3] uses
volumetric path tracing, to simplify many scatter events into a few steps, modelling
the volume as generic collection of polydisperse, heterogeneous grains. Finaly using
shell transport as [4], but modelling the media as continuous instead.
[5] proposes some of the same accceleration techniques, for rendering procedural
granular media, with precomputed solutions to scattering with different grain types.
Using a combination of path tracing, volumetric path tracing and diffusion approx-
imation of long scattering paths, [5] also achieves accelerated, high fidelity result,
when rendering granular media.
This thesis is concerned with multiscatter microscopic absorptive media and the
behaviour of light transportation within such volumes. Unlike the plethora of work
regarding accelerated light transport, this serves more as a exploration of light paths
within procedural transparent media.

Background
To briefly inform the reader of the underlying methods and implementation, that
the subsequent analysis is based on, this section serves as an overview of the back-
ground to these rendering techniques. This entails the methods that are used to
represent and render procedural microgeometry and transparent materials and their
interaction with light in a raytracing perspective.
3.1
Rendering Procedure
The microscopic geometry is rendered using path-tracing1, a raytracing method that
simulates the path of light rays from a source to an observer and the light-material
interaction along this path. Another raytracing procedure, raymarching, is then
used to represent procedural microgeometry with signed distance fields, also to be
described in the latter. Using the OptiX2 framework in accordance with CUDA3,
the simulated light paths are calculated using GPU accelerated rendering, which
will also be touched upon in the following.
3.1.1
Path Tracing
Path tracing is at its core a raytracing method, aiming to solve the rendering equa-
tion4, by calculating the entire ray paths interaction with material, sampled mul-
tiple times, in order to estimate the light contribution at any given point that can
be observed [6]. As this is a Monte Carlo estimation of radiant contribution, many
samples are needed to arrive at satisfying results.
To simulate light paths, rays are to be traced from some initial point, ro, in some
given direction, #»r d. The direction is based on the previous steps of the ray. Rays
are parametrized as in equation 3.1 [10].

3.2.2
Discrete Distance Fields
In order to represent and find the depth within a volume, defined by a trimesh, it
is necessary to generate signed distance fields to use explicitly. Generating discrete
distance fields is described in [14], where the signed distance is sampled into a voxel
grid, so that the value in a voxel is the shortest distance to the trimesh surface.
The discrete signed distacnce field will be referenced as a macroscopic distance field,
while the distance field representing microgeometry is the microscopic distance field.
3.2.3
Voxel Generation
The GEL library[15] can be used to export trimesh models to voxel grids by sampling
the distance to the trimesh at equidistant points, specified by the caller. In this case
a voxel grid of 128 × 128 × 128 is used to save the signed distance at each discrete
point of a voxelized bounding box.
The distance sampling is a simple check of the distance from the voxel coordinate
to the all trimeshes in the scene, saving the minimum distance. The sign of the
discrete distances is calculated by whether the dot product of the trimesh normal
and the direction vector from the voxel position to the closest trimesh, is positive or
negative. If the voxel is inside the trimesh the dot product would be positive, and
thus a negative sign, and vice versa.
3.2.4
SDF and Trimesh Alignment
When the signed distance to a point in space is to be calculated, the position is
mapped from the bounding box of the trimesh into a unit cube, [0, 1]3, as the
generated distance field is proportionate in this way.
Making a lookup in the generated voxel grid is done given a point (x, y, z) and then
interpolating within the grid, to find the smallest distance to the surface at that
given point.
As the position in the unit-box are not exactly mapable to the voxel-grid, the point is
interpolated trilinearly, which consists of two bilinear interpolations of two opposing
sides of the voxel and then a linear interpolation6.

etry the point in world space coordinates, specifically inside the bounding box, is

This section serves to explain the ray path information that is gathered for the

4.1
Ray-path and Microgeometry Interaction
Path tracing light rays through transparent microgeometry, will yield many cal-
culations and recursions; depending on the material specifications, reflections and
refractions. It can be very diﬀicult to simulate the convergent path when tracing
these rays, and so, finding convergent paths are computationally diﬀicult. Illus-
trated in figure 4.1 is the light path interaction with a medium consisting of many
microgeometric primitives. The light rays will interact with the primitives, calculat-
ing new paths according to material specifications. The contribution to pixel value
is calculated along the path, as described in the previous section.
Figure 4.1: Example of a light path through a medium consisting of transparent
microgeometry. a in the illustration is the ray-path hit with the triangle mesh of
the object. b is a reflection of the ray with a primitive, c1 and c2 are refractions
with the primitive and d is the ray escaping from the medium.
How the ray interacts within this scattering volume is important and much infor-
mation can be gained from these scatterings, for example, termination in relation
to the volume and depth penetration into the volume, distance travelled, etc.

Discussion
To thoroughly analyze ray paths and the interactions within microscopic geometry,
requires assessment of many different configurations, such as, objects, materials,
shape and size of the primitives, etc. So it is important to restrict and determine
which important features to focus on, as it will never be an exhaustive examination,
especially when results are to be rendered and compared. This is one of the things
that holds back this thesis; as choices should be made fairly early in the process, to
know what to engage and compare.
It can be argued whether calculating all ray paths after another is better than
rendering frame by frame, when terminating rays early. A frame by frame rendering
will have to wait until all ray paths are terminated until calculating the next frame,
which would mean that, in some cases, the ray termination would have no effect
on render time. Each path traced would still need to write information at render
time, but without waiting on other calculations. The termination criteria could also
factor in very long ray paths, by utilizing the accumulated ray distance examined,
and so avoiding bottlenecks. This would also be interesting to use in conjunction
with the ray convergence rate, adhering information of where more rays should be
cast.
Testing the generality of ray termination criteria, should also be thoroughly explored
in order to solidify and work towards better techniques and criteria. As the ray
cutoffs found in this thesis were only tested on one configuration, it cannot be argued
that this method is very general, why the importance of investigating configuration
specific scenes is prevalent.
The alignment of the macroscopic distance field and the trimesh geometry could
argueably be more precise; one could imagine that a great amount of particles could
lead to ray termination just under the surface of the distance field, which would
mean that the descrepancies would have a visual effect on the renderings. This was
not a problem in this case as terminating rays at the surface would result in too
great of an error.
Another point that is not addressed yet is the choice to not render the volumes of
microstructures to convergence. It does not give a completely accurate represen-
tation of the potential speed up, but was necessary due to time constraints, and
general computational complexity when calculating the high amount of scattering.
Analysis of Light Path Tracing in Microgeometry

The proposed method using a macroscopic distance field for a given trimesh also
poses a problem for generality; as each object composed of microgeometry would
need a macroscopic distance field for distance lookups, and corresponding mapping
between world space and object space.
Examining the effects of terminating rays by some criteria has also not yet been
examined, but would definitely be of interest, as the ray path information would
definitely be affected.
The analysis of light paths in microgeometry is generally diﬀicult to approach, given
possible configurations and complexity of path tracing. But can be used to under-
stand many physical phenomena and properly simulate the interaction between ligth
and micro-particles.
It could be argued that the ray termination is found by sampling too few rays, and
as such not conservative enough. But due to limitations it could not be determined,
if the ray termination methods will work for a great amount of samples.

Perspective and Future Work
Future work would include examination of more structures and specifically different
primitives, to find the effects of ray-paths with these configurations. It would also
be interesting to analyze particles with different distributions and differing particle
size, to understand ligth interaction on a microscopic level.
A method to find termination criteria, at render time would also be highly desirable,
as ray termination criteria can be computed fairly easily, but needs to be material
and scene specific. So by path tracing a few rays, the needed information could be
found and appropriate termination/acceleration techniques applied.
Using microgeometry to represent volumes and the analysis of ray-paths can lead
to rendering materials consisting of microscopic particles distributed non homoge-
neously throughout. The analysis itself can also lead to understanding light interac-
tion with these materials, but faster rendering techniques are still desired, so in the
following are some approaches that could make sense examining with the ray-path
information assessed in this thesis.
Another approach to calculating convergent ray paths faster without actually ter-
minating them within the volume could be a form of path guiding1, where precal-
culating the gradient of the macroscopic distance field and choosing the incident
refraction direction or reflection direction that will lead to the surface, by assessing
the dot product between the direction and gradient distance field.
From the convergence rate of ray-paths an adaptive sampling technique2 seems in-
triquing, as some areas of the volume is seen to have a lower rate of convergence
and could help guiding an adaptive sampling method.
Another technique would entail accelerated transport within a certain depth of the
volume, where another volume could replace the microstructures, but keep the same
material aspects. Rays that penetrate the volume deep enough will in this way be
transported towards the surface, either by reflection or refraction.

Conclusion
This thesis considers many configurations of micro primitives, such as size and ma-
terial, in order to examine the highly scattering paths of light within such micro-
geometry. Simulating these ray paths to convergence is computationally infeasible
why this thesis investigates light ray paths interacting with highly scattering media
to find other methods or information. From the analysis it was found that when
rendering volumes with many microscopic densely packed particles, the effect of
absorption is almost non existent.
Using discrete distance fields to find ray penetration, and thus ray-surface distance,
and analysing the rate of convergence, ray travel distance, lead to finding termi-
nation criteria, for possible early termination of ray paths. The found termination
criteria are based on the ray to surface distance, and ray travel distance, means
that it is possible to accelerate the render time when rendering multiscattering mi-
crogeometry. The termination cutoffs were tested and compared to a groundtruth
rendering using the path tracing without any early ray termination, to compare
speed-up and fidelity. The found relative speed-up when rendering microscopic ice
particles within a volume, using path tracing, was up to 13.2% than explicit path
tracing, while keeping the relative MSE below 5.1e-4.
The analysis of light paths within microgeometry has not been fully exhausted, but
is a step towards using and developing new methods and techniques in order to
accelerate light transportation.
